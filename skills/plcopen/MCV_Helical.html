MCV_Helical

Performs a helical move and additionally allow vector speed to be changed when using multiple moves in the buffer.

Description:

MCV_Helical is used to perform a helical move. It moves 2 orthogonal axes in such a way as to produce a circular arc at the tool point with a simultaneous linear move on a third axis. Additionally, it allows vector speed to be changed when using multiple moves in the buffer.

When MCV_Helical is executed, Axis0 and Axis1 PLCopen states will change to ‘DiscreteMotion’.

Either Axis0 or Axis1 can be used in MC_Stop to stop motion on both axes.

Make sure the input and output parameters are of data type indicated in the table below.

An array input parameter is denoted by "[]"characters following the parameter name. Suitable array input is expected.

MCV_Helical is equivalent to TrioBASIC command MHELICALSP. Please refer to TrioBasic Help for more information.
