MC_ReadMotionState

Gets the status of an axis with respect to the motion currently in progress.

Description:

This function block returns the axis motion status in progress. One or more outputs will be set when ‘Enable’ is TRUE and all inputs are valid.

The output ‘ConstantVelocity’ will be set when velocity is constant. Velocity may be 0. For the actual value, a window is applicable (vendor specific).

This Function Block does not affect the PLCopen state machine.

Make sure the input and output parameters are of data type indicated in the table below.

Different input ‘Source’ may have different output of an axis at the same time.

Inputs/Outputs:

Name

Data Type

Description

INPUTS

Enable

BOOL

Get the value of the parameter continuously while enabled

Axis

AXIS_REF

Reference to the axis

Source

MC_SOURCE

Defines the source of the relevant data: mcCommandedValue, mcSetValue, or mcActualValue.

OUTPUTS

Valid

BOOL

A valid set of outputs is available

Busy

BOOL

The FB is not finished and new output values are to be expected

Error

BOOL

Signals that an error has occurred within the FB.

ErrorID

UINT

Error identification

ConstantVelocity

BOOL

Velocity is constant. Velocity may be 0. For the actual value a window is applicable (vendor specific)

Accelerating

BOOL

Increasing the absolute value of the velocity

Decelerating

BOOL

Decreasing the absolute value of the velocity

DirectionPositive

BOOL

Signals that the position is increasing

DirectionNegative

BOOL

Signals that the position is decreasing

Examples:

Example 1:

 Example

(*Example for MC_ReadMotionState *)
inst_ReadMotionState ( Enable := bEnable,
                        Axis := Axis0Ref, 
                        Source := mSource );
bValid:= inst_ReadMotionState.Valid;
bBusy := inst_ReadMotionState.Busy; 
bError := inst_ReadMotionState.Error; 
ErrorId := inst_ReadMotionState.ErrorID;
IF bValid THEN
    bConstantVelocity := inst_ReadMotionState.constantVelocity;
    bAccelerating := inst_ReadMotionState.Accelerating;
    bDecelerating := inst_ReadMotionState.Decelerating;
    bDirectionPositive := inst_ReadMotionState.DirectionPositive;
    bDirectionNegative := inst_ReadMotionState.DirectionNegative;
END_IF ;
