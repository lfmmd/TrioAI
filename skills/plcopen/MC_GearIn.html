MC_GearIn

Commands a ratio between the VELOCITY of the slave and master axis.

Description:

The slave ramps up to the ratio of the master velocity and locks in when this is reached. Any lost distance during synchronisation is not caught up.

The gearing ratio can be changed while MC_GearIn is running, using a consecutive MC_GearIn command without the necessity to MC_GearOut first.

After ‘ InGear’ changes to TRUE, a position locking or just a velocity locking is system specific.

Make sure the input and output parameters are of data type indicated in the table below.

Examples:

Example 1:

‘Axis1Master’ is the master axis and ‘Axis2Slave’ is the slave axis. The slave axis moves with the master axis based on a special ratio used in MC_GearIn. The slave axis disconnects from the master axis and maintains the last velocity after disengaging to master axis with MC_GearOut execution during the stopping stage of the master axis.

 Example

VAR
 Inst_MC_Power : MC_Power ;
   (*$desc=Enable master axis*)
 Inst_MC_Power1 : MC_Power ;
   (*$desc=Enable Slave axis*)
 Axis1Master : lib:AXIS_REF ;
   (*$desc=Master axis in MC_CamIn which moving in constant velocity*)
 Axis2Slave : lib:AXIS_REF ;
   (*$desc=Slave axis in MC_CamIn which moving in constant velocity*)
 PowerAxes : BOOL := FALSE ;
   (*$desc=To power axes on*)
 Inst_MC_MoveVelocity : MC_MoveVelocity ;
   (*$desc=To move the master axis with constant velocity*)
 MoveMaster : BOOL := FALSE ;
   (*$desc=To start master axis moving*)
 mDir : MC_DIRECTION := mcPositiveDirection ;
   (*$desc=Motion direction definition*)
 mBufferMode : MC_BUFFER_MODE := mcAborting ;
   (*$desc=Buffer mode setting for master axis motion*)
 bInVelo : BOOL := FALSE ;
   (*$desc=Master axis motion status*)
 bStaMaster : BOOL ;
   (*$desc=Power status of master axis*)
 bStaSlave : BOOL := FALSE ;
   (*$desc=Power status of Slave axis*)
 mMasterSrc : MC_SOURCE := mcActualValue ;
   (*$desc=Define the source for Synchronisation for master axis*)
 mStartMode : MC_START_MODE := mcAbsolute ;
   (*$desc=Start mode setting*)
 Inst_MC_Stop : MC_Stop ;
 bStopSlave : BOOL := FALSE ;
   (*$desc=To stop the slave axis*)
 bStopped : BOOL := FALSE ;
   (*$desc=Is slave axis stopped*)
 bOutDone : BOOL := FALSE ;
   (*$desc=Is cam out execution finished*)
 Inst_MC_GearIn : MC_GearIn ;
 Inst_MC_GearOut : MC_GearOut ;
 bGearOut : BOOL := FALSE ;
   (*$desc=Executing disengaing gear motion variable*)
 bGearExe : BOOL := TRUE ;
   (*$desc=To engage the slave axis with the master axis*)
 bInGear : BOOL := FALSE ;
   (*$desc=Gear motion engaging status*)
END_VAR

Example 2:

A machine has an automatic feed on axis 1 which must move at a set ratio to axis 0. This ratio is selected using inputs 1-3 to select a ‘gear’. This ratio can be updated continuously.  Combinations of inputs will select intermediate gear ratios.  For example: when Input1 = OFF, Input2 = ON, and Input3 = ON, the ratio is 6:1.

 Example

VAR
 Inst_MC_Power : MC_Power ;
 Inst_MC_Power1 : MC_Power ;
 Axis0 : lib:AXIS_REF ;
   (*$desc=Previous axis*)
 bPower : BOOL := FALSE ;
   (*$desc=Two axes enable variable which is bound to IO8*)
   (*$profile=IO*)
   (*$embed=(VERS=1    MODE=0    POINT=8)*)
 Axis1 : lib:AXIS_REF ;
   (*$desc=The slave axis in gear engaging motion*)
 bPowerAxis0 : BOOL := FALSE ;
   (*$desc=The power status of axis 0*)
 bPowerAxis1 : BOOL := FALSE ;
   (*$desc=The power status of axis 1*)
 Inst_MC_MoveVelocity : MC_MoveVelocity ;
 bExeAxis0 : BOOL := FALSE ;
   (*$desc=To start axis 0 motion which is bound to IN2*)
   (*$profile=IO*)
   (*$embed=(VERS=1    MODE=0    POINT=2)*)
 mDir : MC_DIRECTION := mcPositiveDirection ;
   (*$desc=Motion direction*)
 mBufferMode : MC_BUFFER_MODE := mcAborting ;
   (*$desc=Buffer mode setting for axis 0*)
 bInVelo : BOOL := FALSE ;
   (*$desc=Is in constant velocity for axis 0*)
 Inst_MC_Stop : MC_Stop ;
 Inst_MC_Stop1 : MC_Stop ;
 bStop : BOOL := FALSE ;
   (*$desc=To stop two axis and it is bound to IN3*)
   (*$profile=IO*)
   (*$embed=(VERS=1    MODE=0    POINT=3)*)
 bStopAxis0 : BOOL := FALSE ;
   (*$desc=Axis0 stopping status, true--stopped*)
 bStopAxis1 : BOOL ;
   (*$desc=Axis1 stopping status, true--stopped*)
 Inst_MC_ReadDigitalInput : MC_ReadDigitalInput ;
 Inst_MC_ReadDigitalInput1 : MC_ReadDigitalInput ;
 Inst_MC_ReadDigitalInput2 : MC_ReadDigitalInput ;
 Input1 : INT := INT#10 ;
   (*$desc=Gear ratio setting input 1*)
 Input2 : INT := INT#11 ;
   (*$desc=Gear ratio setting for Input2*)
 Input3 : INT := INT#12 ;
   (*$desc=Gear ratio setting for input3*)
 bReadGear : BOOL := TRUE ;
   (*$desc=To read the gear ratio input setting*)
 GearRatio : INT := INT#1 ;
   (*$desc=Gear ratio setting value*)
 Inst_MC_GearIn : MC_GearIn ;
 lastGearRatio : INT := INT#0 ;
   (*$desc=Storing the last gear ratio setting value*)
 mValueSrc : MC_SOURCE := mcActualValue ;
   (*$desc=The value source used for gear motion*)
 bInGear : BOOL := FALSE ;
   (*$desc=Synchronisation motion status of gear motion*)
END_VAR
