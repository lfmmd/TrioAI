MC_CamOut

Disengages the slave axis from the master axis immediately.

Description:

MC_CamOut is executed when ‘Execute’ changes to TRUE. If the default value of ‘Execute’ is TRUE, this instruction will not be executed in the first cycle. Re-triggering this FB will be ignored if it is already running.

Make sure the input and output parameters are of data type indicated in the table below.

It is assumed that this command is followed by another command, for instance MC_Stop, MC_GearIn, orany other command. If there is no new command, the default condition should be: maintain last velocity.

After issuing MC_CamOut, there is no FB active on the slave axis until the next FB is issued. This can result in problems because no motion command is controlling the axis. Alternatively, the actual velocity can be read via MC_ReadActualVelocity and MC_MoveVelocity can be issued on the slave axis with the actual velocity as input. The FB is here because of compatibility reasons.

Inputs/Outputs:

Name

Data Type

Description

INPUTS

Execute

BOOL

Start to disengage the slave from the master at rising edge

Slave

AXIS_REF

Reference to the slave axis

OUTPUTS

Done

BOOL

Disengaging completed

Busy

BOOL

The FB is not finished and new output values are to be expected

Error

BOOL

Signals that an error has occurred within the FB

ErrorID

UINT

Error identification

Example 1:

inst_CamOut (Execute := bExecute, Slave := SlaveRef);
bDone := inst_CamOut.Done; 
bBusy := inst_CamOut.Busy; 
bError := inst_CamOut.Error; 
ErrorId := inst_CamOut.ErrorID;

Code:

 

// Cancel all previous moves in MTYPE
Inst_TC_CANCEL( TRUE , 0 , 0 );
Inst_TC_CANCEL1( TRUE , 1 , 0 );

// Cam axis configuration
TCW_UNITS ( 0 , 1 );
TCW_SPEED ( 0 , 10 );
TCW_ACCEL ( 0 , 10 );
TCW_DECEL ( 0 , 10 );
Inst_TC_DEFPOS0( TRUE , 0 , 0 );
TCW_SERVO ( 0 , TRUE );

// Master axis configuration
TCW_UNITS ( 1 , 1 );
TCW_SPEED ( 1 , 1 );
TCW_ACCEL ( 1 , 10 );
TCW_DECEL ( 1 , 10 );
Inst_TC_DEFPOS1( TRUE , 1 , 0 );
TCW_SERVO ( 1 , TRUE );

TCW_WDOG ( 1 );

// Set Cam Table
CamTable.TableStart:= 200 ;
CamTable.TableFinish:= 211 ;
Inst_MC_CamTableSelect( TRUE , CamTable, FALSE , FALSE , FALSE );
CamTableId:=Inst_MC_CamTableSelect.CamTableID;

// MC_CamIn configuration
master.AxisNo:= 1 ;
Slave.AxisNo:= 0 ;
StartMode:=mcAbsolute;
MasterValueSource:=mcCommandedValue;
Inst_MC_CamIn( TRUE , Master, Slave, 0 , 0 , 1 , 1 , 0 , 0 , StartMode, MasterValueSource, CamTableID);

// MC_CamOut setup
bCamOutExecute:= FALSE ;
Inst_MC_CamOut(bCamOutExecute, Slave);

// Run master axis
Inst_TC_FORWARD( TRUE , 1 );
