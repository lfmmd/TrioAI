Parameters

Description:

Parameters defined below have been supported in Trio controllers with its PLCopen function blocks ( MC_ReadParameter , MC_ReadBoolParameter , MC_WriteParameter , MC_WriteBoolParameter ).Users can read or write as application requirement.

To introduce all these axis parameters, every parameter has five parts (Parameter number (PN), data type, Attribute, Value range and Comments) to describe it. Data types used in the following parameter table are:

•         BOOL: Boolean value

•         DINT: 32-bit signed integer

•         REAL: 32-bit float 

•         INT: 16-bit signed integer

•         LINT: 64-bit signed integer

•         LREAL: 64-bit float

•         UINT: 16-bit unsigned integer   

•         UDINT: 32-bit unsigned integer

•         ULINT: 64-bit unsigned integer

Parameter attribute means its read and write attribute. Abbreviation of read and write attribute are:

•         R: read only

•         W: write only

•         R/W: read and write

Value range is used to let users know it value range, the value which out of the parameter value range will not be wrote and an error will generate if trying to write.

Comment for a parameter is used to give a brief description for the parameter.
