MCV_MoveLine2D

Moves linear interpolation for a specified absolute position for 2 axes.

Description:

MCV_MoveLine2D performs linear interpolation for a specified absolute position for 2 axes. The target position is specified as an absolute position.

When MCV_MoveLine2D is executed, Axis0 and Axis1 PLCopen states will change to ‘DiscreteMotion’.

Either Axis0 or Axis1 can be used in MC_Stop to stop motion on both axes.

Make sure the input and output parameters are of data type indicated in the table below.

An array input parameter is denoted by "[]"characters following the parameter name. Suitable array input is expected.
