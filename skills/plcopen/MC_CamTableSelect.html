MC_CamTableSelect

Selects the CAM tables by setting the connections to the relevant tables.

Description:

MC_CamTableSelect selects the CAM tables by setting the connections to the relevant tables.

A virtual axis can be used as master axis.

MC_CamTableSelect makes data available. This can include:

  1. Starting point of a download of a profile
  2. Start to generate a CAM profile

When the ‘Done’ output is SET, the CamTableID is valid and ready for use in a MC_CamIn FB.

Commanding is possible at any time regardless of the state of the master or slave axis.

Make sure the input and output parameters are of data type indicated in the table below.

Inputs/Outputs:

Name

Data Type

Description

INPUTS

Execute

BOOL

Selection at rising edge

CamTable

MC_CAM_REF

Reference to CAM description

Periodic

BOOL

1 = periodic, 0 = non periodic (single-shot)

MasterAbsolute

BOOL

1 = absolute, 0 = relative coordinates

SlaveAbsolute

BOOL

1 = absolute, 0 = relative coordinates

OUTPUTS

Done

BOOL

Pre-selection done

Busy

BOOL

The FB is not finished and new output value is to be expected.

Error

BOOL

Signals that an error has occurred within the Function Block

ErrorID

UINT

Error identification

CamTableID

UINT

Identifier of CAM Table to be used in the MC_CamIn FB

Examples:

Example 1:

 Example

CamTable0.TableStart := 1000 ; (*Set as actual cam table setting *)
CamTable0.TableFinish := 1010 ;

inst_CamTableSelect ( Execute := bExecute,
                        CamTable := CamTable0,
                        Periodic := bPeriod, 
                        MasterAbsolute := bAbsolute, 
                        SlaveAbsolute := bAbsolute );
bDone := inst_CamTableSelect.Done; 
bBusy := inst_CamTableSelect.Busy; 
bError := inst_CamTableSelect.Error; 
ErrorId := inst_CamTableSelect.ErrorID;
IF bDone THEN
    TableID := inst_CamTableSelect.CamTableID;
END_IF ;

Code:

 

// Cancel all previous moves in MTYPE
Inst_TC_CANCEL( TRUE , 0 , 0 );
Inst_TC_CANCEL1( TRUE , 1 , 0 );

// Cam axis configuration
TCW_UNITS ( 0 , 1 );
TCW_SPEED ( 0 , 10 );
TCW_ACCEL ( 0 , 10 );
TCW_DECEL ( 0 , 10 );
Inst_TC_DEFPOS0( TRUE , 0 , 0 );
TCW_SERVO ( 0 , TRUE );

// Master axis configuration
TCW_UNITS ( 1 , 1 );
TCW_SPEED ( 1 , 1 );
TCW_ACCEL ( 1 , 10 );
TCW_DECEL ( 1 , 10 );
Inst_TC_DEFPOS1( TRUE , 1 , 0 );
TCW_SERVO ( 1 , TRUE );

TCW_WDOG ( 1 );

// Set Cam Table
CamTable.TableStart:= 200 ;
CamTable.TableFinish:= 211 ;
Inst_MC_CamTableSelect( TRUE , CamTable, FALSE , FALSE , FALSE );
CamTableId:=Inst_MC_CamTableSelect.CamTableID;

// MC_CamIn configuration
master.AxisNo:= 1 ;
Slave.AxisNo:= 0 ;
StartMode:=mcAbsolute;
MasterValueSource:=mcCommandedValue;
Inst_MC_CamIn( TRUE , Master, Slave, 0 , 0 , 1 , 1 , 0 , 0 , StartMode, MasterValueSource, CamTableID);

// MC_CamOut setup
bCamOutExecute:= FALSE ;
Inst_MC_CamOut(bCamOutExecute, Slave);

// Run master axis
Inst_TC_FORWARD( TRUE , 1 );
