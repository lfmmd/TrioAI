Non-Axis Errors

Description:

Non-axis errors are generated in FBs mostly because of invalid inputs. It can be cleared by providing valid inputs when the FB’s Enable input is TRUE (except MC_Power), or at the rising edge of the FB’s Execute input.

The ErrorID value of non-axis errors is less than 0x8000 and can only represent an error state even though more than one error may have occurred.

Non-axis errors do not affect the PLCopen state diagram and current motion.

ErrorID Range:

Non-axis errors have an ErrorID range of 0x0000 – 0x7FFF.

Details:

Making the axis reference avaliable.
