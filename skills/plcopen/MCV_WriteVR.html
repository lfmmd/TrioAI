MCV_WriteVR

Writes a specified VR variable value.

Description:

MCV_WriteVR writes a value to a VR variable specified by ‘VR_number’ input.

MCV_WriteVR does not affect PLCopen state diagram.

Make sure the input and output parameters are of data type indicated in the table below.

Please refer to TrioBasic Help for more information.

Inputs/Outputs:

Name

Data Type

Description

INPUTS

Execute

BOOL

Start execution at rising edge.

VR_number

UINT

VR variable number

Value

LREAL

Value to be written to VR

OUTPUTS

Done

BOOL

The variable is written successfully.

Busy

BOOL

The FB new output values are to be expected

Error

BOOL

Signals that an error has occurred within the FB

ErrorID

UINT

Error identification
