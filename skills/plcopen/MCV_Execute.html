MCV_Execute

Allows any TrioBASIC commands to be executed as a string parameter.

Description:

MCV_Execute is used to allow any TrioBASIC command to be executed as a string parameter.

Make sure the input and output parameters are of data type indicated in the table below.

Please refer to TrioBasic Help for more information.

Inputs/Outputs:

Name

Data Type

Description

INPUTS

Execute

BOOL

Start execution at rising edge

Command

STRING

TrioBASIC command string

OUTPUTS

Done

BOOL

The command has been finished.

Busy

BOOL

The FB new output values are to be expected

Error

BOOL

Signals that an error has occurred within the FB

ErrorID

UINT

Error identification
