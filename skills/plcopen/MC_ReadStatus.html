MC_ReadStatus

Gets the state of the specified axis, as specified in the Trio PLCopen State Diagram .

Description:

It is used to get the state (as described in the Trio PLCopen State Diagram ) of an axis when ‘Enable’ is TRUE.

There are eight defined states for an axis and the axis is always in one of these states: ErrorStop, Disabled, Stopping, Homing, Standstill, DiscreteMotion, ContinuousMotion, SynchronizedMotion.

This Function Block does not affect the PLCopen state machine.

Make sure the input and output parameters are of data type indicated in the table below. An error is generated when this is not the case.

Examples:

Example 1:

 Example

(*Example for MC_ReadStatus *)
inst_ReadStatus( Enable := bEnable,
                Axis := Axis0Ref );
bValid:= inst_ReadStatus.Valid;
bBusy := inst_ReadStatus.Busy; 
bError := inst_ReadStatus.Error; 
ErrorId := inst_ReadStatus.ErrorID;
IF bValid THEN
    bErrorStop := inst_ReadStatus.ErrorStop;
    bDisabled := inst_ReadStatus.Disabled;
    bStopping := inst_ReadStatus.Stopping;
    bHoming := inst_ReadStatus.Homing;
    bStandstill := inst_ReadStatus.Standstill;
    bDiscreteMotion := inst_ReadStatus.DiscreteMotion;
    bContinuousMotion := inst_ReadStatus.ContinuousMotion;
    bSynchronizedMotion := inst_ReadStatus.SynchronizedMotion;
END_IF ;
