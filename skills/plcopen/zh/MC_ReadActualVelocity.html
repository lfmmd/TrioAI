MC_ReadActualVelocity

获取指定轴的实际速度。

描述

当“Enable”更改为TRUE且“Axis”具有有效值时，“Velocity”将输出指定轴的速度，“Valid”输出变为TRUE。

这个功能块不影响PLCopen状态机。它可以在任何状态下工作。

确保输入和输出参数为下表所示的数据类型。

输入/输出

名称数据类型描述
输入
EnableBOOL为TRUE时连续获取轴的实际速度
AxisAXIS_REF参考轴
输出
ValidBOOL有效的“Velocity”输出可用
BusyBOOLFB尚未完成，预计会有新的输出值。
ErrorBOOL表示FB中发生了错误。
ErrorIDUINT错误标识
VelocityLREAL新绝对速度[单位/秒]

例

例1

例
(*MC_ReadActualVelocity 的示例*)
inst_ReadActualVelocity ( Enable:=bEnable, 
                          Axis:=Axis0Ref );
bValid:= inst_ReadActualVelocity.Valid;
bBusy := inst_ReadActualVelocity.Busy; 
bError := inst_ReadActualVelocity.Error; 
ErrorId := inst_ReadActualVelocity.ErrorID;
IF bValid THEN
    lrVelocity := inst_ReadActualVelocity.Velocity;
END_IF;
