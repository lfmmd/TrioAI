描述

MC_Home命令轴执行“search home”序列。这个序列的细节由FB的输入参数MC_HOME_MODE (Home输入)选择。“Position”输入用于设置检测到参考信号时的绝对位置。

MC_Home将PLCopen状态从“Standstill”更改为“Homing”。

FB必须从轴状态“Standstill”调用。在“Execute”的上升沿，FB将轴状态更改为“homing”。随着回零过程的完成，轴状态回到“Standstill”状态。

如果在回零过程中出现错误，FB将被中止，轴状态将更改为“ErrorStop”。

这个功能块也可以被另一个带有缓存模式的“Aborting”功能块终止。

FB有6种回零方法:

  • **mcForwardSwitch:**轴以程序给定的速度向前移动，直到到达基准开关。然后轴以爬坡速度向后移动，直到基准开关复位。然后将测量位置设置为定义位置。

确保输入和输出参数为下表所示的数据类型。

MC_Home只能从“Standstill”PLCopen状态图开始，否则会产生错误。

它不允许在回零过程进行时改变参考轴。

输入' Home '定义了回零运动参数。这包括' HomeMode '，它定义了回零模式:mcForwardZ, mcReverseZ, mcForwardSwitch, mcReverseSwitch, mcForwardZS, mcReverseZS。

输入/输出

名称数据类型描述
输入
ExecuteBOOL写入上升沿处的参数值
AxisAXIS_REF参考轴
HomeMC_HOME_REF回零运动参数，包括回零模式、回零速度、爬坡速度和回零运动时的加速度、减速度、加加速度、回零开关通道设置。
PositionLREAL检测到参照信号时的绝对位置[单位]
输出
DoneBOOL参照识别并设置成功。
BusyBOOLFB尚未完成，预计会有新的输出值。
ActiveBOOL表示FB对轴具有控制权。
CommandAbortedBOOL“Command”被另一个命令终止。
ErrorBOOL指示FB内部发生错误。
ErrorIDUINT错误识别。

例

例1

首先，将' Power '设置为TRUE，从而为轴0上电。如果没有错误，轴0将转移到“StandStill”状态。其次，将“Home”设置为TRUE，启动轴0的回零过程。轴0状态将转移到“Homing”状态。一旦回零完成，状态将变为“StandStill”。示例代码使用**HomeMode “mcReverseZS”**，“Position”为0。这意味着当前的轴位置将在0单位的位置回零。

例
(*定义MC_Home的示例*)
#define STATE_READY  0
#define STATE_POWER  1
#define STATE_HOMING 2
#define NEXT_STEP    3
#define ERROR_STEP   7
(* 变量声明 *)
VAR
 HomeRef : lib:MC_HOME_REF ;
 State : USINT ;
  (*$desc=当前状态*)
 bPower : BOOL := FALSE ;
  (*$desc=绑定到 IO1的上电开关*)
  (*$profile=IO*)
  (*$embed=(VERS=1    MODE=0    POINT=1)*)
 Inst_MC_Power : MC_Power ;
 Inst_MC_Home : MC_Home ;
 Axis0Ref : lib:AXIS_REF ;
  (*$desc=轴0参照*)
 ErrorID0 : UINT ;
END_VAR
(*变量初始化*)
HomeRef.HomeMode := mcReverseZS;
HomeRef.HomeSpeed := LREAL#0.0;
HomeRef.CreepSpeed := LREAL#30.0;
HomeRef.Acceleration := LREAL#5000.0;
HomeRef.Deceleration := LREAL#5000.0;
HomeRef.Jerk := LREAL#5000.0;

State:= STATE_READY;
Axis0Ref.AxisNo := 0;

(* 循环程序 *) 
CASE State OF  
    (* ...在运动前上电轴... *)
    (********************STATE_READY*********************)
    STATE_READY:
    IF (bPower) THEN
        State:=STATE_POWER;
    END_IF;
    (********************STATE_POWER*********************)
    STATE_POWER:  (*上电轴0*)
    Inst_MC_Power( Axis := Axis0Ref(*AXIS_REF*), 
                    Enable := TRUE(*BOOL*) );
    IF (Inst_MC_Power.Status = TRUE) THEN
        Inst_MC_Power( Enable := FALSE(*BOOL*), 
                        Axis := Axis0Ref(*AXIS_REF*) );
        Inst_MC_Power( Enable := TRUE(*BOOL*) , 
                        Axis := Axis0Ref(*AXIS_REF*));
        
        Inst_MC_Home( Execute := FALSE(*BOOL*),
                        Axis := Axis0Ref(*AXIS_REF*), 
                        Home := HomeRef(*MC_HOME_REF*), 
                        Position := 0.0(*LREAL*) );
        State:=STATE_HOMING;
    ELSIF (Inst_MC_Power.Error) THEN
        (* 在变量中储存错误ID*)
        ErrorID0:=Inst_MC_Power.ErrorID;
        (*在STATE_ERROR前重置使能*)
        Inst_MC_Power( Enable := TRUE(*BOOL*), 
                         Axis := Axis0Ref(*AXIS_REF*));
        State:=STATE_ERROR;
    END_IF;
    (********************STATE_HOMING*********************)
    STATE_HOMING:  (*Homing Axis*)
    (* Makin a rising edge for MC_Home0 execution *)
    Inst_MC_Home( Execute := TRUE(*BOOL*),
                    Axis := Axis0Ref(*AXIS_REF*), 
                    Home := HomeRef(*MC_HOME_REF*), 
                    Position := 0.0(*LREAL*) );
            
    IF (Inst_MC_Home.Done = TRUE) THEN
        State := NEXT_STEP;
    ELSIF (Inst_MC_Home.Error = TRUE) THEN
        (*在变量中储存错误ID *)
        ErrorID0:= Inst_MC_Home.ErrorID;
        (*在STATE_ERROR前重置使能*)
        Inst_MC_Home( Execute := TRUE(*BOOL*),
                        Axis := Axis0Ref(*AXIS_REF*), 
                        Home := HomeRef(*MC_HOME_REF*), 
                        Position := 0.0(*LREAL*) );
                        State := ERROR_STEP;
    END_IF;

    ERROR_STEP:
    (* 当运动出错时，在此添加代码*)
    
    NEXT_STEP:
    (* 在调用MC_Stop后，在此添加代码*)

END_CASE;
