MC_DigitalCamSwitch

指定一组离散的输出位进行切换，类似于一组连接到一根轴上的机械凸轮控制开关。

描述

MC_DigitalCamSwitch类似于电机轴上的开关:它命令一组离散的输出位切换，类似于一组连接到一根轴上的机械凸轮控制开关。允许向前和向后运动。

轨道是一个凸轮，它包含一组与一个输出相关联的开关。“TrackNumber”对应于凸轮号。当“Enable”为TRUE时，“EnableMask”指定每个轨道是启用还是禁用。位0对应于轨道0，位15对应于轨道15。如果一个位被设置为1，对应的轨道被启用，如果这个位被设置为0，对应的轨道被禁用。如果某位的值从1变为0，则对应轨道的输出将关闭。这个功能有时被称为PLS -相位或位置或可编程限位开关。

在“Enable”的上升沿上，初始化配置的“Switches”、“ Outputs”和“TrackOptions”的参数。初始化后，内部切换立即开始。

在“EnableMask”上升沿上，数字输出将被启用。输出将根据配置的“Switches”和“TrackOptions”进行设置和重置。当输出被禁用时，输出被重置。

“EnableMask”用于启用和禁用相应的轨道。如果“FirstOnPositon[n]”等于“LastOnPosition[0]”，则“OutputNumber[n]”中定义的相应输出将始终被设置。

“AxisDirection”用于定义确定电子凸轮方向的轴方向。

当“CamSwitchMode”设置为“Time based”时，使用“Duration”。

上面的时序图是“Position based”的，“AxisDirection”是“Positive (1)”或“Both(0)”。轴是匀速运动的。

当“CamSwitchMode”基于时基时，使用“Duration”。它表示相应的“OutputNumber”设置为TRUE后的输出时间。它是非负值。

上图显示了前三个轨道的输出，所有轨道都处于“OffCompensation”，第二个轨道处于“OnCompensation”。

“OffCompensation”用于补偿输出断开时的输出延时。“OnCompensation”用于补偿接通时的输出延时。“OffCompensation”和“OnCompensation”的索引用于对应的轨道号。它可以是正值，也可以是负值。正值时时间被延迟，负值时时间被提前。

负的时间值将根据实际速度计算。“OnCompensation”和“OffCompensation”可以在“CamSwitchMode”模式“Position based”和“Time based”以及“AxisDircetion”的任何方向上使用。轨迹的输出时间Ttrack计算如下:

Ttrack = Tout – TOnCompensation + TOffCompensation

Ttrack:轨道的输出时间

Tout:没有补偿的轨道的输出时间

TOnCompensation:“OnCompensation”的补偿时间

TOffCompensation:“OffCompensation”的补偿时间
重要
MC_DigitalCamSwitch总是在轴匀速移动时使用。
如果将轴设置为旋转轴，则“OnCompensation”或“OffCompensation”将发生以下操作，并且轴会突然改变。
长时间的“OnCompensation”或“OffCompensation”值可能会导致输出时序不稳定，例如时间等于旋转的时间。
如果“OnCompensation ”的值超过“LastOnPosition ”点+ “ OffCompensation ”点，则输出时序不稳定。
如果“ OffCompensation ”的值超过“FirstOnPosition ”点+“ OnCompensation”点，则输出时序不稳定。

上图提供了一个如何使用hysteresis函数的例子。指定一个hysteresis窗口可以防止在开关位置周围区域出现一系列不需要的脉冲。轨道0对应的输出被设置的位置由“FirstOnPosition”和“LastOnPosition”改变为“FirstOnPosition+ Hysteresis”和“LastOnPosition+ Hysteresis”。

轴正向移动的例子:

在点a，轴位置达到配置的“FirstOnPosition”。当位置达到“FirstOnPosition + Hysteresis”时，输出打开。

在点b，轴位置到达关断位置“LastOnPosition”。当位置达到“LastOnPosition + Hysteresis”时，输出关闭。

“Hysteresis”是为了避免在开关点周围进行多次开关。

确保输入和输出参数为下表所示的数据类型。

MC_CAMSWITCH_REF是一个开关结构，用作ON/OFF开关模式数据。

STRUCT
TrackNumber: ARRAY[0..15] of UINT;(*TrackNumber是 参考轨道*)
OutputNumber: ARRAY[0..15] of UINT;(*输出数*)
FirstOnPosition: ARRAY[0..15] of LREAL;
LastOnPosition: ARRAY[0..15] of LREAL;
AxisDirection: ARRAY[0..15] of MC_AXIS_DIRECTION;(* 0: 两个方向，1:正方向，2:负方向*)
CamSwitchMode: ARRAY[0..15] of MC_CAMSWITCH_MODE;(* 默认0:基于位置;1:基于时间*)
Duration: ARRAY[0..15] of TIME;
END_STRUCT

MC_TRACK_REF是一个包含轨道属性的特定结构体。

STRUCT
OnCompensation: ARRAY [0..15]TIME;(*每个轨道上开关打开的延时的 时间补偿。*)
OffCompensation: ARRAY [0..15]TIME;(*每个轨道上开关断开的延 迟的时间补偿。*)
Hysteresis: ARRAY[0..15]LREAL; (*开关点的距离(正负方向) ，其中直到轴离开该区域才执行开关，以免在开关点周围进行多次开关。 *)
END_STRUCT

例
(*定义MC_DigitalCamSwitch案例*)
#define STATE_READY     0
#define STATE_CAMIN     1
#define STATE_DIGITALCAM 2
#define NEXT_STEP       3
#define STATE_MOVE_POS  9
#define STATE_MOVE_NEG  6
#define STATE_STOP      5
#define ERROR_STEP      7
#define STATE_ERROR     8
VAR
 state : USINT ;
 RefTrack : lib:MC_TRACK_REF ;
 RefSwith : lib:MC_CAMSWITCH_REF ;
 EnableDigiCam : BOOL := FALSE ;
 Inst_MC_DigitalCamSwitch : MC_DigitalCamSwitch ;
 mcSource : MC_SOURCE ;
 Axis0Ref : lib:AXIS_REF ;
 MovePositive : BOOL := TRUE ;
(*$desc=移动正方向启用变量*)
Inst_MC_MoveVelocity1 : MC_MoveVelocity ;
 mDirection : MC_DIRECTION ;
 BufferMode : MC_BUFFER_MODE ;
 ErrorID0 : UINT ;
 MoveNegative : BOOL := FALSE ;
(*$desc=启用负移动变量*)
bStop: BOOL:= TRUE;
(* $ desc =要停止运动,它必须为IO1 *)
  (*$profile=IO*)
  (*$embed=(VERS=1    MODE=0    POINT=1)*)
 Inst_MC_Stop4 : MC_Stop ;
END_VAR
(*变量初始化*)

(******************** 切换设置*********************)
RefSwith.TrackNumber[0] := UINT#0;
RefSwith.OutputNumber[0] := UINT#0;
RefSwith.FirstOnPosition[0] := LREAL#2000.0;
RefSwith.LastOnPosition[0] := LREAL#3000.0;
RefSwith.AxisDirection[0] := mcPositive;
RefSwith.Duration[0] := time#0;
RefSwith.CamSwitchMode[0] := mcPosition;

RefSwith.TrackNumber[1] := UINT#1;
RefSwith.OutputNumber[1] := UINT#1;
RefSwith.FirstOnPosition[1] := LREAL#2500.0;
RefSwith.LastOnPosition[1] := LREAL#3000.0;
RefSwith.AxisDirection[1] := mcNegative;
RefSwith.Duration[1] := time#0;
RefSwith.CamSwitchMode[1] := mcPosition;

RefSwith.TrackNumber[2] := UINT#2;
RefSwith.OutputNumber[2] := UINT#2;
RefSwith.FirstOnPosition[2] := LREAL#3500.0;
RefSwith.LastOnPosition[2] := LREAL#4000.0;
RefSwith.AxisDirection[2] := mcBoth;
RefSwith.Duration[2] := time#0;
RefSwith.CamSwitchMode[2] := mcPosition;

RefSwith.TrackNumber[3] := UINT#3;
RefSwith.OutputNumber[3] := UINT#3;
RefSwith.FirstOnPosition[3] := LREAL#5000.0;
RefSwith.LastOnPosition[3] := LREAL#1000.0;
RefSwith.AxisDirection[3] := mcBoth;
RefSwith.Duration[3] := time#0;
RefSwith.CamSwitchMode[3] := mcPosition;

(******************** 轨道设置 *********************)
RefTrack.OnCompensation[0] := T#0;
RefTrack.OffCompensation[0] := t#0;
RefTrack.Hysteresis[0] := 0;

RefTrack.OnCompensation[1] := T#0;
RefTrack.OffCompensation[1] := t#0;
RefTrack.Hysteresis[1] := 0;

RefTrack.OnCompensation[2] := T#0;
RefTrack.OffCompensation[2] := t#0;
RefTrack.Hysteresis[2] := 0;

RefTrack.OnCompensation[3] := T#0;
RefTrack.OffCompensation[3] := t#0;
RefTrack.Hysteresis[3] := 0;

mcSource := mcCommandedValue;
Axis0Ref.AxisNo := 0;
mDirection := mcPositiveDirection;
BufferMode := mcAborting;


(******************* 循环程序 *********************)
CASE State OF
    (*在运动前上电并归零轴... *)
    (********************STATE_READY****************************)
    STATE_READY:
    IF(EnableDigiCam) THEN 
        Inst_MC_DigitalCamSwitch( Enable := TRUE(*BOOL*), 
                                    Axis := Axis0Ref(*AXIS_REF*), 
                                    Switches := RefSwith(*MC_CAMSWITCH_REF*), 
                                    TrackOptions := RefTrack(*MC_TRACK_REF*),         
                                    EnableMask := UINT#15(*UINT*), 
                                    ValueSource := mcSource(*MC_SOURCE*) );
        
        State:=STATE_DIGITALCAM;
    END_IF;
    (********************STATE_DIGITALCAM****************************)
    STATE_DIGITALCAM: 
    IF (Inst_MC_DigitalCamSwitch.InOperation) AND (MovePositive = TRUE) THEN        
        mDirection := mcPositiveDirection;
        
        Inst_MC_MoveVelocity1( Execute := FALSE (*BOOL*), 
                                Axis := Axis0Ref (*AXIS_REF*),         
                                ContinuousUpdate := TRUE (*BOOL*), 
                                Velocity := LREAL#100.0 (*LREAL*), 
                                Acceleration := LREAL#1000.0 (*LREAL*), 
                                Deceleration := LREAL#1000.0 (*LREAL*), 
                                Jerk := LREAL#500.0(*LREAL*), 
                                Direction := mDirection (*MC_DIRECTION*), 
                                BufferMode := BufferMode (*MC_BUFFER_MODE*) );
        
        
        Inst_MC_MoveVelocity1( Execute := TRUE (*BOOL*), 
                                Axis := Axis0Ref (*AXIS_REF*),         
                                ContinuousUpdate := TRUE (*BOOL*), 
                                Velocity := LREAL#100.0 (*LREAL*), 
                                Acceleration := LREAL#1000.0 (*LREAL*), 
                                Deceleration := LREAL#1000.0 (*LREAL*), 
                                Jerk := LREAL#500.0(*LREAL*), 
                                Direction := mDirection (*MC_DIRECTION*), 
                                BufferMode := BufferMode (*MC_BUFFER_MODE*) );
        
        State := STATE_MOVE_POS;
    ELSIF (Inst_MC_DigitalCamSwitch.Error) THEN
        (*在变量中储存错误ID*)
        ErrorID0:=Inst_MC_DigitalCamSwitch.ErrorID;
        State:= STATE_ERROR;
    END_IF;
    (********************STATE_MOVE_POS****************************)
    STATE_MOVE_POS:    
    IF (Inst_MC_MoveVelocity1.InVelocity) AND (MoveNegative = TRUE) THEN
        mDirection := mcNegativeDirection;
        
        Inst_MC_MoveVelocity1( Execute := FALSE (*BOOL*),
                                Axis := Axis0Ref (*AXIS_REF*), 
                                ContinuousUpdate := TRUE (*BOOL*), 
                                Velocity := LREAL#100.0 (*LREAL*), 
                                Acceleration := LREAL#1000.0 (*LREAL*), 
                                Deceleration := LREAL#1000.0 (*LREAL*), 
                                Jerk := LREAL#500.0(*LREAL*), 
                                Direction := mDirection (*MC_DIRECTION*), 
                                BufferMode := BufferMode (*MC_BUFFER_MODE*) );
        
        Inst_MC_MoveVelocity1( Execute := TRUE (*BOOL*),  
                                Axis := Axis0Ref (*AXIS_REF*),
                                ContinuousUpdate := TRUE (*BOOL*), 
                                Velocity := LREAL#100.0 (*LREAL*), 
                                Acceleration := LREAL#1000.0 (*LREAL*), 
                                Deceleration := LREAL#1000.0 (*LREAL*), 
                                Jerk := LREAL#500.0(*LREAL*), 
                                Direction := mDirection (*MC_DIRECTION*), 
                                BufferMode := BufferMode (*MC_BUFFER_MODE*) );
        
        State:=STATE_MOVE_NEG;
    ELSIF (Inst_MC_MoveVelocity1.Error) THEN
        (*在变量中储存错误ID *)
        ErrorID0:=Inst_MC_MoveVelocity1.ErrorID;
        State:= STATE_ERROR;
    END_IF;
    (********************STATE_MOVE_NEG****************************)
    STATE_MOVE_NEG:
    IF((Inst_MC_MoveVelocity1.InVelocity) AND (bStop = TRUE)) THEN
        Inst_MC_Stop4( Execute := FALSE(*BOOL*), 
                        Axis := Axis0Ref(*AXIS_REF*), 
                        Deceleration := LREAL#2000.0(*LREAL*), 
                        Jerk := LREAL#1000.0(*LREAL*) );
        
        Inst_MC_Stop4( Execute := TRUE(*BOOL*), 
                        Axis := Axis0Ref(*AXIS_REF*), 
                        Deceleration := LREAL#2000.0(*LREAL*), 
                        Jerk := LREAL#1000.0(*LREAL*) );
        
        State:= STATE_STOP;
    ELSIF (Inst_MC_MoveVelocity1.Error) THEN
        (*在变量中储存错误ID *)
        ErrorID0:=Inst_MC_MoveVelocity1.ErrorID;
        State:= STATE_ERROR;
    END_IF;
    (********************STATE_STOP****************************)
    STATE_STOP:  (*停止移动*)
    IF (Inst_MC_Stop4.Done = TRUE) THEN
        Inst_MC_Stop4( Execute := FALSE(*BOOL*),
                        Axis := Axis0Ref(*AXIS_REF*),  
                        Deceleration := LREAL#2000.0(*LREAL*), 
                        Jerk := LREAL#1000.0(*LREAL*) );
        State:=NEXT_STEP;
    ELSIF (Inst_MC_Stop4.Error = TRUE) THEN
        (*在变量中储存错误ID *)
        ErrorID0:= Inst_MC_Stop4.ErrorID;
        Inst_MC_Stop4( Execute := FALSE(*BOOL*), 
                        Axis := Axis0Ref(*AXIS_REF*), 
                        Deceleration := LREAL#2000.0(*LREAL*), 
                        Jerk := LREAL#1000.0(*LREAL*) );
        State:=ERROR_STEP;        
    END_IF;

    ERROR_STEP:
    (* 当运动错误时，在此添加代码*)
    
    NEXT_STEP:
    (*在MC_Stop calling停止后，再次添加代码 *)

END_CASE;
