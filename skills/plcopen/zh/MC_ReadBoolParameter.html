MC_ReadBoolParameter

读取指定轴的BOOL类型参数。

描述

这个功能块返回参数号指定的参数值。当“Enable”更改为TRUE时，“Valid”将更改为TRUE，如果没有发生错误，“Value”将输出参数的值。此功能块不影响PLCopen状态机。它可以在任何状态下工作。

确保输入和输出参数是下表所示的数据类型。

对非BOOL轴参数使用MC_ReadParameter。

输入/输出

名称数据类型描述
输入
EnableBOOL在启用时连续获取参数的值。
AxisAXIS_REF参考轴
ParameterNumberINT指定参数的参数号。
请参见参数。
输出
ValidBOOL在输出“Value”上有一个有效的输出。
BusyBOOLFB尚未完成，预计会有新的输出值。
ErrorBOOL表示FB中发生了错误。
ErrorIDUINT错误号。出错时非零。
ValueLREAL指定参数的LREAL值，如果数据类型为DINT, LINT, REAL，则输出将显式转换为LREAL。

例

例1

例
(*MC_ReadBoolParameter的示例*)
ParNum := 10000;  (* PARBOOLTEST *)
inst_ReadBoolParameter(Axis:=Axis0Ref,Enable:=bEnable,ParameterNumber:=ParNum);
bValid := inst_ReadBoolParameter.Valid;
bBusy := inst_ReadBoolParameter.Busy; 
bError := inst_ReadBoolParameter.Error; 
ErrorId := inst_ReadBoolParameter.ErrorID;
IF bValid THEN
    bValue := inst_ReadBoolParameter.Value;
END_IF;
