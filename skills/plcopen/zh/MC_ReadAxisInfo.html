MC_ReadAxisInfo

读取指定轴的有关信息。

描述

该功能块用于读取指定轴的信息。

如果输入中有错误，“Error”和“ErrorID”将输出，而“Enable”为TRUE。

成功执行后，以下输出将被刷新:

  • HomeAbsSwitch:数字回零开关输入处于激活状态
  • LimitSwitchPos:正向硬件端开关处于激活状态
  • LimitSwitchNeg:反向硬件端开关处于激活状态
  • Simulation: 轴处于仿真模式
  • CommunicationReady:“Network”初始化，准备进行通信
  • ReadyForPowerOn: 驱动器已准备好启动(上电)
  • PowerOn: TRUE表示电源状态已切换为ON
  • IsHomed:轴的绝对参考位置已知(轴已归零)
  • AxisWarning:轴上存在警告

这个功能块不影响PLCopen状态机。它可以在任何状态下工作。

确保输入和输出参数为下表所示的数据类型。

输入/输出

名称数据类型描述
输入
EnableBOOL在启用时不断获取轴信息
AxisAXIS_REF参考轴
输出
ValidBOOL一组有效的输出可用
BusyBOOLFB尚未完成，预计会有新的输出值
ErrorBOOL表示FB中发生了错误。
ErrorIDUINT错误标识
HomeAbsSwitchBOOL数字回零开关输入处于激活状态
LimitSwitchPosBOOL正向硬件端开关处于激活状态
LimitSwitchNegBOOL负向硬件端开关处于激活状态
SimulationBOOL轴处于模拟模式
CommunicationReadyBOOL“Network”已初始化并准备好进行通信
ReadyForPowerOnBOOL驱动器准备启用(上电)
PowerOnBOOL如果TRUE，表示电源状态处于ON
IsHomedBOOL轴的绝对参考位置是已知的(轴已回零)
AxisWarningBOOL轴上存在警告

例

例1

例
(* MC_ReadAxisInfo的示例*)
inst_ReadAxisInfo( Axis := Axis0Ref, Enable := bEnable );
bValid:= inst_ReadAxisInfo.Valid;
bBusy := inst_ReadAxisInfo.Busy; 
bError := inst_ReadAxisInfo.Error; 
ErrorId := inst_ReadAxisInfo.ErrorID;
IF bValid THEN
    bHomeAbsSwitch := inst_ReadAxisInfo.HomeAbsSwitch;
    bLimitSwitchPos := inst_ReadAxisInfo.LimitSwitchPos;
    bLimitSwitchNeg := inst_ReadAxisInfo.LimitSwitchNeg;
    bSimulation := inst_ReadAxisInfo.Simulation;
    bCommunicationReady := inst_ReadAxisInfo.CommunicationReady; 
    bReady4PowerOn := inst_ReadAxisInfo.ReadyForPowerOn;
    bPowerOn := inst_ReadAxisInfo.PowerOn;
    bIsHomed := inst_ReadAxisInfo.IsHomed;
    bAxisWarning := inst_ReadAxisInfo.AxisWarning;
END_IF;
