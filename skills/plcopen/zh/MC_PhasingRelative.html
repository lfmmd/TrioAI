MC_PhasingRelative

相对于现有的相位移动，在从轴的主轴位置创建一个相对相位移动。

描述

此功能块用于改变当前处于同步控制中的主轴的相位差。

与作用于从轴的MC_MoveSuperimposed不同，以从轴视角来看，MC_PhasingRelative作用于主轴。

当轴由以下同步控制指令控制时，可以执行MC_PhasingRelative: MC_CamIn, MC_GearIn, MC_GearInPos, MC_CombineAxes。

请确保输入和输出参数为下表所示的数据类型。

主轴位置相对于实际物理位置发生改变。这类似于将主轴上的联轴器打开一会儿，并用于将轴向其主轴延迟或推进。

以从轴视角进行相位移动。主轴无法获悉从轴经历了相位移动。从轴视角的看到的MasterPos= PhysicalMasterPos + PhaseShiftValueSlaveAxis，相位移动值具有位置偏移的特征。

相位移动会一直存在，直到另一个“Phasing”命令再次改变它。

对于适用的相位移动，相对相位移动会相互相加。

输入/输出

名称数据类型描述
输入
ExecuteBOOL从上升沿开始相位过程。
MasterAXIS_REF参考主轴
SlaveAXIS_REF参考从轴
PhaseShiftLREAL从轴与主轴位置附加相位差[u]。
VelocityLREAL达到相位差的最大速度[u/s]。
AccelerationLREAL达到相位差的最大加速度[u/s2]
DecelerationLREAL达到相位差的最大减速度[u/s2]
JerkLREAL达到相位差的最大加加速度 [u/s3]
输出
DoneBOOL命令相位达到
BusyBOOLFB尚未完成，预计会有新的输出值。
ActiveBOOL表示FB对轴具有控制权
CommandAbortedBOOL“Command”被另一个命令终止
ErrorBOOL表示在功能块中发生了错误
ErrorIDUINT错误标识
CoveredPhaseShiftLREAL连续显示自启动以来所覆盖的相位移动[u]。

例

例1

例
(*MC_PhasingRelative的示例 *)
MasterRef.AxisNo := 0;
SlaveRef.AxisNo := 3;

inst_PhasingRelative ( Execute := bExecute, 
                        Master := MasterRef,
                        Slave := SlaveRef, 
                        PhaseShift := lrPhaseShift, 
                        Velocity := lrVeloShift,
                        Acceleration := lrAccShift, 
                        Deceleration := lrDecShift, 
                        Jerk := lrJerkShift );

bDone := inst_PhasingRelative.Done; 
bBusy := inst_PhasingRelative.Busy; 
bActive := inst_PhasingRelative.Active; 
bCommAborted := inst_PhasingRelative.CommandAborted; 
bError := inst_PhasingRelative.Error; 
ErrorId := inst_PhasingRelative.ErrorID;
lrRelPhaseShift := inst_PhasingRelative.CoveredPhaseShift;
