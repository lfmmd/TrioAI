MC_PositionProfile

指定一个时间-位置固定的运动轮廓。

描述

MC_PositionProfile指定一个时间-位置固定的运动轮廓。这个功能并不意味着它会重复运行一个轮廓。它可以在不同的轮廓之间切换。

当“Execute”更改为TRUE时，“Busy”和“Active ”更改为TRUE。指令不进入缓存区。

当达到指令的位置时，“Done”变为TRUE，并且定位完成。

如果另一条指令中止此指令，“CommandAborted”变为TRUE，“Busy”和“Active ”变为FALSE。

确保输入和输出参数为下表所示的数据类型。

MC_TP_REF的定义如下:

“Time / Position”对的内容以“Delta Time / Pos”表示，其中“Delta”是两个连续点之间的时间差。

STRUCT
TableStart : UDINT;
NumberOfPairs : UDINT;
END_STRUCT

作为此FB的替代方案，可以使用MC_CamIn耦合到虚拟主轴。

输入/输出

名称数据类型描述
输入
ExecuteBOOL从上升沿开始运动
AxisAXIS_REF参考轴
TimePositionMC_TP_REF参考时间/位置
TimeScaleREAL轮廓的总体时间缩放因子
PositionScaleREAL总体位置比例因子
OffsetLREAL轮廓的总体偏移量[u]
BufferModeMC_BUFFER_MODE枚举类型(1-6个值):
0: mcAborting
1: mcBuffered
2: mcBlendingLow
3: mcBlendingPrevious
4: mcBlendingNext
5: mcBlendingHigh
输出
DoneBOOL轮廓完成
BusyBOOLFB尚未完成，预计会有新的输出值。
ActiveBOOL表示FB对轴具有控制权
CommandAbortedBOOL“Command”被另一个命令终止
ErrorBOOL表示在功能块中发生了错误
ErrorIDUINT错误标识

(*MC_PositionProfil的示例 *)
inst_PositionProfile( Execute := bExecute(*BOOL*), 
                        Axis := Axis0Ref (*AXIS_REF*), 
                        TimePosition := TimePositionTable(*MC_TP_REF*), 
                        TimeScale := rTimeScale(*REAL*), 
                        PositionScale := rPosScale(*REAL*), 
                        Offset := lrOffset(*LREAL*), 
                        BufferMode := mcBufferMode(*MC_BUFFER_MODE*) );
bDone := inst_PositionProfile.Done; 
bBusy := inst_PositionProfile.Busy; 
bCommAborted := inst_PositionProfile.CommandAborted;
bError := inst_PositionProfile.Error; 
ErrorId := inst_PositionProfile.ErrorID;
