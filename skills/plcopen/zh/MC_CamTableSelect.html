MC_CamTableSelect

通过设置与关联Table的连接来选择CAM曲线的Table。

描述

MC_CamTableSelect通过设置与关联Table的连接来选择CAM曲线的Table。

虚轴可以用作主轴。

MC_CamTableSelect使数据可用。这可以包括:

  • 凸轮表载入的起始点
  • 开始生成CAM轮廓

当“Done”输出设置时，CamTableID是有效的，可以在MC_CamIn FB中使用。

命令在任何时候都是可用的，无论主从轴的状态如何。

请确保输入和输出参数为下表所示的数据类型。

输入/输出

名称数据类型描述
输入
ExecuteBOOL上升沿选择
CamTableMC_CAM_REF参考CAM
PeriodicBOOL1 =周期性，0 =非周期性(单次)
MasterAbsoluteBOOL1 =绝对坐标，0 =相对坐标
SlaveAbsoluteBOOL1 =绝对坐标，0 =相对坐标
输出
DoneBOOL预选择完成
BusyBOOLFB尚未完成，预计会有新的输出值。
ErrorBOOL表示在功能块中发生了错误
ErrorIDUINT错误标识
CamTableIDUINTMC_CamIn FB中使用的CAM表标识符

例

例1

例
CamTable0.TableStart:= 1000;(*设置为实际凸轮表设置*)
CamTable0.TableFinish:= 1010;

inst_CamTableSelect ( Execute := bExecute,
                        CamTable := CamTable0,
                        Periodic := bPeriod, 
                        MasterAbsolute := bAbsolute, 
                        SlaveAbsolute := bAbsolute );
bDone := inst_CamTableSelect.Done; 
bBusy := inst_CamTableSelect.Busy; 
bError := inst_CamTableSelect.Error; 
ErrorId := inst_CamTableSelect.ErrorID;
IF bDone THEN
    TableID := inst_CamTableSelect.CamTableID;
END_IF;

代码:

//取消MTYPE
Inst_TC_CANCEL(TRUE,0,0);
Inst_TC_CANCEL1(TRUE,1,0);

//凸轮配置
TCW_UNITS(0,1);
TCW_SPEED(0,10);
TCW_ACCEL(0,10);
TCW_DECEL(0,10);
Inst_TC_DEFPOS0(TRUE,0,0);
TCW_SERVO(0,TRUE);

//主轴配置
TCW_UNITS(1,1);
TCW_SPEED(1,1);
TCW_ACCEL(1,10);
TCW_DECEL(1,10);
Inst_TC_DEFPOS1(TRUE,1,0);
TCW_SERVO(1,TRUE);

TCW_WDOG(1);

//设置凸轮表
CamTable.TableStart:=200;
CamTable.TableFinish:=211;
Inst_MC_CamTableSelect(TRUE, CamTable, FALSE, FALSE, FALSE);
CamTableId:=Inst_MC_CamTableSelect.CamTableID;

// MC_CamIn配置
master.AxisNo:=1;
Slave.AxisNo:=0;
StartMode:=mcAbsolute;
MasterValueSource:=mcCommandedValue;
Inst_MC_CamIn(TRUE, Master, Slave, 0, 0, 1, 1, 0, 0, StartMode, MasterValueSource, CamTableID);

// MC_CamOut设置
bCamOutExecute:=FALSE;
Inst_MC_CamOut(bCamOutExecute, Slave);

//运行主轴
Inst_TC_FORWARD(TRUE,1);
