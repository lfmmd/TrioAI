MC_ReadDigitalInput

提供对指定输入的值的访问。

描述

当“Enable”变为TRUE时，如果没有错误输入，FB将连续读取所选输入信号的值。

通过InputNumber选择相应的输入。

“InputNumber”应给定有效值，同时“Valid”为TRUE，否则FB将生成错误。

这个功能块不影响PLCopen状态机。

请确保输入和输出参数为下表所示的数据类型。

不能保证数字信号将被FB看到:在下一个功能块周期发生之前，数字输入上的短脉冲可能会结束。

“InputNumber”范围是基于硬件配置的。

输入/输出

名称数据类型描述
输入
EnableBOOL在启用时连续获取所选输入信号的值。
InputNumberINT选择输入。
输出
ValidBOOL在输出' Value '上有一个有效的输出。
BusyBOOLFB尚未完成，预计会有新的输出值。
ErrorBOOL表示FB中发生了错误。
ErrorIDUINT错误标识
ValueBOOL指定输入信号的值。

例

例1

例
(* MC_ReadDigitalInput的示例*)
InNum := INT#1;
inst_ReadDigitalInput(Enable:=bEnable,InputNumber:=InNum);
bValid:= inst_ReadDigitalInput.Valid;
bBusy := inst_ReadDigitalInput.Busy; 
bError := inst_ReadDigitalInput.Error; 
ErrorId := inst_ReadDigitalInput.ErrorID;
IF bValid THEN
    bValue := inst_ReadDigitalInput.Value;
END_IF;
