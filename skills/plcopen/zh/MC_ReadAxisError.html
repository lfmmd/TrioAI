MC_ReadAxisError

获取与功能块无关的常规轴错误。

描述

此功能块表示与功能块无关的常规轴错误，例如轴错误，驱动器错误，通信错误。

有关“AxisErrorID”输出的详细信息，请参阅轴错误。

此功能块不影响PLCopen状态机。

请确保输入和输出参数为下表所示的数据类型。

输入/输出

名称数据类型描述
输入
EnableBOOL在启用时不断获取轴信息
AxisAXIS_REF参考轴
输出
ValidBOOL如果有有效的输出，则为TRUE
BusyBOOLFB尚未完成，预计会有新的输出值
ErrorBOOL表示FB中发生了错误
ErrorIDUINT错误标识
AxisErrorIDUINT轴错误的值

例

例1

例子
(* MC_ReadAxisError的示例*)
inst_ReadAxisError( Axis := Axis0Ref, Enable := bEnable );
bValid:= inst_ReadAxisError.Valid;
bBusy := inst_ReadAxisError.Busy; 
bError := inst_ReadAxisError.Error; 
ErrorId := inst_ReadAxisError.ErrorID;
IF bValid THEN
    AxisErrorId := inst_ReadAxisError.AxisErrorID;
END_IF;
