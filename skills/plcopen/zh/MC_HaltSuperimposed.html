MC_HaltSuperimposedd

命令停止所有轴的叠加运动。

描述

MC_HaltSuperimposedd用于停止轴的叠加运动。在非缓冲模式下，可以在轴减速期间设置另一个运动命令，这将中止MC_HaltSuperimposedd，并将立即执行。

如果此命令处于激活状态，则可以发出下一个命令。

在“Execute”的上升沿，在轴减速时设置“Busy”。当“Error”或“CommandAborted”更改为TRUE时，“Busy”被重置。

在“Execute”和“Error”的上升沿更改为TRUE时，“Busy ”，“Done”和“CommandAborted”不能为TRUE。

在“Execute”的上升沿，当“Done”变为TRUE时，“Busy”变为FALSE。如果“CommandAborted” 变为TRUE，“Done”变为FALSE。

在“Execute”下降沿处，必须保证相应的输出(“Done”或“Error”)至少设置一个周期。

确保输入和输出参数为下表所示的数据类型。

底层运动在执行后不会中断。

输入/输出

名称数据类型描述
输入
ExecuteBOOL在上升沿开始运动
AxisAXIS_REF参考轴
DecelerationLREAL减速度值[u/s2]
JerkLREAL加加速度值 [u/s3]
输出
DoneBOOL叠加运动停止
BusyBOOLFB尚未完成，预计会有新的输出值
CommandAbortedBOOL“Command”被另一个命令终止
ErrorBOOL表示FB中发生了错误
ErrorIDUINT错误标识

例

例1

下面的例有三个FB: MC_MoveRelative, MC_MoveSuperimposed, 和 MC_HaltSuperimposedd。在启动MC_MoveRelative运动后，MC_MoveSuperimposed将在“AXIS”正在进行时被多次触发。当“AXIS”处于运动状态时，MC_HaltSuperimposed也会被触发。
例
VAR
 Inst_MC_MoveRelative : MC_MoveRelative ;
 Inst_MC_MoveSuperimposed : MC_MoveSuperimposed ;
 Inst_MC_HaltSuperimposed : MC_HaltSuperimposed ;
 refAxis : lib:AXIS_REF ;
  (*$desc=轴0参照*)
 Exe_Rel : BOOL := FALSE ;
  (*$desc=MC_MoveRelative运动执行控制变量*)
 bContUpdate : BOOL := FALSE ;
  (*$desc=ContinuousUpdate函数启用变量*)
 BufferMode : MC_BUFFER_MODE := mcAborting ;
 Exe_MS : BOOL := FALSE ;
  (*$desc=叠加运动执行控制变量*)
 auxAxis : lib:AXIS_REF ;
  (*$desc=叠加运动的辅助轴*)
 Exe_HS : BOOL := FALSE ;
  (*$desc=停止叠加运动的控制变量*)
END_VAR
