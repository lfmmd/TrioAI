MC_TouchProbe

在触发事件时记录轴的位置。

描述

MC_TouchProbe在触发事件时记录轴的位置。实际的位置和速度将被记录在“Execute“的上升沿。

“WindowOnly”用于启用或禁用窗口输入(“FirstPosition”和“LastPosition”)。当“WindowOnly”为TRUE时，只有当轴位置在“FirstPosition”和“LastPosition”指定的范围内时才检测到触发器。当“WindowOnly”为FALSE时，将检测所有轴位置的触发器。

窗口范围由“FirstPosition”和“LastPosition”指定。如果轴是线性轴，则应该定义为“FirstPosition”<“LastPosition”。窗口的范围是[“FirstPosition”，“LastPosition”]。

如果轴以模态模式工作，则触发器事件窗口的范围定义为从起始位置“FirstPosition”到“LastPosition” 沿正向旋转。

“Busy”在“Execute”的上升沿更改为TRUE，并保持TRUE，直到“Done”，“CommandAborted”或“Error”更改为TRUE。

当该指令被MC_AbortTrigger终止时(仅能这样终止)，“CommandAborted”变为TRUE，并保持TRUE，直到' Error '或“Execute”变为FALSE。如果“Execute”为FALSE，它将在一个或多个周期内保持TRUE。

当触发事件被记录时，“Done”更改为TRUE，并保持TRUE，直到“CommandAborted”或“Execute”更改为FALSE。如果“Execute”为FALSE， “Done”将在一个或多个周期内保持TRUE。

输入输出参数“TriggerInput”定义了触发器输入源和模式。它是一个结构类型，如下所示:
STRUCT
Mode : MC_TRIGGER_MODE;
Channel: UINT;
Source: UINT;
Edge: MC_TRIGGER_EDGE;
END_STRUCT

“Edge”的成员定义了记录函数的触发输入的边缘类型。它是一个枚举，值为“mcRisingEdge”和“mcFallingEdge”。

“Mode”的成员是具有“mcHardware”和“mcTime”值的枚举类型。

“Channel”和“Source” 的成员在“Mode”的不同模式下呈现不同的含义。

  • 硬件
channel:0选择通道A(参考TrioBASIC帮助中的REG_INPUTS)。
1选择通道B(参考TrioBASIC帮助中的REG_INPUTS)。
0 .. 511当源设置为4时，数字输入选择。
source:0选择第一个24V输入。
1选择ZMARK。
2选择第二个24V输入。(如果可用)
4选择任意数字输入作为源，在任意轴上使用。
  • mcTime
channel:0…7这是要使用的配准通道(范围0..7)
source:source + 256分配给逻辑通道的物理输入(0x80 +输入号)。

确保输入和输出参数为下表所示的数据类型。

用于单次触发操作，即仅记录“Execute”上升沿后的第一个事件。后续可能发生的事件将被忽略。

一个FB实例应该代表一个探针命令。

当同一探针和轴上有多个实例时，MC_TRIGGER_REF的元素应该扩展为TouchProbeID -唯一的探针命令的标识-这可以链接到MC_AbortTrigger。

数据类型TriggerInput的定义如下:

STRUCT
Mode : MC_TRIGGER_MODE;
Channel: UINT;
Source: UINT;
Edge: MC_TRIGGER_EDGE;
END_STRUCT

输入/输出

名称数据类型描述
输入
ExecuteBOOL在上升沿开始接触探针记录
AxisAXIS_REF参考轴
TriggerInputMC_TRIGGER_REF参考触发信号源。
WindowOnlyBOOL如果设置，只使用窗口(定义如下)来接受触发事件
FirstPositionLREAL起始位置(正方向)触发事件被接受。值包含在窗口中。
LastPositionLREAL窗口的停止位置。值包含在窗口中。
输出
完成保龄球触发事件记录
忙保龄球FB尚未完成，预计会有新的输出值
CommandAborted保龄球“Command”被另一个命令终止(MC_AbortTrigger)
错误保龄球表示FB中发生了错误
ErrorID使用UINT错误标识
RecordedPositionLREAL触发事件发生的位置
RecordedVelocityLREAL触发事件发生的速度

例

例1

例
Axis0Ref.AxisNo := 0;
mTrigInput.Channel := 0; (* 要使用的配准通道* *)
mTrigInput.Edge := mcRisingEdge;  (*上升沿*)
mTrigInput.Mode := mcHardware; 
mTrigInput.Source := 0; (* 选择24V 寄存器输入*)

inst_TouchProbe( Axis := Axis0Ref, (*AXIS_REF*) 
                TriggerInput := mTrigInput, (*MC_TRIGGER_REF*)
                Execute := bExecute, (*BOOL*) 
                WindowOnly := bWindowOnly, (*BOOL*) 
                FirstPosition := lrOnPos, (*LREAL*) 
                LastPosition := lrOffPos ); (*LREAL*)

bDone := inst_TouchProbe.Done; 
bBusy := inst_TouchProbe.Busy; 
bCommAborted := inst_TouchProbe.CommandAborted;
bError := inst_TouchProbe.Error; 
ErrorId := inst_TouchProbe.ErrorID;
IF (bDone = TRUE) THEN
    lrRecordPosition := inst_TouchProbe. RecordedPosition; 
    lrRecordVelocity := inst_TouchProbe. RecordedVelocity;
END_IF;
