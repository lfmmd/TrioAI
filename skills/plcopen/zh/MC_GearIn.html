MC_GearIn

指定从轴和主轴VELOCITY之间的比率。

描述

从轴上升到主轴速度的比率，并在达到这个比率时锁定。同步过程中的任何损失的距离都不会被追赶上。

当MC_GearIn运行时，可以使用连续的MC_GearIn命令改变传动比，而不需要先执行MC_GearOut。

在“_InGear”_更改为TRUE之后，特定系统会发生位置锁定或仅速度锁定。

确保输入和输出参数为下表所示的数据类型。

输入/输出

名称数据类型描述
输入
ExecuteBOOL从上升沿开始齿轮传动过程
MasterAXIS_REF参考主轴
SlaveAXIS_REF参考从轴
ContinuousUpdateBOOLTRUE:当函数块被触发(上升' Execute ')时，它将-只要它保持TRUE -使函数块使用输入变量的当前值并将其应用于正在进行的运动。
FALSE:使用' Execute '输入的上升沿，在整个移动过程中忽略输入参数的变化。
RatioNumeratorINT齿轮比分子
RatioDenominatorUINT齿轮比分母
MasterValueSourceMC_SOURCE定义同步的源:
mcSetValue -主轴设置值同步
mcActualValue -主轴实际值同步
AccelerationLREAL齿轮传动的加速度
DecelerationLREAL齿轮传动的减速度
JerkLREAL齿轮加加速度
输出
InGearBOOLTRUE:设置值=命令值
BusyBOOLFB尚未完成，预计会有新的输出值
ActiveBOOL表示FB对轴具有控制权

例

例1

“Axis1Master”是主轴，“Axis2Slave”是从轴。根据MC_GearIn中使用的特殊比率，从轴与主轴一起移动。从轴与主轴断开，并在主轴停止阶段执行MC_GearOut，保持与主轴断开后的最后速度。
例
VAR
Inst_MC_Power: MC_Power;
(*$desc=启用主轴*)
Inst_MC_Power1: MC_Power;
(*$desc=启用从轴*)
Axis1Master: lib:AXIS_REF;
(*$desc= MC_CamIn中匀速移动的主轴*)
Axis2Slave: lib:AXIS_REF;
(*$desc= MC_CamIn中匀速移动的从轴*)
PowerAxes: BOOL:= FALSE;
(*$desc=为轴上电*)
Inst_MC_MoveVelocity: MC_MoveVelocity;
(*$desc=以恒定速度移动主轴*)
MoveMaster: BOOL:= FALSE;
(*$desc=开始主轴移动*)
mDir: MC_DIRECTION:= mcPositiveDirection;
(*$desc=运动方向定义*)
mBufferMode: MC_BUFFER_MODE:= mcAborting;
(*$desc=缓冲模式设置的主轴运动*)
bInVelo: BOOL:= FALSE;
(*$desc=主轴运动状态*)
bStaMaster: BOOL;
(*$desc=主轴电源状态*)
bStaSlave: BOOL:= FALSE;
(*$desc=从轴电源状态*)
mMasterSrc: MC_SOURCE:= mcActualValue;
(*$desc=定义主轴同步的源*)
mStartMode:MC_START_MODE:= mcAbsolute;
(*$desc=启动模式设置*)
Inst_MC_Stop: MC_Stop;
bStopSlave: BOOL值:= FALSE;
(*$desc=停止从轴*)
bStopped: BOOL:= FALSE;
(*$desc=从轴是否停止*)
bOutDone: BOOL:= FALSE;
(*$desc= cam out是否执行完成*)
Inst_MC_GearIn: MC_GearIn;
Inst_MC_GearOut: MC_GearOut;
bGearOut: BOOL:= FALSE;
(*$desc=执行分离齿轮运动变量*)
bGearExe: BOOL:= TRUE;
(*$desc=使从轴与主轴啮合*)
bInGear: BOOL:= FALSE;
(*$desc=齿轮运动啮合状态*)
END_VAR

例2

一台机器在轴1上有一个自动进给，它必须以设定的比例移动到轴0。这个比率是使用输入1-3来选择一个“齿轮”。这个比值可以连续更新。输入的组合将选择中间传动比。例如:当Input1 = OFF, Input2 = ON, Input3 = ON时，比值为6:1。
