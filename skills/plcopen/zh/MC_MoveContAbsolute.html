MC_MoveContAbsolute

命令一个受控运动以指定的结束速度到达指定的绝对位置。

描述

MC_MoveContAbsolute命令以指定的结束速度控制运动到指定的绝对位置。如果到达命令位置，并且没有将新的运动命令放入缓存区，则轴继续以指定的“EndVelocity”运行。

如果输入“Acceleration”， “Deceleration”和“Jerk”为零，这些值将被设置为最大值。

输入参数“Jerk”可用于平稳加速或减速。

请确保输入和输出参数为下表所示的数据类型。

轴将在执行后改变为状态“ContinuousMotion”(意味着它不会自行停止)。

负速度*负方向=正速度。

“mcShortedWay”用于模态轴运动。

这个过程可以通过FB MC_MoveContAbsolute以下程序实现:
例
VAR
 Inst_MC_Power : MC_Power ;
 Inst_MC_ReadActualPosition : MC_ReadActualPosition ;
 refAxis0 : lib:AXIS_REF ;
(*$desc=轴0的轴引用*)
bPowerEn: BOOL:= FALSE;
(*$desc=启用轴0，它被绑定到IN1*)
 (*$profile=IO*)
 (*$embed=(VERS=1    MODE=0    POINT=1)*)
 readEn : BOOL := TRUE ;
(*$desc=读取轴的实际位置*)
 lrEndCutPos : LREAL := LREAL#50.0 ;
(*$desc=关闭激光切割的结束位置*)
 Inst_MC_WriteDigitalOutput : MC_WriteDigitalOutput ;
 LaserPort : INT := INT#8 ;
(*$desc=激光开关端口号*)
bPowerSta: BOOL值:= FALSE;
(*$desc=轴电源使能状态*)
 Inst_MC_WriteDigitalOutput1 : MC_WriteDigitalOutput ;
 Inst_MC_MoveContAbsolute : MC_MoveContAbsolute ;
 xStartCuttingCycle : BOOL := FALSE ;
(*$desc=开始切割*)
lrStartCutPos: LREAL:= LREAL#100.0;
(*$desc=起始切割位置*)
 lrCutVelocity : LREAL := LREAL#-100.0 ;
 lrFastVelocity : LREAL := LREAL#500.0 ;
(*$desc=快速运动速度*)
 mDir : MC_DIRECTION := mcPositiveDirection ;
 mBufferMode : MC_BUFFER_MODE := mcAborting ;
 Inst_MC_MoveAbsolute : MC_MoveAbsolute ;
 lrIdlePos : LREAL := LREAL#0 ;
END_VAR

在“xStartCuttingCycle”的上升沿处，MC_MoveContAbsolute将在“lrStartCutPos”上以“lrFastVelocity” 移动轴0，当再次以负方向到达“lrStartCutPos”时，返回并具有速度“lrCutVelocity”。在这个时间点上，设置“InEndVelocity”并且打开激光。由于没有其他运动FB打断此运动，MC_MoveContAbsolute将保持轴0以当前速度向负方向行进。一旦轴超过了位置“lrEndPos”，激光被关闭，MC_MoveAbsolute将轴高速移动到空闲位置:
