MC_MoveContRelative

命令一个受控运动以指定的结束速度到达指定的相对位置。

描述

MC_MoveContRelative命令以指定的结束速度控制运动到指定的相对位置。如果到达命令位置，并且没有将新的运动命令放入缓冲区，则轴继续以指定的“EndVelocity”运行。

请确保输入和输出参数为下表所示的数据类型。

轴将在执行后变为“ContinuousMotion”状态(意味着它不会自行停止)。

负速度*负方向=正速度。

“mcShortedWay”用于模态轴运动。

输入/输出

名称数据类型描述
输入
ExecuteBOOL从上升沿开始运动
AxisAXIS_REF参考轴
ContinuousUpdateBOOLTRUE:当功能块被触发(“Execute”上升沿)时，只要它保持TRUE，将使功能块使用输入变量的当前值并将其应用于正在进行的运动。
FALSE:在“Execute”输入的上升沿，整个移动过程中忽略输入参数的变化。
DistanceLREAL运动的相对距离
EndVelocityLREAL终点速度的符号值
VelocityLREAL最大速度值
AccelerationLREAL加速度值
DecelerationLREAL减速度值
JerkLREAL加加速度值
BufferModeMC_BUFFER_MODE定义FB的时间顺序。
输出
InEndVelocityBOOL到达指定距离并以要求的结束速度运行
BusyBOOLFB尚未完成，预计会有新的输出值
ActiveBOOL表示FB对轴具有控制权
CommandAbortedBOOL“Command”被另一个命令终止
ErrorBOOL表示FB中发生了错误
ErrorIDUINT错误标识

VAR
 Inst_MC_Power : MC_Power ;
 mcr1 : MC_MoveContRelative ;
 mcr2 : MC_MoveContRelative ;
 mr1 : MC_MoveRelative ;
 RefAxis : lib:AXIS_REF ;
 bPower : BOOL := FALSE ;
  (*$desc= 参照轴上电*)
 Start : BOOL := FALSE ;
  (*$desc=开始运动*)
 mBufferMode : MC_BUFFER_MODE := mcAborting ;
  (*$desc=带有 mcAborting初始值的缓冲模式变量*)
 Inst_MC_MoveRelative : MC_MoveRelative ;
 bPowerStatus : BOOL := FALSE ;
  (*$desc=轴上电状态*)
END_VAR
