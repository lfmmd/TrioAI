MC_ReadMotionState

获取一个轴当前正在进行的运动状态。

描述

这个功能块返回正在进行的轴运动状态。当“Enable”为TRUE并且所有输入都有效时，将设置一个或多个输出。

当速度为常数时，输出“ConstantVelocity”将被设置。Velocity可能为0。对于实际值，可以使用一个窗口来显示(由供应商定制)。

这个功能块不影响PLCopen状态机。

请确保输入和输出参数为下表所示的数据类型。

不同的输入“源”可能同时具有不同的轴输出。

输入/输出

名称数据类型描述
输入
EnableBOOL在启用时连续获取参数的值
AxisAXIS_REF参考轴
SourceMC_SOURCE定义相关数据的来源:mcCommandedValue、mcSetValue或mcActualValue。
输出
ValidBOOL一组有效的输出可用
BusyBOOLFB尚未完成，预计会有新的输出值
ErrorBOOL表示FB中发生了错误。
ErrorIDUINT错误标识
ConstantVelocityBOOL速度是常数。速度可能为0。对于实际值，可以使用一个窗口来显示(由供应商定制)
AcceleratingBOOL增加速度的绝对值
DeceleratingBOOL降低速度的绝对值
DirectionPositiveBOOL表示位置正在增加
DirectionNegativeBOOL表示位置正在减少

例

例1

例
(*MC_ReadMotionState的示例*)
inst_ReadMotionState ( Enable := bEnable,
                        Axis := Axis0Ref, 
                        Source := mSource );
bValid:= inst_ReadMotionState.Valid;
bBusy := inst_ReadMotionState.Busy; 
bError := inst_ReadMotionState.Error; 
ErrorId := inst_ReadMotionState.ErrorID;
IF bValid THEN
    bConstantVelocity := inst_ReadMotionState.constantVelocity;
    bAccelerating := inst_ReadMotionState.Accelerating;
    bDecelerating := inst_ReadMotionState.Decelerating;
    bDirectionPositive := inst_ReadMotionState.DirectionPositive;
    bDirectionNegative := inst_ReadMotionState.DirectionNegative;
END_IF;
