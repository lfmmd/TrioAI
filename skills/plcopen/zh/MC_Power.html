MC_Power

控制电源状态(ON或OFF)。

描述

该功能块控制电源状态(ON或OFF)。

当“Enable”更改为TRUE时，“Axis”指定的轴准备好操作，然后输出“Status”更改为TRUE。当“Status”为TRUE时，您可以控制轴。

当“Status”为FALSE时，您无法控制轴。如果“Enable”更改为FALSE，则输出“Status”将更改为FALSE。操作命令的输出将停止，轴将不再为操作做好准备。

功能块可以从轴状态“Disabled”或“ErrorStop”被调用，用于启用轴。如果轴上没有错误，当“Enable”为TRUE时，其状态将为“Standstill”。

功能块可以从未使能轴的任何状态被调用。如果轴上没有错误，则当“Enable”更改为FALSE时，其状态将为“Disabled”。

您可以将此功能块用于伺服轴或非伺服轴。每个轴只应存在一个此功能块。否则会产生意想不到的问题。

你可以使用这个指令来禁用轴在运动时的操作，所有的运动命令都会被清除。

当“Enable” 参数变为TRUE时，“Busy”(正在执行) 参数变为TRUE以表示指令已被正确识别。在“Enable” 变为TRUE并且在轴上完成进程之前，轴的“Status” 参数不会更改为TRUE。当“Status” 参数变为TRUE后，轴此时准备好可以运行。当“Enable”变为FALSE时，轴的“Busy”参数变为FALSE，当“Status”参数变为FALSE时，准备状态被清除。“Status” 参数输出轴准备完成的状态，无论“Enable”参数是TRUE还是FALSE。在移动轴之前，确保“Status”参数更改为TRUE。

执行/启用此命令相当于以下TrioBASIC命令:

AXIS_ENABLE AXIS(x) = 1

WDOG = 1

确保输入和输出参数为下表所示的数据类型。

“Enable”输入驱动器中的电源的使能状态，而不是FB本身。

如果MC_Power FB以“Enable”= TRUE调用，则可能需要一些时间来输出“Status”。您不应该在未启用时操作轴(“ Status” = FALSE)。

当“Enable”为TRUE时，您不能更改“Axis”输入，否则将产生错误并禁用轴。

如果电源故障(在操作期间)，它将过渡到“ErrorStop”状态。

每个轴只能包含1个MC_Power功能块。

输入/输出

名称数据类型描述

输出

BusyBOOL上电尚未完成，预计会有新的输出值。
ErrorBOOL指示FB内部发生错误。
ErrorIDUINT错误标识

例1

首先，将MC_Power0设置为TRUE以启用Axis0。一旦Axis0成功通电，其状态将变为“Standstill”。

接下来将MC_Home0设置为TRUE以对轴0执行回零运动。一旦回零进程顺利完成，状态将变为<NEXT_STEP>。否则，如果发生任何错误，状态将更改为<ERROR_STEP>。

例
(*定义MC_Power的示例*)
#define STATE_READY  0
#define STATE_POWER  1
#define STATE_HOMING 2
#define NEXT_STEP    3
#define ERROR_STEP   7
#define STATE_ERROR  8
(*变量声明*)
VAR
 HomeMode : MC_HOME_MODE := mcForwardZS ;
  (*$desc=存储回零方法的变量*)
 State : USINT ;
  (*$desc=当前状态*)
 MC_Power0 : MC_Power ;
 Axis0Ref : lib:AXIS_REF ;
 bPower : BOOL := FALSE ;
(*$desc=开关使能或关闭电源*)
 (*$profile=IO*)
 (*$embed=(VERS=1    MODE=0    POINT=2)*)
ErrorID0:UINT:=UINT#0;
(*$desc=用于存储' ErrorID '的变量*)
MC_Home0: MC_Home;
HomeMode0: lib:MC_HOME_REF;
(*$desc=定义回零进程*)
END_VAR .
(*变量初始化*) 
HomeMode:= mcForwardZS;
State:= STATE_READY;
Axis0Ref.AxisNo := UINT#0;
HomeMode0.HomeMode := HomeMode;
HomeMode0.HomeSpeed := 1000.0;
HomeMode0.CreepSpeed := 30.0;
HomeMode0.Acceleration := 5000.0;
HomeMode0.Deceleration := 5000.0;
HomeMode0.Jerk := 5000.0;

(* 循环程序 *) 
CASE State OF  
    (* ...在运动前给轴上电... *)
    (********************STATE_READY*********************)
    STATE_READY:
    IF (bPower) THEN   (*bPower是绑定到IO2的BOOL变量*)
        State:=STATE_POWER;
    END_IF;
    (********************STATE_POWER*********************)
    STATE_POWER:  (*轴0上电*)
    MC_Power0( Enable := TRUE(*BOOL*), 
                Axis := Axis0Ref(*AXIS_REF*));
    
    IF (MC_Power0.Status = TRUE) THEN        
        MC_Home0( Execute := FALSE(*BOOL*), 
                    Axis := Axis0Ref(*AXIS_REF*),
                    Home := HomeMode0(*MC_HOME_REF*), 
                    Position := 0.0(*LREAL*) );
         
        (*为MC_Home0执行创建上升沿*)
        MC_Home0( Execute := TRUE(*BOOL*),
                    Axis := Axis0Ref(*AXIS_REF*), 
                    Home := HomeMode0(*MC_HOME_REF*), 
                    Position := 0.0(*LREAL*) );
        
        State:=STATE_HOMING;
    ELSIF (MC_Power0.Error) THEN       
        ErrorID0 := MC_Power0.ErrorID; (*在变量中存储ErrorID *)
        (*在STATE_ERROR之前重置“Enable”*)
        MC_Power0( Enable := FALSE(*BOOL*), 
                    Axis := Axis0Ref(*AXIS_REF*) );
        State := STATE_ERROR;
    END_IF;
    (********************STATE_HOMING*********************)
    STATE_HOMING:  (*回零轴*)
    IF (MC_Home0.Done = TRUE) THEN
        State:=NEXT_STEP;
    ELSIF (MC_Home0.Error = TRUE) THEN
        (*在变量中储存错误ID *)
        ErrorID0:= MC_Home0.ErrorID;
        (*在STATE_ERROR之前重置“Enable”*)
        MC_Home0( Execute := FALSE(*BOOL*),
                    Axis := Axis0Ref(*AXIS_REF*), 
                    Home := HomeMode0(*MC_HOME_REF*), 
                    Position := 0.0(*LREAL*) );
        State:=ERROR_STEP;
    END_IF;
    NEXT_STEP:
    (*回零过程完成后，可以根据需要做其他事情。在这里添加代码*)
    
    ERROR_STEP:
    (*回零进程失败，根据您的需要做其他事情。在这里添加代码*)
    
    END_CASE;
