MC_ReadStatus

获取指定轴的状态，在Trio PLCopen状态图中指定。

描述

当“Enable”为TRUE时，它用于获取轴的状态（如Trio PLCopen状态图所述）。

一个轴有八种定义的状态，轴总是处于这些状态之一:ErrorStop,Disabled, stop, Homing, Standstill, DiscreteMotion, ContinuousMotion, SynchronizedMotion。

这个功能块不影响PLCopen状态机。

请确保输入和输出参数为下表所示的数据类型。如果不是这样，就会产生错误。

输入/输出

名称数据类型描述
输入
EnableBOOL在启用时连续获取参数的值
AxisAXIS_REF参考轴
输出
ValidBOOL可用有效的状态输出
BusyBOOLFB尚未完成，预计会有新的输出值。
ErrorBOOL表示FB中发生了错误。
ErrorIDUINT错误标识
ErrorStopBOOL“ErrorStop”状态。参见Trio PLCopen状态图
DisabledBOOL“Disabled”状态。参见Trio PLCopen状态图
StoppingBOOL“Stopping”状态。参见Trio PLCopen状态图
HomingBOOL“Homing”状态。参见Trio PLCopen状态图
StandstillBOOL“Standstill”状态。参见Trio PLCopen状态图
DiscreteMotionBOOL“DiscreteMotion”状态。参见Trio PLCopen状态图
ContinuousMotionBOOL“ContinuousMotion”状态。参见Trio PLCopen状态图
SynchronizedMotionBOOL“SynchronizedMotion”状态。参见Trio PLCopen State Diagram

例

例1

例子
(* MC_ReadStatus的示例*)
inst_ReadStatus( Enable := bEnable,
                Axis := Axis0Ref );
bValid:= inst_ReadStatus.Valid;
bBusy := inst_ReadStatus.Busy; 
bError := inst_ReadStatus.Error; 
ErrorId := inst_ReadStatus.ErrorID;
IF bValid THEN
    bErrorStop := inst_ReadStatus.ErrorStop;
    bDisabled := inst_ReadStatus.Disabled;
    bStopping := inst_ReadStatus.Stopping;
    bHoming := inst_ReadStatus.Homing;
    bStandstill := inst_ReadStatus.Standstill;
    bDiscreteMotion := inst_ReadStatus.DiscreteMotion;
    bContinuousMotion := inst_ReadStatus.ContinuousMotion;
    bSynchronizedMotion := inst_ReadStatus.SynchronizedMotion;
END_IF;
