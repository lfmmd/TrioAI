MC_ReadParameter

读取指定轴的非BOOL数据类型参数。

描述

功能块返回用参数number指定的指定参数的值。

“Value”输出的数据类型是LREAL，即使参数是不同的数据类型，如DINT, LINT, REAL。返回值将显式转换为LREAL。

当“Enable”更改为TRUE时，“Valid”将更改为TRUE，如果没有错误，“Valid”将在“Valid”输出上输出。

当“Enable”为TRUE时，“Valid”将被刷新。

这个功能块不影响PlcOpen状态机。它可以在任何状态下工作。

确保输入和输出参数是下表所示的数据类型。

使用MC_ReadBoolParameter作为轴的BOOL参数。

输入/输出

名称数据类型描述
输入
EnableBOOL在启用时连续获取参数的值。
AxisAXIS_REF参考轴
ParameterNumberINT指定参数的参数号。
参考“参数”
输出
ValidBOOL在输出“Value”上有一个有效的输出。
BusyBOOLFB尚未完成，预计会有新的输出值。
ErrorBOOL指示FB内部发生错误。
ErrorIDUINT错误的数字。出错时非零。
ValueLREAL指定参数的LREAL值，如果数据类型为DINT, LINT, REAL，则输出将显式转换为LREAL。

例

例1

例
(*MC_ReadParameter 的示例*)
ParNum := 1;  (* DPOS *)
Inst_MC_ReadParameter(Enable:=bEnable,Axis:=Axis0Ref,ParameterNumber:=ParNum);
bValid := Inst_MC_ReadParameter.Valid;
bBusy := Inst_MC_ReadParameter.Busy; 
bError := Inst_MC_ReadParameter.Error; 
ErrorId := Inst_MC_ReadParameter.ErrorID;
IF bValid THEN
    lrValue := Inst_MC_ReadParameter.Value;
END_IF;
