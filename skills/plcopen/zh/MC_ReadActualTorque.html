MC_ReadActualTorque

获取指定轴的实际转矩。

描述

当“Enable”变为TRUE且“Axis”具有有效值时，“Torque”将输出指定轴的实际扭矩，“Valid”输出变为TRUE。

此功能块不影响PLCopen状态机。它可以在任何状态下工作。

确保输入和输出参数为下表所示的数据类型。

输入/输出

名称数据类型描述
输入
EnableBOOL为TRUE时连续获取轴的实际扭矩
AxisAXIS_REF参考轴
输出
ValidBOOL有效的' Torque '输出可用
BusyBOOLFB尚未完成，预计会有新的输出值。
ErrorBOOL表示FB中发生了错误。
ErrorIDUINT错误标识
TorqueLREAL实际扭矩的值

例

例1

例
(*MC_ReadActualTorque的示例 *)
inst_ReadActualTorque ( Enable:=bEnable,
                            Axis:=Axis0Ref);
bValid:= inst_ReadActualTorque.Valid;
bBusy := inst_ReadActualTorque.Busy; 
bError := inst_ReadActualTorque.Error; 
ErrorId := inst_ReadActualTorque.ErrorID;
IF bValid THEN
    lrTorque := inst_ReadActualTorque.Torque;
END_IF;
