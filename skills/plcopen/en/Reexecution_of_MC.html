Re-execution of Motion Control Instructions

Description

Re-execution of motion control instructions means that you can change the input variables for a motion control instruction during execution and execute the instruction again to change the target values during operation.

Re-execution of Trio MC_xx motion control instructions is effective for those function blocks with Execute input but without the ContinuousUpdate input.

If the values of the input variables to the same instance which has no ContinuousUpdate input are changed while the motion control instruction is under execution and Execute is toggled from TRUE to FALSE and back to TRUE, operation will follow the new values.

All changed values will be ignored while the function block with the ContinuousUpdate input is under execution.
