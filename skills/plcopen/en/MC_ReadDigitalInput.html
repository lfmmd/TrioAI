MC_ReadDigitalInput

Gives access to the value of the specified input.

Description

When ‘Enable’ changes to TRUE, the FB reads the value of the selected input signal continuously if there are no error inputs.

Select the corresponding input by InputNumbe.

The 'InputNumber' should be given a valid value while ‘Valid’ is TRUE, otherwise the FB will generate an error.

This Function Block does not affect the PLCopen state machine.

Make sure the input and output parameters are of data type indicated in the table below.

It is not guaranteed that the digital signal will be seen by the FB: a short pulse on the digital input could be over before the next Function Block cycle occurs.

The ‘InputNumber’ range is based on hardware configuration.

Inputs/Outputs

NameData TypeDescription
INPUTS
EnableBOOLGet the value of the selected input signal continuously while enabled.
InputNumberINTSelects the input.
OUTPUTS
ValidBOOLA valid output is available on the output ‘Value’.
BusyBOOLThe FB is not finished and a new output value is to be expected.
ErrorBOOLSignals that an error has occurred within the FB.
ErrorIDUINTError identification
ValueBOOLValue of the specified input signal.

Examples

Example 1

Example
(*Example for MC_ReadDigitalInput *)
InNum := INT#1;
inst_ReadDigitalInput(Enable:=bEnable,InputNumber:=InNum);
bValid:= inst_ReadDigitalInput.Valid;
bBusy := inst_ReadDigitalInput.Busy; 
bError := inst_ReadDigitalInput.Error; 
ErrorId := inst_ReadDigitalInput.ErrorID;
IF bValid THEN
    bValue := inst_ReadDigitalInput.Value;
END_IF;
