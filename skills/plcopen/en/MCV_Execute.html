MCV_Execute

Allows any TrioBASIC commands to be executed as a string parameter.

Description

MCV_Execute is used to allow any TrioBASIC command to be executed as a string parameter.

Make sure the input and output parameters are of data type indicated in the table below.

Please refer to TrioBasic Help for more information.

Inputs/Outputs

NameData TypeDescription
INPUTS
ExecuteBOOLStart execution at rising edge
CommandSTRINGTrioBASIC command string
OUTPUTS
DoneBOOLThe command has been finished.
BusyBOOLThe FB new output values are to be expected
ErrorBOOLSignals that an error has occurred within the FB
ErrorIDUINTError identification
