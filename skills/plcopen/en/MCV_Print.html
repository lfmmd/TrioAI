MCV_Print

Prints information in the application program.

Description

MCV_Print is used to output a string via the specified channel.

MCV_Print does not affect PLCopen state diagram.

Make sure the input and output parameters are of data type indicated in the table below.

‘Channel’ number refers to Terminal number available in Motion Perfect.

MCV_Print is equivalent to TrioBASIC command PRINT. Please refer to TrioBasic Help for more information.

Inputs/Outputs

NameData TypeDescription
INPUTS
ExecuteBOOLStart writing at the rising edge
ChannelUINTTerminal number available in Motion Perfect.
Possible values: 0, 5, 6, or 7.
TextSTRINGThe string that will be printed in the Channel
OUTPUTS
DoneBOOLThe text has been written into the channel successfully
BusyBOOLThe FB new output values are to be expected
ErrorBOOLSignals that an error has occurred within the FB
ErrorIDUINTError identification
