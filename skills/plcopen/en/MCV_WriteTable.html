MCV_WriteTable

Writes a specified TABLE variable value.

Description

MCV_WriteTable is used to write a TABLE variable value specified by ‘TableIndex’ input parameter.

MCV_WriteTable does not affect PLCopen state diagram.

Make sure the input and output parameters are of data type indicated in the table below.

Please refer to TrioBasic Help for more information.

Inputs/Outputs

NameData TypeDescription
INPUTS
ExecuteBOOLStart writing at the rising edge
TableIndexUINTTable index to write the value into
ValueLREALValue to be written to the TABLE
OUTPUTS
DoneBOOLThe variable is written successfully into the TABLE
BusyBOOLThe FB new output values are to be expected
ErrorBOOLSignals that an error has occurred within the FB
ErrorIDUINTError identification
