MC_ReadAxisInfo

Reads information concerning the specified axis.

Description

This function block is used to read information of the specified axis.

If there is an error in inputs, ‘Error’ and ‘ErrorID’ will output while ‘Enable’ is TRUE.

On successful execution, the following outputs will be refreshed:

  • HomeAbsSwitch: Digital home switch input is active
  • LimitSwitchPos: Positive hardware end switch is active
  • LimitSwitchNeg: Negative hardware end switch is active
  • Simulation: Axis is in simulation mode
  • CommunicationReady: "Network" is initialized and ready for communication
  • ReadyForPowerOn: Drive is ready to be enabled (power on)
  • PowerOn: If TRUE shows that the power stage is switched ON
  • IsHomed: The absolute reference position is known for the axis (axis is homed)
  • AxisWarning: Warning(s) on the axis is present

This Function Block does not affect the PLCopen state machine. It can work in any states.

Make sure the input and output parameters are of data type indicated in the table below.

Examples

Example 1

Example
(*Example for MC_ReadAxisInfo *)
inst_ReadAxisInfo( Axis := Axis0Ref, Enable := bEnable );
bValid:= inst_ReadAxisInfo.Valid;
bBusy := inst_ReadAxisInfo.Busy; 
bError := inst_ReadAxisInfo.Error; 
ErrorId := inst_ReadAxisInfo.ErrorID;
IF bValid THEN
    bHomeAbsSwitch := inst_ReadAxisInfo.HomeAbsSwitch;
    bLimitSwitchPos := inst_ReadAxisInfo.LimitSwitchPos;
    bLimitSwitchNeg := inst_ReadAxisInfo.LimitSwitchNeg;
    bSimulation := inst_ReadAxisInfo.Simulation;
    bCommunicationReady := inst_ReadAxisInfo.CommunicationReady; 
    bReady4PowerOn := inst_ReadAxisInfo.ReadyForPowerOn;
    bPowerOn := inst_ReadAxisInfo.PowerOn;
    bIsHomed := inst_ReadAxisInfo.IsHomed;
    bAxisWarning := inst_ReadAxisInfo.AxisWarning;
END_IF;
