MCV_WriteTable

Writes a specified TABLE variable value.

Description:

MCV_WriteTable is used to write a TABLE variable value specified by ‘TableIndex’ input parameter.

MCV_WriteTable does not affect PLCopen state diagram.

Make sure the input and output parameters are of data type indicated in the table below.

Please refer to TrioBasic Help for more information.

Inputs/Outputs:

Name

Data Type

Description

INPUTS

Execute

BOOL

Start writing at the rising edge

TableIndex

UINT

Table index to write the value into

Value

LREAL

Value to be written to the TABLE

OUTPUTS

Done

BOOL

The variable is written successfully into the TABLE

Busy

BOOL

The FB new output values are to be expected

Error

BOOL

Signals that an error has occurred within the FB

ErrorID

UINT

Error identification
