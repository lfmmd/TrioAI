MC_CombineAxes

Combines the motion of 2 axes into a third axis with selectable combination method.

Description:

MC_CombineAxes combines the motion of 2 axes into a third axis with selectable combination method. In essence, it is a calculation of a new position setpoint based on the 2 position setpoints of the input axes. This FB is reflected in the state diagram like a synchronized motion type.

MC_CombineAxes supports two combine modes: mcAddAxes and mcSubAxes.

  • mcAddAxes: Slave position = Master1 position (relative position) + Master2 position (relative position).
  • mcSubAxes: Slave position = Master1 position (relative position) - Master2 position (relative position).

MC_CombineAxes is not affected by errors in ‘ Master1’ or ‘ Master2’. After the error is cleared and the Masters are back in motion, the Slave will resume the combined positioning operation. Masters are not affected by errors in Slave.

As application example one can work with a separate profile synchronized to an object on a moving belt, or a rotating knife with flexible covered distance to be cut.

Make sure the input and output parameters are of data type indicated in the table below.

To stop the motion, MC_CombineAxes must be interrupted by another FB issuing a new command.

Inputs/Outputs:

Name

Data Type

Description

INPUTS

Execute

BOOL

Start the combination process at rising edge

Master1

AXIS_REF

Reference to the axis

Master2

AXIS_REF

Reference to the axis

Slave

AXIS_REF

Reference to the resulting combined axis. It can be a virtual axis or linked directly to a real axis

ContinuousUpdate

BOOL

TRUE: when the function block is triggered (rising ‘Execute’),  it  will - as long as it stays TRUE – make the function block use the current values of the input variables and apply it to the ongoing movement.

FALSE: with the rising edge of the ‘Execute’ input, a change in the input parameters is ignored during the whole movement.

CombineMode

MC_COMBINE_MODE

Defines the type of combination applied to AxisOut:

mcAddAxes: Addition of the 2 input axes positions

mcSubAxes: Subtraction of the 2 input axes positions

GearRatioNumeratorM1

INT

Numerator for the gear factor for master axis 1 towards the slave

GearRatioDenominatorM1

INT

Corresponding denominator for master axis 1

GearRatioNumeratorM2

INT

Numerator for the gear factor for master axis 2 towards the slave

GearRatioDenominatorM2

INT

Corresponding denominator for master axis 2

MasterValueSourceM1

MC_SOURCE

Defines the source for synchronization for master axis 1:

mcSetValue: Synchronization on master set value

mcActualValue: Synchronization on master actual value

mcCommandValue: Synchronization on master command value

MasterValueSourceM2

MC_SOURCE

Defines the source for synchronization for master axis 2:

mcSetValue: Synchronization on master set value

mcActualValue: Synchronization on master actual value

mcCommandValue: Synchronization on master command value

OUTPUTS

InSync

BOOL

TRUE if the set value = the commanded value.

Busy

BOOL

The FB is not finished and new output value is to be expected.

Active

BOOL

Indicates that the FB has control on the combined axis

CommandAborted

BOOL

‘Command’ is aborted by another command.

Error

BOOL

Signals that an error has occurred within the Function Block

ErrorID

UINT

Error identification

Examples:

Example 1:

Axis 1 is the master axis of MC_CamIn. Axis 0 is the slave. The master axis will move along the Cam table defined with sinusoid velocity profile. Axis 1 will move with MC_GearIn as axis 0 when it is synchronized with its master axis. When axis 1 is synchronized with axis 0, axis 3 will start with MC_MoveAbsolute. Axis 0 and axis 1 are combined with mcAddAxes mode to axis 3. The following picture shows the velocity profile for these axes.

 

  Example

(* Defines for MC_CombineAxes example *)
#define STATE_READY      0
#define STATE_CAMIN      1
#define STATE_GEARIN     2
#define STATE_COMBINE    3
#define STATE_ERROR      8

 

(* Defines for Variables used in MC_CombineAxes *)

VAR
 state : USINT ;
   (*$desc=Current state*)
 Inst_MC_CamIn : MC_CamIn ;
   (*$desc=MC_CamIn instant*)
 Axis0Ref : lib:AXIS_REF ;
   (*$desc=Axis 0 reference*)
 Axis2Ref : lib:AXIS_REF ;
   (*$desc=Axis 2 reference*)
 CamId : UINT ;
   (*$desc=Cam table ID*)
 bGearStart : BOOL := FALSE ;
   (*$desc=To start Gear motion, it is bound to IO6*)
   (*$profile=IO*)
   (*$embed=(VERS=1    MODE=0    POINT=7)*)
 Inst_MC_GearIn : MC_GearIn ;
 Axis1Ref : lib:AXIS_REF ;
   (*$desc=A slave axis which will move in Gear motion*)
 bStartCombine : BOOL ;
   (*$desc=A slave axis which will move in combine motion*)
   (*$profile=IO*)
   (*$embed=(VERS=1    MODE=0    POINT=7)*)
 Inst_MC_CombineAxes : MC_CombineAxes ;
 Axis3Ref : lib:AXIS_REF ;
 bStop : BOOL := FALSE ;
   (*$profile=IO*)
   (*$embed=(VERS=1    MODE=0    POINT=1)*)
 Inst_MC_Stop1 : MC_Stop ;
 Inst_MC_Stop2 : MC_Stop ;
 Inst_MC_Stop3 : MC_Stop ;
 bStart : BOOL := FALSE ;
   (*$desc=To start comine motion example*)
   (*$profile=IO*)
   (*$embed=(VERS=1    MODE=0    POINT=0)*)
 SrcValue : MC_SOURCE := mcSetValue ;
 StartMode : MC_START_MODE ;
 CombMode : MC_COMBINE_MODE ;
 Inst_MC_MoveAbsolute : MC_MoveAbsolute ;
 mDirection : MC_DIRECTION ;
 BufferMode : MC_BUFFER_MODE ;
END_VAR

 

(* Variables initialization *)
CamId := UINT#25616 ; (* This Cam table should be created before using *)
Axis0Ref.AxisNo := 0 ; (* The slave axis for Cam motion *)
Axis1Ref.AxisNo := 1 ; (* The slave axis for Gear motion *)
Axis2Ref.AxisNo := 2 ; (* The master axis for Cam motion *)
Axis3Ref.AxisNo := 3 ; (* The slave axis for combine motion *)
SrcValue := mcSetValue;
StartMode := mcAbsolute;
CombMode := mcAddAxes;
mDirection := mcPositiveDirection;
BufferMode := mcAborting;


(* cyclic program *)
CASE state OF
     (* ...power and home axis before moving... *)
     (********************STATE_READY*********************)
    STATE_READY:
     (*wait for user to start movement*)
     IF (bStart) THEN
        Inst_MC_CamIn( Execute := FALSE (*BOOL*) ,
                        Master := Axis2Ref (*AXIS_REF*) , 
                        Slave := Axis0Ref (*AXIS_REF*) ,         
                        MasterOffset := LREAL#0.0 (*LREAL*) , 
                        SlaveOffset := LREAL#0.0 (*LREAL*) , 
                        MasterScaling := REAL#1.0 (*REAL*) , 
                        SlaveScaling := REAL#1.0 (*REAL*) , 
                        MasterStartDistance := LREAL#0.0 (*LREAL*) , 
                        MasterSyncPosition := LREAL#0.0 (*LREAL*) , 
                        StartMode := StartMode (*MC_START_MODE*) , 
                        MasterValueSource := SrcValue (*MC_SOURCE*) , 
                        CamTableID := CamId (*UINT*) );
        
        state:= STATE_CAMIN;
     END_IF ;
     (********STATE CAM IN*****************************************)
    STATE_CAMIN:   (*start cam in*)
    Inst_MC_CamIn(Execute := TRUE (*BOOL*) , 
                    Master := Axis2Ref (*AXIS_REF*) , 
                    Slave := Axis0Ref (*AXIS_REF*) , 
                    MasterOffset := LREAL#0.0 (*LREAL*) , 
                    SlaveOffset := LREAL#0.0 (*LREAL*) , 
                    MasterScaling := REAL#1.0 (*REAL*) , 
                    SlaveScaling := REAL#1.0 (*REAL*) , 
                    MasterStartDistance := LREAL#0.0 (*LREAL*) , 
                    MasterSyncPosition := LREAL#0.0 (*LREAL*) , 
                    StartMode := StartMode (*MC_START_MODE*) , 
                    MasterValueSource := SrcValue (*MC_SOURCE*) , 
                    CamTableID := CamId (*UINT*) );
    
     IF ( (bGearStart = TRUE ) AND (Inst_MC_CamIn.InSync)) THEN
        Inst_MC_GearIn( Execute := FALSE (*BOOL*) , 
                        Master  := Axis2Ref (*AXIS_REF*) , 
                        Slave  := Axis1Ref (*AXIS_REF*) ,         
                        ContinuousUpdate := TRUE (*BOOL*) , 
                        RatioNumerator := INT#1 (*INT*) , 
                        RatioDenominator := UINT#1 (*UINT*) , 
                        MasterValueSource := SrcValue (*MC_SOURCE*) , 
                        Acceleration := LREAL#1000.0 (*LREAL*) , 
                        Deceleration := LREAL#1000.0 (*LREAL*) , 
                        Jerk := LREAL#100.0 (*LREAL*) );
          
        state:=STATE_GEARIN;
     ELSIF (Inst_MC_CamIn.Error) THEN         
        Inst_MC_CamIn( Execute := FALSE (*BOOL*) , 
                        Master := Axis2Ref (*AXIS_REF*) , 
                        Slave := Axis0Ref (*AXIS_REF*) ,         
                        MasterOffset := LREAL#0.0 (*LREAL*) , 
                        SlaveOffset := LREAL#0.0 (*LREAL*) , 
                        MasterScaling := REAL#1.0 (*REAL*) , 
                        SlaveScaling := REAL#1.0 (*REAL*) , 
                        MasterStartDistance := LREAL#0.0 (*LREAL*) , 
                        MasterSyncPosition := LREAL#0.0 (*LREAL*) , 
                        StartMode := StartMode (*MC_START_MODE*) , 
                        MasterValueSource := SrcValue (*MC_SOURCE*) , 
                        CamTableID := CamId (*UINT*) );
        
        state:=STATE_ERROR;
     END_IF ;
     (********STATE GEAR IN*****************************************)
    STATE_GEARIN:   (*start cam in*)
    
    Inst_MC_GearIn( Execute := TRUE (*BOOL*) , 
                    Master  := Axis2Ref (*AXIS_REF*) , 
                    Slave  := Axis1Ref (*AXIS_REF*) ,     
                    ContinuousUpdate := TRUE (*BOOL*) , 
                    RatioNumerator := INT#1 (*INT*) , 
                    RatioDenominator := UINT#1 (*UINT*) , 
                    MasterValueSource := SrcValue (*MC_SOURCE*) , 
                    Acceleration := LREAL#1000.0 (*LREAL*) , 
                    Deceleration := LREAL#1000.0 (*LREAL*) , 
                    Jerk := LREAL#100.0 (*LREAL*) );
    
     IF ((bStartCombine = TRUE ) AND (Inst_MC_GearIn.InGear)) THEN
        Inst_MC_CombineAxes( Execute := FALSE (*BOOL*) ,
                            Master1 := Axis0Ref (*AXIS_REF*) ,
                            Master2 := Axis1Ref (*AXIS_REF*) , 
                            Slave := Axis3Ref (*AXIS_REF*) ,                              
                            ContinuousUpdate := TRUE (*BOOL*) , 
                            CombineMode := CombMode (*MC_COMBINE_MODE*) ,
                            GearRatioNumeratorM1 := INT#1 (*INT*) , 
                            GearRatioDenominatorM1 := INT#1 (*INT*) , 
                            GearRatioNumeratorM2 := INT#1 (*INT*) , 
                            GearRatioDenominatorM2 := INT#1 (*INT*) , 
                            MasterValueSourceM1 := SrcValue (*MC_SOURCE*) , 
                            MasterValueSourceM2 := SrcValue (*MC_SOURCE*) );
        
        Inst_MC_MoveAbsolute( Execute := FALSE (*BOOL*) ,
                            Axis := Axis3Ref (*AXIS_REF*) ,
                            ContinuousUpdate := FALSE (*BOOL*) , 
                            Position := LREAL#10000.0 (*LREAL*) , 
                            Velocity := LREAL#1000.0 (*LREAL*) , 
                            Acceleration := LREAL#1000.0 (*LREAL*) , 
                            Deceleration := LREAL#1000.0 (*LREAL*) , 
                            Jerk := LREAL#1000.0 (*LREAL*) , 
                            Direction := mDirection (*MC_DIRECTION*) , 
                            BufferMode := BufferMode (*MC_BUFFER_MODE*) );
        
        state:=STATE_COMBINE;
     ELSIF (Inst_MC_GearIn.Error) THEN         
        Inst_MC_GearIn( Execute := FALSE (*BOOL*) , 
                            Master  := Axis2Ref (*AXIS_REF*) , 
                            Slave  := Axis1Ref (*AXIS_REF*) ,
                            ContinuousUpdate := TRUE (*BOOL*) , 
                            RatioNumerator := INT#1 (*INT*) , 
                            RatioDenominator := UINT#1 (*UINT*) , 
                            MasterValueSource := SrcValue (*MC_SOURCE*) , 
                            Acceleration := LREAL#1000.0 (*LREAL*) , 
                            Deceleration := LREAL#1000.0 (*LREAL*) , 
                            Jerk := LREAL#100.0 (*LREAL*) );
        state:=STATE_ERROR;
     END_IF ;
     (********STATE COMBINE*****************************************)
    STATE_COMBINE:   (*start combine *)     
    Inst_MC_CombineAxes( Execute := TRUE (*BOOL*) ,
                            Master1 := Axis0Ref (*AXIS_REF*) ,
                            Master2 := Axis1Ref (*AXIS_REF*) ,  
                            Slave := Axis3Ref (*AXIS_REF*) ,
                            ContinuousUpdate := TRUE (*BOOL*) , 
                            CombineMode := CombMode (*MC_COMBINE_MODE*) ,
                            GearRatioNumeratorM1 := INT#1 (*INT*) , 
                            GearRatioDenominatorM1 := INT#1 (*INT*) , 
                            GearRatioNumeratorM2 := INT#1 (*INT*) , 
                            GearRatioDenominatorM2 := INT#1 (*INT*) , 
                            MasterValueSourceM1 := SrcValue (*MC_SOURCE*) , 
                            MasterValueSourceM2 := SrcValue (*MC_SOURCE*) );
    
    Inst_MC_MoveAbsolute( Execute := TRUE (*BOOL*) ,
                            Axis := Axis3Ref (*AXIS_REF*) , 
                            ContinuousUpdate := FALSE (*BOOL*) , 
                            Position := LREAL#10000.0 (*LREAL*) , 
                            Velocity := LREAL#1000.0 (*LREAL*) , 
                            Acceleration := LREAL#1000.0 (*LREAL*) , 
                            Deceleration := LREAL#1000.0 (*LREAL*) , 
                            Jerk := LREAL#1000.0 (*LREAL*) , 
                            Direction := mDirection (*MC_DIRECTION*) , 
                            BufferMode := BufferMode (*MC_BUFFER_MODE*) );
    
     IF ( bStop = TRUE ) THEN
        Inst_MC_Stop1( Execute := TRUE (*BOOL*) , 
                        Axis := Axis3Ref (*AXIS_REF*) ,
                        Deceleration := LREAL#1000.0 (*LREAL*) , 
                        Jerk := LREAL#1000.0 (*LREAL*) );
        
        Inst_MC_Stop2( Execute := TRUE (*BOOL*) , 
                        Axis := Axis0Ref (*AXIS_REF*) ,
                        Deceleration := LREAL#1000.0 (*LREAL*) , 
                        Jerk := LREAL#1000.0 (*LREAL*) );
        
        Inst_MC_Stop3( Execute := TRUE (*BOOL*) , 
                        Axis := Axis1Ref (*AXIS_REF*) ,
                        Deceleration := LREAL#1000.0 (*LREAL*) , 
                        Jerk := LREAL#1000.0 (*LREAL*) );
        
        state:=STATE_READY;
     ELSIF (Inst_MC_CombineAxes.Error) THEN
        
        Inst_MC_CombineAxes( Execute := FALSE (*BOOL*) ,
                                Master1 := Axis0Ref (*AXIS_REF*) ,
                                Master2 := Axis1Ref (*AXIS_REF*) , 
                                Slave := Axis3Ref (*AXIS_REF*) ,                                  
                                ContinuousUpdate := TRUE (*BOOL*) , 
                                CombineMode := CombMode (*MC_COMBINE_MODE*) ,
                                GearRatioNumeratorM1 := INT#1 (*INT*) , 
                                GearRatioDenominatorM1 := INT#1 (*INT*) , 
                                GearRatioNumeratorM2 := INT#1 (*INT*) , 
                                GearRatioDenominatorM2 := INT#1 (*INT*) , 
                                MasterValueSourceM1 := SrcValue (*MC_SOURCE*) , 
                                MasterValueSourceM2 := SrcValue (*MC_SOURCE*) );
        
        state:=STATE_ERROR;
     END_IF ;
     (********STATE ERROR*****************************************)
    STATE_ERROR:  
     (* Add user's code here *)

END_CASE ;
