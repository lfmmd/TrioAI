MC_ReadParameter

Reads the specified axis’ non-BOOL data type parameter.

Description:

The function block returns the value of the specified parameter which specified with parameter number.

Data type of the ‘Value’ output is a LREAL, even if the parameter is of a different data type, such as DINT, LINT, REAL. The returned value will be explicit converted to LREAL.

When ‘Enable’ changes to TRUE, ‘Valid’ will changes to TRUE and ‘Value’ will be output on ‘Value’ output if no error occurs.

‘Value’ will be refreshed when it is called while ‘Enable’ is TRUE.

This Function Block does not affect the PlcOpen machine state. It can work in any states.

Make sure the input and output parameters are of data type indicated in the table below.

Use MC_ReadBoolParameter for axis’s Boolean parameters.

Inputs/Outputs:

Name

Data Type

Description

INPUTS

Enable

BOOL

Get the value of the parameters continuously while enabled.

Axis

AXIS_REF

Reference to the axis

ParameterNumber

INT

Parameter number of the specified parameter.

Refer to “ Parameters ”

OUTPUTS

Valid

BOOL

A valid output is available on the output ‘Value’.

Busy

BOOL

The FB is not finished and new output value is to be expected.

Error

BOOL

Signals that an error has occurred within the FB.

ErrorID

UINT

Error number. Non zero when there is an error.

Value

LREAL

Value of the specified parameter in the LREAL, if the data type is DINT, LINT, REAL, the output will be explicit converted to LREAL.

Examples:

Example 1:

 Example

(*Example for MC_ReadParameter *)
ParNum := 1 ;   (* DPOS *)
Inst_MC_ReadParameter(Enable:=bEnable,Axis:=Axis0Ref,ParameterNumber:=ParNum);
bValid := Inst_MC_ReadParameter.Valid;
bBusy := Inst_MC_ReadParameter.Busy; 
bError := Inst_MC_ReadParameter.Error; 
ErrorId := Inst_MC_ReadParameter.ErrorID;
IF bValid THEN
    lrValue := Inst_MC_ReadParameter.Value;
END_IF ;
