MC_ReadActualPosition

Gets the actual position of the specified axis.

Description:

When ‘Enable’ changes to TRUE and ‘Axis’ has valid value, the position of the specified axis is given as ‘Position’ output and ‘Valid’ output becomes TRUE.

This Function Block does not affect the PLCopen state machine. It can work in any state.

Make sure the input and output parameters are of data type indicated in the table below.

Inputs/Outputs:

Name

Data Type

Description

INPUTS

Enable

BOOL

Gets the actual position of the axis continuously while TRUE

Axis

AXIS_REF

Reference to the axis

OUTPUTS

Valid

BOOL

A valid ‘Position’ output is available

Busy

BOOL

The FB is not finished and a new output value is to be expected.

Error

BOOL

Signals that an error has occurred within the FB.

ErrorID

UINT

Error identification

Position

LREAL

New absolute position [units]

Examples:

Example 1:

 Example

(*Example for MC_ReadActualPosition *)  
inst_ReadActualPosition ( Enable:=bEnable,
                            Axis:=Axis0Ref);
bValid:= inst_ReadActualPosition.Valid;
bBusy := inst_ReadActualPosition.Busy; 
bError := inst_ReadActualPosition.Error; 
ErrorId := inst_ReadActualPosition.ErrorID;
IF bValid THEN
    lrPosition := inst_ReadActualPosition.Position;
END_IF ;
