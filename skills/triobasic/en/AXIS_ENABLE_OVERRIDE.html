AXIS_ENABLE_OVERRIDE

Type

Axis Parameter

Description

Can be used to independently enable an axis. OFF by default, this parameter is set to ON to enable the axis even though WDOG and AXIS_ENABLE may be OFF. The purpose is to allow Motion Perfect to Jog and set up the drive tuning of an EtherCAT or similar slave servo-drive.

Requires firmware version 2.0305 or later

Value

Accepts the values ON or OFF, default is OFF.

Example

Enable a single axis for setup purposes.

BASE(axis_number)

AXIS_ENABLE_OVERRIDE = ON  ' Enable the axis (no need to put WDOG ON)
