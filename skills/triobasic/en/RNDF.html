RNDF

Type

Mathematical Function

Syntax

value = RNDF

value = RNDF(upper-limit)

value = RNDF(lower-limit, upper-limit)

Description

The RNDF function returns a pseudo-random floating-point value.

Without parameters, it returns a value from 0 to just less than 1, i.e. 0 ≤ value < 1, or [0, 1).

When an upper limit alone is specified, it returns a value from 0 to just less than the limit, i.e. 0 ≤ value < upper-limit

When a lower and an upper limit are specified, it returns a value from the lower limit to just less than the upper limit, i.e. lower-limit ≤ value < upper-limit.

Note that the sequence generated varies with each program execution, which is usually desirable, but if you require the same sequence generated then use the RANDOMIZE command

Parameters

value:A pseudo-random floating-point value
lower-limit:The lowest value that the value returned may take (this can only be specified when an upper limit is specified)
upper-limit:The highest value that the value returned may approach, i.e. up to this value but excluding this value

Examples

Example 1

Print some random numbers between 0 and 1 exclusive on the command line

>>PRINT RNDF

0.4055

>>PRINT RNDF

0.6248

>>PRINT RNDF

0.7969

>>

Example 2

Print a random number from 0 to 100 exclusive on the command line

>>PRINT RNDF(100)

95.5165

>>PRINT RNDF(100)

13.1009

>>PRINT RNDF(100)

24.2894

>>

Example 3

Print a random number from -0.5 to 0.5 exclusive on the command line

>>PRINT RNDF(-0.5, 0.5)

-0.4366

>>PRINT RNDF(-0.5, 0.5)

0.2732

>>PRINT RNDF(-0.5, 0.5)

-0.1176

>>
