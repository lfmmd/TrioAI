EPROM_STATUS

Type

System Parameter

Description

Returns the status of background FLASH memory programming.
0Not active
1Flash memory is being Erased
2Erasing is suspended

Flash memory programming takes place when programs are loaded or edited. Also when a VR value is changed and when TABLE values are changed in Motion Coordinators that support automatic TABLE store to Flash. As each Flash page becomes full, a new page is erased ready to continue programming.

This parameter can be checked before powering down the Motion Coordinator, however when the power is removed, a power-off detection will finalise any auto-store that is in progress in the time available before the processor shuts down. In normal use it is not necessary to wait for the FLASH_STATUS to show 0 before powering down.

Motion Perfect shows the FLASH_STATUS as an indicator marked “Flash”.
