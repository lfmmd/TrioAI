ADDRESS

Type

System Parameter

Description

Sets the RS485 or Modbus multi-drop address for the controller.

Value

The node’s address, in the range of 1 to 32, or 255.

If it is set to 255 addressing is not used and all 8 characters from the packet are sent through to the user.

Example

Initialise Modbus as node 5

ADDRESS = 5

SETCOM(19200, 8, 1, 2, 1, 4)
