REMAIN

Type

Axis Parameter (Read Only)

Description

This is the distance, in UNITS, remaining to the end of the current move. It may be tested to see what amount of the move has been completed.

When WAIT UNTIL REMAIN command is used after a MOVE command, use a time delay WA command, or use the WAIT LOADED command to allow the MOVE command to be loaded into MTYPE.

Value

The distance remaining in user UNITS of the current move

Examples

Example 1

To change the speed to a slower value 5 mm from the end of a move. A WAIT LOADED command is used to allow motion command to be loaded into MTYPE before REMAIN is polled.

SPEED = 10

MOVE(45)

WAIT LOADED

WAIT UNTIL REMAIN < 5

SPEED = 1

WAIT IDLE

Example 2

Turn on output 0 when it is 25 units away from the end of the move. Time delay is used to allow motion command to be loaded into MTYPE.

BASE(0)

UNITS = 1

SPEED = 10

ACCEL = 10 * SPEED

DECEL = ACCEL



OP(0, OFF)

MOVE(100)

WA(2)

WAIT UNTIL REMAIN < 25

OP(0, ON)
