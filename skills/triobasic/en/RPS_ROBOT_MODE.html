ROBOT_MODE

Type

Robotics Parameter (Read Only)

Syntax

ROBOT_MODE(group)

Description

Returns the operating profile (mode) of the given kinematic group.

ROBOT_MODE requires the RPS runtime FEC

RPS 2.0 kernel must be activated using the RPS_KERNEL_V2 keyword to enable its use.

Value

ModeDescription
0Unset (key switch has not been sampled)
1External (KMI and KAI both unset)
2Manual (KMI set)
3Automatic (MAI set)
4Invalid (KMI and KAI both set)

Examples

Example 1

Detect if robot 0 is in manual.

IF ROBOT_MODE(0) = 2 THEN

   'Robot is in manual mode. Further action here.

ENDIF
