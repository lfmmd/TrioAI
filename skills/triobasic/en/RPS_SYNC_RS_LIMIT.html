SYNC_RS_LIMIT

Type

Robotics Parameter

Description

An end of travel limit may be set up in software thus allowing synchronisation to stop when the robot is going outside the limit. The robot will wait with SYNC_CONTROL = -1 if the controller is waiting for the synchronisation point to move within range.

Value

The absolute position of the software reverse travel limit in user UNITS. (default = 200000000000)

Examples

Example 1

Define the forward synchronisation limit to 100 units. The robot will start synchronisation when the target point position on axis 0 is less than SYNC_RS_LIMIT. SYNC_CONTROL will be -1 before starting synchronisation. Synchronisation will be stopped if the robot end effector travels outside the limits while in sync.

BASE(0)

SYNC_RS_LIMIT = -100

SYNC_TO_OBJECT_FRAME(1, 1000, "pallet", "object_1")
