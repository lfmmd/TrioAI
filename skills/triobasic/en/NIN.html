NIN

Type

I/O Parameter

Description

This parameter returns the number of inputs fitted to the system. The value is set by the firmware taking into consideration the total I/O detected, including module I/O, CAN I/O, Fieldbus I/O and CANopen I/O.

Value

The highest input point + 1 that is available for use.

Example

There are 24 external output points in addition to the 16 built-in I/O points on the controller.

Printing NIN in the terminal:

>>?NIN

40

>>

Note that in this case, the last input point addressable is IN(39).
