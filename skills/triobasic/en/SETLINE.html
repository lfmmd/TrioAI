SETLINE

Type

System Command

Syntax

SETLINE "program", process, line_number, base_index, robot_index

Description

Set program pointer on the selected line of a program.

Parameters

Program nameAny name a program can accept. It has to be specified between double quotes.
Process numberIf the program is already running over a process or the process wants to be specified. In the case of program is not already running set this value as -1 and the system will select an available process.
Line numberLine number where program pointer will be set
Base indexIf the process where the line will be set is not already running, it will not have any BASE selected. This parameter will set the BASE index. Set it as -1 to leave the process BASE as it is.
Robot indexIf the process where the line will be set is not already running, it will not have any ROBOT selected. This parameter will set the ROBOT index. Set it as -1 to leave the process ROBOT as it is.

Example

Set program pointer on line 13, leaving no BASE selected, for ROBOT(0), in program PICK_AND_PLACE that is not already running.

SETLINE "PICK_AND_PLACE_CUPS", -1, 13, -1, 0
