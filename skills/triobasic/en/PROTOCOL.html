PROTOCOL

Type

Port Parameter

Syntax

value = PROTOCOL PORT(port)

Description

This parameter allows the user to check which protocol is running on the specified PORT.

You can write to this parameter however it is advisable to initialise the communication protocol through SETCOM and ANYBUS.

Do not write a value to PORT(0) as you will disable communications with Motion Perfect.

Value

value:The value depends on port hardware and firmware version:
0None
1Firmware download
2Motion Perfect
3Modbus server
4Modbus client
5Transparent
6Datagram
7TCP client
8TCP server
9Hostlink
10Not used
11Modbus TCP server
12Modbus TCP client
13Ethernet IP
14HMI

Example

Check that Modbus is running on the RS485 channel (PORT(2))

IF PROTOCOL PORT(2) <> 3 THEN

  PRINT#user, "MODBUS has stopped"

ENDIF
