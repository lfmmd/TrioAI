FWD_JOG

Type

Axis Parameter

Description

This parameter holds the input number to be used as a forward jog input.

When the FWD_JOG input is active the axis moves forward at JOGSPEED (or SPEED if the FAST_JOG input is assigned and active).

The input used for FWD_JOG is active low.

It is advisable to use INVERT_IN on the input for FWD_JOG so that 0V at the input disables the jog.

FWD_JOG overrides REV_JOG if both are assigned to the same input and activated. If they are assigned to different inputs and activated one after the other, the first command takes precedence, the second command is ignored.

Note that this move is automatically loaded whenever the FWD_JOG input is active and so cannot be cancelled using CANCEL or RAPIDSTOP, so if it is required to forcibly stop the forward jog, then either de-assign its input (FWD_JOG = -1) or set JOGSPEED to zero.

Value

-1Disable the forward jog input (default)
≥0The input number to control forward jog with

Example

Initialise the FWD_JOG so that it is active high on input 7.

INVERT_IN(7, ON)

FWD_JOG = 7
