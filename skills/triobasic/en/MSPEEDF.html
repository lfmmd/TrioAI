MSPEEDF

Type

Axis Parameter (Read Only)

Description

MSPEEDF is the value of MSPEED after a low pass filter has been applied. The filter helps to remove jitter and noise from the value which could otherwise produce misleading results when MSPEED is read in a program. MSPEEDF represents the average speed measured as a change in measured position in user UNITS (per second).

Value

Change in measured position per second in user UNITS.

Example

Monitor MSPEEDF to detect when the speed of axis 3 rises above a threshold value.

BASE(3)

MSPEED_FILTER = 0.5 'Initialise the filter coefficient



WHILE TRUE

  IF MSPEEDF > 5 THEN

    GOSUB start_main

  ENDIF

  ...

WEND
