WORLD_UNITS

Type

Axis Parameter

Description

WORLD_UNITS is a conversion factor that allows the user to scale the edges/ stepper pulses to a more convenient scale. WORLD_UNITS is only used when a FRAME is active and only applies to the parameters in the axis coordinate system (after the FRAME). This includes WORLD_DPOS, WORLD_FS_LIMIT and WORLD_RS_LIMIT.

The number of counts per required units (default =1). Examples:
