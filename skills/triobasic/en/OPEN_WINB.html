OPEN_WINB

Type

Axis Parameter

Alternate Format

OWB

Description

This parameter defines the start position of the second registration window which will be used for registration marks if windowing is specified by the REGIST command (mode 20 only).

Value

Absolute start position of the second registration window in user units.

Example

Set the second registration window between -40 and 40

OPEN_WINB = -40

CLOSE_WINB = 40

OPEN_WIN , CLOSE_WIN , CLOSE_WINB , REGIST
