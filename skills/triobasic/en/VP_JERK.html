VP_JERK

Type

Axis Parameter (Read Only)

Syntax

value = VP_JERK

Description

Returns the instantaneous jerk that the velocity profiler is running at, which varies as moves are being profiled. The jerk is the rate at which acceleration is increased or decreased.

Value

The velocity profile jerk in user UNITS per second per second per second.

Example

Wait until the axis achieves constant acceleration:

MOVE(100)

WAIT UNTIL VP_JERK <> 0

WAIT UNTIL VP_JERK = 0

See Also

VP_ACCEL, VP_SPEED
