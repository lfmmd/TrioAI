DRIVE_CLEAR

Type

Axis Function

Syntax

value = DRIVE_CLEAR(parameter)

Description

DRIVE_CLEAR allows the user to clear alarms in the drive. Currently this is supports only the Panasonic RTEX drives.

DRIVE_READ can be used to read the value of the alarm

Parameters

parameter:0Clear current alarm
1Clear all alarm history
2Clear all external alarms
