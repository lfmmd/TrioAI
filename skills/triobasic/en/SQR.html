SQR

Type

Mathematical Function

Syntax

value = SQR(number)

Description

Returns the square root of a number.

Parameters

value:The square root of the number
number:Any valid TrioBASIC number or variable.

Example

Calculate the square root of 4 on the command line.

>>PRINT SQR(4)

2.0000

>>
