CANIO_STATUS

Type

I/O Parameter

Description

Returns the status of the Trio CAN I/O network. You can set bit 4 to reset the network.

Value

BitDescriptionValue
0Error from the I/O module 0, 3, 6 or 91
1Error from the I/O module 1, 4, 7 or 102
2Error from the I/O module 2, 5, 8 or 114
3Error from the I/O module 12, 13, 14 or 158
4Should be set to re-initialise the CANIO network16
5Is set when initialisation is complete32
6Error from Analogue module64
7Output error (0-3)128
8Output error (4-7)256
9Output error (8-11)512
10Output error (12-15)1024
11Input error (0-3)2048
12Input error (4-7)4096
13Input error (8-11)8192
