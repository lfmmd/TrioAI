IS_ENCRYPTION_KEY

Type

System Function

Syntax

value = IS_ENCRYPTION_KEY(type)

Description

The System Function, IS_ENCRYPTION_KEY returns either TRUE or FALSE to indicate that a USER or OEM key have been set in the controller. The type parameter selects which key type is being requested.

Motion Perfect has tools for managing the Encryption Key.

Parameters

NameDescription
type:0Selects the Controller key status
1Selects the OEM key status
2Selects the USER key status

Examples

Example 1

Get the status of the USER Encryption key.

' Is the USER encryption key set?

IF IS_ENCRYPTION_KEY(2)=FALSE THEN

    PRINT "No USER Encryption Key set in this controller"

ENDIF
