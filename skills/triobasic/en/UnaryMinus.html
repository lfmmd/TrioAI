- Unary minus

Type

Mathematical Operator

Syntax

-<expression1>

Description

Negates expression1.

Parameters

Expression1:Any valid TrioBASIC expression

Example

Evaluate 2.1 multiply by 9 and negate the result, this will then be stored in VR 0, leaving it containing the value -18.9

VR(0) = -(2.1 * 9)
