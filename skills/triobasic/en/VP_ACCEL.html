VP_ACCEL

Type

Axis Parameter (Read Only)

Syntax

value = VP_ACCEL

Description

Returns the instantaneous acceleration that the velocity profiler is running at, which varies as moves are being profiled. The acceleration is the rate at which speed is increased or decreased.

Value

The velocity profile acceleration in user UNITS per second per second.

Example

Wait until the axis stops accelerating:

MOVE(100)

WA(10)

WAIT UNTIL VP_ACCEL = 0

See Also

VP_JERK, VP_SPEED
