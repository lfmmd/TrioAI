ROBOT_TARGET_MODE

Type

Robotics Parameter

Syntax

ROBOT_TARGET_MODE = value

Description

ROBOT_TARGET_MODE is used to control level of restrictions to be applied when using GTAE (Global Targets Array - Extended) or TARGETE data type.

ROBOT_TARGET_MODE requires the RPS runtime FEC.

Value

BitDescriptionValue
10Deactivate robot number restriction2
1Activate robot number restriction
20Deactivate robot configuration restriction4
1Activate robot configuration restriction
30Deactivate object frame restriction8
1Activate object frame restriction
40Deactivate tool offset restriction16
1Activate tool offset restriction
50Deactivate robot frame restriction32
1Activate robot frame restriction
60Deactivate world value of W axis extended range. Default value is between -180 and +180.64
1Activate world value of W axis extended range while still respecting joint limits. Value is between -infinity and +infinity (within last joint limits)

Examples

Example 1

Deactivate all robot target restrictions.

ROBOT_TARGET_MODE = 0 'Turn off all restrictions for TARGETEs

Example 2

Monitor robot used in the target.

ROBOT_TARGET_MODE.1 = 1 'Turn on Robot Number Restriction. 

Example 3

Monitor robot used in the target and the robot configuration for that target.

ROBOT_TARGET_MODE = 6 'Turn on robot number and robot conf restriction.
