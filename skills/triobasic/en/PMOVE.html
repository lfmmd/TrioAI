PMOVE

Type

Process Parameter (Read Only)

Description

Returns the state of the process move buffer.

When a process encounters a motion command, it loads the motion’s requirements into its “process move buffer”. This can hold one motion command for any group of axes, and when the load into the process move buffer is complete the PMOVE parameter becomes TRUE. When the next servo period occurs, if the axis move buffer for all the axes involved in the move has room, this move will be loaded into it. When this second transfer is complete the PMOVE parameter becomes FALSE.

Motion can only be loaded into the process move buffer when PMOVE is FALSE, so execution of a motion command when PMOVE is TRUE will cause this command to stall until the process move buffer becomes free.

Each process has its own PMOVE parameter.

Value

TRUE (-1)the process move buffer is occupied
FALSE (0)the process move buffer is empty
