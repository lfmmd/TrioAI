AXIS_MODE

Type

Axis Parameter

Description

This parameter enables or disables features of an axis by setting or clearing the bit for the relevant feature.

Value

BitDescriptionValue
0Not used1
1Prevent CONNECT from cancelling when a hardware or software limit is reached, instead setting the ratio to 0 while outside its limits2
2Enable 3D direction calculations (2D when unset)4
3Cancel ADDAX on hitting limit8
4Disable look-ahead16
5Not used32
6Use unipolar (unsigned) analogue feedback64
7Not used128
8Use DPOS for software limit256
9Disable following error accumulation during move512
10Use FASTDEC when cancelling RPS jog1024
11Delay position evaluation to when move is loaded2048
12Direction shortest4096
13Monitor following error8192
14Ignore software limits16384
15Dual axis average position32767
16Invert motor direction65536
17Cancel current move on master if ADDAX hits limit131072

When bit 2 is set, all merged moves must be 3 axis moves.

Examples

Example 1

Enable bit 2 so that you can use 3D direction calculations, the OR is used to combine bit 2 into any already set bits.

AXIS_MODE AXIS(18) = AXIS_MODE AXIS(18) OR 4

Example 2

Enable bit 6 so that you can use a 0 to 10 V analogue input as axis feedback. Uses the dot notation for bit number.

BASE(5)

AXIS_MODE.6 = 1

Example 3

As example 3 but using the AXIS modifier.

AXIS_MODE AXIS(5).6 = 1
