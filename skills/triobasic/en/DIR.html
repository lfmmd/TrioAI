DIR

Type

System Command (command line only)

Syntax

DIR [option]

Alternate Format

LS [option]

Description

Prints a list of all programs including their size and RUNTYPE.

Parameters

ParameterFunction
noneDirectory listing of controller memory
dDirectory listing of SD card memory
u1 or

u2		Directory listing of USB memory stick. (Flex-7 only)
sReserved function
xExtended listing of controller memory (used by Motion Perfect).
