CAMLINK

Type

Axis Command

Syntax

CAMLINK(start, entries, mode, multiplier, link_dist, link_axis[, link_opt[, link_pos]])

Description

This move type for flexible cam shapes implements a fixed range of profiles. Shape selection will be supported in Motion Perfect which will output the parameters required. Only Mode 0 is supported.

Mode 0: Based on the input control point entries, and the input link_distance, CAMLINK calculates position, speed and acceleration for a single point from the profile, corresponding to the specified link_distance. It then writes the calculated 3 values at TABLE locations after the last input control point.

This command is for use by Motion Perfect when generating Cam Profiles. See Motion Perfect version 4.

Parameters

startThe start position of the control point data in the TABLE
entriesNumber of points in the TABLE
modeMode of operation
0Off-line calculation mode
multiplierNot used
link_distThe distance the link axis must move to complete CAMLINK profile
link_axisNot used
link_optNot used
link_posNot used

Control points

The profile is defined by a set of control points, each one having several properties - point type, relative X, position, speed and acceleration.

The input set of control point entries must start and end with a "Start" and "End" point types resp. Points between the start and end can be of type "Point", "Line Start" and "Line End". A Line End must be preceded by a Line Start. Parts of the profile between Line Start and Line End, inclusive, have fixed speed and acceleration values.

TABLE values

Each control point is written to TABLE as a block of 10 values starting at i. The first block is written with i=start, the second block at i=start+10 and so on.
IndexDescription
iPoint type
1Start
2Point
3End
4Line start
5Line end
i + 1Relative X
i + 2Position
i + 3Speed
i + 4Acceleration
i + 5 to i + 9Reserved

The profile points are calculated by interpolation with a polynomial order of 5th degree. The polynomial passes through the input control points, and has speed and acceleration values at these control points as specified in the input.

Results output

The results are put in the TABLE at the end of the input data.
IndexDescription
start + entries + 1Position value
start + entries + 2Speed value
start + entries + 3Acceleration value
