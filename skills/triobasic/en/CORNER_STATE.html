CORNER_STATE

Type

Axis Parameter

Description

Allows a BASIC program to interact with the move loading process.

This can be used to facilitate tool adjustment such as knife rotation at sharp corners.

This parameter is only active when CORNER_MODE bit 2 is set. It is also required to set bit 1 of CORNER_MODE with STOP_ANGLE set to less than or equal to RAISE_ANGLE to stop the motion.

Value

BitDescriptionValue
0Ready for corner sequence, XY axes are stopped1
1Load next XY move with 0 speed2
2Next move is loaded and TANG_DIRECTION is ready4
Ramp up speed and continue XY move0

Bits 0 and 2 are set by the system software. Bit 1 must be set by the application program that controls the corner sequence.

Value will be the combination of bits set, 0, 1, 3 and 7.

Example

When a transition exceeds RAISE_ANGLE it is required to lift a cutting knife and rotate it to a new position. The following process is required:

  • System sets CORNER_STATE to 1 to indicate move ready to be loaded with large angle change.
  • BASIC program raises knife.
  • BASIC program sets CORNER_STATE to 3.
  • System will load following move but with speed overridden to zero. This allows the direction to be obtained from TANG_DIRECTION.
  • System sets CORNER_STATE to 7.
  • BASIC program orients knife. Possibly using MOVETANG.
  • BASIC program clears CORNER_STATE to 0.
  • System will ramp up speed to perform the next move.

Axes 0 and 1 are XY, axis 2 is the knife rotation. This program controls the knife raise/lower and rotation synchronised with the state of CORNER_STATE.

BASE(2)

'Set up an automatic tangential move on axis 2

IF MTYPE = 0 THEN

    MOVETANG(0, 0)

ENDIF



BASE(0)

CORNER_MODE = 4 'Enable the CORNER_STATE machine

RAISE_ANGLE = 15 * PI / 180 '15 degrees



REPEAT

    WAIT UNTIL CORNER_STATE > 0 'Normally CORNER_STATE = 1 here

    WAIT UNTIL MTYPE = 0 'Wait for the present XY move to finish



    'Insert BASIC commands to raise the knife here:



    WA(100)



    'Setting CORNER_STATE = 3 by BASIC makes the firmware

    'load the next move but at zero speed:

    CORNER_STATE = 3



    'Firmware sets CORNER_STATE = 7 to indicate next move loaded

    WAIT UNTIL CORNER_STATE = 7





    'In this example we are using MOVETANG, but if not, then insert BASIC commands to rotate knife here:

    WA(2)



    'Allows MOVETANG to start following new TANG_DIRECTION

    WAIT UNTIL REMAIN AXIS(2) = 0



    'Insert BASIC commands to lower the knife here:



    WA(100)



    'Set CORNER_MODE = 0 to ramp up speed in XY

    CORNER_STATE = 0

UNTIL FALSE

Running in parallel with the above program, a separate program feeds XY moves into the motion generator.
