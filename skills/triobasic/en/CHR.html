CHR

Type

String Function

Syntax

value = CHR(number[, format])

Description

CHR returns the ASCII/UTF-8 character as a STRING which is referred to by the number, this can be assigned to a STRING variable or be PRINTed.

Parameters:
value:A STRING containing the character
number:Any valid numerical value for an ASCII/UTF-8 character
format:Format of the number
0ASCII (default)
1UNICODE UTF-8 encoded

Examples

Example 1

Print the character A on the command line

>>PRINT CHR(65)

A

>>

Example 2

Print a line of text terminating only with a carriage return

PRINT#5, "abcdefghijk"; CHR(13)

Example 3

Append a character from the serial port to a STRING variable

DIM value AS STRING

WHILE KEY#5

  GET#5, char

  value = value + CHR(char)

WEND

Example 4

Print a Unicode character UTF-8 encoded:

>>PRINT CHR(2600, 1)

袂

>>
