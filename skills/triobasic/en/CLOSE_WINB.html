CLOSE_WINB

Type

Axis Parameter

Alternate Format

CWB

Description

This parameter defines the end position of the second registration window which will be used for registration marks if windowing is specified by the REGIST command (mode 20 only).

Value

Absolute end position of the second registration window in user units.

Example

Set the second registration window between 100 and 150

OPEN_WINB = 100

CLOSE_WINB = 150

OPEN_WIN , CLOSE_WIN , OPEN_WINB , REGIST
