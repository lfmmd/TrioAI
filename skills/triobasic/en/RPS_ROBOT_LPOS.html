ROBOT_LPOS

Type

Robotics Parameter (read only)

Description

Returns the position of the robot’s active TCP (Tool Centre Point) in Cartesian space as a TARGET in WORLD coordinate system.

Value

Position in user UNITS.

Example

Example 1

GTA(1) = ROBOT_LPOS

Example 2

>>?ROBOT_LPOS     

600.00000,0.00045,6056.94750,180.00000,0.00000,7.37070e-05,""
