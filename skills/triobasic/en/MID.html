MID

Type

String Function

Syntax

value = MID(string, start[, length])

Description

Returns the mid-section of the specified string using the optional length specified, or defaults to the remainder of the string when not specified.

Parameters

string:String to be used
startStart index of string
length:Length of string to be returned, if not specified then the remainder of the string will be used

Examples

Example 1

Pre-define a variable of type string and later print characters: from index 5 to 10

DIM str1 AS STRING(32)

str1 = "TRIO MOTION TECHNOLOGY"

PRINT MID(str1, 5, 6)

Example 2

Transfer a string to an array of VRs so that each VR holds one byte. Print the result using VRSTRING.

DIM str1 AS STRING(32)

DIM vr_start AS INTEGER

DIM string_length AS INTEGER

DIM i AS INTEGER



str1 = "My cat is black"



vr_start = 10

string_length = LEN(str1)



FOR i = 0 TO string_length - 1

    VR(vr_start + i) = MID(str1, i, 1)

NEXT i

VR(vr_start + string_length) = 0 'terminator byte



PRINT VRSTRING(vr_start)
