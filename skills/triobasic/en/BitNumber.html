. Bit operator

Type

Mathematical Operator

Syntax

<expression>.bit_number

Description

Accesses the specified bit of the expression, either returning its value (0 or 1) if read or allowing it to be set or unset if written to.

Typically, this is used to access a specific bit of a variable, but can be used with any numeric expression, even a literal value.

Since the bit operator is also used as a decimal point, be careful that it is not interpreted as part of a value.

There should be no spaces before or after the bit operator.

Parameters

expression:Any valid TrioBASIC expression
bit_number:The bit number of the expression to return

Examples

Example 1

Check the AXISSTATUS for remote drive errors (bit 3)

IF AXISSTATUS.3 = 1 THEN

  PRINT "Remote drive error"

ENDIF

Example 2

Set VR(10) to 54.2, then read bit 2 of 54.

VR(10) = 54.2

PRINT (54).2

Example 3

Show how a local variable can have bits set and cleared in a program.

DIM a AS INTEGER



a = 0



a.8 = 1

PRINT a 'Prints 256

a.2 = 1

PRINT a 'Prints 260

a.8 = 0

PRINT a 'Prints 4

Example 4

The commands CLEAR_BIT, READ_BIT and SET_BIT are specifically for manipulating the bits within a VR variable, but this can be achieved using the bit operator.

VR(23).6 = 0 'Same as CLEAR_BIT(6, 23)

PRINT VR(13).4 'Same as PRINT READ_BIT(4, 13)

VR(7).3 = 1 'Same as SET_BIT(7, 3)

Example 5

Display axis status bit 3 (remote drive error) for axis 1.

PRINT AXISSTATUS AXIS(1).3
