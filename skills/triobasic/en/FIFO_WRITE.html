FIFO_WRITE

Type

Access Type

Syntax

OPEN # channel AS "[location:]name" FOR FIFO_WRITE(length)

Description

See OPEN for details.

Example

A G-Code program has been downloaded to a file on an SD card. It must be transferred to a FIFO so that it can be interpreted into motion. The FIFO file will act as a buffer to smooth out variations in processing speed.

OPEN#41 AS "SD:gc_program1" for INPUT 

OPEN#42 AS "gcodefifo" for FIFO_WRITE(512) 



WHILE KEY#41 

  GET#41, char 

  PRINT#42, char; 

WEND 



CLOSE#42 

CLOSE#41
