IEEE_OUT

Type

Mathematical Function

Syntax

byte_n = IEEE_OUT(value, n[, size])

Description

The IEEE_OUT function returns a single byte extracted from the floating-point value encoded in IEEE 754 format, typically for transmission over a communication bus system. The function is called multiple times to extract each byte in turn, 4 times for IEEE 754 single precision, and 8 times for IEEE 754 double precision.

Parameters

value:Any TrioBASIC floating-point variable or parameter.
n:The byte number (0 - 7) to be extracted.
size:Number of bits in the floating-point value (32 or 64), default 32.

Byte 0 is the high byte of the IEEE 754 floating-point format.

Examples

Example 1

Extract the 4 bytes from MPOS and store then in local variables ready for transmission over a communications bus.

a = MPOS AXIS(2)

byte0 = IEEE_OUT(a, 0)

byte1 = IEEE_OUT(a, 1)

byte2 = IEEE_OUT(a, 2)

byte3 = IEEE_OUT(a, 3)

Example 2

Extract the 8 bytes from MPOS and store then in local variables ready for transmission over a communications bus.

a = MPOS AXIS(2)

byte0 = IEEE_OUT(a, 0, 64)

byte1 = IEEE_OUT(a, 1, 64)

byte2 = IEEE_OUT(a, 2, 64)

byte3 = IEEE_OUT(a, 3, 64)

byte4 = IEEE_OUT(a, 4, 64)

byte5 = IEEE_OUT(a, 5, 64)

byte6 = IEEE_OUT(a, 6, 64)

byte7 = IEEE_OUT(a, 7, 64)
