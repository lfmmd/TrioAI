MPE_MODE

Type

System Parameter (read only)

Syntax

value = MPE_MODE

Description

Read the connection mode in current use on the command line process.

This is normally only used by Motion Perfect, but can be used for user applications.

Parameters

value:0No protocol. Motion Perfect is in Direct Mode or not connected.
1No longer used.
2No longer used.
3Motion Perfect version 3 connected in either tool or synchronised mode.
4Motion Perfect version 4 connected in tool mode.
5Motion Perfect version 4 connected in synchronised mode.

Example

Check the mode before attempting to reset the Motion Coordinator.

IF MPE_MODE = 0 THEN

  ' OK to reset the controller

  EX

ELSE

  PRINT #5,"Cannot reset when Motion Perfect is connected"

  ENDIF
