SYSTEM_ERROR

Type

System Parameter

Description

SYSTEM_ERROR returns an integer value which indicates which active errors have been reported by the firmware.

Any bits set in SYSTEM_ERROR will cause the WDOG to be automatically set to OFF, thus halting all motion.

If it is necessary to prevent the occurrence of particular errors causing a WDOG OFF condition, then SYSTEM_ERROR_MASK may be set.

Masking SYSTEM_ERROR bits must be done with caution as it might create a condition where motors can run out of control.

Unit errors$00ff0000
