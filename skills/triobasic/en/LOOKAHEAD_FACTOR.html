LOOKAHEAD_FACTOR

Type

Axis Parameter

Description

This parameter is used to modify all speeds in the motion buffer by factoring them by value specified.

LOOKAHEAD_FACTOR operates with all SP type interpolated moves MOVESP, MOVEABSSP and MHELICALSP and will factor the axis speeds FORCE_SPEED, ENDMOVE_SPEED and STARTMOVE_SPEED.

This is typically used to reduce speeds by using a factor less than one but can also be used to increase speeds by using a factor greater than one.

Example

Example 1

An analogue input from a potentiometer is used to adjust the overall speed of the motion on axes 2 and 3.

IF VR(200) = 1 THEN

  'Start the operator speed control routine

  BASE(2)

  REPEAT

    speed_factor = AIN(0) / 4096

    LOOKAHEAD_FACTOR = speed_factor

    WA(100)

  UNTIL VR(200) = 0

  LOOKAHEAD_FACTOR = 1.0 '100% full speed

ENDIF

Example 2

Set all look-ahead speeds of axis group starting at axis 4 to 50% of the programmed speed.

LOOKAHEAD_FACTOR AXIS(4) = 0.5
