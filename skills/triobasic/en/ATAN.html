ATAN

Type

Mathematical Function

Syntax

value = ATAN(expression)

Alternate Format

ATN(expression)

Description

Returns the arc-tangent (also known as inverse tangent) of a number, the result being an angle measured in radians in the range -π/2 to π/2 (see PI).

Parameters

value:The angle measured in radians
expression:Any valid TrioBASIC numeric expression

Example

Print the arc-tangent of -1 on the command line

>>PRINT ATAN(1)

0.7854
