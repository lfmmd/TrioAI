FMOD

Type

Mathematical Operator

Syntax

value = expression1 FMOD expression2

Description

Returns the floating point modulus of an expression. The FMOD operation finds the remainder after division of one number by another (called the modulus of the operation). Given two numbers, a and b, a FMOD b is the remainder of the Euclidean division of a by b.

Parameters

value:The modulus of expression 1
expression1:Any valid TrioBASIC expression used as the value to apply the modulus to.
expression2:Any valid TrioBASIC expression used as the modulus

Example

Some FMOD examples typed in terminal #0 command line.

>>PRINT 2023.4 FMOD 356

243.4000

>>PRINT -204.56 FMOD 60

-24.5600

>>PRINT 32760 FMOD 204.8  

196.8000
