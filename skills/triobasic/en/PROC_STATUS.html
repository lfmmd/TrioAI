PROC_STATUS

Type

Process Parameter (Read Only)

Description

Returns the status of another process, referenced with the PROC modifier.

Value

0STOPPEDNo program or task is running on the process
1RUNNINGA task or program is running
2STEPPINGProcess is stepping line by line
3PAUSEDExecution paused (pause button or breakpoint)
4PAUSINGPreparing to go into paused state
5STOPPINGPreparing to stop
6SUSPENDOperating system requested process to suspend
7SUSPENDEDProcess has temporarily been suspended
8SLEEPProcess sleeping, for example during WA, WAIT IDLE, etc.

SUSPEND and SUSPENDED are used internally by the operating system to force programs to stop temporarily while the OS runs some background tasks that would otherwise affect the programs. For example, the OS forces a SUSPEND when a program line is edited while other programs are running.

Example

Run a program in process 12, check for it to start and then for it to complete.

RUN "progname", 12

WAIT UNTIL PROC_STATUS PROC(12) <> 0 'Wait for program to start

WAIT UNTIL PROC_STATUS PROC(12) = 0 'Wait for program to finish



'Program "progname" has now finished
