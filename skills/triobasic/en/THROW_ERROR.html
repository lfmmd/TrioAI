THROW_ERROR

Type

Open Architecture Function

Syntax

THROW_ERROR(error_message)

Description

This command throws an error with a given error message. It generates a RUN_ERROR number 268. When an error is generated, the given error message is displayed.

This command can be used with ‘Open Moves’ to throw an error if a move cannot be loaded. It can also be used in functions or programs to stop a program with a custom error message.

Parameters

error_message:Message to display when throwing an error as STRING.

Examples

Example 1

Throw an error if function parameters are invalid.

FUNCTION set_frame(num as INTEGER)

   IF num > 121 THEN

      THROW_ERROR("FRAME number invalid")

   ELSE

      FRAME = num

   ENDIF

ENDFUNC

Example 2

Throw an error if a move takes too long. This will generate a RUN_ERROR and stop the program.

BASE(0)

MOVEABS(100, 200, 10)

TICKS = 2000

WHILE NOT IDLE

   IF TICKS < 0 THEN

      THROW("Move failed to finish in time")

   ENDIF

WEND
