SPEED_FACTOR

Type

Axis Parameter

Description

This parameter is used to reduce the speed specified in SPEED axis parameter by the per-unit value entered when VP_MODE is 5 or 6.

Speed factor range is 0-1, where 1 represents using 100% of the speed value.

Example

Example 1

An analogue input from a potentiometer is used to adjust the speed of the motion on axes 2.

IF VR(200) = 1 THEN

  'Start the speed control routine

  BASE(2)

  REPEAT

    SPEED_FACTOR = AIN(0) / 4096    

    WA(100)

  UNTIL VR(200) = 0

ENDIF

Example 2

Set the speed of axis 4 to 50% of the programmed speed.

SPEED_FACTOR AXIS(4) = 0.5
