POWER_UP

Type

System Parameter

Syntax

POWER_UP = value

Description

This parameter is included in the token list to allow older programs to compile in the MC4xx/MC5xx/MC6xx Motion Coordinators. It does not have a function because all programs are automatically stored in the Flash EPROM and the Motion Coordinator will always power up from Flash.

Example

Type the command in the Motion Perfect command-line terminal. The command has no function but it returns a value so as to maintain compatibility with previous Motion Coordinator ranges such as the MC206X.

>>?power_up 

0
