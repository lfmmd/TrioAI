VRSIZE

Type

System Parameter (Read Only)

Description

Returns the size of the VR array, which varies across controller types.

The VR array is indexed from zero, so the valid range of indices is 0 to VRSIZE - 1.

Value

The number of elements in the VR array.

Example

Set the last VR element to 123.

>>VR(VRSIZE – 1) = 123

>>
