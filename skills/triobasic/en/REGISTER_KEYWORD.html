REGISTER_KEYWORD

Type

Open Architecture Function

Syntax

REGISTER_KEYWORD(keywordName, funcName, minParams, maxParams)

Description

This command registers a keyword as a part of the TrioBASIC language. Each keyword must be linked to a BASIC function that holds the implementation of the keyword. It enables users to extend the language using their own keywords.
Note that a new keyword cannot be the same as an existing keyword.

Parameters

keywordName:Name of your keyword.
funcName:Name of your BASIC function.
minParams:Minimum number of parameters for BASIC function.
maxParams:Maximum number of parameters for BASIC function.

Examples

Example 1

Register 2 keywords.

In MC_CONFIG :

REGISTER_KEYWORD("RUN_TEST1", "test_int", 0, 2)

REGISTER_KEYWORD("RUN_TEST2", "test_float", 0, 1)


In TEST_LIB.BAL :

FUNCTION test_int(param1 AS INTEGER)

   IF param1 = 2 THEN

      PRINT "test_int passed"

   ELSE

      PRINT "test_int failed"

   ENDIF

ENDFUNC







FUNCTION test_float(param1 AS FLOAT)

   IF param1 = 4.12 THEN

      PRINT "test_float passed"

   ELSE

      PRINT "test_float failed"

   ENDIF

ENDFUNC


In TEST.BAS :

RUN_TEST1(2)

RUN_TEST1(0)

RUN_TEST2(4.12)

RUN_TEST2(2.31)
