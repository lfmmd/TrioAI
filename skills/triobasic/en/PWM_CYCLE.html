PWM_CYCLE

Type

Axis Parameter

Syntax

PWM_CYCLE = value

Description

Defines the number of internal clocks in the combined mark + space period (Maximum value 65535).

A 20MHz clock is used as the base clock for the PWM generation.

Parameters

valueNumber of 20MHz clock pulses for one PWM pulse

Examples

Example 1

If PWM_CYCLE=1000, PWM will be generated at 20 kHz frequency

If PWM_CYCLE=200, PWM will be generated at 100 kHz frequency (but with lower resolution)
