TSIZE

Type

System Parameter (Read Only)

Description

Returns the size of the TABLE, which varies across controller types.

The table is indexed from zero, so the valid range of indices is 0 to TSIZE - 1.

Value

The size of the TABLE

Example

Example 1

Display the size of the table.

>>?tsize

500000

Example 1

Set the last table element to 123.

>>table(tsize - 1, 123)

>>
