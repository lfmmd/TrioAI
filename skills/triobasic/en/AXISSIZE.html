AXISSIZE

Type

System Parameter (Read Only)

Description

Returns the maximum number of axes that a controller may use, which varies across controller types and servo period.

Axes are numbered from zero, so the valid range of axes is 0 to AXISSIZE - 1.

Value

The maximum size, minus 1, that the BASE keyword and the AXIS modifier may take.

Example

Display the number of usable axes on the Flex 6 Nano command line.

>>?AXISSIZE

>>64
