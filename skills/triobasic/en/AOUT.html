AOUT

Type

System Command

Syntax

AOUT(channel)

Description

Writes a value to an analogue output. Analogue outputs available from the CAN Analogue module.

The value sent is the decimal equivalent of the binary number to be written to the D to A converter.

Parameters

channel:Analogue output channel number 0 to 15

Example

An output is to be set to the speed input of an open-loop inverter drive. 10 V is 1500 rpm and the required speed is 300 rpm.

value = 300 * 2048 / 1500

AOUT(1) = value

The analogue output voltage is set to 2 V.

The voltage is approximate and the output must be calibrated by the user if high accuracy is required.
