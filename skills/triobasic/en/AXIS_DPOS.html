AXIS_DPOS

Type

Axis Parameter (Read Only)

Alternate Format

TRANS_DPOS

Description

AXIS_DPOS is the axis demand position at the output of the FRAME transformation.

AXIS_DPOS is equal to DPOS on each axis when the frame transformation is 1:1 for each axis (FRAME = 0). However, some machinery configurations, typically machines such as robotic arms or machines with parasitic motions on the axes, require a frame transformation that is not 1:1, and in this situation (FRAME > 0), AXIS_DPOS returns the demand position for the actual motor.

AXIS_DPOS is set to MPOS when SERVO or WDOG are OFF

Value

The axis demand position at the output of the FRAME transformation in AXIS_UNITS. Default 0 on power up.

Example

Return the axis position in user AXIS_UNITS using the command line.

>>PRINT AXIS_DPOS

125.22

>>
