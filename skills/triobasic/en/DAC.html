DAC

Type

Axis Parameter

Description

Writing to this parameter when SERVO = OFF and AXIS_ENABLE = ON allows the user to force a demand value for that axis. On an analogue axis this will set a voltage on the output. On a digital axis this will be the demand value. For example, an EtherCAT CSV axis that has target speed.

When using a FlexAxis as a stepper or encoder output or anytime with SERVO = OFF the voltage outputs are available for user control.

The WDOG and AXIS_ENABLE must be ON for the demand value to be set. When the WDOG or AXIS_ENABLE is OFF you can write a value to DAC but the actual output (DAC_OUT) will be at 0.

The maximum output value is limited by OUTLIMIT. Set OUTLIMIT to a suitable value that matches the axis type and the connected device.

Value

The demand value for an analogue axis depends on the resolution of the D to A converter. The stand-alone Motion Coordinators, MC403, MC405, MC508 have 12-bit D to A converters. FlexAxis modules P874 and P879 on the MC664(X) have 16-bit D to A converters.

Example

To force a square wave of amplitude ±5 V and period of approximately 500 ms on axis 0.

WDOG = ON

SERVO AXIS(0) = OFF

square:

  DAC AXIS(0) = 1024

  WA(250)

  DAC AXIS(0) = -1024

  WA(250)

GOTO square
