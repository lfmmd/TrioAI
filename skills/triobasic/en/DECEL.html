DECEL

Type

Axis Parameter

Description

The DECEL axis parameter may be used to set or read back the deceleration rate of each axis fitted.

Value

The deceleration rate in UNITS per second per second.

Must be a positive value.

Example

Set the deceleration rate and print it to the terminal.

DECEL = 100

PRINT "Deceleration rate is "; DECEL; " mm/s/s"
