BITCOUNT

Type

Mathematical Function

Syntax

value = BITCOUNT(expression)

Description

Returns the number of bits set (equal to 1) in the expression when processed as a 64-bit integer.

Parameters

value:The number of bits set in the 64-bit integer expression
expression:Any valid TrioBASIC expression representing a numerical value

Example

>>?bitcount(255)

8

>>?bitcount(65534)

15

>>?bitcount($1001200240048008)

8

>>?bitcount(-1) 

64

>>?bitcount(-2)

63

>>?bitcount(4.8892)

1

>>?bitcount($2000+$C9)

5

>>
