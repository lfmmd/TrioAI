^ Power

Type

Mathematical Operator

Syntax

(expression1) ^ (expression2)

Description

Raises expression1 to the power of expression2.

Parameters

Expression1:Any valid numeric expression
Expression2:Any valid numeric expression

Example

Example 1

Raises the 2 to the power of 6 and store it in local variable ‘x’, then print the value of ‘x’, which is 64.

x = 2 ^ 6

PRINT x

Example 2

Determine the cube root of 2 and store it in local variable ‘x’, then print the value of ‘x’, which is 1.2599.

x = 2 ^ (1 / 3)

PRINT x
