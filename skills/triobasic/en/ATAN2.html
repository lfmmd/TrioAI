ATAN2

Type

Mathematical Function

Syntax

value = ATAN2(expression1, expression2)

Description

Returns the arc-tangent (also known as inverse tangent) of the ratio expression1 / expression2, the result being an angle measured in radians in the range -π to +π (see PI) in the correct quadrant.

Use ATAN2 when calculating vectors as it is quicker to execute than ATAN(y/x)

Parameters

value:The angle measured in radians
expression1:Any valid TrioBASIC numeric expression representing the side opposite the angle (in a right-angled triangle)
expression2:Any valid TrioBASIC numeric expression representing the side adjacent to the angle (in a right-angled triangle)

Examples

Print the arc-tangent of -1 divided by 2 on the command line

>>PRINT ATAN2(-1, 2)

-0.4636

Print the arc-tangent of 1 divided by -2 on the command line

>>PRINT ATAN2(1, -2)

2.6779
