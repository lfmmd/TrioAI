COS

Type

Mathematical Function

Syntax

value = COS(expression)

Description

Returns the natural cosine of the supplied angle measured in radians.

Parameters

value:The natural cosine of the expression
expression:Any valid TrioBASIC numeric expression representing an angle measured in radians

Example

Print the cosine of 1 radian on the command line with 3 decimal places

>>PRINT COS(1)[3]

0.540

>>
