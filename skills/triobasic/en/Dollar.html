$ Dollar

Type

Special Character

Syntax

$<hexadecimal-digits>

Description

The $ symbol is used to specify that the following signed 53 bit number (for floating point) or 63 bit (for integer) is in hexadecimal format using the digits 0-9 and A-F.

Examples

Example 1

Store the hexadecimal value of 38F3B into VR element 10 and -A58 into VR element 11

VR(10) = $38F3B

VR(11) = -$A58

Example 2

Turn on outputs 10, 11, 14, 15

OP($CC00)
