HW_TIMER_DONE

Type

Slot Command (Read Only)

Syntax

HW_TIMER_DONE

Description

Indicates whether a requested HW_TIMER is in progress.

Value

0The hardware timer is not in progress (the previous HW_TIMER request has completed)
1The hardware timer is in progress (running the previous HW_TIMER request)

Example

Request enable output of axis 4 to be ON for 500 ms.

HW_TIMER(1, 500000, ON, 1, 4): WAIT UNTIL HW_TIMER_DONE = 0
