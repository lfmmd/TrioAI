SYSTEM_LOAD_AVG

Type

System Parameter

Description

SYSTEM_LOAD_AVG returns an average from the previous 256 cycles of the SYSTEM_LOAD value. The value is expressed as a percentage of the current servo period.

Value

The average percentage of servo period time that is used for system and motion processing.

Example 1

From the terminal 0 command line, read the current average percentage of servo time being used by the system firmware.

>>?SYSTEM_LOAD_AVG

16.073

>>
