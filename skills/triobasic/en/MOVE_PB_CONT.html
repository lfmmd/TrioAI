MOVE_PB_CONT

Type

Axis Parameter

Description

This parameter and MOVE_PA_CONT are two axis parameters which control what the system software does with MOVE_PA/MOVE_PB when the move becomes active or MOVE_PA_IDLE/MOVE_PB_IDLE when the axis becomes idle.

By changing the value of this parameter, the behaviour of the MOVE_PA/MOVE_PB and MOVE_PA_IDLE/MOVE_PB_IDLE will change.

Parameters

mode:1Bits 0 to 11 of MOVE_PB set output #; Bit 12 sets ON/OFF. Output set to setting in MOVE_PB_IDLE when MTYPE = 0
2MOVE_PB is copied into AXIS_DEBUG_B
3Bits 0 to 11 of MOVE_PB set output #; Bit 12 sets ON/OFF. Output set to setting in MOVE_PB_IDLE when VP_SPEED = 0 or MTYPE = 0
4The bit pattern in MOVE_PB is copied to the outputs starting at a given index. The bit pattern defined in MOVE_PB_IDLE is output when MTYPE = 0.
5Analogue channel in MOVE_PB bits 32 to 39 set to 32-bit value in MOVE_PB bits 0 to 31 when move is started.
6Analogue channel in MOVE_PB bits 32 to 39 set to 32-bit value in MOVE_PB bits 0 to 31 when move is started. MOVE_PB_IDLE is used when move is complete.

Mode 4 is available in firmware v2.0284 and later

Modes 5 and 6 are available in firmware v2.0339 and later
