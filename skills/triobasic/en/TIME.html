TIME

Type

System Parameter

Description

Allows the user to set and read the time from the real time clock.

Value

Read = the number of seconds since midnight (24:00 hours)
Write = the time in 24hour format hhss

Examples

Example 1

Sets the real time clock in 24 hour format; hhss

'Set the real time clock

>>TIME = 13:20:00

Example 2

Calculate elapsed time in seconds.

time1 = TIME



'Wait for event

time2 = TIME

timeelapsed = time1 - time2

Example 3

Set the time from within a BASIC program.

hh = 15

mm = 23

ss = 30

TIME = hh:mm:ss
