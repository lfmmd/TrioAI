STOP_ROBOT

Type

Robotics Command

Syntax

STOP_ROBOT(identity)

Description

This command cancels all moves, clears buffers and stops process belonging from the selected robot index. It can be invoked from other processes, making it possible to control a robot or group of robots from a main program.

Parameters

identity:Robot identity number

Examples

Example 1

Move robot 0 between 2 defined points whilst input 303 is active. When input 296 is active, stop the robot move and cancel all buffered moves.

ROBOT(0)



WHILE IN(303) = ON

    MOVEJ GTA("pInfeed_pt1") S:=10 O:="infeed"

    MOVEJ GTA("pOutfeed_pt1") S:=10 O:="outfeed"

    

    IF IN(296) = ON THEN

        STOP_ROBOT(0)

    ENDIF

WEND



ROBOT(-1)
