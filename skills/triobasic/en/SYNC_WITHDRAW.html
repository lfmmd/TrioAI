SYNC_WITHDRAW

Type

Axis Parameter

Syntax

SYNC_WITHDRAW AXIS(x) = 125.5 {distance in axis units}

Description

One a set of axis parameters, including SYNC_PAUSE and SYNC_DWELL, which allow greater control over when the sync motion occurs during the sync time period. The SYNC_WITHDRAW axis parameter allows an additional motion to be specified and can be used for the Z axis motion where a robot moves away from a conveyor or other position.

Value

The distance for the "axis withdraw" motion in axis units. Typically the Z axis will have the withdraw move on it. Used in a XYZ pick and place situation.

The SYNC_PAUSE, SYNC_DWELL and SYNC_WITHDRAW parameters are often set differently on each axis of the SYNC command. For example the SYNC_WITHDRAW parameter may be zero on the X and Y axes of a system, but SYNC_WITHDRAW is set to a value on the Z axis of a system.

Example

BASE(0)



UNITS AXIS(0) = 1000

UNITS AXIS(1) = 1000

UNITS AXIS(2) = 1000

UNITS AXIS(7) = 1000

UNITS AXIS(8) = 1000



ATYPE AXIS(0) = 0

ATYPE AXIS(1) = 0

ATYPE AXIS(2) = 0



SYNC(4, 100) 'Clear any previous sync

WAIT UNTIL SYNC_CONTROL = 0



overall_time = 600

sync_part = 500

flight_part = 50



BASE(2) 'Z Axis SYNC controls:

SYNC_PAUSE = (overall_time - flight_part) / 2

SYNC_DWELL = (overall_time - flight_part) / 2

BASE(1) 'Y Axis SYNC controls:

SYNC_WITHDRAW = 0

SYNC_PAUSE = (overall_time - sync_part) / 2

SYNC_DWELL = (overall_time - sync_part) / 2

BASE(0) 'X Axis SYNC controls:

SYNC_WITHDRAW = 0

SYNC_PAUSE = (overall_time - sync_part) / 2

SYNC_DWELL = (overall_time - sync_part) / 2





DEFPOS(0, 0, 0)

SPEED AXIS(7) = 100

SPEED AXIS(8) = 100

DEFPOS(-200) AXIS(7) 'Axis 7 simulates conveyor A

DEFPOS(-200) AXIS(8) 'Axis 8 simulates conveyor B

IF MTYPE AXIS(7) = 0 THEN FORWARD AXIS(7)

IF MTYPE AXIS(8) = 0 THEN FORWARD AXIS(8)



TRIGGER



'These are the 2 conveyor heights:



conveyor_a_z = 65

conveyor_b_z = 70



'Sync to conveyor B from start point:

SYNC_WITHDRAW AXIS(2) = 5

SYNC(1, overall_time, 0, 8, 60, 115, conveyor_b_z)

WAIT UNTIL SYNC_CONTROL = 3' synchronised

WA(150)



FOR k = 0 TO 1

   'Sync to conveyor A from conveyor B:

   SYNC_WITHDRAW AXIS(2) = -40

   SYNC(10, overall_time, 0, 7, 60, -100, conveyor_a_z)

   WAIT UNTIL SYNC_CONTROL = 3 'Synchronised

   WA(150)



   'Sync to conveyor B from conveyor A:

   SYNC_WITHDRAW AXIS(2) = -40

   SYNC(10, overall_time, 0, 8, 60, 115, conveyor_b_z)

   WAIT UNTIL SYNC_CONTROL = 3 'Synchronised

   WA(150)

NEXT k



'Sync to conveyor A from conveyor B:

SYNC_WITHDRAW AXIS(2) = -40

SYNC(10, overall_time, 0, 7, 60, -100, conveyor_a_z)

WAIT UNTIL SYNC_CONTROL = 3 'Synchronised

WA(150)





'De Sync from conveyor A to start point:

SYNC_WITHDRAW AXIS(2) = -conveyor_a_z

SYNC(10, overall_time, 0, 9, 0, 0, 0) '9 is a dummy axis

WAIT UNTIL SYNC_CONTROL = 3 'Synchronised



STOP
