SYSTEM_WARNING

Type

System Parameter

Description

SYSTEM_WARNING is a bitwise system variable that returns a value which indicates which active warnings have been reported by the firmware.

Value

RFID$000001
EBUS overcurrent$000002
USB1 overcurrent$000004
USB2 overcurrent$000008
Power supply undervoltage$000010
PCMCAT Windows RAM low$000020
MotionPLC slice output driver fault$000040
MotionPLC built-in output driver fault$000080
