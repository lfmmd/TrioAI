WORLD_POS_MAX_JERK

Type

Robotics Parameter

Description

The WORLD_POS_MAX_JERK axis parameter can be used to read and write the maximum possible linear axis JERK in world mode.

JERK is the derivative of acceleration. JERK units are /sec/sec/sec. The way JERK is applied to the axis motion depends on the value of VP_MODE.

Value

The robot axis maximum linear acceleration in mm/ sec3.
