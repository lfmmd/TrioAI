WORLD_RS_LIMIT

Type

Axis Parameter

Description

An end of travel limit may be set up in software thus allowing the program control of the working range of a robot axis. This parameter holds the absolute position of the world forward travel limit in user WORLD_UNITS.

Bit 1 of the ROBOTSTATUS register is set when the axis position is greater than the WORLD_RS_LIMIT.

World software limits are only enabled when FRAME<>0 so that the user can limit the range of motion of the robot.

When WORLD_DPOS reaches WORLD_RS_LIMIT the controller will CANCEL all moves on the FRAME_GROUP, the axis will decelerate at FASTDEC if specified, otherwise at DECEL or. Any SYNC is also stopped. As this software limit uses WORLD_DPOS it will require a negative change in WORLD_DPOS to move off the limit. This might not be a negative movement on DPOS due to the selected FRAME transformation.

WORLD_RS_LIMIT is disabled when it has a value greater than REP_DIST or when FRAME=0.

Value

The absolute position of the software forward travel limit in user UNITS. (default = 400000000000)

Examples

Set up an axis software limit so that the axis operates between -200 and +500 mm.

WORLD_UNITS = FRAME_SCALE

WORLD_FS_LIMIT = 500

WORLD_RS_LIMIT = -200
