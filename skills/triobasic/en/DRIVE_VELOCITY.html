DRIVE_VELOCITY

Type

Axis Parameter

Description

Returns the actual velocity value from a remote drive. For this value to be active, the cyclic data transfer from the drive must be first configured to return the drive actual velocity value. For a drive connected by CANOpen over EtherCAT (CoE) the value will be configured as part of the Process Data Object. (PDO)

Value

The motor speed is returned in drive units.

Example

Example1

Display the motor actual speed to Motion Perfect terminal 5.

PRINT #5,"Drive current value = ";DRIVE_VELOCITY AXIS(2)

Example2

Wait for the motor speed value to go below a pre-defined level.

BASE(16)

WAIT UNTIL DRIVE_VELOCITY < 500
