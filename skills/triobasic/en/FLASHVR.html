FLASHVR

Type

System Function

Syntax

FLASHVR(index)

Description

Copies the specified VR index to permanent Flash memory, a negative index signifying a special action (see table below).

If FLASHVR(-1) is being used then you cannot use FLASHTABLE

This command should only be used on MC2XX and MC3XX, which do not store data in flash memory automatically. Other controllers automatically store VR in flash memory, and FLASHTABLE should be used to backup TABLE data.

Parameters

index:0 to VRSIZE - 1The index of the VR array element to write to flash memory.
-1Stores the entire TABLE to the Flash EPROM and use it to replace the RAM table data on power-up.
-2Stop using the EPROM copy of table during power-up.
-100Force all changed VR’s to be committed to Flash EPROM (non-battery backed controllers only)

After using function -1, any changed table data will be overwritten on the next power up or reset.

Example

Save the entire TABLE data to FLASH memory.

FLASHVR(-1)
