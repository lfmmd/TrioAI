COMBINE_CONTROL

Type

Axis Command

Syntax

COMBINE_CONTROL(mode, source_1, source_2, Numerator_1, Denominator_1, Numerator_2, Denominator_2)

Description

The COMBINE_CONTROL command is used to set up the mode of operation for the COMBINE_AXES function.

Parameters

mode:0DPOS out = source_1 position + source_2 position
1DPOS out = source_1 position – source_2 position
2√ (source_1 MPOS ^2 + source_2 MPOS ^2) [1]
3DPOS out = (source_1 position + source_2 position) / 2
source_10 = DPOS, 1 = MPOS
source_20 = DPOS, 1 = MPOS
numerator_1Applied to source_1 before the value is used in the function
denominator_1Applied to source_1 before the value is used in the function
numerator_2Applied to source_2 before the value is used in the function
denominator_2Applied to source_2 before the value is used in the function

[1] Mode 2 has only 3 parameters:

COMBINE_CONTROL(mode, offset_1, offset_2)
mode2
offset_1Offset applied to source_1 value
offest_2Offset applied to source_2 value

Examples

Example 1

Sub routine to set up an axis to have position defined by the sum of 2 other axes. Set the first source axis to have 0.5 multiplier and the second source to be x2.

DPOS out = MPOS1 * 0.5 + MPOS2 * 2

'=============================

mode_0:
