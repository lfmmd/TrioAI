RPS_KERNEL_V2

Type

Robotics Command (MC_CONFIG only)

Syntax

RPS_KERNEL_V2 = value

Description

RPS kernel is a controller state machine that allows motion only when certain conditions are met for specific operation profile. There are 3 operation profiles: Auto, Manual, and External. Please refer to RPS Operation Profiles document for more information.

The kernel provides faster response as a safety module than the previous RPS solution.

The command can only be set in MC_CONFIG file, but it can be read in Motion iX (TrioBasic) programs and on the command line.

The kernel can only be activated once and cannot be deactivated manually. This is to ensure the integrity of the I/O operations within the operation profiles. To deactivate, a power cycle of the controller is needed.

Parameters

value: BooleanTrue = RPS Kernel is activated.
To deactivate, restart controller.

Examples

Example 1

Activate RPS kernel (from MC_CONFIG).

RPS_KERNEL_V2 = TRUE
