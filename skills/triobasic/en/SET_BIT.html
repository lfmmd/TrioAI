SET_BIT

Type

Logical and Bitwise operator

Syntax

SET_BIT(bit, variable[, mode])

Description

SET_BIT can be used to set the value of a single bit within a VR variable. All other bits are unchanged.

You can change the mode for setting a single bit to TABLE.

Parameters

bit:The bit number to set, valid range is 0 to 52
variable:The VR/TABLE which to operate on
Mode:0Bit sets to VR (default)
1Bit sets to TABLE

Examples

Example 1

Set bit 3 of VR(7)

SET_BIT(3, 7)

Example 2

Set bit 3 of TABLE (1)

SET_BIT(3, 1, 1)
