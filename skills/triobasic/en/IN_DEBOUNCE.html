IN_DEBOUNCE

Type

I/O command

Syntax

IN_DEBOUNCE = time_in_servocycles

Description

The inputs go to the set state after the voltage has been either 24 or 0 for the IN_DEBOUNCE time in servo cycles. The selected debounce time is applied to all inputs when read in a program using the Motion iX IN function or the equivalent IN binding in the IEC 61131-3 language.

Registration input functionality is not affected by the IN_DEBOUNCE value.

Parameters

value:The number of servo cycles for which the input must remain in a steady state for before the ON/OFF value is passed to a program.

Examples

Example 1

SERVO_PERIOD is 500 and the required debounce time is 5 ms:

IN_DEBOUNCE = 10

Example 2

A simple test program to see the effect of IN_DEBOUNCE. Connect the output number 8 to input number 0 externally with a wire. Result is displayed in Terminal #0.

IN_DEBOUNCE = 12 'SERVO_PERIOD is 1000 so we expect 12 ms

WA(200)



TICKS = 0

OP(8, ON)

WAIT UNTIL IN(0) = ON

t = -TICKS

PRINT "HI: "; t[0]

WA(200)



TICKS = 0

OP(8, OFF)

WAIT UNTIL IN(0) = OFF

t = -TICKS

PRINT "LO: "; t[0]
