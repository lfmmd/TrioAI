FASTJERK

Type

Axis Parameter

Description

The FASTJERK axis parameter may be used to set or read the fast jerk rate of each axis fitted.

The fast deceleration and fast jerk rates are used when motion is cancelled, either via the CANCEL or RAPIDSTOP commands, or from a software or hardware limit.

FASTJERK is applied only when using VP_MODE 5 or 6.

FASTJERK must be greater than JERK, otherwise JERK will be used.

As of firmware version 2.0342, the SRAMP keyword may be used instead of FASTJERK, but if SRAMP is assigned a value (even 0), then any assignment to FASTJERK will be ignored. When SRAMP is used, the FASTJERK is calculated from FASTDEC / SRAMP.

Using SRAMP with VP_MODE = 6 is intended solely to aid portability for existing applications, and it is strongly recommended that new applications use JERK and FASTJERK.

Value

The deceleration rate in UNITS/sec/sec/sec and must be a positive value, or zero to not be used.

Example

DECEL = 100            'Set normal deceleration rate

JERK = 200             'Set normal jerk rate

FASTDEC = 1000         'Set fast deceleration rate

FASTJERK = 2000        'Set fast jerk rate

MOVEABS(10000)         'Start a move

WAIT UNTIL MPOS = 5000 'Wait until the move is halfway through

CANCEL                 'Cancel the move (will use fast rates)
