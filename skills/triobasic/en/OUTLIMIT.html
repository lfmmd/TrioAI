OUTLIMIT

Type

Axis Parameter

Description

The output limit restricts the DAC output to a lower value than the maximum. This can be used to limit the analogue outputs or demand value to a digital drive. OUTLIMIT will always limit the DAC output if you are using a servo control or just manually setting DAC.

As it is applied to the output of the closed loop algorithm it is not applied to position-based axis.

Value

The range that the DAC is limited to.

The value required varies depending on whether the axis has a 12-bit or 16-bit DAC. If the voltage output is generated by a 12-bit DAC values an OUTLIMIT of 2047 will produce the full ±10 V range. If the voltage output is generated by a 16-bit DAC values an OUTLIMIT of 32767 will produce the full ±10 V range.

Examples

Example 1

Limit a 12-bit DAC to ±5 V (±1023)

OUTLIMIT AXIS(0) = 1023

Example 2

Adjust the limit to suit a 3-axis EtherCAT system with drives in CSV (cyclic synchronous velocity mode) where the Target Velocity value to the drive is ±300000 for ±3000 RPM.

OUTLIMIT AXIS(3) = 300000

OUTLIMIT AXIS(4) = 300000

OUTLIMIT AXIS(5) = 300000
