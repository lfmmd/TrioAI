ACCEL

Type

Axis Parameter

Description

The ACCEL axis parameter may be used to set or read back the acceleration rate of each axis fitted.

Value

The acceleration rate is in UNITS per second per second.

Must be a positive value.

Example

Set the acceleration rate and print it to the terminal.

ACCEL = 130

PRINT "Acceleration rate is "; ACCEL; " mm/s/s"
