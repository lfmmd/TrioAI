JOGSPEED

Type

Axis Parameter

Description

Sets the jog speed in user units for an axis to run at when performing a jog.

You can set a faster jog speed using SPEED and the FAST_JOG input

Value

The speed in user UNITS per second which an axis will use when being jogged.

Example

Configure an input to be the jog input at 20 mm/s on axis 12

BASE(12)

SPEED = 3000

FWD_JOG = 12

JOGSPEED = 20
