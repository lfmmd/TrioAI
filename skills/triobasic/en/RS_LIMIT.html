RS_LIMIT

Type

Axis Parameter

Alternate Format

RSLIMIT

Description

An end of travel limit may be set up in software thus allowing the program control of the working envelope of the machine. This parameter holds the absolute position of the reverse travel limit in user units.

Bit 10 of the AXISSTATUS register is set when the axis position is less than than the RS_LIMIT.

When DPOS is less than RS_LIMIT the controller will cancel the move, so the axis will decelerate at FASTDEC if specified, otherwise at DECEL.

RS_LIMIT is disabled when it has a value greater than -REP_DIST. The value must take account of the position lag on MPOS.

Value

The absolute position of the software reverse travel limit in user units (default = -200000000000).

Example

After homing a machine set up the reverse software limit so that the axis will stop 5mm away from the hard stop. So, if the hard limit is at -200, with a maximum speed of 80 and a FASTDEC of 1000 the reverse limit will be -188.6.

hard_limit_position = -200

max_speed = 400

FASTDEC = 1000



DATUM(3)

WAIT IDLE

RS_LIMIT= hard_limit_position + 0.5 * (max_speed ^ 2 / FASTDEC) + 5
