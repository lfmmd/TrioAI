WHILE .. WEND

Type

Program Structure

Syntax

WHILE condition

  Commands

WEND

Description

The commands contained in the WHILE..WEND loop are continuously executed while the condition is TRUE. Execution then continues after the WEND. If the condition is FALSE when the WHILE is first executed then the loop will be skipped.

Parameters

condition:Any valid logical TrioBASIC expression
commands:TrioBASIC statements that you wish to execute

Example

While input 12 is off, move the base axis and flash an LED on output 10

WHILE IN(12) = OFF

  MOVE(200)

  WAIT IDLE

  OP(10, OFF)

  MOVE(-200)

  WAIT IDLE

  OP(10, ON)

WEND
