MOVETEST

Type

Axis Command

Syntax

MOVETEST(start/stop, dist0, accel0, speed0, dwell0, dist1, accel1, speed1, dwell1)

Description

Start/stop a test move. This will perform a continuous cycle of 2 moves based on the profile parameters in the function. The axis will move forward and backward cyclically over the two selected distances. Each movement has its own acceleration and speed values which are in user units. The dwell between moves is in milliseconds.

All motion stopping command like CANCEL, RAPIDSTOP will stop the motion

Calling MOVETEST(1, ...) will start the motion

Calling MOVETEST(0, ...) will stop the motion at the end of the second move

Parameters

start/stop:0Stop the motion at the end of the second move
1Start the test motion
dist0:Distance to move in the forward direction.
accel0:Acceleration applied to the forward move.
speed0:Speed applied to the forward move.
dwell0:Dwell time in milliseconds after the forward movement before starting the reverse move.
dist1:Distance to move in the reverse direction.
accel1:Acceleration applied to the reverse move.
speed1:Speed applied to the reverse move.
dwell1:Dwell time in milliseconds after the reverse movement before starting the next forward move.

Examples

Example 1

Set up a cyclic test move on the selected BASE axis. Move will go 10 turns of the motor at 1200 RPM (20 turns per second) with a dwell of 500 μs between each move.

BASE(2)

UNITS = 2 ^ 21 '21-bit encoder in use on the motor 

MOVETEST(1, 10, 100, 20, 500, -10, 100, 20, 500) 



'Continue until input 5 is OFF

WAIT UNTIL IN(5) = OFF
