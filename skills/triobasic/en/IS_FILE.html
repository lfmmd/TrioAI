IS_FILE

Type

System Parameter (Read Only)

Syntax

value = IS_FILE "file_name"

Description

IS_FILE returns the state of whether a specific file exists or not within the project directory of the controller (DIR). The name of the file could be a string literal, a variable of string type and VRSTRING could also be used.

IS_FILE and IS_PROG have the same function.

Parameters

value:TRUE (-1) if the file exists,

FALSE (0) if the file does not exists.
file:Name of the file

Examples

Example 1

file_status = IS_FILE "MAIN_TASK"

Example 2

Remove a text file only if it exists:

DIM textfile_name AS STRING(12)

textfile_name = "textfile_no1"

IF IS_FILE textfile_name THEN 

    DEL textfile_name

ENDIF
