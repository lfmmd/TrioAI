SEEK

Type

File Function

Syntax

SEEK #channel, from, position

Description

Sets the position in the file for the next read/write operation. The file must be opened for INPUT or for READ_WRITE when available.

Parameters

channel:Channel number. e.g. 40 for a file opened with OPEN #40.
from:0position offset from start of the file.
1position offset from current position in the file.
position:Position offset value, in bytes from the selected "from" position.

Examples

Example 1

Save a string to a file and later select a position to start reading the file:

DIM mystring AS STRING(35)



OPEN #40 AS "my_file" FOR OUTPUT(1)

PRINT #40,"Trio Motion Technology MC range of Motion Coordinators"; CHR(10); CHR(13);

CLOSE #40



OPEN #40 AS "my_file" FOR INPUT

SEEK #40, 0, 22

LINPUT #40, mystring



'mystring contains "MC range of Motion Coordinators"

PRINT mystring
