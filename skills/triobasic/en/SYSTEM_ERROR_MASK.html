SYSTEM_ERROR_MASK

Type

System Parameter

Description

Allows selection of which SYSTEM_ERROR bits will trigger a fault indication and set WDOG to OFF.

Masking SYSTEM_ERROR bits must be done with caution as it might create a condition where motors can run out of control.

Value

Default value is $FFFFFFFF (all bits set to 1) to not mask out any errors.

Set bits to 0 to mask out the error bits listed in SYSTEM_ERROR as required.
