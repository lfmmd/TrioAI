= Assignment

Type

Program Variable

Syntax

Variable = expression

Description

The assignment operator allows an expression to be stored in a variable for later use.

The variable may be a simple variable declared with DIM (either a scalar or an array with up to three dimensions), or may be a system variable such as VR and GTA (including GTAE and GTAJ).

Parameters

value:the variable in which to store the value
expression:any valid TrioBASIC expression

Examples

Example 1

Set the sum of 10 and 9 into local variable ‘result’.

result = 10 + 9

Example 2

Assign variable ‘result’ to VR element 10.

VR(10) = result

See Also
