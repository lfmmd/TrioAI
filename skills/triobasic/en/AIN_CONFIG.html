AIN_CONFIG

Type

I/O Command

Syntax

AIN_CONFIG(channel) = value

Description

Sets the mode of operation for an analogue input. This operation can be performed on any external analogue input from a Trio CAN Analogue input module P326/P328.

This function requires CANIO firmware 2.3.2 or later.

The CAN analogue module P326 has 8 x 12 bit inputs.

The CAN analogue module P328 has 8 x 16 bit inputs.

Parameters

channel:Analogue input channel number.
value:0Bipolar, -10 V to +10 V
1Unipolar, 0 to +10 V
2Bipolar, -5 V to +5 V
3Unipolar, 0 to +5 V

Example

Set analogue input 5 to range 0 to 5 V. The 12 bit value range will be 0 to 4095 where 4095 := 5 V.

AIN_CONFIG(5) = 3
