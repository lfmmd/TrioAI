TOKENTABLE

Type

System Command

Description

When entered in the command line in Direct Mode, the TOKENTABLE will output the complete list of Tokens or “key words” that make up the Trio language. The output format is as plain text with associated numerical and string tags.

C144EXECUTE,0,0,1,3,0,0<cre>

The command is used by a remote connection such as the TrioPC Motion ActiveX control, DLL etc. to download the token list for use within the remote application software.

Example

>>tokentable

[TOKENTABLE]

C0TOKENTABLE,0,0,0,1,0,0<cbre>

C58CONSTANT,0,0,0,2,0,0<cre>

C59FILE,0,0,1,3,0,0<cfre>

C60OPEN,0,0,0,0,0,0<cfre>

.....
