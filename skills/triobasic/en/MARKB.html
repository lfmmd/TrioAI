MARKB

Type

Axis Parameter (Read Only)

Description

This parameter can be polled to determine if the registration event has occurred on the second registration channel.

MARKB is reset when REGIST is executed.

Value

FALSEThe registration event has not occurred
TRUEThe registration event has occurred (default)
< -1Quantity of registration events have been logged to the TABLE

When TRUE the REG_POSB is valid.

See Also
