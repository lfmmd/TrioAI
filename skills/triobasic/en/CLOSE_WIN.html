CLOSE_WIN

Type

Axis Parameter

Alternate Format

CW

Description

This parameter defines the end position of the first registration window which will be used for registration marks if windowing is specified by the REGIST command.

Value

Absolute end position of the first registration window in user units.

Example

Set a position window between 10 and 30

OPEN_WIN = 10

CLOSE_WIN = 30

See Also

OPEN_WIN, OPEN_WINB, CLOSE_WINB, REGIST
