VERIFY

Type

Axis Parameter

Syntax

VERIFY AXIS(n) = value

Description

This parameter is included in the token list to allow older programs to compile in the MC4xx/MC5xx/MC6xx Motion Coordinators. It does not have any function, but the value will store the least significant bit of the value.

On MC2xx and before, VERIFY was used to change the encoder port on a pulse count axis to count Quadrature encoder pulses. In the MC⅘/6 ranges, the same action is done by changing the axis type (ATYPE).
