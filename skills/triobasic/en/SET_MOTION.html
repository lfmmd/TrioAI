SET_MOTION

Type

System Function

Syntax

SET_MOTION(axis, speed[, accel[, decel[, jerk]]])

Description

SET_MOTION is used to pre-set the parameter list for a subsequent move using just one command.

This command requires the axis to be specified as a parameter and therefore does not use the current base axis nor will it accept the AXIS modifier.

Parameters

Minimum of the first 2 parameters must be set. All other parameters are optional.
axis:Axis number to be set up
speed:The value for SPEED to be set
accel:The required value for ACCEL
decel:The required value for DECEL
jerk:The required value for JERK

Examples

Example 1

Set up and run a move on axis 3 (setting JERK is not required, so the last parameter is not included).

SET_MOTION(3, 20, 100, 120)

BASE(3)

MOVE(150)

WAIT IDLE
