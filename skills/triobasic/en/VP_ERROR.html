VP_ERROR

Type

Axis Parameter (Read Only)

Syntax

value = VP_ERROR AXIS(n)

Description

The axis parameter VP_ERROR allows the error state of the VPU to be read.

Only the VPU axis will have its VP_ERROR updated.

Only valid for VP_MODE 6 – all other modes will only return 0.

Value

0No error present
1The VPU has not been initialised
2Bad mode change – the VPU’s mode may only be changed when idle
3Bad position demand – a position demand is only allowed when the VPU is in position mode
4Bad speed demand – speed must be greater than zero in position mode
5Bad acceleration/deceleration demand – value must be greater than zero
6Bad acceleration demand – value must be greater than zero
7Bad deceleration demand – value must be greater than zero
8Bad jerk demand – value must be greater than zero
9Bad profile time – value must be greater than zero and less than 4 ms
10Bad resume – may only resume after a pause

Example

The AXISSTATUS has bit 33 set, indicating a VP_ERROR is present, so print the error code:

PRINT VP_ERROR
