MOVECIRCSP

Type

Axis Command

Syntax

MOVECIRCSP(end1, end2, centre1, centre2, direction[, ta[, output]])

Description

Works as MOVECIRC and additionally allows vector speed to be changed when using multiple moves in the look ahead buffer when MERGE=ON, using additional parameters FORCE_SPEED and ENDMOVE_SPEED.

Optional parameters ta and output allow the use of transition curves. See MOVECIRC.

Example

In a series of buffered moves using the look ahead buffer with MERGE = ON, a circular move is required where the incoming vector speed is 40 units/second and the finishing vector speed is 20 units/second.

FORCE_SPEED = 40

ENDMOVE_SPEED = 20

MOVECIRCSP(100, 100, 0, 100, 1)
