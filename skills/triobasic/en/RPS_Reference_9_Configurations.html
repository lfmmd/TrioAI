Configurations

The system can solve the kinematics for a particular position and orientation of the TCP in up to eight different axis configurations. The different configurations can be specified by the embedded motion parameter ‘C’.

SCARA

There are 2 configurations for Scara robots: right-handed and left-handed.
ConfigurationMeaning
0Left-handed
1Right-handed

6 DOF Robot

The embedded ‘C’ parameter can be only used in MOVEJ instruction with values from 0 to 7.

C := 0 – Wrist no flip, elbow above, base forward

C := 1 – Wrist flip, elbow above, base forward

C := 2 – Wrist no flip, elbow below, base forward

C := 3 – Wrist flip, elbow below, base forward

C := 4 – Wrist no flip, elbow below, base backward

C := 5 – Wrist flip, elbow below, base backward

C := 6 – Wrist no flip, elbow above, base backward

C := 7 – Wrist flip, elbow above, base backward
