CANIO_ENABLE

Type

I/O Parameter

Description

CANIO_ENABLE enables the Trio CAN I/O or CANopen protocol.

When using the Trio I/O protocol it is set automatically by the firmware. You must set CANIO_ENABLE = ON manually after configuring CANopen I/O.

Value

ONEnable the CAN protocol (default when CANIO_ADDRESS = 32)
OFFDisable the CAN protocol (default when CANIO_ADDRESS <> 32)
