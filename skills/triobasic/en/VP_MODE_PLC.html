VP_MODE_PLC

Type

System Parameter (MC_CONFIG)

Syntax

VP_MODE_PLC = value

Description

Writing to this parameter allows the selection of the most suitable velocity profiling mode for your PLC (IEC 61131-3) application. The default value is 6 as this includes JERK and other axis parameters suitable for the PLCopen Motion Control (MC) function blocks.

VP_MODE_PLC is an MC_CONFIG keyword in all Motion Coordinators that support PLCopen.

VP_MODE 5 or 6 is required for PLCopen Motion function blocks with prefix “MC”_

Value

0Trapezoidal multi-axis interpolation
5Jerk limited multi-axis interpolation
6Jerk limited multi-axis interpolation (default)

For firmware versions prior to 2.0339, the default value is 5.

Example

The IEC 61131-3 PLC program is to use the Trio TC_ motion functions using the traditional velocity profile mode (VP_MODE) 0. In MC_CONFIG, set the VP_MODE_PLC to 0 so that the required mode is set up at power-on.

VP_MODE_PLC = 0
