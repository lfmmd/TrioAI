MOVEL

Type

Robotics Command

Syntax

MOVEL target [S:= T:= O:= ZT:= ZF:= A:= D:=]

Description

MOVEL is used to move the robot from one point to another along a linear path. All axes reach the destination position at the same time.

KINEMATIC_GROUP requires the RPS runtime FEC

Parameters

ParameterData typeDescription
TargetTARGETCartesian target that the robot is moving to.
NOTE: joint target is not suitable for this move. Error ‘Invalid mix of data type’ will be generated if joint target is used instead.
Embedded parameters:See RPS Move Commands for details

Examples

Example 1

GTA(0) = 350, 0, 50, 180, 0, 0, "p1"

GTA(1) = 400, 100, 20, 180, 0, 0, "p2"

GTA(2) = 450, -300, 60, 180, 0, 0, "p3"

OBJECT_FRAME(1, "of1", -10, 0, 0, 0, 0, 15)

TOOL_OFFSET(1, "tool1", 0, 0, 50, 0, 0, 0)

ROBOT(0)

MOVEL GTA("p1")

MOVEL GTA("p2")

MOVEL GTA("p3")

STOP

This example commands Robot 0 to Move linearly to target point 0. Note that there are no other embedded instructions in the MOVE. This instruction will MOVE the robot to target point 1 using the following parameters.

MOVEL GTA("p1") Z:=67
