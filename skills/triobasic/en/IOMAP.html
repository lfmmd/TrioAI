IOMAP

Type

I/O Command (command line only)

Syntax

IOMAP[(mode)]

Description

Lists the current digital I/O map, analogue I/O map or full I/O map.

Parameters

mode0Show digital I/O only (Default)
1Show analogue I/O only
2Show digital and analogue I/O

Example1

>> IOMAP

Digital Input map :

     0-   7 : Built-in Inputs

     8-  15 : Built-in Bi-Directional IO

    16-  31 : CAN P318 @ Address 0 (fw=v1.3.0)

    32-1023 : Virtual



Digital Output map :

     0-   7 : Virtual

     8-  15 : Built-in Bi-Directional IO

    16-  31 : CAN P327 @ Address 0 (fw=v1.3.0)

    32-1023 : Virtual

Example2

Show analogue I/O connected using CANopen

>>IOMAP(1)

Analogue Input map :

     0-   3 : CANopen @ CAN Buffer 20

     4-   7 : CANopen @ CAN Buffer 21

     8-  11 : CANopen @ CAN Buffer 23

    12-  15 : CANopen @ CAN Buffer 24

    16-  19 : CANopen @ CAN Buffer 26

    20-  23 : CANopen @ CAN Buffer 27

    24-  27 : CANopen @ CAN Buffer 29

    28-  31 : CANopen @ CAN Buffer 30

         32 : Built-in @ channel 0 [12-bit, unipolar, 10V]

         33 : Built-in @ channel 1 [12-bit, unipolar, 10V]

    34-  63 : Reserved



Analogue Output map :

     0-   3 : CANopen @ CAN Buffer 22

     4-   7 : CANopen @ CAN Buffer 25

     8-  11 : CANopen @ CAN Buffer 28

    12-  15 : CANopen @ CAN Buffer 31

    16-  63 : Reserved
