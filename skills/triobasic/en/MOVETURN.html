MOVETURN

Type

Axis Command

Syntax

MOVETURN(mode, axes, x_start, y_start, x_end, y_end, sp_start, sp_end[, acc[, x_po[, y_po[, sp_lim]]]])

MOVETURN(mode, axes, x_start, y_start, z_start, x_end, y_end, z_end[, sp_start[, x_po[, y_po[, sp_lim]]]])

Parameters

mode:0
1, 2
3
4Equivalent of mode 1, but for 3 axes
parameters:Varies based on mode (see below)

mode = 0

MOVETURN performs an exterior change of direction.

MOVETURN(mode, axes, x_start, y_start, x_end, y_end, sp_start, sp_end[, acc])

Parameters

mode:0
axes:Number of axes (must be 2 for this mode)
x_start:Entry vector x component
y_start:Entry vector y component
x_end:Exit vector x component
y_end:Exit vector y component
sp_start:Entry speed of the MOVETURN
sp_end:Exit speed of the MOVETURN
acc:Acceleration/deceleration rate to use (Optional)

mode = 1, 2

MOVETURN performs an interior change of direction.

MOVETURN(mode, axes, x_start, y_start, x_end, y_end, sp_start, sp_end[, acc])

Parameters

mode:1
axes:Number of axes (must be 2 for this mode)
x_start:Entry vector x component
y_start:Entry vector y component
x_end:Exit vector x component
y_end:Exit vector y component
sp_start:Not used (supply 0)
sp_end:Not used (supply 0)
acc:Not used (supply 0)

mode = 3

MOVETURN performs an ? change of direction.

MOVETURN(mode, axes, x_start, y_start, x_end, y_end, sp_start, sp_end[, acc[, x_po[, y_po[, sp_lim]]]])

Parameters

mode:3
axes:Number of axes (must be 2 for this mode)
x_start:Entry vector x component
y_start:Entry vector y component
x_end:Exit vector x component
y_end:Exit vector y component
sp_start:Not used (supply 0)
sp_end:Not used (supply 0)
acc:Acceleration/deceleration rate to use (Optional)
x_po:Position offset from start to end, x component (Optional)
y_po:Position offset from start to end, y component (Optional)
sp_lim:Speed limit (Optional)

mode = 4

Performs 3 axis equivalent of mode 1.

MOVETURN(mode, axes, x_start, y_start, z_start, x_end, y_end, z_end[, sp_start[, x_po[, y_po[, sp_lim]]]])

Parameters

mode:4
axes:Number of axes (must be 3 for this mode)
x_start:Entry vector x component
y_start:Entry vector y component
z_start:Entry vector z component
x_end:Exit vector x component
y_end:Exit vector y component
z_end:Exit vector z component
sp_start:Entry speed of the MOVETURN
x_po:Position offset from start to end, x component (Optional)
y_po:Position offset from start to end, y component (Optional)
sp_lim:Speed limit (Optional)
