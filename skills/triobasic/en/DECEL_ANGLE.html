DECEL_ANGLE

Type

Axis Parameter

Description

This parameter is used with CORNER_MODE, it defines the maximum change in direction of a 2-axis interpolated move that will be merged at full speed. When the change in direction (see CHANGE_DIR_LAST) between two moves is greater than DECEL_ANGLE, the speed will be proportionally reduced using the following relationship:

speed = FORCE_SPEED * (angle - DECEL_ANGLE) / (STOP_ANGLE - DECEL_ANGLE)

When a move needs to slow down, its STARTMOVE_SPEED is set to the above value.

Value

The angle (in radians) beyond which the speed is reduced.

Must be greater than or equal to zero (default value is zero) and should be less than STOP_ANGLE otherwise it will be ignored (motion will stop first).

Example

Decelerate to a slower speed when the transition is between 15 and 45 degrees.

CORNER_MODE.1 = 1

DECEL_ANGLE = 15 * PI / 180

STOP_ANGLE = 45 * PI / 180
