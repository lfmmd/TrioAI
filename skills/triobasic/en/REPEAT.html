REPEAT .. UNTIL

Type

Program Structure

Syntax

REPEAT

  commands

UNTIL expression

Description

The REPEAT..UNTIL construct allows a block of commands to be continuously repeated until an expression becomes TRUE. REPEAT..UNTIL loops can be nested without limit.

The commands inside a REPEAT..UNTIL loop will always be executed at least once, but if you want them to only be executed when the expression if TRUE, you can use a WHILE..WEND loop instead.

Parameters

expression:Any valid TrioBASIC expression
commands:TrioBASIC statements that you wish to execute

Example

A conveyor is to index 100 mm at a speed of 1000 mm/s wait for 0.5 s and then repeat the cycle until an external counter signals it should stop by setting input 4 on.

SPEED = 1000

REPEAT

  MOVE(100)

  WAIT IDLE

  WA(500)

UNTIL IN(4) = ON
