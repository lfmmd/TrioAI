FWD_START

Type

Axis Parameter

Description

Force an immediate forward jog motion at JOGSPEED (or SPEED if the FAST_JOG input is assigned and active).

When FWD_JOG is -1 and FWD_START is set to 1, the axis will run the forward jog (MTYPE 14) on the axis.

If FWD_JOG is set to an input number (not equal to -1), then FWD_START has no function.

Note that this move is automatically loaded whenever the FWD_START is active and so cannot be cancelled using CANCEL or RAPIDSTOP, so if it is required to forcibly stop the forward jog, then either de-assign it (FWD_START = 0) or set JOGSPEED to zero.

Value

FWD_STARTFWD_JOGMTYPE
0-10 (idle)
1-114 (forward jog)
Not Equ 1-10 (idle)
AnyNot Equ -10 (idle) *

* Under control of the designated input

Example

BASE(2)

FWD_JOG = -1

REV_JOG = -1



IF IDLE THEN

  REPEAT

    SELECT_CASE VR(10)

      CASE 0

        FWD_START = 0

        REV_START = 0

      CASE 1

        FWD_START = 1

        REV_START = 0

      CASE 2

        FWD_START = 0

        REV_START = 1

    END_CASE

  UNTIL VR(10) = 99

ENDIF
