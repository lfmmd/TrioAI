RAISE_ANGLE

Type

Axis Parameter

Description

This parameter is used with CORNER_MODE, it defines the maximum change in direction of a 2-axis interpolated move before CORNER_STATE is triggered. When the change in direction is greater than the RAINSE_ANGLE, CORNER_STATE will change so the system can interact with a program.

This can be used to change the angle of a cutting knife

RAISE_ANGLE does not control the speed so it should be set equal or greater than STOP_ANGLE.

Value

The angle (in radians) to start to interact with a program through CORNER_STATE

Must be greater than or equal to zero (default value is zero), and when zero, no corner state interaction will occur.

Example

Turn on corner control and the corner state machine, and configure it to decelerate beyond 15 degrees and stop at 45 degrees. Set the knife raise angle to the stop angle so that the CORNER_STATE routine can run.

CORNER_MODE.1 = 1

CORNER_MODE.2 = 1

DECEL_ANGLE = 15 * PI / 180

STOP_ANGLE = 45 * PI / 180

RAISE_ANGLE = STOP_ANGLE
