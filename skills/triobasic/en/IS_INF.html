IS_INF

Type

Mathematical Function

Syntax

value = IS_INF(expression)

Description

Check the value of a stored variable. Tests to see if the value is a floating-point infinite number and therefore likely to cause an expression to generate a BASIC Error.

Parameters

expressionAny valid numeric expression.

Examples

Example 1

IF IS_INF(TABLE(3000)) THEN

    PRINT "TABLE(3000) is Infinity"

ENDIF

Example 2

Check all VRs in the MC403 and remove Infinite values

FOR i = 0 TO 4095

    IF IS_INF(VR(i)) THEN

        VR(i) = 0

    ENDIF

NEXT i
