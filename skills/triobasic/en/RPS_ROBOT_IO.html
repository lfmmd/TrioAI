ROBOT_IO

Type

Robotics Parameter (Read Only)

Syntax

ROBOT_IO(group)

Description

Returns the states of the kinematic group’s digital I/O as a bit pattern.

The least significant 16 bits contain the kernel’s digital input states. The most significant 16 bits contain the kernel’s digital output states.

ROBOT_IO requires the RPS runtime FEC

RPS 2.0 kernel must be activated using the RPS_KERNEL_V2 keyword to enable its use.

Value

BitCodeMeaning
0KMIKey switch manual input
1KAIKey switch automatic input
2ESIEmergency stop input
3DMIDemand input
4REIReset errors input
5SAISafety input
6DEIDrive enabled input
7MEIMaster enable input
16MPOMotor powered output
17REORobot energised output
18EROError output

Examples

Example 1

Check if robot 0 is enabled.

IF ROBOT_IO(0).7 = 1 THEN

   'Robot is enabled. Further action here.

ENDIF
