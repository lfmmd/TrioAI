ROBOT_ERROR

Type

Robotics Parameter (Read Only)

Syntax

ROBOT_ERROR(group)

Description

Returns the errors present in the RPS 2.0 kernel. There may be multiple errors present, with the value in the table below showing what each bit set represents.

ROBOT_ERROR requires the RPS runtime FEC

RPS 2.0 kernel must be activated using the RPS_KERNEL_V2 keyword to enable its use

Value

BitValueError
01Invalid I/O
12Invalid key switch
24Kinematic group changed while activated
38Kinematic group I/O changed while activated
416Kinematic group activated when not set
532Kinematic group activated when I/O not set
664Operating mode changed while not idle
7128Safety condition triggered
8256Emergency stop triggered
9512Drive not enabled

Examples

Example 1

Detect an emergency stop activation in robot 0.

IF ROBOT_ERROR(0).8 = 1 THEN

   'Emergency stop has been triggered

ENDIF
