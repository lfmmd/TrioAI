MOVE_COUNT_INC

Type

Axis Parameter

Description

MOVE_COUNT_INC sets the MOVE_COUNT to either increment or decrement every time a motion command is loaded and starts. The setting can be used when a pattern of moves is stopped and then reversed, for example in a machine following a series of points through an XY path.

Value

0MOVE_COUNT = MOVE_COUNT + 1 for each new move
1MOVE_COUNT = MOVE_COUNT - 1 for each new move
