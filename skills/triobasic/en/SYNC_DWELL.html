SYNC_DWELL

Type

Axis Parameter

Syntax

SYNC_DWELL AXIS(n) = 60 {time period in servo period units}

Description

SYNC_DWELL is one of 2 new axis parameters that allow greater control over when the sync motion occurs during the sync time period.

Value

The time period for the pause in milliseconds

Sync profile without pause or dwell. Profile velocity vs time.

Effect of SYNC_PAUSE and SYNC_DWELL.

The SYNC_PAUSE, SYNC_DWELL and SYNC_WITHDRAW parameters are often set differently on each axis of the SYNC command.
