FULL_SP_RADIUS

Type

Axis Parameter

Description

This parameter is used with CORNER_MODE, it defines the minimum radius that will be executed at full speed. When a radius to be profiled is smaller than FULL_SP_RADIUS the speed will be proportionally reduced so that VP_SPEED = FORCE_SPEED * radius / FULL_SP_RADIUS.

Value

The full speed radius in user UNITS (default = 0).

Example

In the following program, the FULL_SP_RADIUS is 6, so when the first MOVECIRCSP is reached, the speed remains at 10 because its radius of 8 is greater than the FULL_SP_RADIUS, but when the second MOVECIRCSP is reached, the speed is halved because its radius of 3 is half that of the FULL_SP_RADIUS.

CORNER_MODE = 8

MERGE = ON

SPEED = 10

FULL_SP_RADIUS = 6

DEFPOS(0, 0)



MOVESP(10, 10)

MOVESP(10, 5)

MOVESP(5, 5)

MOVECIRCSP(8, 8, 0, 8, 1)

MOVECIRCSP(3, 3, 0, 3, 1)

MOVESP(5, 5)

MOVESP(10, 5)
