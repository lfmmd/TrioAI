WORLD_ORI_MAX_JERK

Type

Robotics Parameter

Description

The WORLD_ORI_MAX_JERK axis parameter can be used to read and write the maximum possible orientation axis jerk in world mode.

JERK is the derivative of acceleration. JERK units are per second per second per second. The way JERK is applied to the axis motion depends on the value of VP_MODE.

Value

The robot axis maximum orientation acceleration in degrees per second per second per second.
