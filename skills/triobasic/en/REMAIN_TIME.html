REMAIN_TIME

Type

Axis Parameter (Read Only)

Description

This is the amount of time it will take to finish all the MOVES loaded in the buffer. The value can be used to turn ON/OFF outputs for example used in gluing applications.

Value

The time remaining in seconds for the current sequence of buffered moves.

Examples

Example 1

To change the speed to a slower value in the last 750 msecs of the move. A WAIT LOADED command is used to allow the motion command to be loaded onto MTYPE before checking the REMAIN_TIME.

start:

SPEED=10

ACCEL=200 ' 200 mm/s/s give 0.05 second ramp time

DECEL=200

MOVE(45) ' move takes approximately 4.5 seconds

WAIT LOADED

WAIT UNTIL REMAIN_TIME < 0.750

SPEED=1

WAIT IDLE
