SELECT

Type

System Command

Syntax

SELECT "program"[, program_type]

Description

Makes the named program the currently selected program, if the named program does not exist then it makes a program of that name. If no program type is specified, then a BASIC type will be selected.

It is not normally used except by Motion Perfect.

The SELECTed program cannot be changed when programs are running.

When a program is SELECTed any previously selected program is compiled.

When program_type is specified is to make a program of that name and type, it cannot be possible to change the type of an existing program.
