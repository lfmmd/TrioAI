IP_BIND_ADDRESS

Type

System Parameter

Description

IP_BIND_ADDRESS is used to set the Ethernet IPv4 UDP and TCPSERVER bind address for client connections. By default, when a TrioBASIC client connection is opened with the OPEN command, the local socket is bound to the source IP address, 0.0.0.0. This is a synonym for all current local IP addresses.

When a UDP or TCPSERVER client is created, the OPEN statement only specifies one port, which is used to create both the source and destination sockets. If IP_BIND_ADDRESS is 0.0.0.0, then we cannot talk to applications that are running on the same device. To avoid this situation, we set IP_BIND_ADDRESS to a specific local IP address, for example 127.0.0.1.

The default value is 0.0.0.0.

Note that this parameter is applicable to PC-MCAT only.

Value

Network IP address in dot-decimal notation, which consists of four decimal numbers separated by dots, where each number is an 8-bit value (an octet), and so has the range 0 to 255.

Examples

Example 1

IP_BIND_ADDRESS = 192.168.0.250
