DV_STATUSWORD

Type

System Command (Read Only)

Syntax

value = DV_STATUSWORD(channel)

Description

Returns the status word from a remote Flexslice EtherCAT module such as the thermocouple module and RTD module.

Parameters

value:The value read from the device variable’s status word
channel:Channel number

See Also

DV_CONTROLWORD, DV_IN, DV_OUT
