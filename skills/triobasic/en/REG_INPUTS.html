REG_INPUTS

Type

Axis Parameter

Description

Selects which of the hardware registration inputs to use for an axis. When using REGIST modes 3 to 17 the first input is the A channel and the second is the B.

It is recommended to use REGIST(20 to 22) for new projects.

The following defaults are used:

Example

Example 1

On the Flexaxis module, set registration input 2 as the first input and 7 as the second

REG_INPUTS AXIS(9) = $72

Example 2

On the MC405, set up first registration inputs as IN(3) to IN(7) for axes 0 to 5. Second channel registration inputs all on IN(0).

REG_INPUTS AXIS(0) = $03

REG_INPUTS AXIS(1) = $04

REG_INPUTS AXIS(2) = $05

REG_INPUTS AXIS(3) = $06

REG_INPUTS AXIS(4) = $07

Example 3

On the Euro404, set up first registration inputs as IN(0) to IN(3) for axes 0 to 3 and second channel inputs to be IN(4) to IN(7).

REG_INPUTS AXIS(0) = $40

REG_INPUTS AXIS(1) = $51

REG_INPUTS AXIS(2) = $62

REG_INPUTS AXIS(3) = $73
