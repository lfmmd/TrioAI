DRIVE_CONTROL

Type

Axis Parameter

Syntax

DRIVE_CONTROL AXIS(n) = parameter_value



return_value = DRIVE_CONTROL AXIS(n)

Description

This command is for support of the Omron G5 drive over EtherCAT. The setting value identifies the object to be watched by DRIVE_MONITOR at runtime every servo period.

For this drive, a fixed mapping is used to implement DRIVE_MONITOR and DRIVE_CONTROL.

Example

To read the actual following error in the drive, one would do the following:

DRIVE_CONTROL = 5

fe_actual = DRIVE_MONITOR
