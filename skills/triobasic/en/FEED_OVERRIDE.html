FEED_OVERRIDE

Type

Axis Parameter

Description

This parameter can be used to override the feed rate when using an embedded speed profile such as MOVECONTOUR.

The default value is 100%, and a value between 0% and 150% can be set.

The path speed (feed-rate) is ramped up or down to the new value.

Value

100Run at the speed set by the embedded profile parameters.
0-150Run at a percentage of the set speed.

Example

Use an analogue input to control the feed rate from a manual control.

BASE(3) 'Control the feed rate on moves with base axis 3

WHILE TRUE

    feed_rate = AIN(1) * 150 / 2048

    IF feed_rate < 0 THEN feed_rate = 0

    FEED_OVERRIDE = feed_rate

    WA(100)

WEND
