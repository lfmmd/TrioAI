NOT

Type

Logical and Bitwise operator

Syntax

NOT expression

Description

The NOT operator inverts all the bits of the supplied integer value.

When supplied with a floating-point value, this is first truncated to an integer value.

Be wary when using NOT with bit values 0 and 1, as while NOT 0 will work as expected (returning TRUE (because NOT 0 is $FFFFFFFF (-1 in decimal)), NOT 1 will not return the expected 0 (FALSE), instead returning -2 (which is not FALSE) , because NOT 1 is $FFFFFFFE in hexadecimal (-2 in decimal).

Parameter

expression:Any valid TrioBASIC expression.

Examples

Example 1

Perform a bitwise conjunction (AND) of the value 7 with the bitwise inversion (NOT) of 1.5. The value 1.5 is first truncated to 1 prior to bitwise inversion, followed by a bitwise conjunction with 7 (in binary, using 8 bits for brevity, evaluates 00000111 AND 11111110, which equals 00000110).

>>PRINT 7 AND NOT 1.5

6

Example 2

If a function fails, then print an error message and stop the program

IF NOT CAN(0, 9, 13, 1, 8, $6060, 0, $02) THEN

  PRINT #user, "Failed to set velocity mode"

  STOP

ENDIF
