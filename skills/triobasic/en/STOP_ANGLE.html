STOP_ANGLE

Type

Axis Parameter

Description

This parameter is used with CORNER_MODE, it defines the maximum change in direction of a 2-axis interpolated move that will be merged at speed. When the change in direction (see CHANGE_DIR_LAST) between two moves is greater than the STOP_ANGLE, the speed is reduced to zero.

When a move needs to stop its STARTMOVE_SPEED is set to zero.

Value

The angle (in radians) beyond which motion momentarily comes to rest at the move junction.

Must be greater than or equal to zero (default value is zero), and when zero, no stop points will be detected.

Example

Reduce the speed to zero on any move transition greater than 25 degrees.

CORNER_MODE.1 = 1

STOP_ANGLE = 25 * PI / 180
