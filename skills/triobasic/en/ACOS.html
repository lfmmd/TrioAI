ACOS

Type

Mathematical Function

Syntax

value = ACOS(expression)

Description

Returns the arc-cosine (also known as inverse cosine) of a number in the range -1 to +1, the result being an angle measured in radians in the range 0 to π (see PI).

Parameters

value:The angle measured in radians
expression:Any valid TrioBASIC numeric expression with a value in the range -1 to +1

Example

Print the arc-cosine of -1 on the command line

>>PRINT ACOS(-1)

3.1416

>>
