ASC

Type

String Function

Syntax

value = ASC(string[, offset])

Description

ASC returns the ASCII value of the first character in the provided string parameter. If the string is empty, then 0 will be returned.

When the optional second parameter is used, this provides an offset from the start of the string, so an offset of 0 will return the ASCII code of the first character, 1 the second character, 2 the third character, etc. An offset outside of the range of characters in the string will return 0.

This function gets its name because it used to deal exclusively with characters encoded in ASCII. However, TrioBASIC strings use Unicode characters encoded in UTF-8, and so the value returned by ASC will be the Unicode value of the character.

Parameters:
string:Any valid STRING
offset:An integer value (default 0)

Examples

Example 1

Print the ASCII value of character ‘A’ contained within a longer string.

>>PRINT ASC("ABCDEF")

65

>>

Example 2

Find the position of the decimal point in a numerical string.

DIM a AS STRING(20)

DIM i, x, dp_pos AS INTEGER



a = "112.234"

FOR i = 0 TO LEN(a) - 1

  x = ASC(a, i)

  IF x = $2E THEN

    dp_pos = i

  ENDIF

NEXT i

PRINT dp_pos
