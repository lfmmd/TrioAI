MOVE_PA_IDLE

Type

Axis Parameter

Description

This and MOVE_PB_IDLE are two axis parameters which can be freely set by BASIC. When there are no moves available and the axis has MTYPE=0, the value of MOVE_PA_IDLE is used instead of MOVE_PA to perform an action according to MOVE_PA_CONT.

Using control value =3 the output settings set by the MOVE_PA_IDLE will be set if the axis is IDLE or if the VP_SPEED=0.

Parameters

MOVE_PA_CONT = 1 or 3

Bits 0…11:Set the output number to be switched
Bit 12:Sets ON/OFF

MOVE_PA_CONT = 2

Bits 0…31:The value in MOVE_PA is placed in AXIS_DEBUG_A synchronised with the loading of the move.

MOVE_PA_CONT = 4

Bits 0…11:Set the first output index
Bits 12…19:The number of outputs to be set/cleared
Bits 20…51:The output bit pattern

Mode 4 is available in firmware V2.0284 and later

MOVE_PA_CONT = 5 or 6

Bits 0…31:Analogue 32-bit value
Bits 32…39:Analogue channel number

Modes 5 and 6 are available in firmware v2.0339 and later

Examples

Example 1

Switching output number 10 during a move by using the MOVE_PA_IDLE command and control value = 1:

FOR a = 0 TO 1

    UNITS AXIS(a) = 10

NEXT a



SPEED = 10

MERGE = ON

LIMIT_BUFFERED = 16

ACC(SPEED * 20)

TRIGGER



MOVE_PA_CONT = 1



MOVE_PA = 10

MOVE_PA.12 = ON

MOVE_PA_IDLE = 10

MOVE_PA_IDLE.12 = OFF



MOVE(20)

WAIT IDLE

WA(2000)

MOVE(20)

WAIT IDLE

WA(2000)

MOVE(10)

Example 2

Switching output number 10 during a move by using the MOVE_PA_IDLE command and control value = 3. When the move stops once SPEED=0 is executed, the speed must be reset through the command line for the program to continue:

FOR a = 0 TO 1

    UNITS AXIS(a) = 10

NEXT a



SPEED = 10

MERGE = ON

LIMIT_BUFFERED = 16

ACC(SPEED * 20)

TRIGGER



MOVE_PA_CONT = 1



MOVE_PA = 10

MOVE_PA.12 = ON

MOVE_PA_IDLE = 10

MOVE_PA_IDLE.12 = OFF



MOVE(20)

WAIT IDLE

WA(2000)

MOVE(20)

WA(2000)

SPEED = 0

MOVE(20)

WAIT IDLE

WA(2000)

MOVE(10)
