DRIVE_INTERFACE

Type

Axis Command

Syntax

DRIVE_INTERFACE(slot, function)

Description

DRIVE_INTERFACE allows specific operations on digital drive networks. Currently this is supports only Panasonic RTEX drives.

Parameters

slot:The slot number to apply the function to
function:0Restart the digital drive network

Example

Restart the network if not all the axes have been detected

IF ATYPE AXIS(0) <> 50 THEN

  DRIVE_INTERFACE(0,0)

ENDIF
