DRIVE_NEG_TORQUE

Type

Axis Parameter

Description

Sets the negative torque limit value in a remote drive. For this value to be active, the cyclic data transfer from the drive must be first configured to set the NEGATIVE_TORQUE_LIMIT value. For a drive connected by CANopen over EtherCAT (CoE) the value will be configured as part of the Process Data Object (PDO).

Usually, the parameter is mapped to Object $60E1 sub index $00, size 16 bits.

Value

The drive negative torque limit is an INT16 in drive units, often it is percent x 10. See the drive’s manual for details.

Example

Example1

Set the drive’s negative torque limit to 50% from a program.

DRIVE_NEG_TORQUE AXIS(2) = 500
