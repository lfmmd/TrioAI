DEL

Type

System Command

Syntax

DEL "filename"[, delete_source]

Alternate Format

RM "filename"

Description

Deletes the specified file from the controller’s memory.

This command cannot be used from within Motion Perfect when in Sync Mode or Tool Mode.

Parameters

filename:the name of the file to be deleted
delete_source:0Delete the program’s compiled code only, leaving its source code on the controller
1 (default)Delete both the program’s source and compiled code

Example

Delete an old program

>>DEL "oldprog"

OK

>>
