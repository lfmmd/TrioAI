INT

Type

Mathematical Function

Syntax

value = INT(expression)

Description

The INT function returns the integer part of a number by truncating the fractional part towards zero.

To convert a floating-point value to the nearest whole number, use the ROUND function, but note that this returns a floating-point value, so use INT to cast the result of ROUND to integer, if required

Parameters

expression:Any valid TrioBASIC expression.
value:The integer part of the expression

Examples

Example 1

Print the integer part of a number on the command line

>>PRINT INT(1.79), INT(-1.79)

1     -1

Example 2

Print a value to the nearest integer on the command line, either returning the result as a FLOAT or an INTEGER.

>>PRINT ROUND(1.79), INT(ROUND(1.79)), ROUND(-1.79), INT(ROUND(-1.79))

2.0000     2     -2.0000     -2
