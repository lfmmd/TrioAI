EPROM

Type

System Function

Syntax

EPROM(function)

Description

This parameter is included in the token list to allow older programs to compile in the MC4xx/MC5xx/MC6xx Motion Coordinators. It does not have a function because all programs are automatically stored in the Flash EPROM and so are never contained only in the RAM.

Example

Type the command in the Motion Perfect command-line terminal. The command has no function but it returns text strings that are compatible with previous product ranges such as the MC206X.

>>EPROM(0)
