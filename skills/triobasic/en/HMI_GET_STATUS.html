HMI_GET_STATUS

Type

System Function

Syntax

value = HMI_GET_STATUS[(<ip>)]

Description

Return the status of the given HMI Client. If the IP address is not specified then the current page for the lowest active session will be returned.

Parameters

value-1HMI Client is not connected
1HMI Client is Connected
2HMI Page is loading
3HMI Page is running
4HMI Client is in error
IPIP address of the HMI Client to which this message must be sent.

Example

Example 1

Wait for the HMI Client to initialise correctly, change to the start page and wait for the change to complete.

WAIT UNTIL HMI_GET_STATUS=3

HMI_SET_PAGE("START")

WAIT UNTIL HMI_GET_STATUS=3 AND HMI_GET_PAGE="START"

Example 2

Get the status from 2 HMIs and combine them into one VR value. Upper 16 bits for first HMI, lower 16 bits for second HMI.

hmi1 = HMI_GET_STATUS(192.168.0.231)

hmi2 = HMI_GET_STATUS(192.168.0.232)

VR(hmi_stats) = (hmi1<<16) OR hmi2
