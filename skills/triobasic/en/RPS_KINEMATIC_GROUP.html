KINEMATIC_GROUP

Type

Robotics Command

Syntax

KINEMATIC_GROUP(group[, table_offset[, kinematics, axis0, axis1, ..., axisn]])

Description

Allows the activation of RPS on the controller and defines all robot configuration data like its mechanical links, operating modes and anything else required.

The first form is used to activate and deactivate RPS, a positive table index is used to specify where the robot configuration data is located, and an index of -1 is used to deactivate the kinematic group.

The second form is used to display the kinematic group to the terminal.

KINEMATIC_GROUP requires the RPS runtime FEC

Although 8 kinematics can be initialised on a controller it may not be possible to process all 8 at a given SERVO_PERIOD. The number that can be run depends on many factors including, which kinematic is selected, drive connection method, if OBJECT_FRAME and TOOL_OFFSET are enabled and additional factory communications.

The number of axes in the group must match the number of axes used by the KINEMATICS. The axes must also be ascending order though they do not have to be contiguous. If a group is deleted RPS is disabled for all those axes involved.

Parameters

group:The group number, 0 to 7.
When used as the only KINEMATIC_GROUP parameter, it prints the KINEMATIC_GROUP, the active OBJECT_FRAME, ROBOT_FRAME and TOOL_OFFSET information to the currently selected output channel (default channel 0).
table_offset:-1 = Delete group data.
0+ = The index in the table to start storing the FRAME configuration from.
kinematics:Type of robot.
axis0:The first axis in the group.
axis1:The second axis in the group.
axisn:The last axis in the group.

The second form of the KINEMATIC_GROUP command prints the configuration of the kinematic group.

Examples

Example 1

Check if RPS is set by displaying kinematic group index 0 on the command line.

Terminal

>>kinematic_group(0)

0 [0] [19] : 10, 11, 12, 13, 14, 15 TO={0[TO_default]:0.00000, 0.00000, 0.00000, 0.00000, 0.00000, 0.00000} RF={0[RF_default]:0.00000, 0.00000, 0.00000, 0.00000, 0.00000, 0.00000} OF={0[OF_default]:0.00000, 0.00000, 0.00000, 0.00000, 0.00000, 0.00000} CO{3}

Example 2

Disable kinematic group index 0 on the command line.

>>kinematic_group(0, -1)

>>
