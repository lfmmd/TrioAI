REGIST_CONTROL

类型

轴参数

语法

REGIST_CONTROL = value

Variable = REGIST_CONTROL

描述

读取或设置注册控制寄存器中的低级位模式。REGIST_CONTROL是一个8位轴参数，它符合BASE和axis修饰符的标准用法。

参数

value:位描述
Chan A0输入(1)或z标记(0)
1未使用
2启用
3下降(1)或上升(0)
Chan B4输入(1)或z标记(0)
5未使用
6启用
7下降(1)或上升(0)

例1

使用REGIST函数20在下降沿通道B上设置注册。然后检查注册模式是否设置正确。

BASE(2)

REGIST(20, 1, 0, 1, 0)

IF REGIST_CONTROL.4 = 1 THEN PRINT "Input selected"

IF REGIST_CONTROL.6 = 1 THEN PRINT "Registration enabled"

IF REGIST_CONTROL.7 = 1 THEN PRINT "Falling edge selected"

例2

禁用注册通道，以便接收到的任何信号将不会导致标记为TRUE，并且不会存储REG_POS。

BASE(0)

REGIST_CONTROL.2 = 0 '禁用通道A注册表，但保持通道B激活
