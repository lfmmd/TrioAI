STATIC.. AS..

类型

程序变量

语法

STATIC name AS BOOLEAN

STATIC name AS FLOAT

STATIC name AS INTEGER

STATIC name AS STRING

STATIC name AS BOOLEAN[(size1 [,size2 [,size3]])]

STATIC name AS FLOAT[(size1 [,size2 [,size3]])]

STATIC name AS INTEGER[(size1 [,size2 [,size3]])]

STATIC name AS STRING([size1 [,size2 ,]]length)

描述

关键字STATIC(类似于Visual Basic的STATIC关键字)被添加，允许在函数变量被重复调用时保留它的值，也就是说，变量只会在函数第一次被调用时被初始化。

下面的代码片段演示了不同类型的变量是如何被初始化的:

FUNCTION myfunc()

  DIM f1 AS FLOAT ' FLOAT变量将在每次调用时重新初始化为0.0

  STATIC f2 AS FLOAT ' FLOAT变量将保持其值

  DIM array1 AS INTEGER(5,10) ' Array将在每次调用时重新初始化为0

  STATIC array2 AS INTEGER(100) 'Array将保留其值

...

ENDFUNC

仅在函数库中使用

变量类型

BOOLEAN1位二进制值(TRUE或FALSE)
FLOAT64位浮点数(默认)
INTEGER64位带符号整数值
STRINGASCII文本

例

使用局部变量作为标志来跟踪机器的ok状态。

FUNCTION mysum(ival AS INTEGER) AS INTEGER

    STATIC total AS INTEGER = 1000 'total will start at 1000 and then add ival each call



    total = total + ival



    RETURN total

ENDFUNC
