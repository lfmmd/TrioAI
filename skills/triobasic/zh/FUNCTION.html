FUNCTION, RETURN, ENDFUNC

类型

函数

语法

FUNCTION name([param1 AS type[, param2 AS type[, param3 AS type …. ]]])
….

RETURN x
ENDFUNC

描述

用户定义的函数驻留在函数库中，供运动控制器多任务系统中的任何BASIC程序使用。函数至少必须有一个名称，并以ENDFUNC关键字结束。函数的内容可以接受任何BASIC命令，只要它们在所需函数操作的上下文中工作。

在固件2.0301及更高版本中，最多可以定义和使用1024个函数。旧版本的固件支持多达256个函数。

函数需要一个匹配的ENDFUNC关键字来完成函数定义。参数是可选的(最多16个)，应在括号内指定;数据类型相同的参数可以用逗号分隔。

返回参数也是可选的，应该在行尾指定。

功能只能在函数库中使用。

与标准程序文件相比，BASIC库文件支持更小的命令子集。例如，在BASIC库文件中不支持GOSUB和GOTO。

库函数可以有一个对其他库函数调用的嵌套层次结构，最大深度为16层，可以遍历这个层次结构的参数的最大数量为200。

参数

param1:传递给函数的第一个可选参数
param2:如果需要，则为第二个参数
param3:如果需要，则为第三个参数
paramN:最多可以传递16个参数

数组参数

支持将数组传递给函数，可以通过在参数数据类型后使用可选括号()来启用，注意不需要指定数组维度，例如;

FUNCTION f1(data AS INTEGER(), b AS FLOAT) AS BOOLEAN

提供数组属性是为了支持泛型数组。因此，一个函数可以被不同维度的数组重用。

<array name>.dims -指示数组维数的整数值

<array name>.dimsize(n) -指示指定维度长度的整数值

注意，数组属性可以在任何程序中使用，并不局限于BASIC库文件。

局部变量

例如，函数可以使用DIM语句声明自己的局部变量

FUNCTION myfunc AS INTEGER

DIM a, b AS INTEGER

DIM t AS TARGET

DIM x AS FLOAT(5)

… 

ENDFUNC

局部变量名可以在同一个库文件中的多个函数中重用。一个函数中命名为xx的变量与另一个函数中命名为xx的变量是不同的。

过程变量

变量可以在库文件中使用FUNCTION..ENDFUNC结构上下文之外的DIM语句声明，这些变量对库中的所有函数都是可见的，但是每个使用库函数的进程都维护自己的副本。变量中包含的数据是持久的;因此，如果一个函数改变了一个变量，那么另一个函数也会看到改变的值，但只在同一个进程中。

将数据返回给调用者

关键字RETURN并不新鲜，但在函数的上下文中表现不同。在函数中使用此命令将执行返回给调用者，但当函数定义了返回数据类型时，也用于将数据返回给调用者。在单个函数定义中允许使用多个RETURN语句。

例

例1

定义一个函数来简单地改变RS232波特率，同时将位，奇偶校验和停止位都保持恒定值。

库中的函数:

FUNCTION set_rs232(baudrate AS INTEGER)

    SETCOM(baudrate, 8, 1, 0, 1, 0)

ENDFUNC

程序中的函数调用:

set_rs232 (38400)

例2

定义一个函数来计算给定DECEL和SPEED值的最小减速距离。

库中的函数:

FUNCTION get_decel_distance(f_decel AS FLOAT, f_vel AS FLOAT) AS FLOAT

  DIM dec_dist AS FLOAT

  '使用 v^2 = u^2 + 2as计算减速距离

  dec_dist = f_vel ^ 2 / (2 * f_decel)

  RETURN dec_dist

ENDFUNC

程序中的函数调用:

BASE(2)

s_minimum = get_decel_distance(DECEL, SPEED)

例3

如果轴号在0到12之间，设置轴参数并返回-1 (TRUE)。否则，返回0。

FUNCTION init_axis(ax AS INTEGER) AS INTEGER

  IF ax > 12 THEN

    RETURN 0

  ELSE

    UNITS AXIS(ax) = 200

    SPEED AXIS(ax) = 50

    ACCEL AXIS(ax) = 100

    DECEL AXIS(ax) = 100

    RETURN -1

  ENDIF

ENDFUNC

例4

扫描一个数组并返回最大条目的值。

FUNCTION max_i(data AS INTEGER()) AS INTEGER

  DIM i, max AS INTEGER



  max = data(0)

  FOR i = 1 TO data.dimsize(0) - 1

    IF data(i) > max THEN

      max = data(i)

    ENDIF

  NEXT i

  RETURN max

ENDFUNC

例5

使用数组从函数返回多个值。

'============================================================

'函数返回数组中的多个值

' 调用: get_multi_vr(index, number_of_values, array_pointer)

'示例 获取100个VR

'   DIM my_array AS FLOAT(100)

'   get_multi_vr(200, 100, my_array)

'   PRINT my_array(54)

'============================================================

FUNCTION get_multi_vr(index, size AS INTEGER, return_array AS FLOAT())

    DIM a_index, v_index AS INTEGER

    a_index = 0

    FOR v_index = index TO index + size - 1

        return_array(a_index) = VR(v_index)

        a_index = a_index + 1

    NEXT v_index

ENDFUNC
