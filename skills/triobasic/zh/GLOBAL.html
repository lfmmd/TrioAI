GLOBAL

类型

系统命令

语法

GLOBAL ["name"[, vr_number]]

描述

全局变量可以在控制器中声明，所有程序都可以使用。GLOBAL将名称声明为对全局VR变量之一的引用。这个名称可以在包含GLOBAL定义的程序和运动控制器项目中的所有其他程序中使用。

它们必须在启动时声明。声明全局变量的程序必须是唯一在启动时运行的进程。

一旦GLOBAL被分配，它就不能被更改，即使您更改了分配它的程序。

型号全局变量的最大数目
MC4641024
MC403(Z)、MC405、Euro404、Euro408、MC4N-ECAT、MC4N-RTEX4096
MC664(X)、MC508、Flex-6 Nano、MC6N-ECAT8192

在开发过程中，您可能希望清除或更改GLOBAL。您可以通过单独使用第一个参数来清除单个GLOBAL。所有GLOBALs都可以通过发出GLOBAL来清除。您可以使用LIST_GLOBAL查看所有GLOBALS。

当程序运行时，将在GLOBAL和CONSTANT表中搜索已定义变量的名称。如果预定义了大量的GLOBAL/CONSTANT名称，这可能需要几秒钟的时间。当使用GLOBAL和CONSTANT系统时，不建议依赖于临时运行然后停止程序。最好在机器启动时运行所有必需的程序，并使它们处于运行状态。

参数

name:任何用户定义的包含小写字母、数字或下划线(_)字符的名称。
vr_number:要与名称关联的VR的编号。

例

例1

初始化两个GLOBALs并使用它们来调整机器参数。

GLOBAL "screw_pitch",12

GLOBAL "ratio1",534



ratio1 = 3.56

screw_pitch = 23.0

PRINT screw_pitch, ratio1

例2

初始化3个GLOBALs，删除其中一个GLOBALs，然后删除所有GLOBALs。

>>GLOBAL "pitch",12

>>GLOBAL "roll",534

>>GLOBAL "yaw",922

>>LIST_GLOBAL ' Display GLOBALs

Global                           VR

----------------                 ----

pitch                            12

roll                             534

yaw                              922

Constant                         Value

----------------                 -------





>>GLOBAL "roll"  '删除 roll

>>LIST_GLOBAL    '展示GOBALs

Global                           VR

----------------                 ----

pitch                            12

yaw                              922

Constant                         Value

----------------                 -------





>>GLOBAL         '删除所有GLOBALs

>>LIST_GLOBAL    '展示 GLOBALs

Global                           VR

----------------                 ----

Constant                         Value

----------------                 -------

>>

例3

初始化GLOBAL和CONSTANT值的程序在上电时使用默认进程号运行。一旦完成，它就会运行主程序来启动机器操作。

'机器设置

GLOBAL "initial_speed", 10

GLOBAL "run_speed", 11

'视觉数据通过Modbus

GLOBAL "vision_x", 51

GLOBAL "vision_y", 51

GLOBAL "vision_r", 52

GLOBAL "vision_status", 53



CONSTANT "x_axis", 0

CONSTANT "y_axis", 1

CONSTANT "v_status_ok", 8

CONSTANT "v_status_error", 4



RUN "MAIN", 2 ' 运行进程#2上的主程序
