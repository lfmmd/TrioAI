SYNC_TO_OBJECT_FRAME

类型

机器人命令

语法

SYNC_TO_OBJECT_FRAME(control, time[, object frame, GTA])

描述

机器人TCP可以与移动的物体或另一台机器或机器人同步。这意味着机器人TCP将以与同步对象相同的方向移动或旋转，并在其上进行绝对运动。

同步有两种类型:

  • 对象框架同步
  • GTA同步

对于对象框架同步，在机器人能够同步之前，必须首先将对象框架附加到至少一个轴上。

由于同步和附着过程不会被加载到移动缓冲区中，因此不会被CANCEL或RAPIDSTOP命令取消。为此，必须执行去同步或分离过程。当达到软件或硬件限制时，同步和连接立即停止，不减速。

参数

参数数据类型描述
controlInteger1 =启动同步
4 =停止同步。
10 =重新同步到另一个附加对象框架
timeInteger以毫秒为单位完成同步的时间。
(optional) object frameInteger/String对象框架已经附加。
值=对象框架的索引或名称。
不需要控制值4(停止同步)。
(optional) GTAInteger/String与GTA相关的对象框架同步。
值= GTA的索引或名称，或-1表示没有GTA。
不需要控制值4(停止同步)。
