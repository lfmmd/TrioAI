AIN

类型

系统命令

语法

AIN(channel)

描述

从模拟量输入中读取一个值。模拟量输入由运动控制器内置输入、CAN模拟量模块或EtherCAT模拟量输入模块产生。

返回值是从模拟量数字量（A到D）转换器读取的二进制数的十进制值。

内置模拟量输入和EtherCAT模拟量输入在每个伺服周期内被更新，由SERVO_PERIOD设置。

在标准模式下，CAN模拟量输入每10ms更新，在快速模式下，更新速率为2ms但是这限制了CANbus网络上的最大输入/输出数量。

参数

  • 运动协调器固件版本为 2.0252 及以下。
通道模拟量输入通道号0...35
0 - 31CAN 模拟量输入通道号
32 - 35内置模拟量输入通道号

如果没有安装CAN模拟量模块，AIN(0) 和AIN(1)将读取前两个内置通道以保持与以前版本的兼容性。

  • 固件版本 2.0252 至 2.0267。
通道模拟量输入通道号0...63
0 - 63CAN/EtherCAT模拟量输入通道号
  • 固件版本 2.0268 及以上。
通道模拟量输入通道号0...127
0 - 127CAN/EtherCAT模拟量输入通道号

例

物料匀速从辊子上发出。有一个超声波高度传感器，当辊子空时返回4V，当辊子满时返回0V。BASIC中写入一个lazy loop来控制滚转速度。

MOVE(-5000)

REPEAT

  a = AIN(1)

  IF a < 0 THEN a = 0

  SPEED = a * 0.25

  WA(10)

UNTIL MTYPE = 0

检查模拟量输入值以确保它大于0，即使它始终应该是正的。这是为了允许输入信号上的任何噪声，这些噪声可能使得值为负并导致错误，因为负速度对除 FORWARD 或 REVERSE之外的任何移动类型都无效。
