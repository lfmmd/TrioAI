MID

类型

字符串函数

语法

value = MID(string, start[, length])

描述

使用指定的可选长度返回指定字符串的中间部分，或者在未指定时默认为字符串的剩余部分。

参数

string:要使用的字符串
start字符串的起始索引
length:要返回的字符串长度，如果未指定则使用字符串的剩余部分

例

例1

预先定义一个字符串类型的变量，然后打印从索引5到10的字符

DIM str1 AS STRING(32)

str1 = "TRIO MOTION TECHNOLOGY"

PRINT MID(str1, 5, 6)

例2

将字符串传输到VR数组，以便每个VR保存一个字节。使用VRSTRING打印结果。

DIM str1 AS STRING(32)

DIM vr_start AS INTEGER

DIM string_length AS INTEGER

DIM i AS INTEGER



str1 = "My cat is black"



vr_start = 10

string_length = LEN(str1)



FOR i = 0 TO string_length - 1

    VR(vr_start + i) = MID(str1, i, 1)

NEXT i

VR(vr_start + string_length) = 0 '终止字节



PRINT VRSTRING(vr_start)
