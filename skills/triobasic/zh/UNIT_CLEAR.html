UNIT_CLEAR

类型

系统命令

描述

清除UNIT_ERROR系统参数中的所有位。该命令通常用于重置由EtherCAT引发的SYSTEM_ERROR或重置MC664选项模块的模块错误。

当出现UNIT_ERROR时，带有LCD显示屏的运动控制器将显示UNIT_ERROR。 UNIT_CLEAR可以用来重置显示。

值

该命令不带值

例

例1

清除UNIT_ERROR位，然后检查哪个模块或多个模块可能出错。

UNIT_CLEAR

WA(10)

PRINT UNIT_ERROR[0]

例2

重置MC6N-ECAT中的SYSTEM_ERROR标志，作为错误恢复例程的一部分。

IF SYSTEM_ERROR <> 0 THEN

  UNIT_CLEAR ' 尝试清除错误

  WA(50)

  IF SYSTEM_ERROR <> 0 THEN

    PRINT "Error bits cannot be cleared: ";HEX(SYSTEM_ERROR)

  ENDIF

ENDIF
