CORNER_MODE

类型

轴参数

描述

允许程序控制拐角动作。

自动拐角速度控制使系统能够根据减速角和停止角降低速度

CORNER_STATE状态机允许与TrioBASIC程序交互，并根据RAISE_ANGLE加载缓冲动作

自动半径速度控制使系统能够根据其松紧度降低半径周围的速度。这可以通过两种方式实现，一种是将速度分解为转角半径与FULL_SP_RADIUS的比率，另一种是限制向心加速度(v2 / r)，这可以将向心加速度限制在FULL_SP_RADIUS或ACCEL。第一种方法是为了向后兼容，因为由于半径限制，它不能准确地建模力，而向心模式可以做到这一点。

您可以启用任何速度控制位的组合，但只有在设置了第3位时，第4位才有效，而只有在设置了第4位时，第5位才有效。

当第4位和第5位被设置时(向心加速限制为ACCEL的向心模式)，FULL_SP_RADIUS的值不被使用，可以保留其默认值零。

值

16位值，每一位代表一个不同的拐角模式。
位描述值
0保留1
1自动拐角速度控制2
2启用CORNER_STATE状态机4
3自动半径速度控制8
4向心
模式		0 =使用移动半径与FULL_SP_RADIUS的比率来控制速度
1 =使用向心加速度的速度控制
忽略，除非设置了位3		16
5使用ACCEL0 =将向心加速度限制在FULL_SP_RADIUS处
1 =限制向心加速度不超过ACCEL
忽略，除非设置了3位和4位		32

例

例1

开启拐角状态机和自动转角速度控制。

CORNER_MODE = 2 + 4

例2

执行与例1相同的操作，但使用位操作符。

CORNER_MODE.1 = ON

CORNER_MODE.2 = ON

例3

使用移动半径与FULL_SP_RADIUS的简单比率来启用拐角半径速度控制。

CORNER_MODE.3 = ON

CORNER_MODE.4 = OFF

示例4

通过限制向心加速度不大于FULL_SP_RADIUS，启用拐角半径速度控制。

CORNER_MODE。3 = on

CORNER_MODE。4 = on

CORNER_MODE。5 = off

例5

通过限制向心加速度不大于ACCEL(不使用FULL_SP_RADIUS)来启用拐角半径速度控制。

CORNER_MODE.3 = ON

CORNER_MODE.4 = ON

CORNER_MODE.5 = OFF
