CO_READ

类型

系统命令

语法

CO_READ(slot, address, index, subindex, type[, vr_number])

描述

这个函数从远程驱动器或I/O设备获取一个CANopen-over-EtherCAT (CoE)对象。对象的索引和子索引用于请求一个值，该值要么放置在VR中，要么在Motion Perfect终端中显示（如果VR数字被设置为-1）。

参考远程设备的手册获取可用对象的列表。如果成功返回对象值，该命令返回TRUE(-1)。否则，在请求值时出现错误的情况下，命令返回FALSE(0)。

参数

slot:EtherCAT模块槽位号。
address:远端设备在网络中的节点地址
index:CANopen对象索引
subindex:CANopen对象子索引
Type:1Boolean
2Integer 8
3Integer 16
4Integer 32
5Unsigned 8
6Unsigned 16
7Unsigned 32
8Real 32
17Real 64
21Integer 64
27Unsigned 64
vr_number:当VR值介于0和最大值之间时，结果将被存储。

(-1表示该值将被打印到终端)

例

例1

读取远程驱动的操作模式并显示到终端

>> co_read (0,1， $6061, 0,2， -1)

8

>>

例2

获取远程驱动器插补时间，对象$60C2子索引1和子索引2，并放置在VR(200)和VR(201)中。

'读取对象$60C2:01 无符号 8

CO_READ(0, 5, $60C2, 1, 5, 200)

'读取对象$60C2:02符号8

CO_READ(0, 5, $60C2, 2, 2, 201)

PRINT "Drive at node 5: "; VR(200)[0]; "x 10^"; VR(201)[0]
