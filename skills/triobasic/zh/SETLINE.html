SETLINE

类型

系统命令

语法

SETLINE "program", process, line_number, base_index, robot_index

描述

在程序的选定行上设置程序指针。

参数

Program name程序可以接受的任何名称。它必须在双引号之间指定。
Process number如果程序已经在一个进程上运行，或者该进程想要被指定。在程序尚未运行的情况下，将此值设置为-1，系统将选择一个可用的进程。
Line number将设置程序指针的行号
Base index如果要设置行的进程尚未运行，则不会选择任何BASE。该参数将设置BASE索引。将其设置为-1以保持进程BASE不变。
Robot index如果要设置行的进程尚未运行，则不会选择任何ROBOT。该参数将设置ROBOT索引。将其设置为-1以使进程ROBOT保持原样。

例

将程序指针设置在第13行，对于尚未运行的程序PICK_AND_PLACE中的ROBOT(0)，不选择BASE。

SETLINE "PICK_AND_PLACE_CUPS",-1,13,-1,0
