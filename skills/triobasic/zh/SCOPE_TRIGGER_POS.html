SCOPE_TRIGGER_POS

类型

系统参数

语法

value = SCOPE_TRIGGER_POS

描述

当SCOPE_DELAY设置为预触发或延迟触发时，SCOPE_TRIGGER_POS返回触发点的索引位置。

如果SCOPE_DELAY为负值，则发生预触发。如果捕获的点少于设置的SCOPE_DELAY，则使用SCOPE_TRIGGER_POS在TABLE中查找实际的触发器索引。

值

最近的scope触发点所在的TABLE索引。

例

例1

将Scope设置为500 ms的预触发。打印一条scope完成和触发位置消息到终端#5。

'在两个通道上捕获1000点

'预触发在TRIGGER之前捕获500点 

'和在TRIGGER之后捕获500点

SCOPE_DELAY = -500

SCOPE(ON, 1, 10001, 12000, MPOS AXIS(0), MPOS AXIS(1))

WA(320)

TRIGGER

WAIT UNTIL SCOPE_POS = 1000

PRINT #5, "Scope capture complete"

PRINT #5, "Actual Trigger at TABLE "; 10000 + SCOPE_TRIGGER_POS

这是故意不给抓取500点留下足够的时间。它展示了SCOPE_TRIGGER_POS的用法。
