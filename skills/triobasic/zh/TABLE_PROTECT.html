TABLE_PROTECT

类型

系统命令

语法

TABLE_PROTECT (first [, last/lock [, lock]])

描述

设置一个TABLE值的范围为写保护(即只读)。可以用来定义TABLE中存储设置数据的部分，这些数据不能被任何程序、数据连接或直接用户输入所更改。

TABLE_PROTECT 不适用于 Euro404/408, MC403, MC403-X, MC403-Z, MC404-Z, MC405, MC405-Y 和 P157。

参数

first:第一个被锁定的TABLE地址
last:最后一个被锁定的TABLE地址
lock:锁定/解锁 TABLE值为只读
0允许读写(默认)
1只读

例1

将TABLE值10设置为终端中的只读值:

>>TABLE(10, 222) 

>>?TABLE(10)

222.0000

>>TABLE_PROTECT(10, 1)

>>?TABLE(10)         

222.0000

>>TABLE(10, 111)      

%[COMMAND 224] TABLE location write protected  ((10))

例2

设置TABLE值从10到20为只读值，然后在终端检查:

'程序设置保护区域

TABLE_PROTECT(10,20,1)

在Motion Perfect终端0中，检查它是否为只读

>>?TABLE(10)         

222.0000

>>?TABLE(13)         

125.0000



>>TABLE(13, 111)      

%[COMMAND 224] TABLE location write protected  ((13))

>>TABLE(16, 111)      

%[COMMAND 224] TABLE location write protected  ((16))
