DRIVE_PARAMETER

类型

轴参数

语法

DRIVE_PARAMETER AXIS(n) = value



value = DRIVE_PARAMETER AXIS(n)

描述

该参数是一个通用名称，用于允许通过EtherCAT PDO访问某些命名对象。DRIVE_PARAMETER可以通过使用EC_EXTEND EtherCAT扩展文件映射到TxPDO或RxPDO对象。

EC_EXTEND RxPDO包含TARGET_MAX_CURRENT

这里显示的PDO中的参数将映射到DRIVE_PARAMETER。

EC_EXTEND SDO初始化列表。(Pre-Op -> Safe-Op)

TARGET_MAX_CURRENT是PDO中的第3个对象，将映射到驱动器对象0x6073，子索引0x00，如图所示。
