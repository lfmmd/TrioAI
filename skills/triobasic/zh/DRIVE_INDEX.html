DRIVE_INDEX

类型

轴参数

语法

DRIVE_INDEX AXIS(n) = value

描述

DRIVE_INDEX用于将EtherCAT伺服驱动器中的其他PDO参数映射到VR变量中。 给定的值是映射的基准VR地址。 非标准PDO参数在每个VR中被映射，从标准对象之后的第一个PDO参数开始。

这个轴参数可以添加到MC_CONFIG中。

在使用此功能之前，必须为EtherCAT驱动器配置特定于应用程序的配置文件。

参数

value:PDO数据被映射的VR地址索引

例

例1

在PDO报文中循环地向驱动器传输应用程序数据。EtherCAT轴已预先配置为可在驱动器中运行的特殊应用软件。

DRIVE_INDEX = 100



'获取传入循环数据

user_status_1 = VR(100)

user_status_2 = VR(101)



'设置输出数据

VR(102) = user_control_word

VR(103) = winder_mode

VR(104) = ref_value_1

VR(105) = ref_value_2

VR(106) = correction_value

VR(107) = program_state
