NW_NODE_INDEX

类型

系统数组(MC_CONFIG)

语法

NW_NODE_INDEX(slot, alias_address) = value

描述

这个2D数组可用于将指针设置为EtherCAT节点使用的VR块。它可用于定义来自不同数据源(包括EtherCAT节点中的布尔和整型数据)的特定于用户的输入输出映射。

每个PDO对象都映射了一个VR，从从站到主站的值开始(例如从站实际值，DIN，状态字，实际位置等)，然后从主站到从站的值(例如从站目标值，DOUT，控制字，目标位置等)。

参数

slot:EtherCAT端口槽位号。(MC4N、MC6N和Flex-6 Nano上为0)
alias_address:已配置的从站设备别名地址。
value:0 ~ 65535EtherCAT循环数据被映射到从该VR索引开始的VR块。(MC664、MC6N、Flex-6 Nano)
0 ~ 4095EtherCAT循环数据被映射到从这个VR索引开始的VR块。(MC4N)

例

一个通用的EtherCAT设备连接到主机，并通过虚拟机交换其所有数据。VR的输出基准地址为50，输入基准地址为100。用户将实际地址设置为1050和1100。设备配置别名地址4000。

NW_NODE_INDEX(0, 4000) = 1000
