MHELICAL

类型

轴命令。

语法

MHELICAL(end1, end2, centre1, centre2, direction, distance3 [,mode])

替代格式

MH()

描述

执行螺旋式移动。

以这样的方式移动两个正交轴，在工具点处产生一个圆弧，同时在第三个轴上进行线性移动。前5个参数类似于MOVECIRC命令的参数。第六个参数定义同步线性移动。

多轴插值使用BASE轴的UNITS、速度、加速和减速。为了正确的XYZ矢量操作，所有3个轴的UNITS必须相同。

参数

end1:完成在BASE上的位置。
end2:在BASE数组的下一个轴上完成的位置。
centre1:在BASE轴上移动的位置。
centre2:在BASE数组中要移动的下一个轴上的位置。
direction:0圆弧沿逆时针方向插补
1弧线按顺时针方向插补
distance3:以用户为单位在BASE阵列轴的第三个轴上移动的距离
mode:0计算路径速度时，将3轴与2轴进行插内插。(螺旋路径)
1只插值前2个轴的路径速度，但移动第三个轴与其他2个轴协调。(圆形路径跟随第三个轴)

前4个距离参数根据BASE轴的当前单位转换因子进行缩放。第六个参数使用自己的轴单位。

例

例1

命令序列遵循轴为1和2的圆角矩形路径。轴3是刀具旋转，使刀具始终垂直于产品。轴3的UNITS设置使轴以度数校准。

REP_DIST AXIS(3) = 360

REP_OPTION AXIS(3) = ON '所有3个轴必须在开始前归位

MERGE = ON

MOVEABS(360) AXIS(3) '点轴3在正确的起始方向

WAIT IDLE AXIS(3)

MOVE(0, 12)

MHELICAL(3, 3, 3, 0, 1, 90)

MOVE(16, 0)

MHELICAL(3, -3, 0, -3, 1, 90)

MOVE(0, -6)

MHELICAL(-3, -3, -3, 0, 1, 90)

MOVE(-2, 0)

MHELICAL(-3, 3, 0, 3, 1, 90)

例2

PVC切割机使用的2个轴类似于xy绘图仪，第三个轴用来控制刀的切割角度。为了在切割曲线时保持x轴和y轴的合成切割速度相同，将模式1应用于螺旋命令。

BASE(0, 1, 2)

MERGE = ON           '混合移动成一个连续的动作

MOVE(50, 0)

MHELICAL(0, -6, 0, -3, 1, 180, 1)

MOVE(-22, 0)

WAIT IDLE

MOVE(-90) AXIS(2)    '停在转角后旋转刀

WAIT IDLE AXIS(2)

MOVE(0, -50)

MHELICAL(-6, 0, -3, 0, 1, 180, 1)

MOVE(0, 50)

WAIT IDLE            '再次暂停以旋转刀

MOVE(-90) AXIS(2)

WAIT IDLE AXIS(2)

MOVE(-22, 0)

MHELICAL(0, 6, 0, 3, 1, 180, 1)

WAIT IDLE
