STICK_WRITEVR

类型

系统功能

语法

value = STICK_WRITEVR(flash_file, vr_start [,length [,format]])

描述

用于将VR数据以两种格式之一存储到SD卡上。

如果这个文件已经存在，就会被覆盖。

如果您想在不丢失任何精度的情况下存储数据，请使用二进制格式。

参数

value:TRUE=函数成功
FALSE=函数不成功
flash_file:一个数字，当附加到字符“SD”将形成数据文件名。
硬编码的字符串或string变量将形成数据文件名。
vr_start:将从VRs中传输数据值的起始点。
length:要传输的VR值的数量(默认128个值)
format:0 =二进制64位浮点格式，BIN文件(默认)
1 = ASCII码逗号分隔值，文件，无索引CSV
2 = ASCII CSV文件，格式为:索引，值<cr>

当以format=0存储时，数据以IEEE浮点二进制格式little-endian存储，即最不重要的字节优先。

例

例1

将从VR(1000)开始的2000个值传输到称为SD1501.BIN的SD卡文件

success = STICK_WRITEVR (1501, 1000, 2000, 0)

例2

将从VR(1000)开始的2000个值传输到名为MYDATALOG1_VR.BIN的SD卡文件

DIM flash_file AS STRING(20)

flash_file = "MYDATALOG1_VR"

success = STICK_WRITEVR (flash_file, 1000, 2000, 0)

例3

将从VR(2080)开始的20个值传输到SD卡文件DATA1_VR.CSV

DIM flash_file AS STRING(20)

DIM s AS BOOLEAN

flash_file = "DATA1_VR"

s = STICK_WRITEVR(flash_file, 2080, 20, 2)

IF s = TRUE THEN

  PRINT flash_file; " saved OK"

ELSE

  PRINT "Error saving "; flash_file

ENDIF



FILE "Type" flash_file + ".CSV"
