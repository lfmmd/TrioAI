ROBOT_STATE

类型

机器人参数(只读)

语法

ROBOT_STATE(group)

描述

允许读取RPS 2.0内核的状态。

ROBOT_STATE需要RPS run-time FEC

RPS 2.0内核必须使用RPS_KERNEL_V2关键字激活才能使用。

值

状态名称描述
0初始化将RPS内核内部数据设置为初始值
1初始化初始值设置
2配置等待运动组及其I/O被设置
3配置运动组完全指定
4验证检查运动组是否有错误
5验证运动学组无误差
6激活运动组要求固件启动Frame变换
7运动组激活Frame变换已经开始
8等待操作模式按键开关正在采样
9工作模式设置采样完成，操作模式设置
10操作操作配置文件正在执行
11错误的仲裁器确定错误的性质并设置ERO
12轻微的错误跳转到状态“重置错误”
13主要的错误需要做紧急停止
14紧急停止中止运动和机器人程序
15监控停止等待运动停止
16停止运动已经停止
17重置错误等待REI脉冲高、低(降边触发)
18错误复位清除ERO并跳转到状态“配置”
19关闭设置输出低并跳转到状态“初始化”

例

例1

检测机器人(0)的运动是否停止。

IF ROBOT_STATE(0) = 16 THEN

   '机器人运动已停止，在此进行操作。

ENDIF
