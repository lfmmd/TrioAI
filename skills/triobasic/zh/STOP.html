STOP

类型

命令

语法

STOP "progname"[, process_number]

描述

在当前行停止一个程序。可以指定一个特定的程序名和一个可选的进程号。如果运行的程序有一个以上的实例，则需要进程号。如果没有指定名称或进程号，则当前选定的程序将被停止。

参数

progname:要停止的程序名称。
process_number:可选进程号，当程序的多个实例正在运行并且只有一个要停止时使用。

例

例1

从命令行停止一个名为“axis_init”的程序。注意，引号是可选的，除非程序名也是BASIC关键字。

>>STOP axis_init

例2

当数字输入关闭时，停止指定的程序。

IF IN(12) = OFF THEN

  STOP "hmi_handler"

  STOP "motion1"

ENDIF

例3

停止指定程序的指定实例，同时保持其他实例运行。

'让进程停止(通过HMI将其放入VR中)

proc_a = VR(45)



'停止所需的test_program实例

STOP "test_program", proc_a

HALT, SELECT, SETLINE, STEPINTO, STEPLINE, RUN
