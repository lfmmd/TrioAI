IS_ENCRYPTION_KEY

类型

系统函数

语法

value = IS_ENCRYPTION_KEY(type)

描述

系统函数，IS_ENCRYPTION_KEY返回TRUE或FALSE，表示已经在控制器中设置了USER或OEM密钥。type参数选择被请求的密钥类型。

Motion Perfect有管理加密密钥的工具。

参数

名称描述
type:0选择Controller key状态
1选择OEM key状态
2选择用户key状态

例

例1

获取用户加密密钥的状态。

'是否设置了用户加密密钥?

IF IS_ENCRYPTION_KEY(2)=FALSE THEN

    打印"此控制器中未设置用户加密密钥"

ENDIF
