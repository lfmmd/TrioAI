IS_PROG

类型

系统功能

替代

IS_FILE

语法

value = IS_PROG [path:]filename

描述

如果所选路径上存在具有指定名称的文件，则IS_PROG返回一个非零值。可选的路径名可能表明该文件位于本地内存或扩展内存中，例如SD卡。如果省略path，则使用本地RAM目录。

参数

value:0文件/程序不存在。
1文件/程序不存在且未打开。
2文件/程序已存在，并已打开以供读取。
3该文件/程序确实存在，并且可以写入。
4文件/程序确实存在，并且可以读写。
path:RAM:内部存储器的可选路径名。
SD:可选SD卡的路径名。
Filename:路径是RAM或未应用;文件名没有扩展名。
路径为SD;带扩展名的完整文件名。例如file1.txt, program1.bas等等。

例

例1

检查文件是否在本地内存中。

file_status = IS_PROG "program1"

替代语法

file_status = IS_PROG("program1")

例2

显示创建文件名的各种方法，并结合扩展名和SD:路径指示符。

DIM result AS INTEGER

DIM filename AS STRING(20)



result = IS_PROG "SD:file1.txt"

PRINT result



result = IS_PROG "SD:"+"file1.txt"

PRINT result



filename = "file1.txt"

result = IS_PROG "SD:"+filename

PRINT result



IF IS_PROG "SD:"+filename THEN

    PRINT "Yes file ";filename;" is there"

ENDIF



VR(30000)=$66

VR(30001)=$69

VR(30002)=$6c

VR(30003)=$65

VR(30004)=$31

VR(30005)=$00



filename = VRSTRING(30000)

PRINT filename



IF IS_PROG "SD:"+filename+".txt" THEN

    PRINT "Checked file ";filename;" via VRSTRING"

ENDIF

例3

检查SD卡上的文件，并向终端5报告文件状态。

DIM result AS INTEGER

DIM filename AS STRING(20)



filename = "file1.txt"

result = IS_PROG "SD:"+filename

SELECT_CASE result



CASE 0

  PRINT#5，"文件未找到"



CASE -1

  PRINT#5，"文件存在于SD上，但未打开" 

CASE -2

  PRINT#5，"文件打开以供阅读"

CASE -3

  PRINT#5，"文件打开用于写入"



CASE -4

  PRINT#5，"文件打开用于读写"



END_CASE
