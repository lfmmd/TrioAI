DATE

类型

系统函数

描述

返回或设置实时时钟持有的当前日期。

设置日期

语法

DATE = dd:mm:yy

描述

使用两位数字年格式或四位数字年格式设置日期。

参数

dd:以两位数字格式表示的日
mm:两位数字格式的月
yy:年份的最后两位数字，使用00-99表示2000-2099
或
使用2000-2099范围的年份的完整四位数字

2000-2099范围以外的年份无效。

例

将日期设置为 the 20th October 2012

>>DATE = 20:10:12

或

>>DATE = 20:10:2012

读取日期

语法

Value =DATE({mode})

描述

从实时时钟中读取日期值作为数字。

参数

模式价值
none从01/01/2000开始的天数(01/01/2000 = 0)
0本月的日
1本年的月
2本年

例

例1

打印自2000年1月1日以后的天数(第一天为第0天)。

>>PRINT DATE

4676

>>

例2

设定一个日期，然后用US格式打印出来。

>>DATE=05:08:2008

>>PRINT DATE(1); "/"; DATE(0); "/"; DATE(2) '以US格式打印

08/05/2008

>>

例3

在BASIC程序中使用VR中的值设置日期。

VR(20) = 15

VR(21) = 11

VR(22) = 2012

DATE = VR(20):VR(21):VR(22)
