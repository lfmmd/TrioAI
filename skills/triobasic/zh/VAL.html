VAL

类型

字符串函数

语法

value = VAL(string)

描述

将字符串转换为数值。如果字符串不是数值，则VAL返回0。

参数

string:要转换的字符串

例

例1

预先定义一个字符串类型的变量，然后将其当前值转换为存储在VR中的数值。VR的结果是-132.456。

DIM str1 AS STRING(20)

str1 = "-123.456"

VR(100) = VAL(str1)

例2

预先定义一个字符串类型的变量，然后将其当前值转换为存储在局部变量中的整型数值。在局部变量中得到的结果是1110。

DIM str2 AS STRING(10)

DIM number AS INTEGER

str2 = "987"

number = INT(VAL(str2)) + 123
