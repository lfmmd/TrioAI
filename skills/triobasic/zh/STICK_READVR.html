STICK_READVR

类型

系统功能

语法

value = STICK_READVR(flash_file, vr_start [,format])

描述

从SD卡读取VR数据到控制器。

任何现有的VR数据都将被覆盖。

Binary格式提供了最好的数据精度。

参数

value:TRUE =函数成功
FALSE =函数不成功
flash_file:一个数字，当附加到字符“SD”将形成数据文件名。
硬编码的字符串或string变量将形成数据文件名。
vr_start:数据值将被传输到的VRs中的起始点。(格式2忽略)
format:0 =二进制64位浮点格式，BIN文件(默认)
1 = ASCII码逗号分隔值，文件，无索引CSV
2 = ASCII CSV格式:索引，值<cr>

当以format=0存储时，数据以IEEE浮点二进制格式little-endian存储，即最不重要的字节优先。

例

例1

从SD卡读取二进制文件SD002012.bin，并将数据复制到VR内存中，从VR(101)开始。

success = STICK_READVR(2012, 101, 0)

IF success=TRUE THEN

  PRINT #5,"SD card read OK"

ENDIF

例2

从SD卡读取二进制文件MYDATALOG1_VR.BIN，并将数据复制到从VR(101)开始的VR内存中。

success = STICK_READVR("MYDATALOG1", 101, 0)

IF success=TRUE THEN

  PRINT #5,"SD card read OK"

ENDIF

例3

从SD卡上读取CSV文件DATA1_VR.CSV，并根据文件中的索引将数据拷贝到VR内存中。格式为index, value <cr>。

DIM flash_file AS STRING(20)

DIM s AS BOOLEAN

flash_file = "DATA1_VR"



s = STICK_READVR(flash_file, 0, 2)



IF s = TRUE THEN

  PRINT flash_file; " read OK"

ELSE

  PRINT "Error reading "; flash_file

ENDIF
