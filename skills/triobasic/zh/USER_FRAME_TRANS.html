USER_FRAME_TRANS

类型

数学函数

语法

USER_FRAME_TRANS(user_frame_in, user_frame_out, tool_offset_in, tool_offset_out, table_in, table_out, [scale])

描述

这个功能可以让你将一组位置从一个Frame转换到另一个Frame。这可以用于从视觉系统中获取一组位置，并对其进行转换，使其成为相对于传送带的一组位置。

USER_FRAME_TRANS需要启动运动学runtime FEC

要使用这个函数，需要设置FRAME_GROUP和USER_FRAME。如果您不希望使用实轴设置FRAME_GROUP，您可以使用虚拟轴。

USER_FRAME计算是在原始的整数位置数据上执行的。table数据通过尺度参数进行缩放，为获得最佳分辨率，应将尺度设置为机器人的UNITS。

由于所有USER_FRAME转换都使用相同的坐标比例，因此位置是作为原始位置提供还是按UNITS缩放都无关紧要。

参数

user_frame_in:输入点的USER_FRAME标识
user_frame_out:点被转换为的USER_FRAME标识
tool_offset_in:提供点的TOOL_OFFSET标识
tool_offset_out:点被转换为的TOOL_OFFSET标识
table_in:输入位置的开始
table_out:生成位置的开始
scale:这个参数允许你缩放table值(默认1000)

table_in需要12个值。任何不需要的都应该设置为0表示位置，1表示比例。
table_in第一轴位置
table_in + 1第二轴位置
table_in + 2第三轴位置
table_in + 3第四轴位置
table_in + 4第五轴位置
table_in + 5第六轴位置
table_in + 6第一轴FRAME_SCALE
table_in + 7第二轴FRAME_SCALE
table_in + 8第三轴FRAME_SCALE
table_in + 9第四轴FRAME_SCALE
table_in + 10第五轴FRAME_SCALE
table_in + 11第六轴FRAME_SCALE

例

USER_FRAME(视觉)已配置为相对于机器人原点的视觉系统。传送带已在USER_FRAME(传送带)中配置要使用传送带USER_FRAME上的视觉系统位置。它们必须通过USER_FRAME_TRANS进行转换。

USER_FRAME_TRANS(vision, conveyor, 0, 0, 200,300)
