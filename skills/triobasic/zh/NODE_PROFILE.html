NODE_PROFILE

类型

系统数组(MC_CONFIG)

语法

NODE_PROFILE(slot, node)= value

描述

这个2D数组用于在内部数据库中设置EtherCAT配置文件，以使用选定的配置文件。每个配置文件都提供额外的功能，并且是特定于供应商和产品代码的。请查阅为您连接的从属设备提供的额外技术说明。

阵列是二维的，第一维是主站槽标识符，第二维是节点在主站网络中的位置。

参数

slot:EtherCAT端口槽位号。(MC4N、MC6N和Flex-6 Nano上为0)
node:由节点在EtherCAT网络中的位置决定的地址。第一个节点是地址0。
value:0使用默认节点配置文件/配置(默认)
> = 1使用指定的EtherCAT配置文件/配置
