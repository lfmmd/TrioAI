开放运动

Trio控制器支持用户描述的关键字。这解释了在使用此功能开发时如何设置，使用和调试。一个新的命令REGISTER_MOVE允许用户注册他们的关键字。

设置一个开放的移动

使用命令可以注册一个关键字

REGISTER_MOVE(moveName, minAxes, maxAxis, posPerAxis, optParams)
参数数据类型描述
moveNameSTRING运动的名字。
minAxesINTEGER此运动的最小轴数
maxAxisINTEGER此运动的最大轴数
posPerAxisINTEGER每
optParamsINTEGER可选参数个数

开放运动的新命令

随着开放运动的发布，新的命令被添加到固件中，以帮助开发新的用户自定义运动。

THROW_ERROR

该命令将抛出一个运行时错误(编号268)。

THROW_ERROR(errorString)

Example:

THROW_ERROR("Move distance too short")

LOAD_OPEN_MOVE

这个命令会将一个移动加载到movebuffer中。这个命令只允许在Real-Time Basic库中使用。

LOAD_OPEN_MOVE(moveDist, moveData)
参数数据类型描述
moveDistFLOAT以单位移动距离
moveDataFLOAT预计算存储在移动缓冲区中，以供伺服功能使用

例:

FUNCTION move_sine_load(baseaxis AS INTEGER, posdemand AS FLOAT(), optparams AS FLOAT())
    DIM movedata AS FLOAT(10)
    DIM movedist AS FLOAT

    movedist = posdemand(0) + posdemand(1) ' units
    movedata(0) = posdemand(0)
    movedata(1) = posdemand(1)

    IF (movedist < 5) THEN
        THROW_ERROR("Move distance too short")
    ELSE
        LOAD_OPEN_MOVE(movedist, movedata)
    ENDIF
ENDFUNC

UPDATE_OPEN_MOVE_POS

此命令将给定轴的DPOS设置为给定值。这个命令只允许在Real-Time Basic库中使用。

例:

FUNCTION move_sine_servo(f AS FLOAT, baseaxis, currentaxis AS INTEGER, movedata AS FLOAT())

    p = f * (movedata(0))
    q = f * (-movedata(0))

    OPEN_MOVE_DEBUG_A AXIS(baseaxis) = f

    IF currentaxis = baseaxis THEN
        UPDATE_OPEN_MOVE_POS(baseaxis, p)
    ELSE
        UPDATE_OPEN_MOVE_POS(currentaxis, q)
    ENDIF
ENDFUNC

限制

  • 新的运动名称不能与现有关键字同名。在这种情况下控制器会抛出一个错误。
  • TrioBASIC函数中定义的参数数量必须与传递给关键字的参数数量相匹配。
  • 任何时候只能注册5个运动。
  • REGISTER_MOVE命令只能在MC_CONFIG中设置。

例

在 MC_CONFIG.MCC

REGISTER_MOVE("MOVE_SINE", 2, 2, 1, 1)

在 MYMOVE.BTBLIB

' 该函数被调用以将移动加载到缓冲区中。
' 请执行计算并调用'LOAD_OPEN_MOVE'函数将移动加载到缓冲区中。
FUNCTION move_sine_load(baseaxis AS INTEGER, posdemand AS FLOAT(), optparams AS FLOAT())
    DIM movedata AS FLOAT(10)
    DIM movedist AS FLOAT

    movedist = posdemand(0) + posdemand(1) ' units
    movedata(0) = posdemand(0)
    movedata(1) = posdemand(1)

    IF (movedist < 5) THEN
        THROW_ERROR("Move distance too short")
    ELSE
        LOAD_OPEN_MOVE(movedist, movedata)
    ENDIF
ENDFUNC

'此函数在执行您的移动时实时调用。
' f 是一个从0到1的因子。0代表开始，1代表结束。
'  这个因子直接来自VPU。
' 注意:请将此功能保持尽可能小，以减少系统负载。
FUNCTION move_sine_servo(f AS FLOAT, baseaxis, currentaxis AS INTEGER, movedata AS FLOAT())

    p = f * (movedata(0))
    q = f * (-movedata(0))

    OPEN_MOVE_DEBUG_A AXIS(baseaxis) = f

    IF currentaxis = baseaxis THEN
        UPDATE_OPEN_MOVE_POS(baseaxis, p)
    ELSE
        UPDATE_OPEN_MOVE_POS(currentaxis, q)
    ENDIF
ENDFUNC
