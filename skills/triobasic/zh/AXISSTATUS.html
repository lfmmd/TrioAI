AXISSTATUS

类型

轴参数(只读)

描述

AXISSTATUS轴参数用于读取轴的状态，每个位表示轴所处的特定状态。

Motion Perfect使用字符在“轴参数”窗口中显示错误，对于没有字符的位，则显示其位数。

在通电后，Kinematic FRAMEs会运行1小时。要在1小时后继续，添加Feature Enable Code 22。

第27位仅针对P874/P879。

功能

L -轴VP_SPEED由MOVESP/MOVEABSSP/MOVECIRCSP等控制。它不在SPEED设置下。

W - FE值在FE_RANGE设置的警告值之外。

A -轴上发生通讯错误。 EtherCAT，RTEX或绝对编码器（EnDat，BiSS）出现通信问题。

M -远程驱动器（EtherCAT，RTEX等）已发出信号，表明内部错误。 检查驱动器显示和软件。

F -轴已触发由FWD_IN设置的正向限位开关。

R -轴已触发由REV_IN设置的反向限位开关。

D - DATUM程序正在进行中。

H -进给保持输入是有效的，轴VP_SPEED设置为FHSPEED。这不适用于MOVESP类型或链接的移动。

E -跟随误差FE超过了FE_LIMIT轴参数设置的限制。

X -轴DPOS大于FS_LIMIT轴参数设置的限制。

Y -轴DPOS小于RS_LIMIT轴参数设置的限制。

C -当前移动已被取消，并以FASTDEC(如果指定)否则以DECEL的速率减速。

O -轴的DEMAND_SPEED超过了步进脉冲硬件产生脉冲的能力。典型的最大脉冲速率为2 MHz和8 MHz，具体取决于产品。

T -轴上的MOVETANG正在减速。

V -轴开启了VOLUME_LIMIT命令设置的3D体积限制，并且机器人末端执行器的位置在设置的限制之外。

I -在机器人FRAME中，AXIS_DPOS大于AXIS_FS_LIMIT。这通常是指关节角度极限。

J -在机器人FRAME中，AXIS_DPOS小于AXIS_RS_LIMIT。这通常是指关节角度极限。

P -由编码器连接器供电的连接编码器或其他硬件的5v电源因过载而关闭。所有编码器端口共用一个电源。因此，P将显示在同一硬件上运行的所有轴。

N -表示硬件P-switch FIFO在轴上的缓冲区不为空。它在缓冲区中有1个或多个位置。

B -表示硬件P-switch FIFO缓冲区已满。当缓冲区已满时，不能再执行新的HW_PSWITCH命令。

K -运动学FRAME已停止运行。WDOG被设置为OFF, FRAME值被设置回0。如果运动学FEC或机器人FEC不存在，则在每次重置后1小时后产生。

22 - BISS编码器状态标志。当使用Renishaw Resolute时，此位表示由于刻度上有污垢而导致数据不可靠。

23 -在机器人FRAME中，WORLD_DPOS大于WORLD_FS_LIMIT。

24 -在机器人FRAME中，WORLD_DPOS小于WORLD_RS_LIMIT。

25-模式3 CANCEL发生。移动取消并立即合并到下一个移动。

26 -表示用户自定义错误，例如在EtherCAT上，EtherCAT函数$79将在DRIVE_STATUS中定义位，这些位将组合以设置此AXISSTATUS位。

27 - hw_pswitch FIFO为半满或更多时设置。需要硬件支持。仅在某些运动控制器上可用。

28 -设置轴被运行在主移动顶部的MOVEADD偏移的时间。

29 -当MOVEADD被取消并逐渐下降时设置。坡道降落完成后清除。

30 -当EtherCAT远程驱动器发送EtherCAT紧急消息时设置，有关紧急消息响应的详细信息，请参阅ETHERCAT函数$E0。

31 -未使用。

32 -当执行open frame 的TrioBasic代码出现错误时设置。

33 -当 VPU处于错误状态时设置，可以使用VP_ERROR读取。

34 -当缓冲区运动被取消时设置（当运动停止时将被删除）。

35 -当取消缓冲区运动时设置，减速将在缓冲距离内停止（因为停止距离大于当前运动的剩余距离）。

36 -当取消缓冲区运动时设置，减速将在当前运动结束时停止（因为停止距离小于当前运动的剩余距离）。

37 -当最后一个加载的运动被取消时设置。

ERRORMASK是一个按位掩码，它决定哪些AXISSTATUS位将生成MOTION_ERROR并将WDOG或AXIS_ENABLE设置为OFF

例

例1

检查第4位，看轴是否在正向限制中。

IF (AXISSTATUS AND 16) > 0 THEN 

  PRINT "In forward limit"

ENDIF

例2

检查第3位，看是否有远程驱动器错误。尝试复位EtherCAT驱动器故障。

BASE(ecat_axis_1)

IF AXISSTATUS.3 = 1 THEN 

  PRINT "Remote drive error"

  ETHERCAT($64, ecat_axis_1)

  WA(50)

  DATUM(7)

  IF AXISSTATUS.3 = 0 THEN

    PRINT "Error was cleared"

  ENDIF

ENDIF

例3

发生运动错误时，检查每个出错轴的轴状态。

IF MOTION_ERROR <> 0 THEN

  FOR a = 0 to 23

    IF MOTION_ERROR.a = 1 THEN

      IF AXISSTATUS AXIS(a).2 = 1 THEN

        PRINT "Communication error on axis "; a[0]

      IF AXISSTATUS AXIS(a).3 = 1 THEN

        PRINT "Remote drive error on axis "; a[0]

      IF AXISSTATUS AXIS(a).8 = 1 THEN

        PRINT "FE exceeded FE_LIMIT on axis "; a[0]

      ENDIF

    ENDIF

  NEXT a

ENDIF
