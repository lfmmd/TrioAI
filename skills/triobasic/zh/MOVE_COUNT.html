MOVE_COUNT

类型

轴参数

描述

每当运动命令加载到MTYPE缓冲区或当命令自动重新加载(如MOVELINK, CAMBOX和FLEXLINK)时，使用选项值4时，move_count都会增加。(自动重复)

可以写入move_count来设置一个初始值。

值

加载到MTYPE缓冲区中的移动次数。

长度为零的移动不会被加载到MTYPE缓冲区中，因此不会被计数。

Fwd和Rev Jog移动不被计算在内。

例

例1

运行动作程序，然后在加载了10步之后打开OP(11)。

MOVE_COUNT = 0

RUN "MOTION"

WAIT UNTIL MOVE_COUNT > 10

OP(11,ON)

例2

连续运行MOVELINK，并在250个循环后停止，在最后一个完整的MOVELINK长度结束时停止。

motion_axis = 0

link_axis = 1

BASE(motion_axis)

MOVE_COUNT = 0

MOVELINK(10,12,2,2,link_axis,4)

IF IDLE AXIS(link_axis) THEN

    FORWARD AXIS(link_axis)

ENDIF

WAIT UNTIL MOVE_COUNT >= 250

REP_OPTION.1 = 1

WAIT IDLE
