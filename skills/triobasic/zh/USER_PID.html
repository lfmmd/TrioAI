USER_PID

类型

系统功能

语法

USER_PID(mode, input data VR index, output data VR index)

描述

USER_PID是一个通用的PID函数。该功能反映了典型IEC 61131-3 PLC编程系统中的PID。该函数使用全局变量VR将输入、输出和控制值传递给USER_PID。

在模式1中，每次从BASIC程序调用USER_PID时，它都会运行一个周期。它被设计成在具有一致可重复周期的程序循环中使用。这通常是通过在循环结束时使用WA(time_in_ms)或TICKS设置时间等待来实现的。

在模式2中，USER_PID只被调用一次，并以SERVO_PERIOD的速率在内核中运行。

功能图

参数

Mode:1 =每次调用运行一次PID控制。
2 =在控制器中以SERVO_PERIOD速率连续自动运行。
Input_data_index:包含输入数据的VRs块的基址。
Output_data_index:包含输出数据的VRs块的基址。

输入数据

输入值(VR偏移量(input_data))
VR偏移名字描述
0AUTOTRUE = 自动 / FALSE = 手动
1PV输入
2SP设定点
3XOUT_MANU手动模式的输出值
4KP增益
5TI积分时间常数
6TD微分时间常数
7TS扫描时间
8XMIN最小允许输出值
9XMAX最大输出值
10I_SEL如果为FALSE，则忽略积分项
11INT_HOLDTRUE =保持积分和
12I_ITL_ONTRUE =重置积分和为I_ITLVAL
13I_ITLVAL积分和复位值
14DEADB_ERRPV上的滞后。如果大于(PVprev - DEADBAND_W)而小于(PRprev + DEADBAND_W)，则认为PV不变。
15FFD输出上的干扰

输出数据

输出值(VR的偏移量(output_data))
VR偏移名字描述
0XOUT输出值
1ER错误值
2XOUT_P最后一次计算的比例值
3XOUT_I最后一次计算的积分值
4XOUT_D最后一次计算的导数值
5XOUT_HLM如果达到上限，则为TRUE
6XOUT_LLM如果达到下限，则为TRUE
7LASTCALL保留-仅内部使用
8ERR保留-仅内部使用
9INT保留-仅内部使用
10OVER保留-仅内部使用

例1

使用虚拟轴模拟要由USER_PID控制的进程。

input_data_index = 100

GLOBAL "p_auto", 100

GLOBAL "p_pv", 101

GLOBAL "p_sp", 102

GLOBAL "p_xout_manu", 103

GLOBAL "p_kp", 104

GLOBAL "p_ti", 105

GLOBAL "p_td", 106

GLOBAL "p_ts", 107

GLOBAL "p_xmin", 108

GLOBAL "p_xmax", 109

GLOBAL "p_i_sel", 110

GLOBAL "p_int_hold", 111

GLOBAL "p_i_itl_on", 112

GLOBAL "p_i_itlval", 113

GLOBAL "p_deadb_err", 114

GLOBAL "p_ffd", 115



output_data_index = 120

GLOBAL "p_xout", 120

GLOBAL "p_er", 121

GLOBAL "p_xout_p", 122

GLOBAL "p_xout_i", 123

GLOBAL "p_xout_d", 124

GLOBAL "p_xout_hlm", 125

GLOBAL "p_xout_llm", 126



'初始化 PID 设置

p_xout_manu = 0

p_kp = 70.0

p_ti = 300

p_td = 2

p_ts = 1

p_xmin = -2048

p_xmax = 2048

p_i_sel = TRUE

p_int_hold = FALSE

p_deadb_err = 0

p_ffd = 0



'开始虚拟轴 "目标进程"

GOSUB set_test_target



'重置 USER_PID

p_auto = FALSE

p_i_itlval = 0

p_i_itl_on = TRUE

USER_PID(1, input_data_index, output_data_index)



p_auto = TRUE

p_i_itl_on = FALSE



WHILE TRUE

  BASE(10)

  p_pv = MPOS

  USER_PID(1, input_data_index, output_data_index)

  SPEED = p_xout

  ACCEL = ABS(SPEED) * 40

  DECEL = ACCEL

  WA(10)

WEND



set_test_target:

'设置虚拟轴像被控制的过程值一样运动

BASE(10)

DEFPOS(0)

ATYPE = 0

UNITS = 1000

SERVO = ON

WDOG = ON

SPEED = 0 '初始化速度为 0

ACCEL = 10000000 '即时加速度

DECEL = ACCEL

IF IDLE THEN FORWARD

'PID输出将设置轴速度

'MPOS将为PV值的反馈输出

RETURN

例2

使用AIN(0)作为外部过程的反馈，AOUT(2)作为控制输出。

input_data_index = 100

GLOBAL "p_auto", 100

GLOBAL "p_pv", 101

GLOBAL "p_sp", 102

GLOBAL "p_xout_manu", 103

GLOBAL "p_kp", 104

GLOBAL "p_ti", 105

GLOBAL "p_td", 106

GLOBAL "p_ts", 107

GLOBAL "p_xmin", 108

GLOBAL "p_xmax", 109

GLOBAL "p_i_sel", 110

GLOBAL "p_int_hold", 111

GLOBAL "p_i_itl_on", 112

GLOBAL "p_i_itlval", 113

GLOBAL "p_deadb_err", 114

GLOBAL "p_ffd", 115



output_data_index = 120

GLOBAL "p_xout", 120

GLOBAL "p_er", 121

GLOBAL "p_xout_p", 122

GLOBAL "p_xout_i", 123

GLOBAL "p_xout_d", 124

GLOBAL "p_xout_hlm", 125

GLOBAL "p_xout_llm", 126



'初始化PID设置

p_xout_manu = 0

p_kp = 5.0

p_ti = 40

p_td = 0.5

p_ts = 1

p_xmin = -2048

p_xmax = 2048

p_i_sel = TRUE

p_int_hold = FALSE

p_deadb_err = 0

p_ffd = 0



'重置USER_PID

p_auto = FALSE

p_i_itlval = 0

p_i_itl_on = TRUE

USER_PID(1, input_data_index, output_data_index)



p_auto = TRUE

p_i_itl_on = FALSE



WHILE TRUE

  TICKS = 20 '设置周期时间

  p_pv = AIN(0)

  USER_PID(1, input_data_index, output_data_index)

  AOUT(2) = p_xout

  REPEAT

    WA(1) '在剩余的周期时间内休眠

  UNTIL TICKS <= 0

WEND
