# 哈希

类型

特殊字符

语法

command #channel

描述

#符号用于指定用于串行输入/输出命令的通信通道。

参数

通道设备
0以太网端口0(命令行)
1RS232端口1
2RS485接口2
5_Motion Pefect_用户通道
6_Motion Pefect_用户通道
7_Motion Pefect_用户通道
8用于_Motion Pefect_内部操作
9用于_Motion Pefect_内部操作
20 - 29使用OPEN命令配置用于协议使用的通道
40-48使用OPEN命令配置的通道，用于文件和字符串变量访问
49保留
50第1个Anybus模块
51第2个Anybus模块
52第3个Anybus模块
53第4个Anybus模块
54第5个Anybus模块
55第6个Anybus模块
56第7个Anybus模块

通道5到9是逻辑通道，由Motion Perfect叠加到端口0上。

例

例1

将ASCII字符串打印到不同的通道

PRINT #1, "将数据打印到RS232通道"

PRINT #5, "将数据打印到Motion Perfect 终端 5"

例2

在通道6检查和接收字符

WHILE KEY #6

  GET #63, VR(123)

WEND

查看

Get, key, input, open, outdevice, PRINT
