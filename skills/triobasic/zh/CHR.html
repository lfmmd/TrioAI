CHR

类型

字符串函数

语法

value = CHR(number[, format])

描述

CHR将ASCII/UTF-8字符作为STRING返回，该字符串由数字引用，可以将其分配给STRING变量或打印。

参数:
value:一个包含字符的STRING
number:ASCII/UTF-8字符的任何有效数值
format:数字的格式
0ASCII码(默认)
1UNICODE UTF-8编码

例

例1

在命令行上打印字符A

>>PRINT CHR(65)

A

>>

例2

打印一行仅以回车符结束的文本

PRINT#5, "abcdefghijk"; CHR(13)

例3

将一个字符从串行端口附加到一个STRING变量

DIM value AS STRING

WHILE KEY#5

  GET#5, char

  value = value + CHR(char)

WEND

例4

打印一个编码为UTF-8的Unicode字符:

>> print CHR(26,1)

袂

>>
