OP

类型

I / O命令

描述

设置输出，并允许读取前32个输出的状态。数字I/O通常指的是运动控制器附带的24V输入和输出。没有硬件连接的I/O点可以用作“虚拟”输出和输入，可以用OP设置，用IN读取。

OP命令有四种操作模式，最多使用三个参数:

  • 读基本块
  • 写基本块
  • 设置单个输出
  • 写块

Mode = Read Base Block

语法

value = OP

描述

以二进制模式返回前32个输出的状态。

参数

value前32个输出的二进制模式

Mode = Write Base Block

语法

OP(state)

描述

同时设置前32个输出具有二进制模式的状态。

参数

State要在输出上设置的二进制数的十进制等效值

Mode = Set Single Output

语法

OP(output, state)

描述

设置单个输出的状态

参数

输出要设置的输出数。
状态0或OFF
1或ON

Mode = Write Block

语法

OP(start, end, state)

描述

同时用状态的二进制模式设置一组定义好的输出。

参数

start组内第一个输出
end组内最后输出
state要在群上设置的二进制数的十进制等价物

例

例1

打开单个输出44

OP(44, 1)

这相当于:

OP(44, ON)

例2

在前5个物理输出上设置位模式10010，输出13-31将被清除。注意位模式是如何移动8位来设置第一个可用的输出的，因为0到7不存在。

OP(18 << 8)

例3

读取前32个输出，清除0-7，因为它们只是输入和16-32。然后设置16-32，让8-15保持原来的状态。

read_output:

  VR(0) = OP

  '清除0-7和16-32

  VR(0) = VR(0) AND $0000FF00

  '设置 $1A42 在输出 16-32,

  '8-15将保持其原始状态

  VR(0) = VR(0) OR $1A420000

  OP(VR(0))

例4

同时设置输出10到13全开。

OP(10, 13, $F)
