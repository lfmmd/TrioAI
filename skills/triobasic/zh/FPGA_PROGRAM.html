FPGA_PROGRAM

类型

系统功能

语法

value = FPGA_PROGRAM(program)

描述

此功能允许您在支持FPGA重编程的控制器上可用的不同FPGA程序之间进行选择。

我们建议使用Motion Perfect中的工具来选择FPGA，而不是使用此命令。

参数

variant:1显示存储在本地控制器闪存中的FPGA图像
>=0要加载的程序编号，参见下表或检查FPGA_PROGRAM(-1)以查看可用选项。
value:TRUEFPGA编程成功

MC403

FPGA_PROGRAM特性笔记
0伺服，步进，HW_PSWITCH, SSI默认程序
1伺服，步进，HW_PSWITCH，多摩川
2伺服，步进，HW_PSWITCH, EnDATHW_PSWITCH仅在前2轴可用

MC405

FPGA_PROGRAM特性笔记
0伺服，步进，HW_PSWITCH, SSI，多摩川默认程序
1伺服，步进，HW_PSWITCH, SSI, EnDAT
2保留

例

检查可用的FPGA程序，然后加载程序1，以便可以使用EnDAT编码器。不要忘记断电重启。

>>FPGA_PROGRAM(-1)

0 : (00C) Servo,Stepper,PSwitch,SSI,Tamagawa                

1 : (00C) Servo,Stepper,PSwitch,SSI,EnDAT                   

>>FPGA_PROGRAM(1)
