IP_MEMORY_CONFIG

类型

系统参数(MC_CONFIG)

描述

MC464以太网端口有分配的内存来缓冲传入和传出的数据报文。每个缓冲页使用1600字节的内存。如果使用IP_PROTOCOL_CONFIG关闭某些端口，则可以使用IP_MEMORY_CONFIG重新分配未使用的内存，并为传入和传出数据提供更大的缓冲区大小。

缺省情况下，分配给Tx和Rx的字节分别为2 x 1600和2 x 1600。IP_MEMORY_CONFIG的值是$22(或2 + 32的十进制)，在大多数网络中，这个缓冲区大小足以处理所有的网络流量。

值

IP_MEMORY_CONFIG是一个字节，它被分成2块。

位描述价值
0 - 3Rx缓冲区的大小;1600字节页面数。$01 到 $09
4 - 7Tx缓冲区的大小;1600字节页面数。$10 到 $90

不要将任何一个nibble设置为小于1，否则将不会分配内存并且Motion Perfect将无法使用。

例

为传入的Rx以太网流量分配更多的缓冲空间，以应对繁忙网络中频繁的广播报文。

'禁用以太网IP和文本文件加载器端口

Ip_protocol_config = $37



'将释放的内存空间分配给Rx网络缓冲区

Ip_memory_config = $29
