MOVEJ

类型

机器人命令

语法

MOVEJ target [S:= T:= O:= C:=]

描述

MOVEJ用于沿非线性路径将机器人从一个点移动到另一个点。所有轴同时到达目的地位置。这是最快的运动类型，因为轴会根据精确的度数移动到所需位置。

移动的速度是插补速度，而不是任何单个轴或末端执行器的速度。

例如，机器人是以50的速度从关节位置的10.5、0、20、0、0、0移动到90.21、44.2、24.1、4.1、0.0、12.2。

总位置变化为144.33，具体如下:

J1: 90.21 - 10.5 = 79.71

J2: 44.22 - 0 = 44.22

J3: 24.1 - 20 = 4.1

J4: 4.1 - 0 = 4.1

J5: 0 - 0 = 0

J6: 12.2 - 0 = 12.2

机器人将以50的速度跑完144.33度的距离。单个轴可能以高于或低于50的速度移动，但所有轴将同时到达末端位置。

KINEMATIC_GROUP需要RPS运行FEC

参数

参数数据类型描述
TargetTARGET机器人移动到的笛卡尔目标或关节目标。
Embedded parameters:有关详细信息，请参见RPS移动命令

例

例1

GTA(0) = 350, 0, 50, 180, 0, 0, "p1"
GTA(1) = 400, 100, 20, 180, 0, 0, "p2"
GTA(2) = 450, -300, 60, 180, 0, 0, "p3"
OBJECT_FRAME(1,"of1", -10, 0, 0, 0, 0, 15)
TOOL_OFFSET(1,"tool1", 0, 0, 50, 0, 0, 0)

ROBOT(0)
MOVEJ GTA("p1")
MOVEJ GTA("p2")
MOVEJ GTA("p3")
STOP

三个移动指令命令机器人0从目标点0(名称为“p1”的GTA 0)线性移动到目标点2(名称为“p3”的GTA 2)。请注意，在任何MOVEJ指令中都没有其他嵌入参数。因此，该指令将使用以下参数移动机器人。
嵌入参数单位参数
Object Frame-当前激活的对象框架
Tool Offset-当前激活的工具偏移
Speed度/秒WORLD_ORI_SPEED * LOOKAHEAD_FACTOR
Acceleration度/秒2WORLD_ORI_MAX_ACC
Jerk度/秒3WORLD_ORI_MAX_JERK
Blend radius毫米0
Configuration-0

上面的对象框架和工具偏移量声明不会自动将对象框架和工具偏移量设置为当前激活的对象框架和工具偏移量。当前激活的对象框架和工具偏移量未被修改。

例2

MOVEJ GTA("p1") S:=100
类型单位参数
Speed度/秒100 * LOOKAHEAD_FACTOR

例3

MOVEJ GTA("p1") 0:="of1"
类型单位参数
Object Frame-of1
Speed度/秒WORLD_ORI_SPEED * LOOKAHEAD_FACTOR

Of1未设置为活动对象框架。这个指令只考虑了of1这个移动。

例4

MOVEJ GTA("p1") T:="tool1"
类型单位参数
Object Frame-当前活动对象框架
Tool Offset-tool1

Tool1未设置为活动工具框。此指令仅将tool1考虑到此移动。

例5

MOVEJ GTA("p1") Z:=67
MOVEJ GTA("p2")
类型单位参数
Tool offset-当前有效工具偏移量
Blend radius毫米67毫米

机器人将向目标p1移动。67mm在到达目标之前，它会混合下一个动作，走向目标p2。

例6:
MOVEJ GTA("p1") S:=150 O:="of1" T:="tool1" Z:=42 C:=0
MOVEJ GTA("p2") S:= 100 C:=1

Object Frame-当前激活的对象Frame当前激活的对象框架
Tool Offset-当前激活的工具偏移量当前激活的刀具偏移量
Speed度/秒150 * LOOKAHEAD_FACTOR100 * LOOKAHEAD_FACTOR
Acceleration度/秒2WORLD_POS_MAX_ACC * JOG_ACC_FACTORWORLD_POS_MAX_ACC * JOG_ACC_FACTOR
Jerk度/秒3WORLD_POS_MAX_JERKWORLD_POS_MAX_JERK
Blend radius毫米42毫米0
Configuration-01
