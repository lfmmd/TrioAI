LENB

类型

字符串函数

语法

value = LENB(string)

描述

返回指定字符串中使用的字节数。

请注意，当存在UTF-8字符时，这将不同于字符串中的字符数。

参数

string:被测量的字符串。

例

例1

定义字符串类型的变量，为其分配一个包含UTF-8字符的字符串，然后显示其字节长度和字符长度。

DIM str1 AS STRING(30)

str1 = "Здравей свят" '保加利亚语中的"Hello world" 

PRINT LENB(str1), LEN(str1) '将显示23和12
