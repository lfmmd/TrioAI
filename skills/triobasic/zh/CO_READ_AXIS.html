CO_READ_AXIS

类型

系统命令

语法

CO_READ_AXIS(axis_number, index, subindex, type[, vr_number])

描述

这个函数从远程驱动器或I/O设备获取一个CANopen-over-EtherCAT (CoE)对象。对象的索引和子索引用于请求一个值，该值要么放置在VR中，要么在Motion Perfect终端中显示，如果VR数字被设置为-1。

参考远程设备的手册获取可用对象的列表。如果成功返回对象值，该命令返回TRUE(-1)。否则，在请求值时出现错误的情况下，命令返回FALSE(0)。

参数

Axis_number:EtherCAT驱动器的轴号。
index:CANopen对象索引
subindex:CANopen对象子索引
Type:1Boolean
2Integer 8
3Integer 16
4Integer 32
5Unsigned 8
6Unsigned 16
7Unsigned 32
8Real 32
17Real 64
21Integer 64
27Unsigned 64
vr_number:当VR值介于0和最大值之间时，结果将被存储。
(-1表示该值将被打印到终端)

例

例1

打印对象0x6064子索引00的值，实际位置值。这是一个Int32类型的对象，CANopen类型4与之相同。

>> co_read_axis (3， $6064, 0,4， -1)

5472

>>

示例2

从远程驱动器获得比例增益和速度前馈增益，并置于VR(200)和VR(201)中。执行检查，确保驱动器支持该对象。

IF CO_READ_AXIS(2, $60FB, 1, 6, 200) = FALSE THEN

  PRINT "Error reading Object $60FB:01"

ELSE

  PRINT "Drive P Gain = "; VR(200)[0]

ENDIF

IF CO_READ_AXIS(2, $60FB, 2, 6, 201) = FALSE THEN

  PRINT "Error reading Object $60FB:02"

ELSE

  PRINT "Drive VFF Gain = "; VR(201)[0]

ENDIF
