INTEGER_READ

类型

数学函数

语法

INTEGER_READ(source, least_significant, most_significant)

描述

INTEGER_READ对64位寄存器执行低级访问，将其分成两个32位字段。

这可以用于从高分辨率编码器读取位置

参数

source:将被读取的2位值，可以是VR、TABLE或系统变量。
least_significant:存储最低有效位(最右边)32位的变量，可以是本地变量，VR或TABLE
most_significant:存储最高有效位(最左边)32位的变量，可以是本地变量，VR或TABLE

例

例1

读取存储在VR(1)中的浮点数为2个整数值，并以十六进制格式显示。

VR(1) = 1.0

INTEGER_READ(VR(1), VR(4), VR(5))

PRINT VR(1); "    ";

PRINT "MSW "; HEX(VR(5)), "LSW "; HEX(VR(4))

例2

读取ENCODER值作为一个整数。

INTEGER_READ(ENCODER AXIS(2), VR(4), VR(5))

PRINT "Encoder Value "; VR(5)[0]
