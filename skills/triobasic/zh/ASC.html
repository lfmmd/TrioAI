ASC

类型

字符串函数

语法

value = ASC(string[, offset])

描述

ASC返回所提供的字符串参数中第一个字符的ASCII值。如果字符串为空，则返回0。

当使用可选的第二个参数时，它提供了从字符串开始的偏移量，因此偏移量为0将返回第一个字符的ASCII码，1为第二个字符，2为第三个字符，等等。字符串中字符范围之外的偏移量将返回0。

这个函数之所以得名，是因为它过去专门处理以ASCII码编码的字符。但是，TrioBASIC字符串使用以UTF-8编码的Unicode字符，因此ASC返回的值将是该字符的Unicode值。

参数:
string:任何有效的字符串
offset:整数值(默认为0)

例

例1

打印包含在较长字符串中的字符' A '的ASCII值。

>>PRINT ASC("ABCDEF")

65

>>

例2

查找数字字符串中小数点的位置。

DIM a AS STRING(20)

  DIM i, x, dp_pos AS INTEGER



  a = "112.234"

  FOR i = 0 TO LEN(a) - 1

    x = ASC(a, i)

    IF x = $2E THEN

      dp_pos = i

    ENDIF

  NEXT i

  PRINT dp_pos
