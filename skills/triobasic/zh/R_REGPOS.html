R_REGPOS

类型

轴参数(只读)

语法

R_REGPOS(expression)

描述

以用户单位存储在轴上看到注册标志的最新位置。该参数与REGIST命令设置的基于时间的注册通道一起使用。

这是一个轴参数，因此您需要确保将此参数与用于设置REGIST的相同轴一起使用，以确保使用正确的UNITS。

参数

Expression:任何有效的TrioBASIC表达式。表达式的结果应该是一个有效的整数通道号。

值

在用户UNITS中发生注册事件的绝对位置。

例

切纸机使用凸轮轮廓形状，通过伺服驱动的滚筒快速绘制纸张，然后在切割时停止。纸张上印有注册标志。检测到这个标记后，通过使用命令的第三个参数缩放凸轮轮廓来调整下一张纸的长度:

'使用CAM拉伸的注册程序示例:

'设置窗口打开和关闭:

Length = 200 

OPEN_WIN = 100

CLOSE_WIN = 130

GOSUB initial



loop:

  TICKS = 0 '设置毫秒计数器为0

  IF R_MARK(0) THEN

    Offset = R_REGPOS(0)

    '下一行如果在工作表末尾，使offset值减去ve:

    IF ABS(offset - length) < offset THEN offset = offset - length

    PRINT "Mark seen at:"; offset[5, 1]

  ELSE

    Offset = 0

    PRINT "Mark not seen"

  ENDIF



  ' 每次移动前重置注册:

  DEFPOS(0)

  REGIST(32, 0, 0, 0, 1) '允许标记在100到130之间被捕捉

  CAM(0, 50, (length + offset * 0.5) * cf, 1000)

  WAIT UNTIL TICKS < -500

GOTO loop

(变量“cf”是一个常量，将根据每个编码器脉冲的机器绘制长度来计算)
