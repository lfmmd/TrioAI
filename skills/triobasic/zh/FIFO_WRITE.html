FIFO_WRITE

类型

访问类型

语法

OPEN # channel AS "[location:]name" FOR FIFO_WRITE(length)

描述

有关详细信息，请参见OPEN。

例

已将G-Code程序下载到SD卡上的文件中。它必须被转移到FIFO，这样它才能被解释为运动。FIFO文件将作为缓冲来平滑处理速度的变化。

OPEN#41 AS "SD:gc_program1" for INPUT 

OPEN#42 AS "gcodefifo" for FIFO_WRITE(512) 



WHILE KEY#41 

  GET#41, char 

  PRINT#42, char; 

WEND 



CLOSE#42
