<> 不等于

类型

比较运算符

语法

<expression-1> <> <expression-2>

描述

如果<expression-1>不等于<expression-2>返回TRUE，否则返回FALSE。

参数

Expression1:任何有效的TrioBASIC表达式
Expression2:任何有效的TrioBASIC表达式

例

如果轴不是空闲的，运行' scoop '子例程(MTYPE为0表示轴是空闲的)。

IF MTYPE <> 0 THEN GOSUB scoop

查看

= 等于, < 小于, <= 小于等于, >= 大于等于, > 大于
