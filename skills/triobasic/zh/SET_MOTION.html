SET_MOTION

类型

轴功能

语法

SET_MOTION(axis, speed[, accel[, decel[, jerk]]])

描述

SET_MOTION用于仅使用一个命令预先设置后续移动的参数列表。

该命令要求将轴指定为参数，因此不使用当前的基准轴，也不会接受axis修饰符。

参数

必须设置前2个参数的最小值。其他参数均为可选参数。
axis:要设置的轴号
speed:要设置的SPEED值
accel:ACCEL所需值
decel:DECEL所需值
jerk:JERK所需值

例

例1

在3轴上设置并运行一个移动。不设置JERK，所以最后一个参数不包括在内。

' SET_MOTION(axis, speed, accel, decel)

SET_MOTION(3, 20, 100, 120)

BASE(3)

MOVE(150)

WAIT IDLE
