DRIVE_CW_MODE

类型

轴参数

描述

默认固件控制周期性发送到远程驱动器的控制字的操作。例如，控制字通常在WDOG和AXIS_ENABLE参数的控制下，这样就可以通过软件启用和禁用驱动器。可选地，如果DRIVE_CW_MODE被设置为非零，控制字可以由用户程序设置。

值

驱动控制字的操作方式。
0系统根据WDOG和AXIS_ENABLE的状态设置控制字的值。(默认)
1用户程序通过DRIVE_CONTROLWORD控制控制字。
2用户程序通过DRIVE_CONTROLWORD控制第11位到第15位。
允许更改制造商特定的位，同时使能位在WDOG和AXIS_ENABLE的控制下。

例

例1：

接管周期性地发送到EtherCAT网络上作为0轴连接的驱动器的CoE控制字，然后切换复位的数据位。

BASE(0)

DRIVE_CW_MODE = 1 '手动控制控制字

DRIVE_CONTROLWORD = $06 '禁用驱动器

WA(10)

DRIVE_CONTROLWORD = $86 '复位驱动器

WA(10)

DRIVE_CONTROLWORD = $06

例2：

接管周期性地发送到EtherCAT网络上作为2轴连接的驱动器的CoE控制字，然后发送指令以开始回零。

BASE(2)

SERVO = OFF

DRIVE_CW_MODE = 1 '设置控制字为用户模式

DRIVE_CONTROLWORD = $06 '禁用驱动器

'将驱动器设置为DS402回零模式

CO_WRITE_AXIS(ax, $6060, $00, 2, -1, 6)

'等待回零模式被接受

VR(100) = 0

REPEAT

  CO_READ_AXIS(ax, $6061, $00, 2, 100)

UNTIL VR(100) = 6



'设置回零方式(1为正方向，2为反方向)

fwd = 1

rev = 2

CO_WRITE_AXIS(ax, $6098, $00, 2, -1, fwd)



DRIVE_CONTROLWORD = $1f '开始回零

WA(20)



'等待回零完成标志(第12位)

REPEAT

  WA(1)

UNTIL DRIVE_STATUS.12 = 1

WA(20)

DEFPOS(ENCODER) '将轴位置设置为驱动器的值

SERVO = ON

WDOG = ON

'将驱动器设置为位置模式

CO_WRITE_AXIS(ax, $6060, $00, 2, -1, 8)

'将控制字设置为正常启用状态

DRIVE_CONTROLWORD = $2f

DRIVE_CW_MODE = 0 '将控制字设置回wdog模式
