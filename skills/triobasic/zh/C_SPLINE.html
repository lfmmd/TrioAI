C_SPLINE

类型

命令

语法

C_SPLINE(mode, index_in, number_points, dimensions expand_ratio, index_out {,tension, bias})

描述

C_SPLINE命令在两种模式下运行，使用Cubic Hermite 样条将源点扩展为更平滑的更高分辨率剖面。这可以用来平滑运动轮廓，可以使用MOVEABS, MOVABSSEQ或类似的序列运行。

expand参数给出了每个位置被扩展的点数。所以总的输出点数是(((点数-1)* expand) + 1)

参数

mode:1和3Catmull Rom 样条线
2和4全基数样条
index_in输入点在TABLE内存中的位置(分组例如x1,y1,z1,x2,y2,y3…)
number_pointsTABLE内存中输入点的数目(每个点有2..)8维度)
dimensions数据的维数(2..8)
expand_ratio每个输入点之间生成的中间点数(2到100)
index_out输出点在TABLE内存中的位置(格式取决于“Mode”)
tension仅用于基数样条。(-1到1)-1表示“最松”，1表示“最紧”
bias仅用于基数样条。(-1到1，通常为0)

Mode = 1; Mode = 3

语法

C_SPLINE(mode, index_in, number_points, dimensions expand_ratio, index_out)

描述

Catmull-ROM(模式1和3)是一种常用的完整基数样条方程(模式2和4)的特殊情况。如果“张力”和“偏差”参数被设置为零，基数样条将与Catmull-ROM相同。

模式3生成点，其中不同维度的输出被分组在一起。模式1生成的点将相同维度的输出分组在一起。例如对于3维XYZ和点1,2,3,4。

模式3将因此按顺序输出数据X1,Y1,Z1,X2,Y2,Z2,X3,Y3,Z3,X4,Y4,Z4…

模式1将因此按顺序输出数据X1,X2,X3,X4,Y1,Y2,Y3,Y4,Z1,Z2,Z3,Z4…

参数

index_in:输入点在TABLE内存中的位置(例如x1,y1,z1,x2,y2,y3…)
number_points:TABLE内存中输入点的数目(每个点有2..)8维度)
dimensions:数据的维数(2..8)
expand_ratio:每个输入点之间生成的中间点数(2到100)
总输出点=(点数-1)*扩展+ 1
(即，如果源配置文件为100点，扩展比例设置为10，则生成的配置文件将为1010点((100-1)* 10+1)。
index_out:输出点在TABLE内存中的位置(格式取决于“Mode”)

例

扩展一个二维配置文件，在控制器内存中TABLE 1000位置处定义7个点:

npoints = 7

TABLE(1000,0,0) 

TABLE(1002,100,0) 

TABLE(1004,100,100) 

TABLE(1006,200,100) 

TABLE(1008,300,100) 

TABLE(1010,300，-100)

TABLE(1012,150，-100)



inp=1000 

outp=4000 

expand=20' Max 100, min 2



'参数:Mode, index_in, number_points, dimension, expand_ratio, 'index_out



C_SPLINE(3,inp,npoints,2,expand,outp)

Mode = 2; Mode = 4

语法

C_SPLINE(mode, index_in, number_points, dimensions expand_ratio, index_out ,tension, bias)

描述

模式2和4将在多维(从2维到8维)点集上执行Cubic Hermic样条，根据扩展率获得更平滑的路径。

当MOVEABS或MOVEABSSEQ的轮廓过于粗糙，需要外推到更大数量点的情况下，使用这些模式。

模式4生成点，其中不同维度的输出被分组在一起。模式2生成的点将相同维度的输出分组在一起。例如对于3维XYZ和点1,2,3,4。

模式4因此将按顺序输出数据X1,Y1,Z1,X2,Y2,Z2,X3,Y3,Z3,X4,Y4,Z4…

因此模式2将按顺序输出数据X1,X2,X3,X4,Y1,Y2,Y3,Y4,Z1,Z2,Z3,Z4…

参数

index_in输入点在TABLE内存中的位置(例如x1,y1,z1,x2,y2,y3…)
number_pointsTABLE内存中输入点的数目(每个点有2..)8维度)
dimensions数据的维数(2..8)
expand_ratio每个输入点之间生成的中间点数(2到100)
总输出点=(点数-1)*扩展+ 1
(即，如果源配置文件为100点，扩展比例设置为10，则生成的配置文件将为1010点((100-1)* 10+1)。
index_out输出点在TABLE内存中的位置(格式取决于“Mode”)
tension仅用于基数样条。(-1到1)-1表示“最松”，1表示“最紧”
bias仅用于基数样条。(-1到1，通常为0)

例

TABLE(1002,136,206) 

TABLE(1004,150,220) 

TABLE(1006,152,220)

inp = 1000  

outp = 2000  



expand = 40 'Max 100, min 2



 PRINT "output x start = "; outp

  PRINT "output y start = "; outp + (npoints - 1) * expand + 1

  PRINT "number of output points = "; (npoints - 1) * expand + 1
