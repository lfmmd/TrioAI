NW_NODE_PROFILE

类型

系统数组(MC_CONFIG)

语法

NW_NODE_PROFILE(slot, alias_address) = value

描述

这是一个数组系统参数，可以在MC_CONFIG中使用。该参数用于定义应该将哪个配置文件编号映射到特定的EtherCAT从站。如果选择的配置文件不可用或超出范围，则EtherCAT协议启动将失败。

NW_NODE_PROFILE配置文件ID值被初始化为零，因为任何配置文件ID值只有在从站设备使用相同的非零配置站别名设置时才会被使用。零值的站别名地址不被EtherCAT协议使用。

如果设置为非零，则对同一从设备使用NODE_PROFILE将优先于使用NW_NODE_PROFILE。

参数

slot:EtherCAT端口槽位号。(MC4N、MC6N和Flex-6 Nano上为0)
alias_address:节点的配置地址。
value:0使用默认的节点概要文件/配置(default)
> = 1使用指定的EtherCAT配置文件/配置

例

将别名地址为123的P375的配置文件设置为2。相当于DRIVE_PROFILE = 2。

在MC_CONFIG:

NW_NODE_PROFILE(0, 12) = 2

启动EtherCAT网络。终端#0输出显示:

EtherCAT State: Init (0)

EtherCAT State: Init (0)

EtherCAT State: Pre-Operational (0)

EtherCAT State: Safe-Operational (0)

EtherCAT State: Operational (0)

EtherCAT Configuration (0):


P600: 0 : 0 : 2000

P366 Coupler: 1 : 0 : 2001

P375 3 Axes Stepper Output: 2 : 12 : 12 (11:2:0,12:2:0,13:2:0)

从终端查看配置文件号。

? nw_node_profile(0,12)

2

P375显示值为:
