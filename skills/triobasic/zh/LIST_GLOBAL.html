LIST_GLOBAL

类型

系统命令(仅限命令行)

语法

LIST_GLOBAL [offset[, length]]

描述

将选定的GLOBAL和CONSTANT定义打印到当前输出通道。

参数

none:打印所有定义的GLOBAL和CONSTANT值。
offset:要打印的第一个GLOBAL或CONSTANT值的数目。
length:要打印的GLOBAL和CONSTANT值的数目。如果length为-1，则打印所有值。

例

检查已设置了以下GLOBAL和CONSTANT的应用程序中的所有全局数据。

CONSTANT "cutter", 23

GLOBAL "conveyor",5



>>LIST_GLOBAL

Global                           VR

----------------                 ----

conveyor                         5

Constant                         Value

----------------                 -------

cutter                           23.00000

>>LIST_GLOBAL 1

Global                           VR

----------------                 ----

conveyor                         5

>>LIST_GLOBAL 0,1

Constant                         Value

----------------                 -------

cutter                           23.00000

>>

查看：

CONSTANT, GLOBAL
