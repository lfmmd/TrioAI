FLASH_TABLE_FIRST

类型

系统参数

语法

FLASH_TABLE_FIRST = value

描述

FLASH_VR_FIRST, FLASH_VR_LAST, FLASH_TABLE_FIRST和FLASH_TABLE_LAST是允许用户配置保存在闪存中的VR/TABLE位置范围的系统参数。当电源断开时，超出定义范围的VR和TABLE位置将不保留。

需要固件V2.0313及更高版本。

参数

Value:在flash中存储全局表值的TABLE起始索引。

注1:MC403、MC405、MC4N不自动保存TABLE值。FLASH_DATA可以用TABLE而不是VR来配置运动控制器。在这种情况下，FLASH_TABLE_LAST与FLASH_VR_LAST具有相同的最大值。

例

在Flex-6 Nano中，将范围10000 ~ 39999的TABLE值保存到闪存中。因此，TABLE(0)到TABLE(9999)和TABLE(40000)到TABLE(511999)将是易失的，在下次上电时设置为0。

FLASH_TABLE_FIRST = 10000

FLASH_TABLE_LAST = 39999
