MODULEIO_BASE

类型

I/O参数(MC_CONFIG)

描述

该参数设置任意级联模块I/O通道的起始地址。与CANIO_BASE、DRIVEIO_BASE和NODE_IO一起，I/O分配方案可以替换和扩展MODULE_IO_MODE的行为，但是如果MODULE_IO_MODE的值被更改为2 (CANIO后跟MODULEIO)，则优先考虑。

值

1模块I/O禁用
0模块I/O自动分配(默认)
> = 8模块I/O位于此I/O点地址，截断为最接近的8的倍数

例子

具有MC464、松下模块(插槽0)和CANIO模块的系统将具有以下I/O分配:

Moduleio_base = 0 +DRIVEIO_BASE = 0 + CANIO_BASE= 0
