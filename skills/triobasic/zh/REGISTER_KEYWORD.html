REGISTER_KEYWORD

类型

开放式架构功能

语法

REGISTER_KEYWORD(keywordName, funcName, minParams, maxParams)

描述

这个命令注册一个关键字作为TrioBASIC语言的一部分。每个关键字必须链接到保存关键字实现的BASIC函数。它使用户能够使用自己的关键字扩展语言。
请注意，新关键字不能与现有关键字相同。

参数

keywordName:关键字的名称。
funcName:BASIC函数的名称。
minParams:BASIC函数的最小参数数量。
maxParams:BASIC函数的最大参数数量。

例

例1

注册2个关键词。

在MC_CONFIG中:

REGISTER_KEYWORD("Run_test1"， "test_int"， 0,2)

REGISTER_KEYWORD("run_test2"， "test_float"， 0,1)


在TEST_LIB.BAL :

FUNCTION test_int(param1 AS INTEGER)

   IF param1 = 2 THEN

      PRINT "test_int passed"

   ELSE

      PRINT "test_int failed"

   ENDIF

ENDFUNC



FUNCTION test_float(param1 AS FLOAT)

   IF param1 = 4.12 THEN

      PRINT "test_float passed"

   ELSE

      PRINT "test_float failed"

   ENDIF

ENDFUNC


在 TEST.BAS :

RUN_TEST1(2)

RUN_TEST1(0)

RUN_TEST2(4.12)

RUN_TEST2(2.31)
