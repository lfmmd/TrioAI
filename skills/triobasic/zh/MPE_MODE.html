MPE_MODE

类型

系统参数(只读)

语法

value = MPE_MODE

描述

在命令行进程中读取当前使用的连接模式。

这通常只用于Motion Perfect，但也可以用于用户应用程序。

参数

value:0没有协议。Motion Perfect处于直连模式或未连接。
1不再使用。
2不再使用。
3Motion Perfect版本3连接在工具或同步模式。
4在工具模式下连接Motion Perfect版本4。
5Motion Perfect版本4连接在同步模式。

例

在尝试重置运动控制器之前检查模式。

IF MPE_MODE = 0 THEN

  ' OK 重置控制器

  EX

ELSE

  PRINT #5," 连接Motion Perfect时无法重置"

  ENDIF
