TEXT_FILE_LOADER_PROC

类型

系统参数(MC_CONFIG)

描述

当TrioPC ActiveX启动文本文件传输到_运动控制器_时，在最高可用进程上启动TEXT_FILE_LOADER_PROGRAM。TEXT_FILE_LOADER _PROC可以设置为为TEXT_FILE_LOADER _PROGRAM指定一个不同的进程。如果定义的进程正在使用中，那么下一个较低的可用进程将被使用。

TEXT_FILE_LOADER _PROC可在MC_CONFIG脚本文件中设置。

值

1使用最高可用进程(默认)
0到最大进程在已定义进程上运行

例

例1

将TEXT_FILE_LOADER _PROGRAM设置为从进程19或更低的进程启动(使用命令行终端)。

>> TEXT_FILE_LOADER_PROC=19

>>

例2

删除TEXT_FILE_LOADER _PROC设置，以便TEXT_FILE_LOADER _PROGRAM在默认进程中启动(使用MC_CONFIG)。

'MC_CONFIG脚本文件

TEXT_FILE_LOADER_PROC = -1  'Start on default process on connection
