MPE

类型

系统命令

语法

MPE(mode)

描述

设置要在命令行上执行的通道握手类型。

这通常仅由Motion Perfect程序使用，但可以用于异步模式下的PCMotion ActiveX控件的用户应用程序。

只有一个命令行进程。如果Trio PCMotion连接并使用Motion Perfect命令行端口，则Motion Perfect将无法同时连接。

参数

mode:0没有协议。Motion Perfect处于直接模式或未连接。
1不再使用。
2不再使用。
3Motion Perfect版本3连接在工具或同步模式。
4在工具模式下连接Motion Perfect版本4。
5Motion Perfect版本4连接在同步模式。

例

在尝试重置运动控制器之前检查模式。

IF MPE = 0 THEN

  ' OK，重置控制器

  EX

ELSE

  PRINT #5," 连接Motion Perfect时无法重置"

  ENDIF
