SYNC_WITHDRAW

类型

轴参数

语法

SYNC_WITHDRAW AXIS(x) = 125.5 {distance in axis units}

描述

一组轴参数，包括SYNC_PAUSE和SYNC_DWELL，它们可以更好地控制同步运动在同步时间段内发生的时间。SYNC_WITHDRAW轴参数允许指定一个额外的运动，并且可以用于机器人从传送带或其他位置移动的Z轴运动。

值

“轴撤回”运动的距离，以轴为单位。通常在Z轴上会有撤回运动。用于XYZ拾取和放置的情况。

SYNC_PAUSE、SYNC_DWELL和SYNC_WITHDRAW参数通常在SYNC命令的每个轴上设置不同。例如，SYNC_WITHDRAW参数在系统的X轴和Y轴上可能为零，但SYNC_WITHDRAW在系统的Z轴上被设置为一个值。

例

BASE(0)



UNITS AXIS(0) = 1000

UNITS AXIS(1) = 1000

UNITS AXIS(2) = 1000

UNITS AXIS(7) = 1000

UNITS AXIS(8) = 1000



ATYPE AXIS(0) = 0

ATYPE AXIS(1) = 0

ATYPE AXIS(2) = 0



SYNC(4, 100) '清空之前的同步

WAIT UNTIL SYNC_CONTROL = 0



overall_time = 600

sync_part = 500

flight_part = 50



BASE(2) 'Z 轴同步控制:

SYNC_PAUSE = (overall_time - flight_part) / 2

SYNC_DWELL = (overall_time - flight_part) / 2

BASE(1) 'Y 轴同步控制:

SYNC_WITHDRAW = 0

SYNC_PAUSE = (overall_time - sync_part) / 2

SYNC_DWELL = (overall_time - sync_part) / 2

BASE(0) 'X 轴同步控制:

SYNC_WITHDRAW = 0

SYNC_PAUSE = (overall_time - sync_part) / 2

SYNC_DWELL = (overall_time - sync_part) / 2





DEFPOS(0, 0, 0)

SPEED AXIS(7) = 100

SPEED AXIS(8) = 100

DEFPOS(-200) AXIS(7) '轴7模拟输送机A

DEFPOS(-200) AXIS(8) '轴8模拟输送机B

IF MTYPE AXIS(7) = 0 THEN FORWARD AXIS(7)

IF MTYPE AXIS(8) = 0 THEN FORWARD AXIS(8)



TRIGGER



'这些是2个输送机高度:



conveyor_a_z = 65

conveyor_b_z = 70



'从起点同步到输送机B:

SYNC_WITHDRAW AXIS(2) = 5

SYNC(1, overall_time, 0, 8, 60, 115, conveyor_b_z)

WAIT UNTIL SYNC_CONTROL = 3' 同步

WA(150)



FOR k = 0 TO 1

   '从输送机B同步到输送机A:

   SYNC_WITHDRAW AXIS(2) = -40

   SYNC(10, overall_time, 0, 7, 60, -100, conveyor_a_z)

   WAIT UNTIL SYNC_CONTROL = 3 '同步

   WA(150)



   '从输送机A同步到输送机B:

   SYNC_WITHDRAW AXIS(2) = -40

   SYNC(10, overall_time, 0, 8, 60, 115, conveyor_b_z)

   WAIT UNTIL SYNC_CONTROL = 3 '同步

   WA(150)

NEXT k



'从输送机A同步到输送机B

SYNC_WITHDRAW AXIS(2) = -40

SYNC(10, overall_time, 0, 7, 60, -100, conveyor_a_z)

WAIT UNTIL SYNC_CONTROL = 3 '同步

WA(150)





'从输送机 A到起始点的不同步

SYNC_WITHDRAW AXIS(2) = -conveyor_a_z

SYNC(10, overall_time, 0, 9, 0, 0, 0) '9 is a dummy axis

WAIT UNTIL SYNC_CONTROL = 3 '同步



STOP
