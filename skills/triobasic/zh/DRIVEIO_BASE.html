DRIVEIO_BASE

类型

I/O参数(MC_CONFIG)

描述

该参数设置任何驱动器I/O通道的起始地址。与CANIO_BASE、MODULEIO_BASE和NODE_IO一起，I/O分配方案可以取代和扩展MODULE_IO_MODE的功能。

值

1驱动器I/O禁用(默认)
0驱动器I/O自动分配
> = 8驱动器I/O位于此I/O点地址，缩短为最接近的8的倍数

例

带有MC464、松下模块(插槽0)和CANIO模块的系统将具有以下I/O分配:

DRIVEIO_BASE = 0 +MODULEIO_BASE = 0 + CANIO_BASE= 0
0-7内置输入
8-15内置双向I/O
16-23松下模块输入
24-39CANIO双向I/O
40-47松下驱动输入
48-1023虚拟I / O

DRIVEIO_BASE = -1 +moduleio_base = 0 + canio_base= 0
0-7内置输入
8-15内置双向I/O
16-23松下模块输入
24-39CANIO双向I/O
40-1023虚拟I / O

DRIVEIO_BASE = 200 +moduleio_base = 80 + canio_base= 400
0-7内置输入
8-15内置双向I/O
16-79虚拟I / O
80-87松下模块输入
88-199虚拟I / O
200-207松下驱动输入
208-399虚拟I / O
400-415CANIO双向I/O
416-1023虚拟I / O
