>大于

类型

比较运算符

语法

<expression-1> > <expression-2>

描述

如果<expression-1>大于<expression-2>返回TRUE，否则返回FALSE。

比较运算符通常用于条件语句中，如IF、REPEAT和WHILE，但也可以在表达式有效的任何地方使用。

参数

Expression1:任何有效的TrioBASIC表达式
Expression2:任何有效的TrioBASIC表达式

例

例1

程序将等待，直到测量的位置大于200。

WAIT UNTIL MPOS > 200

例2

如果变量' x '大于0，则将值TRUE写入VR元素0，否则为FALSE。

VR(0) = x > 0

查看

= 等于, <> 不等于, < 小于, <= 小于等于, >= 大于等于
