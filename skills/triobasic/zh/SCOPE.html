SCOPE

类型

系统命令

语法

SCOPE(enable[, period])

SCOPE(enable, period, table_start, table_stop, p0 [,p1 [,p2 [,p3 [,p4 [,p5 [,p6 [,p7]]]]]]])

SCOPE(enable, period, fifo_name, fifo_size, samples, p0 [,p1 [,p2 [,p3 [,p4 [,p5 [,p6 [,p7]]]]]]])

描述

SCOPE命令可以在每个采样周期内捕获多达8个参数。采集样本，直到填满表格范围。触发器用于开始捕获。

加载SCOPE后，必须通过TRIGGER命令启动。TRIGGER可以指定单发或自动。

确保在你的程序使用的任何table数据之外分配table范围。

使用Motion Perfect来分配SCOPE命令是正常的，但有时手动执行也是有用的。table数据可以读回PC并显示在Motion Perfect示波器上，使用Motion Perfect或STICK_WRITE保存。

参数

enable:1或ON启用软件SCOPE(至少需要5个参数)
0或OFF禁用范围
period:数据样本之间的伺服周期数
table_start:开始在table数组中存储数据的位置
table_stop:要使用的table范围结束
fifo_name:要写入的FIFO文件的名称。如果这个文件已经存在于运动控制器中，那么它必须是一个FIFO文件。
fifo_size:要创建的FIFO文件的大小。如果FIFO文件已经以不同的大小存在，那么它将被删除并重新创建正确的大小。
samples:要存储到FIFO文件的样本数。
p0:存储的第一个参数
p1:存储的第二个参数
p2:存储的第三个参数
p3:存储的第四个参数
p4存储的第五个参数
p5存储的第六个参数
p6存储的第七个参数
p7存储的第八个参数

例

例1

这个例子将SCOPE设置为每10毫秒在5轴5轴上存储MPOS和DPOS (SERVO_PERIOD = 1000)。MPOS将存储在table值0..表中的DPOS值为500到999。直到执行TRIGGER命令后才开始采样。

SCOPE(ON,10,0,1000,MPOS AXIS(5), DPOS AXIS(5))

例2

禁用SCOPE以防止TRIGGER开始捕获

SCOPE(OFF)

例3

在抛掷中更改周期

SCOPE(ON,10,0,1000,MPOS AXIS(5), DPOS AXIS(5))

TRIGGER

WA(100)

SCOPE(ON,20)

例4

延迟捕获开始，直到TRIGGER后20毫秒

SCOPE(ON,10,0,1000,MPOS AXIS(5), DPOS AXIS(5))

SCOPE_DELAY=20

WAIT UNTIL IN(10)=1

TRIGGER

例5

触发前存储20ms数据。

SCOPE(ON,10,0,1000,MPOS AXIS(5), DPOS AXIS(5))

SCOPE_DELAY=-20

WAIT UNTIL IN(10)=1

TRIGGER

例6

将50个样本存储到FIFO文件“SCOPE_FIFO”中，并打印到终端#5。

SCOPE(ON, 1, "SCOPE_FIFO", 2048, 50, TICKS, MPOS AXIS(0), DPOS AXIS(0))

 TICKS = 0

 TRIGGER



 OPEN #40 AS "SCOPE_FIFO" FOR FIFO_READ

 length = 0

 WHILE length < 150

   IF KEY #40 THEN

     GET #40, char

     IF char = 13 THEN

       PRINT #5, ""

       length = length + 1

     ELSE

       PRINT #5, CHR(char);

     ENDIF

     IF char = $2C THEN

       length = length + 1

     ENDIF

   ENDIF

 WEND



 CLOSE #40

FIFO中存储的数据为ASCII CSV格式。每行有一个捕获点的数据。如。

0.00000,45.00000,45.00000
-1.00000,47.00000,47.00000
-2.00000,51.00000,51.00000
-3.00000,56.00000,55.00000
-4.00000,60.00000,60.00000
-5.00000,63.00000,62.00000
