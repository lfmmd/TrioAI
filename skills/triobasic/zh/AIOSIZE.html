AIOSIZE

Type

System Parameter (Read Only)

Description

Returns the total number of analogue I/O channels, both real and virtual, which varies across controller types.

The analogue I/O is numbered from zero, so the valid range is 0 to AIOSIZE - 1.

Value

The maximum number of analogue I/O channels.

Example

Print the last analogue output value.

>>PRINT AOUT(AIOSIZE – 1)

>>
