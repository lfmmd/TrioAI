DLINK

类型

系统命令

语法

DLINK(function,…)

描述

这是一个专门的命令，允许访问SLM™数字驱动器接口。在接口可以用于控制外部驱动器之前，轴参数必须由DLINK函数2命令初始化。

当前的SLM软件规定，驱动器必须在电源应用到运动控制器/ SLM后上电。

参数

Function:指定所需的功能。
0保留功能
1保留功能
2检查SLM模块是否存在
3检查SLM伺服驱动器是否存在
4为SLM通道分配一个_运动控制器_轴
5读取SLM参数
6写一个SLM参数
7写一个SLM命令
8读取驱动器参数
9返回与轴相关联的槽和通信通道
10读取EEPROM参数

Function = 2

语法

value = DLINK(2, slot, com)

描述

检查电机后方是否存在SLM模块。

参数

value:如果SLM存在，返回1，否则返回0。
slot:模块连接的通信槽位。
com:模块中连接轴的通信通道。

例

检查槽0上的SLM模块，通信通道0。

>>? DLINK(2,0,0)

1.0000

>>

Function = 3

语法

value = DLINK(3, slot, com)

描述

检查SLM伺服驱动器是否存在，如MultiAx。

参数

value:如果驱动器正在应答，则返回1，否则返回0。
slot:模块连接的通信槽位
com:模块中连接轴的通信通道

例

检查槽0上的SLM驱动器，通信通道0。

>>? DLINK(3,0,0)

0.0000

>>

Function = 4

语法

value = DLINK(4, slot, com, axis)

描述

为SLM通道分配一个运动控制器轴。
value:如果成功返回TRUE，否则返回FALSE
slot:模块连接的通信槽位
com:模块中连接轴的通信通道
axis:与此驱动器相关联的轴。如果这个轴已经被分配，那么它将失败。该轴的ATYPE将被设置为11。

例

将轴0分配给连接到槽位0和通信通道0的驱动器

>>DLINK(4,0,0,0)

Function = 5

语法

value = DLINK(5, axis, parameter)

描述

读取SLM参数

参数

value:返回的SLM值，如果命令失败返回-1
axis:与驱动器相关联的轴号
parameter:要读取的参数号。SLM通常在0…127的范围内。有关更多信息，请参阅驱动器文档。

例

从轴0打印SLM参数5的值。

>> print DLINK (5,0,1)

463.0000

>>

Function = 6

语法

value = DLINK(6, axis, parameter, value)

描述

写一个SLM参数

参数

value:如果成功返回TRUE，否则返回FALSE
axis:与驱动器相关联的轴号
parameter:要读取的SLM参数。通常在0…127的范围内。有关更多信息，请参阅驱动器文档
value:要写入参数的值

例

将SLM参数0设置为0轴上的值0。

>>DLINK(6,0,0,0)

>>

Function = 7

语法

value = DLINK(7, axis, command)

描述

写一个SLM命令。

参数

value:如果成功返回TRUE，否则返回FALSE
axis:与驱动器相关联的轴号
command:命令号。(参见驱动器文档)

例

将SLM命令250写入0轴

>> print DLINK (7,0,250)

1.0000

>>

Function = 8

语法

value = DLINK(8, axis, parameter)

描述

读取驱动器参数

参数

value:从驱动器返回的值，如果命令失败则返回-1
axis:与驱动器相关联的轴号
parameter:要读取的驱动器参数号。通常在0…127的范围内。有关更多信息，请参阅驱动器文档。

例

读取轴0的驱动器参数53248

>>PRINT DLINK(8,0,53248)

20504.0000

>>

Function = 9

语法

value = DLINK(9, axis)

描述

返回轴关联的槽和通信通道

参数

value:10 ×槽位号+通信通道，如果命令失败返回-1
axis:与驱动器相关联的轴号。

例子

读取轴2 SLM信息

>> print DLINK (9,2)

> > 11.0000

这个例子是针对1号槽，通信通道1

Function = 10

语法

value = DLINK(10, axis, parameter)

描述

读取EEPROM参数

参数

value:EEPROM的值，如果命令失败返回-1
axis:与驱动器相关联的轴号。
parameter:EEPROM参数号。(参见驱动器文档)

例

返回EEPROM参数29，来自轴0的磁通角。

>>PRINT DLINK(10,0,29)

>>62128.0000
