DRIVE_READ

类型

轴函数

语法

value = DRIVE_READ(parameter [,vr_index])

描述

DRIVE_READ允许控制器从数字总线连接的驱动器读取参数。目前支持松下A4N, A5N和A6N驱动器。

参数指标和细节可以在Motion Perfect智能驱动工具中找到。

参数

值描述
值:1DRIVE_READ操作成功
0DRIVE_READ失败
如果不使用vr_index，则返回值为参数值
参数:parameter_numberA4N参数号读取
(class * 256)+ parameter_number
或
(class * $100) + parameter_number		A5N/A6N参数号读取
65536 + SSID_code
或
$10000 + SSID_code		将System ID读入VRSTRING
131072 + (alarm_index * 4096) + alarm_function
或
$20000 + (alarm_index * $1000) + alarm_function		读取一个Alarm code
196608 + (index * 4096) + monitor_number
或
$30000 + (index * $1000) + monitor_number		读取一个Monitor Value
vr_index:将返回值存储在其中的VR

System ID、Alarm codes和Monitor Commands适用于A4N、A5N和A6N驱动器。

系统字符串ID码
SSID_code描述
$010驱动器供应商
$120驱动型号
$130驱动器序列号
$140驱动器固件版本
$220电机型号
$230电机序列号
$310外部规模供应商
$320外部规模模型号。
报警功能
报警代码描述指数
$000读报警要读取的报警索引
$001清除电流报警0
$011清除所有报警0
$012清除外部报警0

DRIVE_CLEAR可用于清除报警

例

例1

从A4N驱动器读取参数124，外标方向

success = DRIVE_READ(124,0)

IF success = 0 THEN 

  PRINT "读取驱动器参数错误"

ELSE

  PRINT "外标方向= "; VR(0)[0]

ENDIF

示例2

从A5N驱动器读取第3类参数26，外标方向

成功= DRIVE_READ(3 * 256 + 26,0)

IF成功= 0THEN 

  PRINT"读取驱动器参数错误"

其他的

  PRINT "外标方向= ";VR (0)[0]

ENDIF

例3

通过读取系统ID而找到的松下伺服驱动器的序列号从 VR(0) 开始存入VRSTRING。

success = DRIVE_READ($10000 + $130,0)

IF success = 0 THEN

  PRINT "读取驱动器参数错误"

ELSE

  PRINT "驱动程序序列号 = ";VRSTRING(0)

ENDIF

例4

从松下伺服驱动器读取历史报警。

PRINT " AXIS(";axis_no[0];")报警读取"

FOR past_alarm = 0 TO 14

  DRIVE_READ($20000 + past_alarm * 4096 + 0 ,0)

  PRINT "报警历史索引 "; past_alarm[0];" = ";VR(0)[0]

NEXT past_alarm

例5

读取监视器类型代码102以找到松下伺服驱动器的编码器分辨率。

success = DRIVE_READ($30000 + $102, 0)

IF success = FALSE THEN

  PRINT "读取驱动器参数错误"

ELSE

  PRINT "编码器分辨率 = ";VR(0)[0]

ENDIF

例6

下面的例程可以使用A4N来定位电机编码器上的Z脉冲。这是通过等待在驱动器上看到Z脉冲然后读取机械角度来工作的。 


pos = DRIVE_READ($30201)

oneturn=10000' 一圈的距离，取决于编码器类型



IF pos <> -1 THEN

  PRINT "机械补偿:";pos[0]

ELSE

  PRINT "驱动器尚未看到Z脉冲"

  MOVE(oneturn)

  WAIT UNTIL DRIVE_READ($30201)<>-1

  CANCEL

  WAIT IDLE

  pos = DRIVE_READ($30201)

  PRINT "机械补偿:";pos[0]

ENDIF

DEFPOS(pos)
