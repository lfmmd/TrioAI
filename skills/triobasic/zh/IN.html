IN

类型

I / O命令

语法

value = IN[(input_no[, final_input])]

描述

IN用于读取数字输入的状态。数字I/O通常是指附着在运动控制器上的24V输入和输出。没有硬件连接的I/O点可以用作“虚拟”输出和输入，可以用OP设置，用IN回读。

如果在没有参数的情况下调用，IN将返回前32个输入的二进制和。如果带一个参数调用，则返回该特定输入通道的状态(1或0)。如果用2个参数调用IN，则返回输入组的二进制和。

在2个参数的情况下，输入的间隔应该小于32。

IN等价于IN(0,31)

参数

value:所选输入或输入范围的状态
none:返回前32个输入的二进制和
input_no:输入返回输入组的值/起始
final_input:组的最后一个输入

例

例1

在这个例子中，测试一个单一的输入:

WAIT UNTIL IN(4) = ON

GOSUB place

例2

移动到拇指轮上设置的距离乘以一个系数。拇指轮连接到输入4,5,6,7，并以二进制编码的十进制输出。

move命令的构造顺序如下:

步骤1:IN(4,7)将得到一个数字0..15

步骤2:乘以1.5467得到所需的距离

步骤3:绝对移动到这个位置

WHILE TRUE

  MOVEABS(IN(4, 7) * 1.5467)

  WAIT IDLE

WEND

例3

测试输入2或3是否有一个是ON。

If (IN AND 12) <> 0 THEN GOTO start

'(Bit 2 = 4 + Bit 3 = 8) so mask = 12
