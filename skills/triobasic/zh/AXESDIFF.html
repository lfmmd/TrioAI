AXESDIFF

类型

轴命令

语法

AXESDIFF(axis1, axis2)

描述

AXESDIFF命令用于配置对ATYPE=85的轴上执行的2个轴的监视。ATYPE=85的轴将根据' axis2 '的MPOS减去' axis1 '的MPOS之间的差异产生FE输出，也将产生DAC输出。

如果需要对FE_LIMIT和FE_RANGE进行操作，则将AXIS_MODE的Bit13设置为1。有了这个设置，当FE超出设置的限制时，AXISSTATUS中的第1位和第8位将做出响应。

指定的轴可以是任何轴， 不必是系统中的物理轴

参数

Axis1:第一轴监控。

-1断开与另一轴的连接。
Axis2:第二轴监控。

-1断开与另一轴的连接。

例

例1

监控轴3和轴7。

BASE(10)

ATYPE = 85

AXESDIFF(3,7)

FE_RANGE = 25 '设置警告范围

FE_LIMIT = 50 ' 设置触发WDOG OFF的限制值

PRINT "轴位置差是";FE;" mm"

例2

使用AXESDIFF检查Trio Flexslice P375控制的步进电机和单独编码器的跟随误差，也反馈到P375。步进电机和编码器都有相同的每转步长和计数。

P375需要256倍频，它有一个内部分频器来平滑脉冲。

BASE(axis_f1)

ENCODER_RATIO(-256, 1)

STEP_RATIO(-1, 1)

UNITS = 20000 * 256 / 45

SERVO = ON

DEFPOS(0)



BASE(axis_f1_enc)

ENCODER_RATIO(256, 1)

UNITS = 20000 * 256 / 45

SERVO = ON

DEFPOS(0)



BASE(axis_f1_diff)

ATYPE = 85

AXIS_MODE.13 = 1 ' 启用 FE_LIMIT

UNITS = 20000 * 256 / 45

AXESDIFF(axis_f1, axis_f1_enc)

SERVO = ON

DEFPOS(0)
