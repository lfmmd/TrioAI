NW_TIMEOUT

类型

系统数组(MC_CONFIG)

语法

NW_TIMEOUT(slot, param_index) = value

描述

NW_TIMEOUT是一个基于插槽的系统参数，用于定义网络启动时EtherCAT网络超时参数。可以为启动序列的每个阶段或整个EtherCAT启动设置超时值。

参数

slot:EtherCAT端口的槽位索引。
param_index:标识要配置的超时参数。
索引描述
0整体ESM*启动超时
1初始化到预操作超时
2预操作到安全操作超时
3安全操作到操作超时
value:超时值。(单位取决于参数)

* ESM = EtherCAT状态机。

例

从站节点从预操作状态到安全操作状态需要4秒的时间。EtherCAT网络运行在运动控制器的0号槽位。将NW_TIMEOUT添加到MC_CONFIG中:

NW_TIMEOUT(0, 2) = 4000
