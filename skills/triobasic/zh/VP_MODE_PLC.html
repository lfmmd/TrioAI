VP_MODE_PLC

类型

系统参数

语法

VP_MODE_PLC = value

描述

系统参数VP_MODE_PLC设置PLC (IEC 61131-3)程序使用的备选速度轮廓模式类型。默认值为5，因为这包括适用于PLC-Open运动(MC)功能块的JERK和其他轴参数。

VP_MODE_PLC在所有支持PLC-Open的运动控制器中显示为MC_CONFIG关键字。

前缀为MC_的PLC-Open运动功能块需要VP_MODE 5

值

0 .. 6(默认= 5)

具体请参见VP_MODE。

例

IEC 61131-3 PLC程序Trio TC_ motion功能块使用传统的速度轮廓模式(VP_MODE) 0。在MC_CONFIG中设置VP_MODE_PLC为0，以便在上电时设置所需的模式。

VP_MODE_PLC = 0
