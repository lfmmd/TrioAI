FILE

类型

系统命令

语法

value = FILE "function" [parameters]

描述

该命令使用户能够管理SD卡上的数据。

当命令打印到所选通道时，可以使用OUTDEVICE选择该通道

参数

function:CD改变当前工作目录
COMPARE比较两个文件
COPY复制文件
CRC16打印文件的CRC-16(循环冗余校验)。
DEL删除文件
DETECT检查SD卡
DISK_FREE获取SD卡上的可用空间量
DISK_SIZE获取SD卡大小
DIR打印当前目录内容
FIND_CLOSE结束查找会话
FIND_FIRST在指定文件类型的目录结构中查找第一个条目
FIND_NEXT在指定文件类型的目录结构中查找下一个条目
FIND_PREV在指定文件类型的目录结构中查找上一个条目
FORMAT使用FAT32结构格式化SD卡
LOAD_PROGRAM将指定的程序加载到控制器内存中
LOAD_PROJECT将指定的项目加载到控制器内存中
LOAD_SYSTEM将指定固件加载到控制器中
MD创建目录
PWD打印工作(当前)目录
RD移除(删除)目录
REN重命名文件
SAVE_PROGRAM将指定程序保存到SD卡上
SAVE_PROJECT从控制器保存所有程序到SD卡
SIZE返回SD卡上文件的大小
TYPE打印选中的文件
parameters:依赖于函数
value:如果函数成功则返回TRUE，否则返回FALSE

当FILE命令在Motion Perfect终端0中使用时，它不返回值。相反，它会向终端报告一条错误信息。

Function = CD

语法

value = FILE "CD" "directory"

描述

更改到给定目录。控制器上有一个活动目录，所有SD卡命令都相对于这个目录。

参数

directory:string要移动到的子目录的名称
\移动到根目录
..向上移动一级到父目录

例

例1

使用命令行切换到新目录

>>file "CD" "new_directory"

OK \NEW_DIRECTORY

>>

例2

使用命令行切换到以下3层的新目录

>>file "CD" " project1\\project2\\project3"

OK \PROJECT1\PROJECT2\PROJECT3

>>

例3

使用命令行移动到根目录

>>file "CD" "\\"

OK \

>>

Function = COMPARE

语法

value = FILE "COMPARE" "file1" "file2"

描述

比较以字符串" file1 "命名的文件与以字符串" file2 "命名的文件。如果两个文件相同，则返回TRUE(-1)，否则返回FALSE(0)。

只能比较同一目录(文件夹)中的文件。

参数

File1:第一个文件的名称，包括文件的扩展名
File2:第二个文件的名称，包括文件的扩展名

例

比较DATA1.TXT和DATA2.TXT文件。

DIM filesSame AS BOOLEAN

filesSame = FILE "COMPARE" "DATA1.TXT" "DATA2.TXT"

IF filesSame THEN

   PRINT "文件相同"

ELSE

   PRINT "文件不同"

ENDIF

Function = COPY

语法

value = FILE "COPY" "file1" "file2"

描述

将字符串“file1”命名的文件复制到字符串“file2”命名的新文件。

文件只能复制到与源文件相同的目录(文件夹)中。

参数

File1:源文件的名称，必须包含文件扩展名
File2:包含扩展名的复制文件的名称

例

在SD卡上创建一个文本文件的副本。副本将与原始文件在同一目录下。

FILE "COPY" "13OKLZ-9.TXT" "13OKLZ-A.TXT"

Function = CRC16

语法

value = FILE "CRC16" "file"

描述

将当前目录中给定文件的CRC-16输出到当前输出通道。

参数

文件:要计算其CRC-16的文件的名称

例

使用命令行显示STARTUP.BAS文件的CRC-16。

>>FILE "CRC16" "STARTUP.bas"

51612

>>

在OUTDEVICE中给出了一个如何获取CRC作为值的示例。

Function = DEL

语法

value = FILE "DEL" "file"

描述

删除当前目录下的给定文件。

参数

file:要删除的文件名，必须包含文件扩展名

例

使用命令行从SD卡上删除BASIC程序。

>>FILE "DEL" "STARTUP.bas"

OK

>>

Function = DISK_FREE

语法

value = FILE "DISK_FREE"

描述

以整数值的形式返回SD卡上的空闲内存量，单位为字节。

例

检查已插入的SD卡大小。

DIM sd_free AS INTEGER

sd_free = FILE "DISK_FREE"

PRINT "SD card Free memory = "; sd_free / 1024;" kB"

Function = DISK_SIZE

语法

value = FILE "DISK_SIZE"

描述

以整数值返回SD卡的大小(以字节为单位)。

例

检查已插入的SD卡大小。

DIM sd_size AS INTEGER

sd_size = FILE "DISK_SIZE"

IF sd_size < 30000000000 THEN

  PRINT "Please insert 32GB or greater"

ENDIF

Function = DETECT

语法

value = FILE "DETECT"

描述

检查插槽中是否有SD卡

返回值

如果正确检测到SD卡，则为TRUE，否则为FALSE。

例

在保存表数据之前，请检查是否有SD卡。

IF FILE "DETECT" THEN

  STICK_WRITE(1501, 1000, 2000, 0)

ENDIF

Function = FIND_CLOSE

语法

value = FILE "FIND_CLOSE"

描述

关闭内部FIND结构。当以FIND_NEXT和FIND_PREVIOUS结束时使用。

Function = FIND_FIRST

语法

value = FILE "FIND_FIRST", type, vr_index

描述

初始化内部FIND结构并定位给定类型的第一个目录条目。找到的目录条目名称存储在一个VRSTRING中

参数

value:如果找到一个目录条目，则为TRUE，否则FALSE
type:1文件
2目录
vr_index:VRSTRING存储在VR内存中的起始位置

如果初始化内部FIND结构出错，则该函数返回FALSE。

Function = FIND_NEXT

语法

value = FILE "FIND_NEXT", vr_index

描述

查找对应的FIND_FIRST命令中给定类型的下一个目录条目。

参数

value:如果找到目录条目，则为TRUE，否则FALSE
vr_index:VRSTRING存储在VR内存中的起始位置

如果初始化内部FIND结构时出错，则该函数返回FALSE。

Function = FIND_PREV

语法

value = FILE "FIND_PREV", vr_index

描述

查找对应的FIND_FIRST命令中给定类型的上一个目录条目。

参数

value:如果找到目录条目，则为TRUE，否则FALSE
vr_index:VRSTRING存储在VR内存中的起始位置

如果初始化内部FIND结构时出错，则该函数返回FALSE。

Function = FORMAT

语法

value = FILE "FORMAT" "volume_name"

描述

使用FAT32标准格式化SD卡。注意，此操作将擦除SD卡上的所有文件。“volume_name”中给出的文本将应用于卡名。只能使用字母数字和下划线(_)字符。

参数

volume_name:要用作卡的顶层名称的名称。最多11个字符。

例

例1

格式化一个新的SD卡，并给它一个卷名。

>>FILE "FORMAT" "MACHINE_1"

100%

OK

>>

Function = LOAD_PROGRAM

语法

value = FILE "LOAD_PROGRAM" "file" ["txt"]

描述

将给定程序加载到_运动控制器_中。如果可选的"txt"字符串被去掉，则该命令将加载一个.BAS文件到BASIC程序。如果“txt”存在，则目标文件将作为程序类型Text加载到运动控制器中。

参数

file:要加载的文件的名称。

如果Motion Perfect以同步模式连接，则固件将阻止以这种方式加载文件。为了允许文件在开发期间被加载，运行函数COORDINATOR_DATA(53,1)'允许在MPv4的同步连接模式下加载文件。

文件加载后，必须重新同步Motion Perfect到项目。

Function = LOAD_PROJECT

语法

value = FILE "LOAD_PROJECT" "name"

描述

读取给定的_Motion_ Perfect项目文件并将所有程序加载到_运动控制器_中，一旦加载，任何RUNTYPEs都会自动设置。

参数

name:您希望加载的项目的名称。

Function = LOAD_SYSTEM

语法

value = FILE "LOAD_SYSTEM" "name"

描述

将系统固件加载到控制器上。

参数

name:您希望加载的固件文件的名称。

加载不正确的固件会导致控制器无法运行

参数

>>

Function = PWD

语法

value = FILE "PWD"

描述

将工作目录(当前在其中的目录)打印到当前输出通道。

Function = SAVE_PROGRAM

语法

value = FILE "SAVE_PROGRAM" "name" ["extension"]

描述

将控制器内存中的命名文件保存到SD卡中。如果不输入扩展名，则使用默认的文件扩展名BAS、TXT或TEMP。

参数

name:要保存到SD卡上的文件的名称。
extension:可选，定义要使用的文件扩展名

Function = SAVE_PROJECT

语法

value = FILE "SAVE_PROJECT" "name"

描述

在当前目录内创建一个给定名称的_Motion_ Perfect项目。这意味着在这个目录下创建目录以及相应的项目和程序文件。

参数

name:您要在SD卡上创建的项目的名称

Function = SIZE

语法

value = FILE "SIZE" "name"

描述

以字节为单位获取命名文件的大小。

参数

name:SD卡上文件的名称

Function = TYPE

语法

value = FILE "TYPE" "name"

描述

读取当前目录内文件的内容，并将其打印到当前输出通道。

参数

name:要打印的文件的名称
