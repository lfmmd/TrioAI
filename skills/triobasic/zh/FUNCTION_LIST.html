FUNCTION_LIST

类型

命令与功能

语法

有两种不同的语法:

第一个参数是字符串

FUNCTION_LIST(function[, parameter]])

第一个参数不是字符串

FUNCTION_LIST[([option[, library[, start index[, end index]]]])]

描述

FUNCTION_LIST输出带有参数和返回数据定义的当前可用函数列表。

参数

第一个参数是字符串
function:如果只有一个参数，则打印此函数的参数个数。
parameter:要打印其数据类型的函数参数的从0开始的索引。

第一个参数不是字符串
option:如果option为1，则打印函数名，否则打印函数详细信息。
library:如果library为空，则打印所有库中的函数，否则打印指定库中的函数。
start index:第一个要打印的函数以0为基础的索引。
end index:最后一个要打印的函数以0为基础的索引。

例

例1

在Motion Perfect命令行终端输入FUNCTION_LIST:

>>FUNCTION_LIST

f1(param1:INTEGER,param2:INTEGER,param3:BOOLEAN):INTEGER

enable_axes(ax0:INTEGER,ax1:INTEGER,ax2:INTEGER):NONE

disable_axes(ax0:INTEGER,ax1:INTEGER,ax2:INTEGER):NONE

set_rs232(baudrate:INTEGER):NONE

blink_3(op_number:INTEGER):NONE

get_decel_distance(f_decel:FLOAT,f_vel:FLOAT):FLOAT

f_cube(param1:FLOAT):FLOAT

>>

例2

DIM funclist AS STRING(256)
