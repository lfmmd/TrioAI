MOVEPICK

类型

轴命令

语法

MOVEPICK(mode, x, y, z, withdraw, overlap_control_A, overlap_control_B , u, v, w ,withdraw_control, xy_control, approach_control)

描述

MOVEPICK是一种新的移动类型，旨在使快速拾取和放置操作更简单。关联移动类型PICKLINK是一种链接移动，相当于MOVEPICK，它以类似于MOVELINK或FLEXLINK的方式链接到另一个轴上的测量运动。

MOVEPICK是3..6轴的绝对移动。然而，该动作包括一个增加的“撤回”量和移动速度轮廓的特殊控制，以允许快速拾取和放置移动。

多轴插值使用BASE轴的UNITS、SPEED、ACCEL和DECEL。

参数

mode:指定轴数0 = xyz, 1 = xyzu, 2 = xyzuv, 3 = xyzuvw
x:目标x绝对位置
y:目标y绝对位置
z:目标z绝对位置
withdraw:从z轴开始位置的增量退出距离
overlap_control_A:0 . .0.9指定角A的圆角(可选)
overlap_control_B:0 . .0.9指定角B的圆角(可选)
u:目标u绝对位置(可选)
v:目标v绝对位置(可选)
w:目标w绝对位置(可选)
withdraw_control:0 . . 1调整撤回速度(可选)
xy_control:0 . . 1调整xy截面的速度(可选)
approach_control:0 . . 1调整接近速度(可选)

弧度控制参数

重叠控制A - 0到 0.9;更大的数字用于第一个角的更大圆角(默认= 0.5)

重叠控制B - 0到0.9; 更大的数字用于第二个角的更大圆角(默认= 0.5)

注意，重叠控制参数应始终小于1.0，建议取值0->0.8。

速度覆盖控制参数

以下可选参数可用于覆盖3个运动部分中的每个部分的速度。值为1会产生指定的SPEED值。数值越小，速度越低。

withdraw_control -调整撤回部分的速度
xy_control -调整xy部分的速度
approach_control -调整接近部分的速度

在MOVEPICK中，加速速率由最短移动的长度控制。因此，如果其中一个动作明显小于其他动作，则可能导致高加速率。这3个参数:withdraw_control, xy_control和approach_control允许将较慢的速度分配给较短的移动，以允许较低的加速率。

MOVEPICK目前不支持较低的速度在接近运动结束。这将在未来的版本中得到支持。

例

例1

使用MOVEPICK转到xyz数组中设置的拾取位置，然后转到放置位置0,0,0。z轴从每个结束位置撤回500。

DIM xyz AS FLOAT(3) 



UNITS AXIS(0) = 1000 

UNITS AXIS(1) = 1000 

UNITS AXIS(2) = 1000 



SPEED = 5000 

ACC(SPEED * 50) 



'绝对位置:

xyz(0) = -1500 

xyz(1) = 0 

xyz(2) = -100 



overlap1 = 0.6 

overlap2 = 0.6 



DEFPOS(0,0,0) 



REPEAT 

    TRIGGER 

    MOVEPICK(0,xyz(0),xyz(1),xyz(2),500, overlap1, overlap2, 0, 0, 0, 1.0, 1.0, 1.0) 

    WAIT IDLE 

    WA(500) 

    MOVEPICK(0, 0, 0, 0, 600, overlap2, overlap1, 0, 0, 0, 1.0, 1.0, 1.0) 

    WAIT IDLE 

    WA(500) 

UNTIL FALSE

例2

3轴拾取和放置显示了撤回控制和xy控制的使用。

  BASE(axis1, axis2, axis3)

  DEFPOS(0, 0, 50)

  SPEED = 200

  ACCEL = 5000

  DECEL = 5000



  ' move到起始位置

  MOVEABS(100, 80, 50)

  WAIT IDLE

  ' 设置OP(16,ON)在终端开始重复序列

  WAIT UNTIL IN(16) = ON



  first_time = TRUE



  REPEAT

    SPEED = 500

    target_x = 150

    target_y = 100

    target_z = 20

    IF first_time = TRUE THEN

      withdraw = 10

      first_time = FALSE

    ELSE

      withdraw = 50 'Z将去的高度(相对距离 )

    ENDIF

    ov_ctrl_a = 0.5 '设置在0和0.9之间，越大的值意味着更弯曲

    ov_ctrl_b = 0.5 '设置在0和0.9之间，更大的值意味着更弯曲的方法

    wd_ctrl = 1 '撤回速度，设置在0.001和1.0之间

(使用的速度是SPEED * wd_ctrl)

    xy_ctrl = 0.6 'XY的移动速度，设置在0.001和1.0之间

    app_ctrl = 1 '接近的速度，设置在0.001和1.0之间



    TRIGGER



    MOVEPICK(0, target_x, target_y, target_z, withdraw, ov_ctrl_a, ov_ctrl_b, 0, 0, 0, wd_ctrl, xy_ctrl, app_ctrl)

    WAIT IDLE

    GOSUB pick



    SPEED = 500

    target_x = 50

    target_y = 80

    target_z = 0

    withdraw = 30

    ov_ctrl_a = 0.5

    ov_ctrl_b = 0.5

    wd_ctrl = 0.7

    xy_ctrl = 0.6

    app_ctrl = 1



    MOVEPICK(0, target_x, target_y, target_z, withdraw, ov_ctrl_a, ov_ctrl_b, 0, 0, 0, wd_ctrl, xy_ctrl, app_ctrl)

    WAIT IDLE

    GOSUB place



  UNTIL IN(16) = OFF

STOP



pick:

  WA(pick_delay)

  OP(32, ON) ' 吸入打开

  WA(suction_delay)



RETURN





place:

  WA(place_delay)

  OP(32, OFF)

  WA(drop_delay)



RETURN
