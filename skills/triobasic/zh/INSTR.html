INSTR

类型

字符串函数

语法

value = INSTR([offset-index, ]string, search-string[, wildcard-char])

描述

用于搜索字符串并返回字符串第一次出现的索引(从零开始的)，如果没有找到字符串则返回-1。

参数

Offset index:开始搜索的可选整数偏移量
string:被搜索的字符串
Search string:要搜索的字符串
Wildcard char:可选的通配符，在搜索字符串中使用，表示为单个字符串或数值ASCII值

例

例

预定义一个string类型的变量，并在其中搜索各种子字符串:

DIM str1 AS STRING(32)

str1 = "TRIO MOTION TECHNOLOGY"

PRINT INSTR(str1, "MOTION") 'value = 5

PRINT INSTR(6, str1, "MOTION") 'value = -1

PRINT INSTR("Value = 123.45E10", "###.##E##", "#") 'Value = 8

PRINT INSTR("this is my string", "is *y", 42) 'Value = 5

PRINT INSTR(3, str1, "IO") 'Value = 8
