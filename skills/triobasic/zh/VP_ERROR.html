VP_ERROR

类型

轴参数(只读)

语法

value = VP_ERROR AXIS(n)

描述

轴参数VP_ERROR允许读取VPU的错误状态。

只有VPU轴的VP_ERROR会被更新。

仅对VP_MODE 6有效-所有其他模式将只返回0。

值

0无错误发生
1VPU未初始化
2模式改变错误— VPU的模式只能在空闲时改变
3位置给定错误—只有在VPU处于位置模式时才允许有位置给定
4速度给定错误—在位置模式下速度必须大于零
5加/减速给定错误—值必须大于零
6加速给定错误—值必须大于零
7减速给定错误—值必须大于零
8加加速度给定错误—值必须大于零
9轮廓时间错误—值必须大于零且小于4ms
10继续运行错误—可能只会在暂停之后才继续

例

AXISSTATUS设置了33位，表示存在VP_ERROR，因此打印错误代码:

PRINT VP_ERROR
