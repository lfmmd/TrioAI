() 数组索引

类型

程序变量

语法

variable(index-0[, index-1[, index-2]]])

描述

允许通过指定所需元素的索引来访问数组的元素。

数组在DIM或STATIC语句中通过在数据类型后面指定每个维度的大小来声明。维度的索引从0到维度的大小减1。

参数

variable:被索引的变量
index-0:第一维的索引
index-1第二维的索引
index-2第三维的索引

例

例1

设置数组' a '的元素0为0.0，然后显示它的值。

DIM a AS FLOAT(6)



a(0) = 0.0

PRINT "a(0) = "; a(0)

例2

显示一个二维数组的所有元素。

DIM a AS FLOAT(2, 3)

DIM i, j AS INTEGER



FOR i = 0 TO 1

    FOR j = 0 TO 2

        PRINT a(i, j);

        IF j < 2 THEN PRINT ", ";

    NEXT j

    PRINT

NEXT i

例3

单独设置二维数组的每一行。

DIM a AS FLOAT(2, 3)



a(0) = 1, 2, 3

a(1) = 4, 5, 6
