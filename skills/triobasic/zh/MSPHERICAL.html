MSPHERICAL

类型

轴命令

语法

MSPHERICAL({parameters}, mode [, gtpi][, rotau][, rotav][, rotaw])

描述

沿着球面路径移动BASE中定义的三轴组，其矢量速度BASE数组第一个轴中设置的由speed决定。有2种操作模式，可以选择在不同于起始点的端点完成移动，或者返回起始点完成一个圆。3D空间中移动的路径可以通过指定路径上某处的一个点来定义，也可以通过指定球体的中心来定义。

参数

mode:0在曲线上指定终点和中点。
1指定球体的端点和中心。
2指定两个中点，曲线完成一个完整的圆。
3指定曲线中点和球心，曲线完成一个完整的圆。
gtpi:如果这个可选参数为非零，模式0和模式1将执行一个移动，沿着相反的方向绕360度圆到达相同的端点。
rotau:如果此可选参数非零，则在球面移动的同时，4th轴将执行线性插值。轴是BASE序列中的下一个。移动距离不影响路径长度或移动所需的时间。路径长度只是根据球面距离计算的。
rotav:如果这个可选参数是非零的，一个5th轴将在球面移动的同时执行线性插值。
rotaw:如果这个可选参数是非零的，一个6th轴将在球面移动的同时执行线性插值。

如果您将第三个轴的参数指定为0并将其分配给虚拟轴，则可以使用MSPHERICAL来执行圆周运动。这允许你在不知道中心点的情况下指定圆弧。

Mode = 0

语法

MSPHERICAL(endx, endy, endz, midx, midy, midz, 0)

描述

移动三个轴，通过指定穿过球体的一个部分的曲线上的终点和中点，设置BASE数组。

参数

endx:第一轴的结束位置
endy:第二轴的结束位置
endz:第三轴的结束位置
midx:第一轴的中间位置
midy:第二轴的中间位置
midz:第三轴的中间位置

Mode = 1

语法

MSPHERICAL(endx, endy, endz, centrex, centrey, centrez, 1)

描述

通过指定球体的端点和中心，移动BASE数组中设置的三轴，穿过球体的一个部分。轮廓线将始终走到端点的最短路径，这可能是顺时针或逆时针。

中心点和终点的坐标不能共线。半圆不能用模式1来定义，因为球体中心将与端点共线。如果指定了共线性点，控制器将以RUN_ERROR停止程序。

参数

endx:第一轴的结束位置
endy:第二轴的结束位置
endz:第三轴的结束位置
centrex:第一轴的位置
centrey:第二轴的中心位置
centrez:第三轴的中心位置

Mode = 2

语法

MSPHERICAL(midx1, midy1, midz1, midx, midy, midz, 2)

描述

通过指定曲线的两个中点，移动BASE数组中设置的三轴，在球体上穿过一个完整的圆。轮廓线将通过第一个中点移动，然后是第二个，最后回到起点。

参数

midx1:第一轴的第二个中间位置
midy1:第二轴的第二个中点位置
midz1:第三轴的第二个中点位置
midx:第一轴的第一个中间位置
midy:第二轴的第一个中间位置
midz:第三轴的第一个中间位置

Mode = 3

语法

MSPHERICAL(midx, midy, midz, centrex, centrey, centrez, 3)

描述

通过指定一个中点和球体的中心，移动BASE数组中设置的三轴在球体上穿过一个完整的圆。轮廓线将以距离中点最短距离的方向开始，这使您可以定义方向。

中心点和中点的坐标不能共线。如果指定了共线性点，控制器将以RUN_ERROR停止程序。

参数

midx:第一轴的中间位置
midy:第二轴的中间位置
midz:第三轴的中间位置
centrex:第一轴的位置
centrey:第二轴的中心位置
centrez:第三轴的中心位置

例

例1

移动需要遵循一个球形路径，在Z方向向上30毫米结束:

radius = 30 '球半径

angle= 0 '弧度



startx = 0

starty = 0

startz = 0 



endx = startx + radius * COS(angle)

endy = starty + radius * SIN(angle)

endz = startz + radius



midx = (endx - startx) * (1 - COS(PI / 4))

midy = (endy - starty) * (1 - COS(PI / 4))

midz = (endz - startz) * SIN(PI / 4)



BASE(3,4,5)

MSPHERICAL(endx, endy, endz, midx, midy, midz, 0)

例2

类似的移动，沿着球面路径，但与Y轴成45度，在XY平面上方30毫米处结束:

radius = 30 '球半径

angle = PI/4 '弧度



startx = 0

starty = 0

startz = 0



endx = startx + radius * COS(angle)

endy = starty + radius * SIN(angle)

endz = startz + radius



midx = (endx - startx) * (1 - COS(PI / 4))

midy = (endy - starty) * (1 - COS(PI / 4))

midz = (endz - startz) * SIN(PI / 4)



BASE(0, 1, 2)

MSPHERICAL(endx, endy, endz, midx, midy, midz, 0)
