ENCODER_WRITE

类型

轴功能

语法

Value = ENCODER_WRITE (address, data)

描述

在EnDat绝对值编码器上写一个内部寄存器到绝对值编码器。

参数

value:如果写成功返回TRUE，如果写失败返回FALSE
address:要写入的EnDat编码器寄存器的地址
data:要写入指定寄存器的值。

例

将值写入EnDat编码器并检查是否已写入，然后将编码器设置回位置模式

IF NOT ENCODER_WRITE (endat_address, setvalue) THEN

  PRINT "写入编码器失败"

ENDIF
