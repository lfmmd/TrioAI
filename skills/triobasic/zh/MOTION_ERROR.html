MOTION_ERROR

类型

系统参数(只读)

描述

motion_error系统参数提供了一个或多个轴出错的指示，并为每个出错的轴设置了一个位。

值

表示每个出错轴的位的总和。
位值轴
010
121
242
383
…
31214748364831

例

例1

轴0、轴1和轴3有错误，轴3上的错误首先发生。

>>?MOTION_ERROR

11

>>?ERROR_AXIS

3

例2

当运动错误发生时，这个程序例程检查每个出错轴的轴状态，并向终端5打印一条消息。

IF MOTION_ERROR <> 0 THEN

  FOR a = 0 to 23

    IF MOTION_ERROR.a = 1 THEN

      IF AXISSTATUS AXIS(a).2 = 1 THEN

        PRINT #5, "Communication error on axis "; a[0]

      IF AXISSTATUS AXIS(a).3 = 1 THEN

        PRINT #5, "Remote drive error on axis "; a[0]

      IF AXISSTATUS AXIS(a).8 = 1 THEN

        PRINT #5, "FE exceeded FE_LIMIT on axis "; a[0]

      ENDIF

    ENDIF

  NEXT a

ENDIF
