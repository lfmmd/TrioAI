: -冒号

类型

特殊字符

描述

冒号字符用作标签结束符和命令分隔符。

标签终结符

语法

label:

描述

冒号字符用于终止用作GOTO和GOSUB命令目标的标签。

标签也可以用来提高代码的可读性。

参数

Label可以是任意长度的字符串，但只有前32个字符是有效的。标签必须是一行的第一项，不应有前导空格。

例

使用ON…GOTO结构根据局部变量“attempts”为VR 10分配一个值。

ON attempts GOTO label1, label2, label3

GOTO continue



label1:

VR(10)=1

GOTO continue



Label2:

VR(10)=5

GOTO continue



Label3:

VR(10)=2

GOTO continue



continue:

命令分离器

语法

statement: statement

描述

冒号还用于在多语句行中分隔TrioBASIC语句。

参数

statement:任何有效的TrioBASIC语句。在多行IF..THEN结构中，冒号分隔符不能在THEN命令之后使用。

如果多语句行包含一个GOTO，其余语句将不会被执行。GOTO类似于GOSUB，因为子例程的调用返回到下一行。

例

例1

在行中使用GOTO意味着它后面的任何命令都不会被执行。这可以用作调试，但通常是由于编程错误而发生的。

PRINT "Hello":GOTO Routine:PRINT "Goodbye"

“Goodbye”不会被打印。

例2

设置速度，在表中的位置和执行移动都在一行中。

SPEED=100:TABLE(10,123):MOVE(TABLE(10)
