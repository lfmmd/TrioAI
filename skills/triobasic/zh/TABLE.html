表格

类型

系统命令

语法

value = TABLE(address[, data0..data35])

描述

TABLE命令可用于加载和读取内部TABLE值。由于table可以被写入和读取，它可以被用来保存信息，作为变量的替代。

table的值是浮点数，因此可以是小数。

您可以使用NEW " TABLE "清除TABLE

一些TABLE元素会根据控制器类型自动备份。使用FLASHTABLE命令强制存储TABLE数据。

参数

value:返回存储在地址上的值，如果用于写操作，则返回-1
address:写的第一点的地址，或读的地址
data0:写入地址的数据
data1:写入地址+ 1的数据
data2:写入地址+ 2的数据
…
data35写入地址+ 35的数据

例

例1
