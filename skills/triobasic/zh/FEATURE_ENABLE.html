FEATURE_ENABLE

类型

系统命令

语法

FEATURE_ENABLE[(feature_number[, "password"])]

描述

运动控制器可以通过输入“Feature Enable Code”或FEC来解锁其他功能。此功能用于启用受保护的功能，例如数字网络或其他编程语言上的附加远程轴。这只能在命令行上运行。

建议使用Motion Perfect来输入和存储功能启用代码。

password参数为可选参数，如果省略则命令会提示输入。

您可以从Trio网站或通过经销商购买额外的功能代码，您将需要控制器的SERIAL_NUMBER。

如果您输入错误的密码3次，控制器将进入保护状态，停止通信。通过对控制器重新上电，即可恢复正常操作。

参数

feature_number:None打印安全代码和当前启用的功能。
01个远程轴
12个远程轴
24个远程轴
38个远程轴
416个远程轴
532个远程轴
6-11保留使用
121个远程轴
132个远程轴
144个远程轴
158个远程轴
1616个远程轴
1732个远程轴
18-20保留使用
21IEC runtime
22-31轴升级
24-31保留使用
password:所需功能码的密码

启用功能时，需要输入密码。

输入密码时，请始终以大写字母输入。注意检查0(零)不会与O混淆，1(一)不会与I混淆。

例

例1

查看控制器上已启用的功能

>>FEATURE_ENABLE

Security code=17980000000028

Enabled features: 0 1

功能0和1被启用，因此在模块包含的内置轴上有额外的3个轴。

例2

使能额外的轴4 (功能2)。对于这个控制器和这个功能，密码为5P0APT。

>>FEATURE_ENABLE(2)

Feature 2 Password=5P0APT

>>

>>FEATURE_ENABLE

Security code=17980000000028

Enabled features: 0 1 2
