INT

类型

数学函数

语法

value = INT(expression)

描述

INT函数返回一个数字的整数部分。

要将一个正数四舍五入到最接近的整数值，取(数+ 0.5)的函数INT

参数

expression:任何有效的TrioBASIC表达式。
value:表达式的整数部分

例

例1

在命令行上打印数字的整数部分

>>PRINT INT(1.79)

1

例2

函数将值四舍五入到最接近的整数。

在函数库中:

FUNCTION r_int(value AS FLOAT) AS INTEGER

    IF value>0 THEN

      rounded = INT(value + 0.5)

    ELSE

      rounded = INT(value - 0.5)

ENDIF

    RETURN rounded

ENDFUNC

BASIC程序中的示例调用:

X = r_int(1.625)
