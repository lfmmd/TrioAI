ROBOT_ERROR

类型

机器人参数(只读)

语法

ROBOT_ERROR(group)

描述

返回RPS 2.0内核中存在的错误。可能存在多个错误，下表中的值显示了每个位集代表的内容。

ROBOT_ERROR需要RPS run-time FEC

RPS 2.0内核必须使用RPS_KERNEL_V2关键字激活才能使用。

值

位值错误
01无效的I / O
12无效的按键切换
24运动组在激活时改变
38运动组I/O在激活时改变
416运动组未设置时激活
532未设置I/O时激活运动组
664工作模式在未空闲时改变
7128安全条件触发
8256触发紧急停止
9512未启用驱动器

例

例1

在机器人(0)中检测到紧急停止激活。

IF (ROBOT_ERROR(0) AND 256) <> 0 THEN

   '紧急停止已被触发

ENDIF
