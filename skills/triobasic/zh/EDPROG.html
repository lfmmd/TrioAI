EDPROG

类型

系统命令

语法

EDPROG [parameters,] function

替代格式

& function[, parameters]

描述

这是一个特殊的命令，可以用来操作控制器上的SELECT程序。

除Motion Perfect外，通常不使用该命令。

功能

1I插入字符串
2S搜索字符串
3D删除行
4l打印行
5N打印行数
6A打印标签地址
7C打印当前选定程序的名称
8R替换行
9K打印校验和
10Z打印指定程序的校验和
11X打印目标代码校验和
12Q检查控制器目录是否损坏
13V打印变量列表
14M提交修改

Function A

语法

EDPROG 6, to_line, from_line

替代语法

& from_line, to_line A

描述

打印“SELECT”程序中定义的区域中的所有标签名称。

参数

from_line:要搜索的SELECT程序的第一行
to_line:要搜索的SELECT程序的最后一行

Function C

语法

EDPROG c

替代语法

& C

描述

打印当前SELECT程序的名称。

Function D

语法

EDPROG 3, line_no

替代语法

& line_no D

描述

删除指定行

参数

line_no:SELECT程序的任何有效行号

Function I

语法

EDPROG string, 1, line_no .

替代语法

& line_no I,string

描述

将文本字符串插入当前选定程序的指定行。

除非需要插入到程序中，否则不应该将字符串括在引号中。

参数

line_no:插入字符串的行
string:要插入到SELECT程序中的文本字符串

Function K

语法

EDPROG 10

替代语法

& K

描述

打印系统软件的校验和

Function L

语法

EDPROG 4, end, start

替代语法

& start, end L

描述

在开始和结束之间打印当前选定程序的行

参数

start:从SELECT程序中打印的第一行
end:要从SELECT程序中打印的最后一行

Function M

语法

EDPROG 14

替代语法

& M

描述

将所有程序更改保存到flash中。

Function N

语法

EDPROG 5

替代语法

& N

描述

打印当前SELECT程序中的行数

Function Q

语法

EDPROG 12

替代语法

& Q

描述

返回控制器程序内存的状态。

返回值

0控制器内存OK
1控制器内存损坏

Function R

语法

EDPROG string, 8, line

替代语法

& line R, string

描述

将当前SELECT程序中的行<line>替换为文本<string>。

除非需要插入到程序中，否则不应该将字符串括在引号中。

参数

line_no:要替换的行
string:要替换SELECT程序中的行的文本字符串

Function S

语法

EDPROG string, 2, to_line, from_line

替代语法

& from_line, to_line S string

描述

打印字符串在SELECT程序中定义的区域中第一次出现的行号。

参数

from_line:要搜索的SELECT程序的第一行
to_line:要搜索的SELECT程序的最后一行
string要搜索的字符串

Function V

语法

EDPROG 13

替代语法

& V

描述

打印SELECT程序中定义的所有变量。

Function X

语法

EDPROG 11

替代语法

& X

描述

打印SELECT程序的16位CRC校验和。

Function Z

语法

EDPROG progname, 10

替代语法

& Z, progname

描述

打印指定程序的CRC校验和。

返回值

使用标准CCITT 16位生成器多项式返回校验和。
