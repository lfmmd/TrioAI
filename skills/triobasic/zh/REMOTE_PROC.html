REMOTE_PROC

类型

系统参数(MC_CONFIG / FLASH)

描述

当TrioPC ActiveX打开与_运动控制器_的同步连接时，REMOTE_PROGRAM在最高可用进程上启动。可以设置REMOTE_PROC，为REMOTE_PROGRAM指定一个不同的进程。如果定义的进程正在使用中，那么下一个较低的可用进程将被使用。

REMOTE_PROC存储在Flash EPROM中，也可以在MC_CONFIG脚本文件中设置。

值

-1使用最高可用进程(默认)
-2关闭Active X处理

设置为此值时将无法连接
0 to max process在已定义进程上运行

建议在MC_CONFIG中设置与TrioPC Motion ActiveX有连接的REMOTE_PROC参数。

例

例1

将REMOTE_PROGRAM设置为在进程19或更低的进程上启动(使用命令行终端)。

>> REMOTE_PROC =19

>>

在命令行终端设置完值后重置运动控制器。

例2

删除REMOTE_PROC设置，以便REMOTE_PROGRAM在默认进程上启动(使用MC_CONFIG)。

'MC_CONFIG脚本文件

REMOTE_PROC = -1 '连接时启动默认进程
