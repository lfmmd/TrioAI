V_LIMIT

类型

轴参数

语法

V_LIMIT AXIS(n) = value

描述

此参数为轴设置速度限制，以便根据需要降低矢量路径速度，以确保具有限制设置的每个轴的速度不超过其限制。

当与MOVECIRC和MOVECIRCSP命令一起使用时，路径速度在圆周上是恒定的，速率由移动中每个轴的最低V_LIMIT确定。

值

以用户UNITS为单位的特定轴允许的最大速度，或小于或等于零的值以禁用速度限制。

例

设置轴速限制，使x轴的运行速度不超过40mm /s, y轴的运行速度不超过80mm /s。

DIM x AS INTEGER = 0

DIM y AS INTEGER = 1



V_LIMIT AXIS(x) = 40

V_LIMIT AXIS(y) = 80



BASE(x, y)

SPEED = 200

SERVO = ON



MOVE(0, 500) '沿着y轴移动，所以以80毫米/秒的速度移动

WA(100)

MOVE(500, 0) '沿着x轴移动，所以以40毫米/秒的速度移动

WA(10)

MOVE(500, 500) '以45度移动，所以VP_SPEED是56.5685 mm/s
