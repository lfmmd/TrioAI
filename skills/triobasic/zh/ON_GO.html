ON .. GOSUB/GOTO

类型

程序结构

语法

ON expression GOxxx label[, label2[, ...]]

...

label1:

commands

RETURN

...

label2:

commands

RETURN

其中GOxxx可以是GOSUB或GOTO

描述

对表达式求值，然后使用整数部分从列表中选择一个标签。如果表达式的值为1，则使用第一个标签，2则使用第二个标签，依此类推。一旦选择了标签，它就可以与GOSUB或GOTO一起使用

如果表达式的值小于1或大于标签数，则命令将逐级执行而不进行任何操作。一旦选择了标签，就执行GOSUB。

参数

expression:任何有效的TrioBASIC表达式都应该返回1或更大的值
commands:要执行的TrioBASIC语句
label:在程序中出现的有效标签。
GOxxxGOSUB或GOTO

如果标签不存在，将在运行时显示错误信息，并停止程序执行。

例

例1

REPEAT

  GET #3, char

UNTIL char >= 1 AND char <= 3

ON char GOSUB mover, stopper, change

例2

使用PLC的输入来决定运行哪个程序。

ON (IN(4, 6) + 1) GOTO prog0, prog1, prog2, prog3, prog '选择程序

GOTO continue '如果选择了未知输入，则跳过progs



prog0:

  RUN "tuning", 2

  GOTO continue

prog1:

  RUN "cutting", 2

  GOTO continue

prog2:

  RUN "packing", 2

  GOTO continue

prog3:

  RUN "moving", 2

  GOTO continue

Prog4:

  RUN "lifting", 2

  GOTO continue



continue:

  ...

示例3

在终端0打印“2”和“完成”。

DIM a AS INTEGER



a = 2



ON a GOSUB one, two, three

PRINT "Done"



STOP '当机器停止时防止运行到子程序。



one:

PRINT "1"

RETURN



two:

PRINT "2"

RETURN



three:

PRINT "3"

RETURN

例4

当程序发生错误时，将错误打印到终端并停止所有运动，只有在错误例程由于程序正常停止而未执行的情况下。

ON BASICERROR GOTO error_routine

'(程序的剩余部分)



'Run time错误处理

error_routine:

  IF RUN_ERROR <> 31 THEN

    '如果有一个运行时错误，然后停止机器按照程序停止

        RAPIDSTOP (2)



    '如果程序没有正常停止，打印运行错误

    PRINT "RUN_ERROR = "; RUN_ERROR[0], "ERROR_LINE = "; ERROR_LINE[0]

  ENDIF

STOP
