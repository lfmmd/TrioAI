CPU_TEMP_MAX

类型

系统参数

语法

value = CPU_TEMP_MAX

CPU_TEMP_MAX = value

描述

这可以记录CPU达到的最高温度。

它可以通过写入一个以摄氏度(°C)为单位的值来重置，通常为零。

CPU_TEMPMAX不适用于 Euro404/408, MC403, MC403-Z, MC405, MC405-Y, MC4NE 和MC4N-RTEX.

例

例1

在命令行上显示CPU已达到的最高温度。

PRINT CPU_TEMP_MAX

将显示:

58.1

例2

重置最高温度。

CPU_TEMP_MAX = 0
