IP_PROTOCOL_CTRL

类型

系统参数(MC_CONFIG)

描述

该参数允许用户禁用以太网上一个或多个通信端口的操作。如果某位为0，则表示该端口已启用。如果该位为1，则端口被禁用，并且当客户端试图打开它时不会响应。当BASIC程序在上电时配置协议设置时，它可以用来暂时禁用端口。设置完成后，该端口将在程序中启用。

默认情况下，所有端口都是启用的。

值

最多可选择2位，默认值为0。
位描述值
0Motion Perfect (Telnet)n/a
1PCMotionn/a
2Modbus4
3EthernetIP8
4IEC61131-3编程n/a
5Uniplayn/a
6传输协议文本文件加载器n/a
7保留位n/a
8保留位n/a
9MC4xx协议文本文件加载器n/a

不能禁用任何标记为n/a的端口。

例1

关闭Modbus TCP端口，直到它在BASIC启动程序中设置为32位数据大小。

  • 在MC_CONFIG设置中:
IP_PROTOCOL_CTRL = 4
  • 在启动BASIC程序设置中:
ETHERNET(1, -1, 14, 0, 2, 1) '32位整数支持

ETHERNET(1, -1, 14, 0, 1, 4) '数据进出TABLE内存

ETHERNET(1, -1, 14, 0, 6, 1) '使用"地址减半"



IP_PROTOCOL_CTRL = 0 '启动Modbus TCP协议

例2

禁用以太网IP端口，直到在BASIC启动程序中设置了数据端点。

  • 在MC_CONFIG设置中:
IP_PROTOCOL_CTRL = 8
  • 在启动BASIC程序设置中:
'配置*PLC输入*实例(100)，数据从Trio到PLC。

ETHERNET(1, -1, 14, 1, 0, 200) '200 = 设置VR开始地址

ETHERNET(1, -1, 14, 1, 1, 3) '3 = 使用VR定位数据

ETHERNET(1, -1, 14, 1, 2, 1) '1 = 使用32位整型数据

ETHERNET(1, -1, 14, 1, 3, 120) '120 = 数据值号码



'——配置*PLC输出*实例(101)，数据从PLC到Trio。

ETHERNET(1, -1, 14, 2, 0, 400) '400 = 设置VR开始地址

ETHERNET(1, -1, 14, 2, 1, 3) '3 = 使用VR定位数据

ETHERNET(1, -1, 14, 2, 2, 1) '1 = 使用32位整型数据

ETHERNET(1, -1, 14, 2, 3, 120) '120 = 数据值号码



'-------------------------------------------------------------



IP_PROTOCOL_CTRL = 0 '启用以太网 IP
