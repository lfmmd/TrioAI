SET_EIP_TAG

类型

系统功能

语法

SET_EIP_TAG(name, mem_area, address, data_type)

描述

定义一个新的CIP标签条目(符号段数据类型)，或者如果已经定义了标签名称，则更新现有标签的配置。

如果在没有任何参数的情况下调用此命令，则所有当前定义的标签将被删除。

参数

name:标签的名称(数据类型为字符串)
mem_area:1: VR, 2: table
address:标签内存区域内的地址
data_type:标签的数据类型，使用CIP规范卷1 C-6.1节中定义的值。
目前支持的数据类型包括:
数据类型名称数据类型代码描述
SINT0xC2有符号8位整数
INT0xC3有符号16位整数
DINT0xC4有符号32位整数
USINT0xC5无符号8位整数
UINT0xC6无符号16位整数
UDINT0xC7无符号32位整数

返回值

TRUE (-1) : Successful, FALSE (0) : Failed

例

例1

为VR(200)和VR(201)中的x和y位置设置标签名称

SET_EIP_TAG("x_pos",1,200,$c3)

SET_EIP_TAG("y_pos",1,201,$c7)

在终端中按命令显示标签:

>>GET_EIP_TAG

x_pos: Memory: 1 (Vr), Address: 200, Datatype: 0xC3

y_pos: Memory: 1 (Vr), Address: 201, Datatype: 0xC7

例2

更改示例1中' y_pos '标签的地址

>>SET_EIP_TAG("y_pos",1,150,$c7)

显示标签

>>GET_EIP_TAG                   

x_pos: Memory: 1 (Vr), Address: 200, Datatype: 0xC3

y_pos: Memory: 1 (Vr), Address: 150, Datatype: 0xC7

>>

例3

删除所有准备重新开始配置标签的标签。

SET_EIP_TAG

显示标签存储，确认为空。
>>GET_EIP_TAG
>>
