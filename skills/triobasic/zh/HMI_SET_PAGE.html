HMI_SET_PAGE

类型

系统命令

语法

HMI_SET_PAGE(<name>[,<ip>])

描述

请求HMI客户端更改为给定的页面。如果没有指定IP地址，请求将被发送到所有当前连接的客户端。在提交新请求之前，该命令将等待所有挂起的HMI客户机请求完成，但它不会等待HMI客户机完成请求。这意味着控制器将继续运行软件，而无需等待所请求的页面显示在HMI客户机上。

参数

nameHMI客户机中HMI设计页面的名称。此名称区分大小写。
IP该消息必须发送到的HMI客户机的IP地址。

例

例1

以5秒的时间间隔自动滚动三个页面。如果一个页面是手动选择的，那么保持这个页面30秒。将页面值从HMI设置为大于3的值，以使页面处于手动模式。

page = 0

page_time = 5000

manual_time = 30000



WHILE(1)

  IF page = 0 THEN

    HMI_SET_PAGE("PAGE1")

    page = 1

    WA(page_time)

  ELSEIF page = 1 THEN

    HMI_SET_PAGE("PAGE2")

    page = 2

    WA(page_time)

  ELSEIF page = 2 THEN

    HMI_SET_PAGE("PAGE3")

    page = 3

    WA(page_time)

  ELSE

    '手动模式

    page = 0

    TICKS = manual_time

    WHILE TICKS>0

      IF page <> 0 THEN

        TICKS = manual_time

        page = 0

      ENDIF

      WA(1)

    WEND

  ENDIF

WEND

例2

设置要在连接到运动控制器的多个Uniplay hmi上显示的页面。

HMI_SET_PAGE("PAGE1","10.200.0.20")

HMI_SET_PAGE("PAGE1A","10.200.0.21")

HMI_SET_PAGE("PAGE3B","10.200.0.22")

一次只能有一个HMI与一个页面进行交互。如果2个或更多hmi必须显示相同的信息，最佳实践是有多个页面副本。例如，PAGE1和PAGE1A是运动控制器中同一页面的两个副本。
