FLAGS

类型

逻辑和位运算符

语法

value = FLAGS([state])

描述

读取或设置32位FLAGS作为块。

提供FLAGS命令是为了帮助与早期控制器兼容，不建议用于新程序。

参数

value:无参数=返回所有标志位的状态
带参数=返回-1
state:要设置标志的位模式的十进制等效值

例

例1

设置标志1、4和7为ON，其他所有为OFF
Bit #76543210
Value1286432168421

FLAGS(146)' 2 + 16 + 128

例2

测试是否设置了FLAG 3。

IF (FLAGS and 8) <>0 then GOSUB somewhere
