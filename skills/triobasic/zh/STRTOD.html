STRTOD

类型

字符串函数

语法

value = STRTOD(format, ...)

描述

STRTOD命令读取一个字符序列并将它们转换为一个数值。转换在输入字符流中发现的第一个无效字符处停止。字符可以从VR数组或TrioBASIC I/O通道中读取。

值可以是整数或浮点数(带和不带前导/尾数字)，可以是有符号的，可以包括指数(带和不带符号)。以下是有效值的示例:

123

123.456

+12.6

-0.12

.125

12.

12.7E-3

0.1e10

1.6e+3

参数

格式

这是一个按位字段，指定数据源和数字格式。
格式:描述:
位0源
0VR数组
1TrioBASIC I/O通道
位 1..2数字格式
0浮点
1整数。如果数字不是整数，则返回0。
2格式是自动选择的，以提供最佳的分辨率。
3格式为字母+值。

Source = 0

语法

value = STRTOD(format, vr_start, vr_index)

描述

将VR数组中的字符转换为数字。

参数

参数:描述:
vr_start数字字符串在VR数组中的第一个字符的位置。
vr_index在VR数组中的位置，存储找到的第一个非数字字符的索引。

Source = 1

语法

value = STRTOD(format, channel, vr_length, vr_index)

描述

将TrioBASIC通道中的字符转换为数字。

参数

参数:描述:
channelTrioBASIC I/O通道读取。这可以是任何有效的TrioBASIC I/O通道:标准通信通道、ANYBUS通道或文件通道。
vr_length在VR数组中的位置，以存储已解析的数字字符串的长度。
vr_index用于存储终止转换的字符的ASCII码的VR数组中的位置。

Number Format = 3

语法

value = STRTOD(format, channel, vr_length, vr_term, vr_index)

描述

将具有LETTER+VALUE格式的字符串(例如A123.456)从源通道转换为VR内存中从vr_index开始的一系列数字。返回一个值，该值是一个位模式，表示在函数调用期间已经解析了哪些代码值。

找到的每个字母在VR中都有相应的值，其中VR按字母顺序从vr_index开始。例如，如果在字符串中找到M123，则将值123放在VR(vr_index + 13)中，因为M是字母表中的第13th个字母。在返回值中也设置了第13位，以表示存在M值。

这种格式对于解析CNC应用中的G-Code字符串很有用。参见例4。

参数

参数:描述:
channelTrioBASIC I/O通道读取。这可以是任何有效的TrioBASIC I/O通道:标准通信通道、ANYBUS通道或文件通道。
vr_lengthVR数组中的位置，用于存储已解析字符串的长度。
vr_termVR数组中的位置，用于存储在字符串末尾找到的终止字符。
vr_indexVR数组中的位置，以存储代码列表中第一个值的索引。第一项为A码值，最后一项为Z码值。(分别为vr_index + 0和vr_index + 25)。

例1

打开文件并输出一个浮点值并关闭它，然后重新打开文件并使用STRTOD将其作为浮点数读取。

OPEN #40 AS "N" FOR OUTPUT(1)

PRINT #40, "123.456"



CLOSE #40

OPEN #40 AS "N" FOR INPUT

VR(100) = STRTOD(1, 40, 101, 102)



PRINT #5, "Value = "; VR(100)

PRINT #5, "Length of string = "; VR(101)[0]

PRINT #5, "Terminating character = "; VR(102)[0]



CLOSE #40

DEL "N"

产生输出:

Value = 123.4560

Length of input = 7

Terminating character = 13

值为123.4560，从7个输入字符中读取，并以回车符结束(ASCII码13)。

例2

打开文件并输出一个浮点值并关闭它，然后重新打开文件并使用STRTOD将其作为整数读取。

OPEN #40 AS "N" FOR OUTPUT(1)

PRINT #40, "123.456"



CLOSE #40

OPEN #40 AS "N" FOR INPUT

VR(100) = STRTOD(3, 40, 101, 102)



PRINT #5, "Value = "; VR(100)

PRINT #5, "Length of input = "; VR(101)[0]

PRINT #5, "Terminating character = "; VR(102)[0]



CLOSE #40

DEL "N"

产生输出:

Value = 0.0000

Length of input = 7

Terminating character = 13

该值为零，因为指定读取整数，但遇到浮点值，从7个输入字符中读取，并以回车符结束(ASCII码13)。

PRINT #5, "Terminating character = "; VR(102)[0]



CLOSE #40

DEL "N"

产生输出:

Value = 123.0000

Length of input = 3

Terminating character = 13

该值为123.0000，从7个输入字符中读取，并以回车符结束(ASCII码13)。

例4

打开一个文件并打印一个整数值并关闭它，然后重新打开文件并使用STRTOD将其作为整数读取。

DIM mask AS INTEGER



OPEN #40 AS "N" FOR OUTPUT(1)

PRINT #40, "N16X2.345Y-12.273Z+123.802"

CLOSE #40



OPEN #40 AS "N" FOR INPUT



mask = STRTOD(7, 40, 101, 102, 103)



PRINT #5, mask

PRINT #5, VR(116)

PRINT #5, VR(126)

PRINT #5, VR(127)

PRINT #5, VR(128)



CLOSE #40

DEL "N"

产生输出:

58728448

16.0000

2.3450

-12.2730

123.8020

返回值的掩码等于‘(1 << 13)OR (1 << 23) OR (1 << 24) OR(1 << 25)’(即字母表中的13th、23rd、24th和25th个字母从0开始编号)，其他值分别为“N”、“X”、“Y”和“Z”的掩码。

例5

打开一个包含有效行G-Code文本的文件。解析每一行，查找G01命令并将值放入TABLE中。

OPEN #40 AS "GCODE" FOR INPUT



DIM i, ch, g_val, mcount, movesteps AS INTEGER

DIM x_val, y_val, z_val, a_val, b_val AS FLOAT



VR(101) = 0



TICKS = 0

debug = ON'关闭

mcount = 0

mstride = 5

movesteps = 1



WHILE VR(101) >= 0

    mask = STRTOD(7, 40, 101, 102, 103)



    IF mask.6 THEN

        g_val = VR(109) ' G 寄存器



        IF debug THEN PRINT "G"; g_val[0];



        IF g_val = 1 THEN

            x_val = VR(126)

            y_val = VR(127)

            z_val = VR(128)

            a_val = VR(103)

            b_val = VR(104)



            TABLE(1000 + mcount * mstride, x_val, y_val, z_val, a_val, b_val)



            IF debug THEN PRINT " X"; x_val; " Y"; y_val; " Z"; z_val; " A"; a_val; " B"; b_val



            mcount = mcount + 1



            IF mcount >= 25000 THEN

                PRINT "Stopping to limit memory use"

                STOP

            ENDIF

        ELSE

            IF debug THEN PRINT ""

        ENDIF

    ENDIF



    IF VR(102) = 13 THEN

        '行结束

    ELSEIF VR(102) = 10 THEN

        '行结束

    ELSEIF VR(102) = ASC("(") THEN

        '符号

        PRINT "~~";

        GET #40, ch



        WHILE ch <> ASC(")")

            PRINT CHR(ch);

            GET #40, ch

        WEND



        PRINT "~~"

    ELSEIF VR(102) = 0 THEN

        'NULL 是文件的末尾. (假定输入是 ASCII 并且不包括NULL)

        PRINT "EOF"

    ELSE

        '数据流中的位置字符

        PRINT "FAILED to understand ASCII";

        PRINT VR(102); " : "; CHR(VR(102))

    ENDIF

WEND
