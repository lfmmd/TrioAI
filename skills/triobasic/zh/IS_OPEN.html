IS_OPEN

类型

文件函数(只读)

语法

value = IS_OPEN #channel

描述

IS_OPEN返回所选通道的状态。通道可以是通信端口、文件或临时缓冲区。IS_OPEN状态TRUE并不意味着以太网连接是有效的和打开的，它只是指控制器固件中的通道，以便发现一个通道号是否已经在使用。

参数

value:如果通道打开，TRUE(-1);

如果通道关闭，FALSE(0)。
channel:使用# channel表示的通道号

例

例1

DIM comm_status AS BOOLEAN



comm_status = IS_OPEN #20

例2

检查通道21上的TCP/IP连接的状态，如果它可用，那么在端口1000上作为TCP客户端打开它。

IF FALSE = IS_OPEN #21 THEN 

    OPEN #21 AS "TCP:192.168.0.230(1000)" FOR READ_WRITE

    WA(20)

ENDIF
