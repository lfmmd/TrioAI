NOT

类型

逻辑和按位符

语法

NOT表达式

描述

NOT函数将只保留整数，并将剩余整数的所有位取反。

参数

expression:任何有效的TrioBASIC表达式。

位运算符varname。Bit返回1或0。NOT将对结果进行位反转，因此返回-1 (TRUE)或-2。NOT 1是十六进制的$FFFF FFFE，或十进制的-2。

例

例1

计算 AND 7 NOT 1.5。先取1.5的整数位1，并对 1 进行取反，然后和 7 进行与运算。

PRINT 7 AND NOT(1.5)

  6.0000

例2

如果函数失败，则打印错误信息并停止程序

IF NOT CAN(0,9,13,1,8,$6060,0,$02) THEN

  PRINT #user, "Failed to set velocity mode"

  STOP

ENDIF
