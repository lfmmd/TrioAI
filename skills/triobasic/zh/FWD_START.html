FWD_START

类型

轴参数

描述

强制立即正向点动。当FWD_JOG为-1且fwd_start设置为1时，轴将在轴上运行MTYPE 14 forward_jog。如果FWD_JOG设置为输入数字(不等于-1)，则fwd_start不起作用。

值

FWD_STARTFWD_JOGMTYPE
0-10 (Idle)
1-114 (forward_jog)
不等于 1-10 (Idle)
任意值不等于-10 (Idle) *

*在指定输入的控制下

例

BASE(2)

FWD_JOG = -1

REV_JOG = -1



IF IDLE THEN

  REPEAT

    SELECT_CASE VR(10)

    CASE 0

      FWD_START = 0

      REV_START = 0

    CASE 1

      FWD_START = 1

      REV_START = 0

    CASE 2

      FWD_START = 0

      REV_START = 1

    END_CASE

  UNTIL VR(10) = 99

ENDIF
