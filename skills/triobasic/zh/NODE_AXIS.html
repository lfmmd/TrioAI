NODE_AXIS

-类型

系统数组(MC_CONFIG)

语法

NODE_AXIS(slot, node) = value

描述

这个二维阵列可用于覆盖任何EtherCAT节点轴的驱动器寻址。这可以用来定义一个用户特定的轴映射，以固定来自不同来源的轴。

这个数组是二维的，第一维是主站槽标识符，第二维是节点在主站网络内的位置。

如果在启动EtherCAT协议时请求的轴已在使用，则会引发错误。

值

0自动分配EtherCAT轴(默认)
> = 1EtherCAT驱动器位于此轴

例

配置单轴驱动TRIO DX4从2号轴到20号轴和P374 2伺服轴分别从0号和1号轴到10号和11号轴。

111
