READ_OP

类型

I / O命令

语法

value = READ_OP(output[, finaloutput])

描述

返回数字输出逻辑的状态。

如果带一个参数调用，则返回该特定输出通道的状态(1或0)。如果带两个参数调用，则READ_OP返回输出组的二进制和。

READ_OP检查输出逻辑的状态。输出可能是虚拟的或未通电的，您仍然会看到逻辑状态。

参数

value:所选输出的二进制模式
output:输出以返回输出组的起点或值
finaloutput:组的最后一个输出

输出到最终输出的范围不能超过32

例

例1

在这个例子中，测试的是单个输出:

test:

  WAIT UNTIL READ_OP(12) = ON

  GOSUB place

例2

检查这组8个输出，如果其中任何一个是ON的，调用一个例程。

op_bits = READ_OP(16,23)

IF op_bits <> 0 THEN

  GOSUB check_outputs

ENDIF
