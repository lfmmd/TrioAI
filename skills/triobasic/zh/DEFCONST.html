DEFCONST

类型

程序常量

语法

DEFCONST name value

描述

DEFCONST 用一个值定义文本名称，而不是整数或浮点数。

这些文字能被用于数组声明或切换语句或任何BASIC普通行。它们能够加到INCLUDE文件并被INCLUDE文件本身参考，或使用INCLUDE文件的主程序。

DEFCONST 在编译时被求值。这限制了它在特定运行功能的使用。例如，“DEFCONST process_number PROCNUMBER”将不起作用，因为DEFCONST不知道程序运行时处于哪个进程。

参数

name:任何用户定义的包含小写字母、数字或下划线 (_) 字符的名称。
value:赋给名称的值或求值表达式。

例：

例 1

声明4个常量文本。

DEFCONST mysize 20

DEFCONST fval SIN(90)

DEFCONST i1 7

DEFCONST i2 i1 * 1000



DIM myarray AS INTEGER(mysize)

例 2

通过SELECT_CASE使用文本。

DEFCONST errorval 100



SELECT_CASE swval



    CASE 0

        PRINT "valid data"



    CASE errorval

        PRINT "SWVAL invalid"

END_CASE
