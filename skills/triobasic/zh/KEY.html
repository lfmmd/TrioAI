KEY

类型

文件函数

语法

value = KEY [#channel]

描述

Key用于检查通道缓冲区中是否有字符。该命令不读取字符，但允许程序测试是否有字符到达。

当使用GET读取字符时，TRUE结果将被重置。

参数

#channel:完整通道列表见#(省略时默认为0)
value:如果channel是以字符模式打开的FIFO，则value是一个负值，表示通道缓冲区中的字符数。
如果channel是以行模式打开的FIFO，则value是一个负值，表示通道缓冲区中的行数。
当作为FIFO工作时，仅适用于通道#40至#48(文件通道)。关于FIFO模式的描述请参见OPEN。
-1(TRUE)如果缓冲区中除上述特定情况外还有其他字符。
如果没有字符则为0(FALSE)。

例

如果在通道1上接收到一个字符，则调用子程序

main: 

  IF KEY #1 THEN GOSUB read

...

read: 

  GET #1, k

RETURN
