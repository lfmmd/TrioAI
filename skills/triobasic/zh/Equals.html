= 等于

类型

比较运算符

语法

<expression-1> = <expression-2>

描述

如果<expression-1>等于<expression-2>返回TRUE，否则返回FALSE。

在比较浮点值时，请记住，除了在特殊情况下，它们往往不会完全相等，因此您通常必须使用替代方法来比较它们，例如，将绝对或相对差异与适当的小公差相结合。

比较运算符通常用于条件语句中，如IF、REPEAT和WHILE，但也可以在表达式有效的任何地方使用。

参数

Expression1:任何有效的triobasic表达式
Expression2:任何有效的triobasic表达式

以下triobasic表达式是有效的:
Expression 1Expression 2描述
数字数字浮点数，整数，IP
字符串字符串区分大小写的词法比较
IP字符串将字符串转换为忽略空格的整数
字符串IP将字符串转换为忽略空格的整数

不允许比较数组。

例

例1

如果输入7为ON，则程序将在以“label:”开头的行继续执行。

IF IN(7) = ON THEN GOTO label

例2

当模拟量输入0读取2.1 V时，调用子程序' threshold_1 '。

本例避免直接比较浮点值(IF AIN(0) = 2.1 THEN…)，因为无法为真，而是使用绝对差的绝对值和比较容差。

IF ABS(AIN(0) – 2.1) < 0.5e-6 THEN GOSUB threshold_1
