MSPEED_FILTER

类型

轴参数

描述

MPSEED_FILTER是MSPEEDF使用的过滤系数值。MSPEED_FILTER可以取0到1.0之间的任何值。较小的值意味着更长的时间常数，因此滤波效果更好。当设置为1时，MSPEEDF的值将与MSPEED相同。

过滤器方程近似如下，每SERVO_PERIOD处理一次。

MSPEEDF = (1 - MSPEED_FILTER) * MSPEEDF + MSPEED_FILTER * MSPEED

值

过滤时间常数系数，可以取0到1之间的值(默认值)。

例

监视MSPEEDF以检测轴3的速度何时上升到阈值以上。

BASE(3)

MSPEED_FILTER = 0.5 '初始化过滤系数



WHILE TRUE

  IF MSPEEDF > 5 THEN

    GOSUB start_main

  ENDIF

  ...

WEND
