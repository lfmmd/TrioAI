DRIVE_WRITE

类型

轴函数

语法

result = DRIVE_WRITE (parameter, value)

描述

DRIVE_WRITE允许控制器从数字总线连接的驱动器写入参数。目前支持松下A4N, A5N和A6N RTEX驱动器。

参数编号和详细信息可以在Motion Perfect智能驱动工具中找到。

参数

result:1DRIVE_WRITE操作成功
0DRIVE_WRITE失败
parameter:parameter_numberA4N参数写入
class * 256 + parameter_numberA5N/A6N参数写入
128将所有驱动器参数存储到EPROM中
129将所有驱动器参数重置为默认值
value:要写入参数的值。
(参数号128 & 129使用0)

例

例1

在A4N驱动器上写入参数122，编码器范围

success = DRIVE_WRITE(122, 10000)

If success = 0 THEN

  PRINT "写入驱动器参数错误"

ELSE

  PRINT "编码器范围设置"

ENDIF

例2

在A5N驱动器上写入class 0参数8，编码器范围

success = DRIVE_WRITE(0 * 256 + 8, 15000)

If success = 0 THEN

  PRINT "写入驱动器参数错误"

ELSE

  PRINT "编码器范围设置"

ENDIF

例3

在EPROM中存储所有驱动器参数

success = DRIVE_WRITE(128, 0)

IF success = 0 THEN

  PRINT "存储驱动器参数的错误至EPROM"

ELSE

  PRINT "驱动器参数存储在EPROM "

ENDIF

例4

将所有驱动器参数重置为默认值

success = DRIVE_WRITE(129, 0)

IF success = 0 THEN

  PRINT "复位驱动器参数错误"

ELSE

  PRINT "驱动器参数重置为默认值"

ENDIF
