MOVETANG

类型

轴命令

语法

MOVETANG(absolute_position[, link_axis])

描述

使用轴的编程SPEED, ACCEL和DECEL将轴移动到所需位置。移动方向是通过计算到达位置的最短路径来确定的，假设轴是旋转的，并且REP_DIST被设置为PI弧度(180度)，并且REP_OPTION = 0。

REP_DIST值将取决于UNITS值和表示PI弧度的步数。例如，如果旋转轴有4000个脉冲/转并且UNITS= 1，则REP_DIST值将为2000。

当MOVETANG完成移动时，它不会从MTYPE中清除。这样你就可以在一个紧密的循环中使用它，通过再次调用MOVETANG来更新结束位置。当使用link_axis时，结束位置自动从链接轴的TANG_DIRECTION更新。

参数

absolute_position:要设置为移动终点的绝对位置。值必须在-PI到+PI的范围内，以旋转轴为单位。例如，如果旋转轴有4000个脉冲/转，UNITS值=1，所需的角度是PI/2(90度)，那么位置值将是1000。
link_axis可以指定一个可选的链接轴。当指定一个链接轴时，系统软件根据链接轴TANG_DIRECTION计算每个伺服周期所需的绝对位置。TANG_DIRECTION乘以REP_DIST/PI来计算所需的位置。注意，当使用link_axis时，absolute_position参数将不再使用。每个伺服周期复制位置，直到MOVETANG被取消。

例

例1

X-Y定位系统有一个必须转动的触控笔，以便它在行进时始终面向同一方向。切向控制程序在一个单独的过程中运行。

BASE(0, 1)

WHILE TRUE

  angle = TANG_DIRECTION

  MOVETANG(angle) AXIS(2)

WEND

例2

X-Y定位系统有一个必须转动的触控笔，以便它在行进时始终面向同一方向。

XY轴对是轴4和轴5。切向触控笔轴为2:

MOVETANG(0,4)AXIS(2)

例3

X-Y切割台有一个“比萨轮”切割机，必须控制它，使其始终与移动方向对齐。主X轴和Y轴由_运动控制器_轴0和1控制，比萨轮由轴2转动。

比萨轮的控制是在独立于主X-Y运动程序的程序中完成的。在这个例子中，转向程序也会进行轴的初始化。

程序TC_SETUP.BAS

'为切向控制设置3个轴

WDOG = OFF



BASE(0)

P_GAIN = 0.9

VFF_GAIN = 12.85

UNITS = 50  '设置Units位mm

SERVO = ON



BASE(1)

P_GAIN = 0.9

VFF_GAIN = 12.30

UNITS = 50 '所有轴的Units必须相同

SERVO = ON



BASE(2)

UNITS = 1  '设置 rep_dist的Units为1

REP_DIST = 2000 '编码器每转2000个脉冲

REP_OPTION = 0

UNITS = 4000 / (2 * PI) '为半径设置Units

SERVO = ON



WDOG = ON

'将第3个轴回零到它的Z标记

DATUM(1) AXIS(2)

WAIT IDLE

WA(10)



'开始正切控制进程

BASE(0, 1) '定义X和Y的轴组

'开始正切控制

BASE(2)

MOVETANG(0, 0) '使用轴0和轴1作为链接的轴组

程序MOTION.BAS

'用程序裁出一个带圆角的正方形

MERGE = ON

SPEED = 300



nobuf = FALSE   '当TURE时，运动不进入缓存区

size = 120      '正方形的每一边的大小

c = 30          '四分之一圆的大小(半径)



DEFPOS(0, 0)

WAIT UNTIL OFFPOS = 0

WA(10)



MOVEABS(10, 10 + c)

REPEAT

  MOVE(0, size)

  MOVECIRC(c, c, c, 0, 1)

  IF nobuf THEN WAIT IDLE: WA(2)

  MOVE(size, 0)

  MOVECIRC(c, -c, 0, -c, 1)

  IF nobuf THEN WAIT IDLE: WA(2)

  MOVE(0, -size)

  MOVECIRC(-c, -c, -c, 0, 1)

  IF nobuf THEN WAIT IDLE: WA(2)

  MOVE(-size, 0)

  MOVECIRC(-c, c, 0, c, 1)

  IF nobuf THEN WAIT IDLE: WA(2)

UNTIL FALSE
