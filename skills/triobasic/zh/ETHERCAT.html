ETHERCAT

类型

系统命令

语法

ETHERCAT(function, slot [,parameters…])

描述

ETHERCAT命令用于对EtherCAT网络进行高级操作。在正常使用中，EtherCAT网络将自动启动，而不需要启动程序中的任何命令。在调试和设置EtherCAT系统时，一些ETHERCAT命令函数可能很有用，因此这里描述一个小子集。

如果执行成功，ETHERCAT命令返回TRUE(-1);如果执行错误，返回FALSE(0)。返回值的函数必须要么将值放入VR中，要么将其打印到当前输出终端。

参数

function:要执行的功能
$00启动EtherCAT网络
$01停止EtherCAT网络
$02在线检查从站
$03检查从站数量
$04获取从站地址
$05获取从轴
$21设置EtherCAT状态
$22获取EtherCAT状态
$30设置备用ESC寄存器值(自动)
$31获取备用ESC寄存器值(自动)
$64将复位序列发送到驱动器
$79设置状态掩码和触发模式
$7A获得状态掩码和触发模式
$87显示网络配置
$91设置Sync0开始时间
$93设置DC Master模式
$E0设置紧急消息控制
$E1读取紧急消息控制值
slot:设置EtherCAT端口槽位号
MC4N, MC6N0
Flex-6 Nano0
MC464P876槽位号(0 ~ 7)
MC664内部槽位号或-2
如果安装了P876槽位号

Function = $00:启动EtherCAT网络

语法

ETHERCAT(0, slot, [,option_flags [,MAC_retries]])

描述

初始化EtherCAT网络，并使其进入运行模式。

参数

槽:EtherCAT端口槽位号。
option_flags:位状态描述
00不打印进度信息
1打印进度信息(默认)
10正常状态改变(默认)
1强制重新扫描网络
20标准启动
1快速启动；不要等待PLL在状态改变之间锁定
MAC_retries:设置主站尝试重启以太网自动交换的次数。Default = 2

例

检查EtherCAT的状态，如果不是操作模式，则重新启动EtherCAT，并设置输出，表示重新启动正在进行中。

'——如果需要，启动EtherCAT。

Slt = 0

ecs_vr = 30 '使用VR 30作为返回值

chk = ETHERCAT($06, slt, ecs_vr) '测试状态



IF chk <> TRUE OR VR(ecs_vr) <> 3 THEN

  OP(9, ON)

  WA(15000) '等待15秒驱动启动

  ETHERCAT(0, slt) '初始化EtherCAT

ENDIF

Function = $02:在线查看从站

语法

ETHERCAT(2, slot, slave_position, vr_index)

描述

检查给定从站是否在线。如果vr_index <> -1，则将结果写入给定的VR。如果vr_index = -1则将结果写入标准输出流。返回函数是否成功执行。

在检查状态之前，从站必须已经初始化并在网络上。

参数

slave_position:要检查的从站在网络上的位置。
vr_index:0 - VRmax结果被放置在选定的VR变量中。
1结果显示在当前输出路径中。(默认终端0)

例

检查网络位置0到7的从站是否脱机。

FOR posn = 0 TO 7

  ETHERCAT(2, 0, posn, 20 + posn)

  IF VR(20 + posn) = FALSE THEN

    PRINT "Drive at "; posn[0]; " is offline"

  ENDIF

NEXT posn

参数

vr_index:0 - VRmax结果被放置在选定的VR变量中。

Function = $04:获取从站地址

语法

ETHERCAT(4, slot, slave_position, vr_index)

描述

确定由其在网络上的位置标识的从站的配置地址。如果vr_index <> -1，则将结果写入给定的VR。如果vr_index = -1则将结果写入标准输出流。返回函数是否成功。

在检查配置的地址之前，从站必须已经初始化并在网络上。

参数

slave_position:要检查的从站在网络上的位置。
vr_index:0 - VRmax配置的地址放在所选的VR变量中。
1在当前输出路径中显示“配置的地址”。(默认终端0)

例

报告网络中每个节点的配置地址。

ETHERCAT(3, 0, 20) '获取总线上的从站数

FOR posn = 0 TO VR(20) - 1

  ETHERCAT(4, 0, posn, 30 + posn)

  PRINT "Drive at position "; posn[0]; " has address "; VR(30+posn)[0]

NEXT posn

Function = $21:设置EtherCAT状态

语法

ETHERCAT($21, slot, state, display)

描述

控制EtherCAT状态机。它(ESM)请求主节点改变到给定的EtherCAT状态，并因此将所有从节点改变到相同的状态。当改变到一个更高的状态时，EtherCAT网络将通过中间状态进入到新的状态，以允许正确启动网络。

参数

state:EtherCAT状态请求
1保留
0初始化 (EtherCAT ESC值0x01)
1预操作(0x02)
2安全操作(0x04)
3操作(0x08)
display:函数
1将状态变化信息写入标准输出流。(默认)
0不写状态变化信息。

例

将EtherCAT更改为安全操作，并取消将打印到终端的信息。

ETHERCAT($ 21,0,2,0)

Function = $64:将复位序列发送到驱动器

语法

ETHERCAT($64, axis_number[， mode[， timeout]])

描述

重置一个从站错误。这个函数在驱动器控制字上运行错误复位序列。DRIVE_CONTROLWORD位8先高后低。这将指示驱动器重置驱动器中错误，删除导致错误的原因。

对复位序列的响应将取决于驱动器及其遵循CoE DS402规范的程度。

参数

axis_number:待复位驱动器的轴号。
mode:0DS402控制字的“故障复位”(位7)设为高，然后在硬编码超时后再次设为低。(默认)
1第7位被设置为高位，直到状态字的“Fault Flag”(第3位)变低，或者发生超时。
timeout:可选的超时时间，以毫秒为单位，用于模式1操作。默认为100毫秒。范围为1 ~ 10000ms。

例

例1

发送控制字复位序列到轴8的驱动器。

ETHERCAT($ 64,8)

例2

发送控制字复位序列到轴2的驱动器。使用模式1强制复位位保持高位直到状态3变低或强制复位位在60毫秒后再次变低，即使状态位仍然是高电平。

ETHERCAT($ 64,2,1,60)

ETHERCAT $79:设置状态掩码和触发模式

类型

EtherCAT配置设置。从控制器交互的运行时定义。

语法

ETHERCAT($79, slot, addr, vr_index [， mask, trigger_pattern])

描述

该命令允许用户定义一个CoE状态字(0x6041:00)掩码和触发模式，用于提高控制器AXISSTATUS“远程驱动器状态字错误”标志(第26位)。

掩码定义CoE状态字中的哪些位是相关的，触发模式定义这些位(0或1)的状态，这些位将触发控制器AXISSTATUS标志。

如果在设置此AXISISTATUS标志时需要重置WDOG(主站使能)，则必须相应地设置轴ERRORMASK。

从站掩码和模式可以在 预操作到网络的操作状态(在配置的地址被分配之后) 设置和修改，。如果网络重新启动，必须重置。

从站服务器必须支持PDO循环数据中的CoE状态字(0x6041:00)。

如果Mask为零，控制器将不会对CoE状态字执行任何运行时检查。

远程驱动器状态字错误标志位为高电平时必须将WDOG清除为OFF，则必须设置轴ERRORMASK参数的第26位。

参数

Slot:模块槽位号例如:0
Addr:配置的从站地址
vr_index:1使用参数5和6中给定的mask和trigger_pattern值。
0 . .max_vr_index-1包含掩码的VR的索引，下一个VR (vr_index+1)包含trigger_pattern。
Mask:CoE statusword mask可选
Trigger_pattern:定义将触发控制器AXISSTATUS“远程驱动器状态字错误”标志的相关位的状态(位26)。可选

返回值

-1:掩码和模式设置正确

0:设置掩码和模式失败

例

我们希望设置AXISSTATUS“远程驱动器状态字错误”(第26位)标志，并在从站驱动器设置“禁用开关”标志(CoE状态字的第6位)或清除“准备打开开关”标志(CoE状态字的第0位)时将WDOG置于OFF。

因此

'Mask = 0x41

'触发模式 = 0x40



'设置Mask和Trigger模式 

'Slot = 0, slave configured address = 1 and axis = 0



'Params(func, slot, addr, vr_index, mask, trigger pattern)

ETHERCAT($79, 0, 1, -1, $41, $40)



'设置ERRORMASK以确保AXISSTATUS位26在触发时将清除WDOG。

BASE(0)

ERRORMASK.26 = 1

现在，如果将CoE状态字位0(准备启用)设置为低，或将6位(切换至禁用)设置为高，则将设置AXISSTATUS“远程驱动器状态字错误”标志(位26)并清除WDOG。

ETHERCAT $91:设置EtherCATsync0开始时间

类型

EtherCAT配置设置。

语法

ETHERCAT($91, slot, vr_index[， value])

描述

设置网络默认的Sync0开始时间偏移。如果vr_index <> -1，则使用给定VR中的值，如果vr_index = -1，则将该值写入第三个参数。

默认的开始时间是1秒(1000000微秒)。加减所需偏移量(以微秒为单位)。例如1000150是150 μs的偏移量，999930是-70 μs的偏移量。偏移量不得超过伺服周期的±50%。

函数根据操作是否成功返回TRUE/FALSE。

该命令必须在EtherCAT网络启动前执行。

参数

Slot:模块槽位号
vr_index:1使用参数3中的值。
0 . .max_vr_index-1与指定的VR数一致。
value起始时间的可选值。

例

'设置EtherCAT Sync0开始时间，偏移量为500微秒。

e_flag = ETHERCAT($91, 0, -1, 1000500)

IF e_flag = FALSE THEN 

    PRINT #5,"设置Sync0开始时间失败"

ENDIF

ETHERCAT $93:设置分布式时钟主模式

类型

EtherCAT配置设置。

语法

ETHERCAT($93, slot, dc_function, vr_index[， value])

描述

设置EtherCAT主节点，使第一个启用DC的节点成为分布式时钟主节点。由于这需要Trio运动控制器同步到外部DC主机，因此它不适用于P876, MC4N-ECAT或MC664(X)，因为这些具有基于FPGA的固定定时硬件用于EtherCAT端口。

函数根据操作是否成功返回TRUE/FALSE。

该命令必须在EtherCAT网络启动前执行。

参数

Slot:模块槽位号
dc_function:分布式时钟功能。2 =使用第一个启用DC的节点作为时钟主节点。
vr_index:-1使用参数3中的值。
0 . .max_vr_index-1使用指定的VR编号中的值。
valueDC模式的可选直接值。0 =使用第一个启用DC的节点作为时钟主节点。

例

将第一个具有DC能力的驱动器设置为分布式时钟主机，然后启动网络。

'使用第一个DC使能的驱动器作为主时钟

ETHERCAT($93, 0, 2, -1, 0)

'启动EtherCAT网络

ETHERCAT(0, 0)
