IS_INF

类型

数学函数

语法

value = IS_INF(expression)

描述

检查存储变量的值。测试该值是否为浮点无限大数，从而可能导致表达式生成BASIC错误。

参数

expression任何有效的数值表达式。

例

例1

IF IS_INF(TABLE(3000)) THEN

    PRINT "TABLE(3000) is Infinity"

ENDIF

例2

检查MC403中的所有VR并移除无限值

FOR i = 0 TO 4095

    IF IS_INF(VR(i)) THEN

        VR(i) = 0

    ENDIF

NEXT i
