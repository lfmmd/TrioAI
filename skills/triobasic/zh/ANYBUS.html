ANYBUS

类型

系统功能

语法

ANYBUS(function, slot [, parameters…])

描述

该函数允许用户配置激活的Anybus模块，使网络处于运行状态。某些网络对数据类型和大小有限制，详情请参考Anybus数据表。

无源模块不需要设置，将作为通信通道出现，然后可以与PRINT, GET等一起使用。这些模块可以使用SETCOM命令进行配置。

参数

功能:0配置映射
1配置模块和启动协议
2停止协议
3读状态字节
4自动配置映射

Function = 0

语法

value = ANYBUS(0,slot [, map, source [, index, type, count, direction [,endian]]])

描述

为通过网络更新的内存区域分配一个VR或表点。可以使用前4个参数删除单个或所有映射。

当前映射可以使用前2个参数打印到终端。

参数

value:TRUE=命令执行成功
FALSE=命令执行失败
slot:安装Anybus的模块槽
map:映射号，使用-1删除所有映射
source:MC464上的数据位置
1删除映射
0VR
1表格
index:数据源中的起始位置
type:通过总线发送的数据的大小和类型
0布尔
1有符号8位整数
2有符号16位整数
3有符号32位整数
4无符号8位整数
5无符号16位整数
6无符号32位整数
7字符
8枚举
9日至15日(保留)
16有符号64位整数
17无符号64位整数
18浮点/实数
count:映射的数据类型数量
direction:数据方向
0数据读入控制器
1从控制器传输的数据
endian0使用来自网络的默认端序(默认)
1交换尾数法

语法

FALSE命令不成功
slot:安装Anybus的模块槽
address:模块地址，节点号，id。MAC等_(Profinet_不需要_)_
baud:波特率CC Link 必须设置

1250 kbps

波特率Profibus -自动，非必需

Function = 2

语法

value = ANYBUS(2,slot)

描述

停止循环数据传输。

参数

value:TRUE命令成功
FALSE命令不成功
slot:安装Anybus的模块槽位

Function = 3

语法

value = ANYBUS(3,slot)

描述

从Anybus模块读取状态字节。

参数

value:Anybus状态字节:
位0 - 2:Anybus状态:
0设置
1NW_INIT
2WAIT_PROCESS 等待进程
3空闲
4PROCESS_ACTIVE 进程激活
5错误
6(保留)
7异常
位3监督位:
0模块无监督
1模块由另一个网络设备监督
位4 - 7(保留)
slot:安装Anybus的模块槽

Function = 4

语法

value = ANYBUS(4,slot [, address], type, inoff, outoff [,endian])

描述

自动配置并启动循环网络。使用函数0仍然可以读取映射。

此功能仅适用于Profibus和Profinet。Profinet不需要地址参数。

参数

value:TRUE命令成功
FALSE命令不成功
slot:安装Anybus的模块槽
address:模块地址，节点号，MAC id.等(仅限Profibus)
type:数据类型和位置
0VR整数
1Table整数
2VR浮点数
3Table浮点数
inoff:输入偏移量
outoff:输出偏移量
endian0使用来自网络的默认端序(默认)
1交换尾数法

例

例1

配置设备网具有2个16位整数输入和2个16位整数输出。该数据使用“轮询连接”方法循环传输。确保配置主服务器与从服务器相同，否则数据将无法传输。

device_net:

  slotnum=0 '带有模块槽位号的局部变量



'映射数据

  map=FALSE

'映射接收数据

  map= ANYBUS(0, slotnum, 1, 0, 0, 2, 4, 0) '4*16-bit Int Rx

  IF map=TRUE THEN

    '映射传输数据

    map= ANYBUS(0, slotnum, 2, 0, 4, 2, 4, 1) '4*16-bit Int Tx 

  ENDIF



  IF map=FALSE THEN

    PRINT#term, "Mapping failed"

    STOP

  ENDIF



'将映射的数据打印到终端

  ANYBUS(0,slotnum)



'开始网络

  map= ANYBUS(1, slotnum, 3, 2)  'MAC ID=3, Baud=500k

  IF map=FALSE THEN

    PRINT#term, "启动网络失败"

    STOP

    ELSE

    PRINT#term, "网络启动"

  ENDIF

  RETURN

例2

配置2站CC-Link，16位输入，16位输出，2个SINT16输入和2个SINT16输出。确保主站配置相同，握手位实现。

cc_link:

'函数0 -设置映射

'站1

  map = ANYBUS(0, slotnum, 0, 0, 0, 0, 16, 0) '16*BOOL Rx

  map = ANYBUS(0, slotnum, 1, 0, 1, 0, 16, 1) '16*BOOL Tx

  map = ANYBUS(0, slotnum, 2, 0, 2, 2, 2, 0)'2*16-bit Int Rx

  map = ANYBUS(0, slotnum, 3, 0, 4, 2, 2, 1) '2*16-bit Int Tx

'站2

  map = ANYBUS(0, slotnum, 4, 0, 6, 0, 16, 0) '16*BOOL Rx

  map = ANYBUS(0, slotnum, 5, 0, 7, 0, 16, 1) '16*BOOL Tx

  map = ANYBUS(0, slotnum, 6, 0, 8, 2, 2, 0) '2*16-bit Int Rx

  map = ANYBUS(0, slotnum, 7, 0, 10, 2, 2, 1) '2*16-bit Int Tx



  ANYBUS(0,slotnum) '将映射的数据打印到终端





'函数1 -启动协议

  IF map = FALSE THEN

  map = ANYBUS(1, slotnum, 1, 2)

  ENDIF
