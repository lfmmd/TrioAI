MOVEC

类型

机器人命令

语法

MOVEC centre_point end_point [S:= T:= O:=]

描述

MOVEC用于沿着圆形路径将机器人从一点移动到另一点。所有轴同时到达目的位置。

KINEMATIC_GROUP需要RPS run-time FEC

参数

参数数据类型描述
centre_pointTARGET定义圆周移动中心的笛卡尔目标。这个点沿着执行的圆周移动遍历。
注意:关节目标不适合这个运动。如果使用关节目标则会生成错误“无效混合数据类型”。
end_pointTARGET定义圆周移动结束的笛卡尔目标。
注意:关节目标不适合这个运动。如果使用关节目标则会生成错误“无效混合数据类型”。
Embedded parameters:详见RPS Move Commands

例

例1

GTA(0) = 350, 0, 50, 180, 0, 0, "p1"

GTA(1) = 400, 100, 20, 180, 0, 0, "p2"

GTA(2) = 450, 0, 60, 180, 0, 0, "p3"

OBJECT_FRAME(1,"of1", -10, 0, 0, 0, 0, 15)

TOOL_OFFSET(1,"tool1", 0, 0, 50, 0, 0, 0)

ROBOT(0)

MOVEL GTA("p1")

MOVEC GTA("p2") GTA("p3")

STOP

这个例子命令Robot 0直线移动到p1，然后沿着p2圆周移动到p3。请注意，代码中没有其他嵌入指令。因此，机器人将使用以下参数执行代码。
类型参数
Object Frame当前活动的对象框架
Tool Offset当前活动的工具偏移量
Speed [mm/s]WORLD_ORI_SPEED * LOOKAHEAD_FACTOR
Acceleration [mm/s^2]WORLD_ORI_MAX_ACC * JOG_ACC_FACTOR
Jerk [mm/s^3]WORLD_ORI_MAX_JERK
Blend radius [mm]0

例2

MOVEC GTA("p1") GTA("p2") S:=100
类型参数
Speed[]100 * LOOKAHEAD_FACTOR

例3

MOVEC GTA("p1") GTA("p2") O:="of1"
类型参数
Object Frame对象框架1
Speed[]WORLD_ORI_SPEED * LOOKAHEAD_FACTOR

例4

MOVEC GTA("p1") GTA("p2") T:="tool1
类型参数
Object Frame当前活动的对象框架
Tool Offset工具框架1

例5

MOVEC GTA("p1") GTA("p2") Z:=67

MOVEL GTA("p3")
类型参数
Tool Offset当前活动的工具偏移量
Blend radius[]67毫米

机器人会朝着目标点0和67前进，在到达目标点之前，它会混合下一个动作，走向目标点1。
