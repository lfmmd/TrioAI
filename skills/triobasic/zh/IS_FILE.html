IS_FILE

类型

系统参数(只读)

语法

value = IS_FILE "file_name"

描述

IS_FILE返回一个特定文件是否存在于控制器的项目目录(DIR)的状态。文件名可以是字符串文本，字符串类型的变量和VRSTRING也可以使用。

IS_FILE与IS_PROG功能相同。

参数

value:如果文件存在，TRUE(-1) ;

如果文件不存在，FALSE(0)。
file:文件名称

例

例1

file_status = IS_FILE "MAIN_TASK"

例2

删除一个存在的文本文件:

DIM textfile_name AS STRING(12)

textfile_name = "textfile_no1"

IF IS_FILE textfile_name THEN 

    DEL textfile_name

ENDIF
