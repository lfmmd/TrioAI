PROG_TYPE

类型

系统参数(只读)

语法

value = PROG_TYPE "file_name"

描述

PROG_TYPE系统参数返回文件的类型。如果文件不存在，则返回-1。

值

以下PROG_TYPE是当前激活的值。
值描述
0BASIC
1加密的BASIC
2GCode
3文本
4IEC
5FIFO
6临时
7项目
8二进制文件
9MC配置
10IEC库
11HMI设计
12BASIC库
13加密的BASIC库
14HMI库
15易失性文本
16ROBOT
17ROBOT BASIC
-1文件不存在

例

例1

如果一个文件或一个程序存在，并且是一个TrioBASIC文件，那么运行它:

triobasic_type = 0

IF IS_PROG "MAIN_TASK" THEN

    IF (PROG_TYPE "MAIN_TASK") = triobasic_type THEN

    RUN "MAIN_TASK"

ENDIF

ENDIF
