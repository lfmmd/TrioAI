MOVEABS

类型

轴命令。

语法

MOVEABS(position1[, position2[, position3[, position4, …]]])

替代格式

MA()

描述

绝对位置移动。将一个轴或多个轴移动到相对于零(主)位置引用的位置。列表中的第一个参数被发送到使用AXIS命令指定的轴或当前BASE轴，第二个参数被发送到下一个轴，依此类推。

在多轴形式中，用于运动的速度、加速度和减速取自BASE组中的第一个轴。控制每个轴的速度，使运动的结果矢量以SPEED设置运行。

多轴插补使用BASE轴的UNITS、SPEED、加速和减速。为了正确的XYZ矢量操作，所有3个轴的UNITS必须相同。

如果需要将某个轴不包含在矢量速度计算中，则可以将该轴的INTERP_FACTOR设置为0。

通过简单地在每个轴上独立放置MOVEABS命令，可以实现非插补，非同步的多轴运动。如果需要，可以使用AXIS命令指定单个MOVEABS的目标轴。这只覆盖了一个MOVEABS的BASE轴设置。

指定的值使用单位转换因子轴参数进行缩放;UNITS。因此，例如，如果一个轴有400个编码器脉冲/mm，则该轴的UNITS为400。然后命令MOVEABS(6)将移动到距离零位6毫米的位置。当MERGE设置为ON时，绝对移动和相对移动将合并，以实现连续的路径移动。

坐标轴的零位(原点)位置的位置可以通过命令OFFPOS,DEFPOS, REP_DIST, REP_OPTION和DATUM来改变。

如果移动中的轴数与之前的移动不同，则它们不会合并。同样，FORWARD，REVERSE或任何链接的移动(例如，MOVELINK)将不会与移动或MOVEABS MERGE。MOVEABS会转到错误的终点。

参数

position1:要移动到基准轴上的位置。
position2:在BASE数组的下一个轴上移动到的位置。
position3:在BASE数组的下一个轴上移动到的位置。
position4:在BASE数组的下一个轴上移动到的位置

MOVEABS命令可以插补到控制器上可用的全部轴数。

例

例1

根据2个开关的选择，机器必须移动到3个位置中的一个。选项为home，位置1和位置2，其中两个开关分别关闭，第一个开关打开，第二个开关打开。位置2优先于位置1。

'定义绝对位置

Home = 1000

position_1 = 2000

position_2 = 3000

WHILE IN(run_switch) = ON

  IF IN(6) = ON THEN       ' 开关 6选择位置2

    MOVEABS(position_2)

    WAIT IDLE

  ELSEIF IN(7) = ON THEN   ' 开关 7选择位置1

    MOVEABS(position_1)

    WAIT IDLE

  ELSE

    MOVEABS(home)

    WAIT IDLE

  ENDIF

WEND

例2

X-Y绘图仪具有一个笔式传送带，其位置相对于绘图仪绝对零位置是固定的。 若要更改笔，则在命令时，无论绘图位置如何，绝对移动到转盘位置都将找到目标。

MOVEABS(28.5, 350) '移动到笔架区域外

WAIT IDLE

SPEED = pen_pickup_speed

MOVEABS(20.5, 350) '进去拿笔

例3

托盘由一个6 × 8的网格组成，其中用包装机间隔185毫米插入气罐。这些气罐从一个固定点捡起。使用DEFPOS命令将托盘中的第一个位置定义为位置(0,0)。在托盘中定位罐的程序部分是:

FOR x = 0 TO 5

  FOR y = 0 TO 7

    MOVEABS(-340, -516.5)     '移动到拾取点

    WAIT IDLE

    GOSUB pick                '调用 pick up 子程序

    PRINT "Move to Position: "; x * 6 + y + 1

    MOVEABS(x * 185, y * 185) '移动到网格中的位置

    WAIT IDLE

    GOSUB place               '调用place down子程序

  NEXT y

NEXT x

例4

使用MOVEABS与REP_DIST移动到最终位置。

Repdist = 360

DEFPOS(0)

MOVEABS(300)    '将从300度移动到300度

MOVEABS(200)    '将向后移动100度至200度

MOVEABS(370)    '将穿过170度到10度，穿过repdist

MOVEABS(350)    '将从340度移动到350度

如果你想以最短的方向移动到绝对位置，使用MOVETANG

例5

3轴插补移动，仅使用X和Y进行矢量速度计算。Z轴紧随其后。

INTERP_FACTOR AXIS(x) = 1

INTERP_FACTOR AXIS(y) = 1

INTERP_FACTOR AXIS(z) = 0

BASE(x, y, z)

SPEED = 10

MOVE(100, 150, 20)

设置x轴和y轴的速度，使沿XY运动路径的速度等于10。Z将移动20毫米，花费与XY对相同的时间，因此将与它们同步。
