MOVELINK

类型

轴命令。

语法

MOVELINK (distance, link_dist, link_acc, link_dec, link_axis[, link_options[, link_pos[, base_dist]]]).

替代格式

ML()

描述

链接的move命令设计用于控制以下动作:

  • 与传送带同步
  • 飞剪机
  • 梳螺纹、攻丝等。
  • 线圈绕组

该运动由线性运动组成，具有单独可变的加速和减速阶段，通过软件变速箱连接到另一个轴的测量位置(MPOS)。该命令使用BASE和AXIS，并以与其他移动命令类似的方式使用单位转换因子。

“链接”轴可以向任何一个方向移动，以驱动输出运动。指定的链接距离总是正的。

运动轮廓

标准梯形轮廓

正弦S曲线(Bit 4 = 1, Bit 10 & Bit 11 = 0)

5次多项式S曲线(位4 = 1，位 10 & 位 11 = 3)

参数:
distance:要在当前基准轴上移动的用户单位的增量距离，这是驱动移动的“输入”轴上测量的移动的结果。
link dist:用户单位的正增量距离，需要在“链接”轴上测量，以导致在基准轴上的运动。
link acc:输入轴上用户单位的正增量距离，基准轴在其上加速。
link dec:输入轴上用户单位的正增量距离，基准轴在其上减速。
link axis:指定要“链接”到的轴。它应该设置为一个介于0和可用轴数之间的值。
link_options:位值选项，以自定义您的MOVELINK如何运作
位0当链接轴上发生注册事件 MARK时，链接完全开始1
位1链接从链接轴上的绝对位置开始（请参阅link_pos以获取开始位置）2
位2当设置该位时，MOVELINK会自动双向重复。(此模式可通过设置轴参数的第1位清除)4
位4如果设置了该位，则将使用“ S”速度曲线而不是梯形速度曲线构建MOVELINK加速和减速阶段。16
位5链接只有在链接轴正向移动时才会激活32
位8当链接轴上发生注册事件MARKB时，链接才完全开始256
位9当链接轴上发生注册事件R_MARK时，链接才完全开始。 （有关频道号，请参见link_pos）512
位10S-Ramp模式选择，当第4位设置时使用。图()显示了与线性斜坡相比的峰值加速度。
0 - SIN速度分布图(1.55)
1 -9次多项式速度(2.42)
2 -7次多项式速度(2.16)
3 -5次多项式速度(1.86)
位11
位13从轴跟随主轴DPOS
点14在从阈值开始的正向移动期间，链接处于激活状态。阈值更新为最高的正数值。
link_pos:link_option位1 -在用户UNITS的链接轴上，MOVELINK将开始的绝对位置。
Link_option位9 -开始移动的注册通道
base_dist固件V2.0253起。“基本距离”是要以基本速度行驶的距离，是总第一个“距离”参数的一部分。参见下面的例4和5。
在使用基本距离时不能使用MOVELINK_MODIFY。

如果参数3和参数4的和大于参数2，它们都按比例减少，直到它们等于参数2。

link_dist位于链接轴的用户单位中，应始终指定为正距离。

开始的链接选项(位1、2、8和9)可以与重复的链接选项(位4和8)和方向组合。

start_pos不能在REP _DIST位置的一个伺服周期内移动。

例

例1

飞剪每隔160米将一张长纸剪成卡片，同时以材料的速度移动。剪切机的移动距离可达1.2米，其中本例中使用的是1米。纸张距离由编码器测量，单位转换因子被设置为在两个轴上给出米的单位:(注意，7轴是链接轴)

WHILE IN(2) = ON

  MOVELINK(0, 150, 0, 0, 7)        '停留(不移动)150m

  MOVELINK(0.3, 0.6, 0.6, 0, 7)    '加速到纸张速度

  MOVELINK(0.7, 1.0, 0, 0.6, 7)    '跟踪纸张然后减速

  WAIT LOADED '等待，直到加速移动链接完成

  OP(8, ON)   '激活刀

  MOVELINK(-1.0, 8.4, 0.5, 0.5, 7) '缩回割刀启动

  WAIT LOADED

  OP(8, OFF)  '在向外冲程结束时关闭刀具

在这个程序中，控制器首先等待轧辊在第一行送出150米。在这段距离之后，剪切机加速到与纸张的速度相匹配，以相同的速度移动，然后在1米行程内减速停止。此移动是使用两个单独的MOVELINK命令指定的。这允许程序等待下一个移动缓冲区清空，NTYPE= 0，表示加速阶段完成。请注意，测量轴上的距离(每个MOVELINK命令中的链接距离):150,0.8,1.0和8.2加起来为160 m。

为了确保切割过程中刀具与纸张的速度和位置匹配，MOVELINK命令的参数必须正确:通常最简单的方法是分别考虑加速、匀速和减速阶段，然后根据需要将它们组合起来:

规则1

在达到匹配速度的加速阶段，链接距离应该是移动距离的两倍。因此，加速阶段可以单独指定为:

MOVELINK(0.3,0.6,0.6,0,1) 'Move都是accel

规则2

在速度匹配的等速阶段，两个轴移动的距离相同，所以移动的距离应该等于连杆的距离。因此，恒速相位可以指定为:

MOVELINK(0.4,0.4,0,0,1) '全等速

在这种情况下，减速阶段被设置为与加速度相匹配:

MOVELINK(0.3,0.6,0,0.6,1) '所有decel

每个阶段的移动现在可以添加到总移动中。

MOVELINK(1,1.6,0.6,0.6,1) '和上面的3个运动一样

但在上面的例子中，加速阶段是分开的:

MOVELINK(0.3,0.6,0.6,0,1)

MOVELINK(0.7,1.0,0,0.6,1)

这允许在加速阶段结束时打开输出。

例2

精确比例变速箱

MOVELINK可用于在两个轴之间创建精确的传动比齿轮箱。假设需要创建4000 / 3072的齿轮箱连杆。这个比率是不精确的(1.30208333)，如果输入CONNECT命令，轴将慢慢地脱离同步。将“链接选项”设置为4，可以连续重复MOVELINK以消除此问题:

MOVELINK(4000, 3072, 0,0, linkaxis, 4)

例3

线圈绕组

在本例中，设置了单位转换因子UNITS，以便出料运动以毫米为单位，主轴位置以转数为单位进行测量。因此，出料眼在主轴25转的范围内移动50mm，命令如下:

MOVELINK(50,25,0,0, linkax)

如果希望在第一次主轴转数上加速，在最后3转数上减速，则命令为

MOVELINK(50, 25, 1, 3, linkax)

OP(motor, ON)   '打开主轴电机

FOR layer = 1 TO 10

  MOVELINK(50, 25, 0, 0, 1)

  MOVELINK(-50, 25, 0, 0, 1)

NEXT layer

WAIT IDLE

OP(motor, OFF)
