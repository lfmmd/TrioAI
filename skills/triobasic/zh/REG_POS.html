REG_POS

类型

轴参数(只读)

替代格式

RPOS

描述

以用户UNITS为单位存储在每个轴上看到注册标记的最新位置。此参数用于第一个(A)硬件注册通道，或仅用于Z标记。

值

在用户UNITS中发生注册事件的绝对位置。

例

切纸机使用凸轮轮廓形状，通过伺服驱动的滚筒快速绘制纸张，然后在切割时停止。纸张上印有注册标志。检测到这个标记后，通过使用CAM命令的第三个参数缩放凸轮轮廓来调整下一张纸的长度:

'使用拉伸的注册程序示例:CAM

'设置窗口的打开和关闭:

  Length = 200 

  OPEN_WIN = 10

  CLOSE_WIN = length - 10

  GOSUB Initial

Loop:

  TICKS = 0 '设置毫秒计数器为0

  IF MARK THEN

    offset = REG_POS

    '下一行如果在工作表末尾，使offset值减去ve:

    IF ABS(offset - length) < offset THEN offset = offset - length

    PRINT "Mark seen at:"; offset[5.1]

  ELSE

    offset = 0

    PRINT "Mark not seen"

  ENDIF



 '每次移动前重置注册:

 DEFPOS(0)

  REGIST(3 + 768) 'Allow mark at first 10mm or last 10mm of sheet

  CAM(0, 50, (length + offset * 0.5) * cf, 1000)

  WAIT UNTIL TICKS < -500

  GOTO Loop

(变量“cf”是一个常量，将根据每个编码器脉冲的机器绘制长度来计算)
