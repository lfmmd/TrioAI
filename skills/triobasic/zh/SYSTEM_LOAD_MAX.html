SYSTEM_LOAD_MAX

类型

系统参数

描述

SYSTEM_LOAD_MAX返回自上电以来的最大值，或者自SYSTEM_LOAD_MAX上次设置为0以来的最大值。如果SYSTEM_LOAD_MAX大于100，则在某些时候固件系统和运动处理已经超出了伺服周期。应该减少轴的数量或将SERVO_PERIOD设置为更高的值。

值

用于系统和运动处理的伺服周期时间的最大百分比。

例1

从终端0命令行，读取系统固件正在使用的伺服时间的最大百分比。

>>?SYSTEM_LOAD_MAX

56.9780

>>

例2

重置SYSTEM_LOAD_MAX值，以便它可以记录自重置以来的新最大值。

>>SYSTEM_LOAD_MAX = 0

>>
