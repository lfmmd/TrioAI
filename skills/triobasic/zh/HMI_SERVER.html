HMI_SERVER

类型

系统命令

语法

HMI_SERVER[(function[, parameters, ...])]

描述

该命令允许从TrioBASIC程序控制、配置和询问Trio HMI服务器。

如果没有参数，则函数为0，参数为0。

参数

Function0运行HMI_SERVER协议
1读取HMI客户端错误数据
2写入HMI_SERVER事件标志
3读取HMI_SERVER状态数据
4设置HMI轮询超时时间
5读取HMI Client版本信息

Function = 0

语法

HMI_SERVER

HMI_SERVER(0[, debug])

描述

该函数启动HMI_SERVER协议。这个函数永远不会停止，所以程序中这个命令之后的TrioBASIC语句不会被执行。

HMI_SERVER程序通常在HMI Client连接到运动控制器时自动启动。如果你想指定它应该在哪个进程上运行，以及是否应该打印调试信息，你可以手动调用它。

如果您手动执行HMI_SERVER，它所运行的程序将挂起在HMI_SERVER所在行。因此，HMI_SERVER应该是程序执行的最后一行。

参数

Debug0无调试信息
1调试信息打印到通道0(仅在Trio请求时使用)

Function = 1

语法

value = HMI_SERVER(1, error_parameter)

描述

当HMI Client发生错误时，如果可能，将此事件发送到HMI服务器。该命令将返回关于HMI Client中发生的最后一个错误的数据。

参数

error_parameter0错误号特定于HMI 客户端操作系统
1错误字符串特定于HMI客户端操作系统
2错误的程序当错误发生时，HMI客户端正在与之通信的运动控制器上的程序名称。
3错误的过程如果适用，在发生错误时与HMI客户端通信的运动控制器上的程序的进程号。

例

在HMI客户端上报告错误

'检查错误

IF HMI_SERVER(1, 0) THEN

    PRINT "HMI Client reports error"

    PRINT "HMI Error="; HMI_SERVER(1, 0)

    PRINT "HMI Description="; HMI_SERVER(1, 1)

    PRINT "MC Program="; HMI_SERVER(1, 2)

    PRINT "MC Process="; HMI_SERVER(1, 3)

ENDIF

Function = 2

语法

HMI_SERVER(2, parameter[, string[, client_ip]])

描述

HMI服务器可以通知HMI客户某些事件已经发生。这些事件被MotionPerfect使用。可选的client_ip目前被HMI_SERVER命令忽略。字符串参数取决于parameter的值。

参数

parameter0没有事件
1运动控制器有一个更新的HMI设计文件，HMI客户端必须请求它。字符串是运动控制器上要读取的文件的名称。
2请求HMI客户端发送它的当前配置文件。字符串是运动控制器上要写入的文件的名称。
4运动控制器有一个更新的HMI配置文件，HMI客户端必须请求它。字符串是运动控制器上要读取的文件的名称。
8运动控制器有一个更新的HMI客户端固件文件，HMI客户端必须请求它。字符串是运动协调器上要读取的文件的名称。
32在HMI客户端上设置当前页面，下面的参数指定页面名称。String为要选择的页面名称。

例

以5秒的时间间隔自动滚动三个页面。如果一个页面是手动选择的，那么保持这个页面30秒。将页面值从HMI设置为大于3的值，以使页面处于手动模式。

Page = 0

Page_time = 5000

Manual_time = 30000



WHILE(1)

  IF page = 0 THEN

    HMI_SERVER(2, 32, "PAGE1")

    page = 1

    WA(page_time)

  ELSEIF page = 1 THEN

    HMI_SERVER(2, 32, "PAGE2")

    page = 2

    WA(page_time)

  ELSEIF page = 2 THEN

    HMI_SERVER(2, 32, "PAGE3")

    page = 3

    WA(page_time)

  ELSE

    '以手动模式

    page = 0

    TICKS = manual_time

    WHILE TICKS>0

      IF page <> 0 THEN

        TICKS = manual_time

        page = 0

      ENDIF

      WA(1)

    WEND

  ENDIF

WEND

Function = 3

语法

value = HMI_SERVER(3, parameter, return_type)

描述

读取HMI 客户端状态信息。

参数

parameter0客户状态:
0断开连接
1连接
2HMI页面加载
3运行
4错误
1当前HMI设计页面
return_type0整数
1字符串

Function = 4

语法

HMI_SERVER(4, parameter)

描述

设置HMI服务器在终止客户端连接之前等待的无活动的毫秒数。

Function = 5

语法

value = HMI_SERVER(5, parameter)

描述

返回HMI客户端描述。在协议初始化期间，HMI客户端将这些数据发送给HMI服务器。

参数

parameter0HMI客户端引擎主版本号
1HMI客户端引擎次要版本号
2HMI客户端通信协议主要版本号
3HMI客户端通信协议次要版本号
4HMI客户操作系统ID:
0Windows CE
1Windows桌面
5HMI客户端操作系统版本:
位0-15次要版本号
位16-31主设备号
6HMI客户端画面大小:
位0-15宽度(像素)
位16-31高度(像素)
