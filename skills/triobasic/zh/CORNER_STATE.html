CORNER_STATE

类型

轴参数

描述

允许BASIC程序与运动加载过程交互。

这可以用来方便刀具的调整，比如尖角处的刀具旋转。

当设置了CORNER_MODE位2时，此参数才有效。还需要将CORNER_MODE的位1与STOP_ANGLE设置为小于或等于RAISE_ANGLE来停止运动。

值

位描述值
0准备拐角序列，XY轴停止1
1以0速度加载下一个XY移动2
2下一步动作已加载，TANG_DIRECTION已准备好4
加速并继续XY移动0

位0和位2由系统软件设置。位1必须由控制转角序列的应用程序设置。

值将是位集0、1、3和7的组合。

例

当过渡超过RAISE_ANGLE时，需要抬起切刀并将其旋转到新的位置。需要进行以下过程:

  • 系统将CORNER_STATE设置为1，以指示运动准备加载大角度变化。
  • BASIC程序提刀。
  • BASIC程序将CORNER_STATE设置为3。
  • 系统将加载跟随移动，但速度覆盖为零。这允许从TANG_DIRECTION。中获得方向。
  • 系统设置CORNER_STATE为7。
  • BASIC程序定位刀。可能用的是MOVETANG。
  • BASIC程序将CORNER_STATE清除为0。
  • 系统将加速执行下一步动作。

轴0和轴1是XY，轴2是刀的旋转。这个程序控制刀的上升/下降和旋转与CORNER_STATE的状态同步。

BASE(2)

'在轴2上设置一个自动切向移动

IF MTYPE = 0 THEN

    MOVETANG(0, 0)

ENDIF



BASE(0)

CORNER_MODE = 4 '启动CORNER_STATE状态机

RAISE_ANGLE = 15 * PI / 180 '15 degrees



REPEAT

    WAIT UNTIL CORNER_STATE > 0 ''正常情况下此处CORNER_STATE = 1

    WAIT UNTIL MTYPE = 0 '等待当前的XY移动完成



'插入BASIC命令在这里抬刀:



    WA(100)



    '通过BASIC设置CORNER_STATE = 3使固件

    '加载下一个动作，但以零速度进行:

    CORNER_STATE = 3



    '固件设置CORNER_STATE = 7来指示下一个动作加载

    WAIT UNTIL CORNER_STATE = 7





    '在这个例子中，我们使用MOVETANG，如果没有，插入BASIC命令来旋转刀:

    WA(2)



    '允许MOVETANG开始遵循新的TANG_DIRECTION

    WAIT UNTIL REMAIN AXIS(2) = 0



    '插入基本命令来降低刀:



    WA(100)



    '设置CORNER_MODE = 0来加速XY

    CORNER_STATE = 0

UNTIL FALSE

与上述程序并行运行时，一个单独的程序将XY移动输入到运动生成器中。
