FLASH_TABLE_LAST

类型

系统参数

语法

FLASH_TABLE_LAST = value

描述

FLASH_VR_FIRST, FLASH_VR_LAST, FLASH_TABLE_FIRST和FLASH_TABLE_LAST是允许用户配置保存在闪存中的VR/TABLE位置范围的系统参数。当电源断开时，超出定义范围的VR和TABLE位置将不保留。

需要固件V2.0313及更高版本。

参数

Value:在flash中存储Table值的TABLE结束索引。

注1:MC403、MC405、MC4N不自动保存TABLE值。FLASH_DATA可以用TABLE而不是VR来配置运动协调器。在这种情况下，FLASH_TABLE_LAST与FLASH_VR_LAST具有相同的最大值。

例

在MC508中，设置自动保存前32000个索引(TABLE值范围从0到31999)到闪存。因此，TABLE(32000)到TABLE(511999)将是不稳定的，因此在下次上电时设置为0。

FLASH_TABLE_FIRST = 0

FLASH_TABLE_LAST = 31999
