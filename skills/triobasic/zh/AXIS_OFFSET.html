AXIS_OFFSET

类型

槽位参数(MC_CONFIG / FLASH)

描述

AXIS_OFFSET是槽试图分配给轴的第一个轴号。如果该轴已经被使用（它的ATYPE非零），那么该轴将被分配给下一个空闲轴。控制器将根据插槽和模块类型按以下顺序分配轴:

  • EtherCAT和Panasonic轴将通过SLOT分配到从AXIS_OFFSET开始的第一个可用轴(加上EtherCAT的节点地址-1)
  • 然后，FlexAxis将由SLOT分配到从AXIS_OFFSET开始的第一个可用轴
  • 将内置轴分配给从AXIS_OFFSET开始的第一个可用轴
  • 最后，根据BASIC程序分配任意BASIC轴。这包括SLM和SERCOS以及在BASIC中配置的任何EtherCAT或Panasonic轴。

轴分配仅在上电时执行。AXIS_OFFSET应该放在MC_CONFIG脚本中，以便立即生效。

值

模块尝试分配的第一个轴，range = 0到最大轴，default = 0。

例

例1

SLOT-1=内置，AXIS_OFFSET=0
SLOT 0 = EtherCAT, 4轴，不设置节点地址，AXIS_OFFSET=0
AXIS(0-3) EtherCAT
AXIS(4)内置----------

AXIS_OFFSET SLOT(0)=0

AXIS_OFFSET SLOT(-1)=0

这是默认情况。

例2

SLOT-1 =内置，AXIS_OFFSET=2
SLOT 0 = EtherCAT, 4轴，无节点地址集，AXIS_OFFSET=0
AXIS (0-3) EtherCAT
AXIS(4)内置

AXIS_OFFSET SLOT (0)=0

AXIS_OFFSET SLOT(-1)=2

内置仍然是最后一个，因为它是最后分配的，控制器尝试将内置轴分配给从2开始的第一个可用轴，即4。

例3

SLOT-1 =内置，AXIS_OFFSET=0
SLOT 0 = EtherCAT, 4轴，无节点地址集，AXIS_OFFSET=1
AXIS(0)内置
AXIS (1-4) EtherCAT

AXIS_OFFSET SLOT(0)=1

AXIS_OFFSET SLOT(-1)=0

偏移量将EtherCAT推出一个轴，因此在分配内置轴时AXIS(0)仍然是空闲的

例4

SLOT-1 =内置，AXIS_OFFSET=0
SLOT 0 = EtherCAT, 4轴，驱动器上的节点开关设为2,3,4,5,AXIS_OFFSET=0
AXIS(0)内置
AXIS (1-4) EtherCAT

AXIS_OFFSET SLOT(0)=0

AXIS_OFFSET SLOT(-1)=0

EtherCAT轴从它们的节点地址-1+AXIS_OFFSET设置

例5

SLOT-1 =内置，AXIS_OFFSET=0
SLOT 0 = EtherCAT, 4轴，节点设置为2,3,4,5,AXIS_OFFSET=1
AXIS(0)内置
AXIS (2-5) EtherCAT

AXIS_OFFSET SLOT(0)=1

AXIS_OFFSET SLOT(-1)=0

EtherCAT轴从它们的节点地址-1+AXIS_OFFSET设置

例6

SLOT-1 =内置，AXIS_OFFSET=0
SLOT 0 = FlexAxis, 8轴模块，AXIS_OFFSET=1
AXIS(0)内置
AXES(1-8) FlexAxis

AXIS_OFFSET SLOT(-1)=0 

AXIS_OFFSET SLOT(0)=1

内置轴通常放在柔性轴之后。在这里，柔性轴被迫从轴1开始，因此内置轴可以取轴0。
