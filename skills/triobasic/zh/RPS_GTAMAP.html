GTAMAP

类型

机器人命令

语法

value = GTAMAP

描述

返回一个字符串，显示GTA索引表示笛卡尔坐标(即使用GTA或GTAE设置显式点类型为0)。每个字符以十六进制字符(' 0 '到' F ')的形式表示 半个字节(或4位)，低GTA索引存储在高阶位中，如下所示。
GTA01234567891011
字符012
位321032103210

建议优先使用GTAEMAP而不是GTAMAP，因为它以统一的方式处理所有点类型，因此GTAEMAP(0)在功能上等同于GTAMAP，因为点类型0表示笛卡尔坐标。

例

GTA(0) = 1, 2, 3, 4, 5, 6

GTA(1) = 1, 2, 3, 4, 5, 6

GTAJ(2) = 1, 2, 3, 4, 5, 6

GTA(3) = 1, 2, 3, 4, 5, 6

GTAJ(4) = 1, 2, 3, 4, 5, 6

GTA(5) = 1, 2, 3, 4, 5, 6

GTA(6) = 1, 2, 3, 4, 5, 6

GTAJ(7) = 1, 2, 3, 4, 5, 6

GTAJ(8) = 1, 2, 3, 4, 5, 6

GTAJ(9) = 1, 2, 3, 4, 5, 6

GTA(10) = 1, 2, 3, 4, 5, 6

GTAJ(11) = 1, 2, 3, 4, 5, 6



PRINT "GTAMAP = "; LEFT(GTAMAP, 10)

PRINT "GTAJMAP = "; LEFT(GTAJMAP, 10)

PRINT "GTAEMAP(0) = "; LEFT(GTAEMAP(0), 10)

PRINT "GTAEMAP(1) = "; LEFT(GTAEMAP(1), 10)
