MOVECIRC

类型

轴命令。

语法

MOVECIRC(end1, end2, centre1, centre2, direction [,ta [,output]])

替代格式

MC()

描述

移动两个正交轴，以便在工具点处产生圆弧。 圆弧的长度和半径由命令行中的五个参数定义。 移动参数始终相对于最后指定的移动的结束。 这是圆圆周上的起始位置。 轴1是当前的 BASE 轴。 轴2是 BASE 数组中的下一个轴。 前4个距离参数根据 BASE轴的当前单位转换系数进行缩放。

通过添加过渡角和可选的输出偏移参数，可以启用使用过渡曲线的增强型MOVECIRC功能。 从版本2.0221开始的固件中可用。

为了正确执行MOVECIRC（）命令，生成圆弧的两个轴必须具有相同数量的编码器脉冲/线性轴距离。 如果不是这种情况，则可以在许多情况下通过使用ENCODER_RATIO 或 STEP_RATIO.来调整编码器比例。

如果指定的终点不在圆弧上。 圆弧将以中心和终点之间的直线指定的角度结束。

在MOVECIRC期间，两个轴都不能越过设置的绝对重复距离(REP_DIST)。这样做可能会导致一根或两根轴跳动或它们的FE 值超过 FE_LIMIT.

参数

end1:在BASE轴上的位置结束。
end2:在BASE数组中下一轴上的位置结束。
centre1:在BASE上移动的位置。
centre2:在BASE数组中下一个要移动的轴上的位置。
direction:0弧沿逆时针方向插补
1弧沿顺时针方向插补
2使用到端点的最短路径对弧进行插值
3使用到端点的最长路径对弧进行插值

ta弧度过渡角
output从TABLE（0）开始将过渡曲线的详细信息输出到TABLE的选项
0修订弧中心X
1修订弧中心Y
2过渡曲线计算得出的比例因子
3修正圆弧半径
4过渡曲线长度
5总长度
6使用的过渡角度（如果圆弧角度不足，则可以小于请求的角度）

在进入圆弧或从圆弧离开时需要减小加速力的地方，可以应用过渡曲线。

沿弧线传播的加速度由下式给出：

a = v * v / r

NB:曲率是1 /半径

从该方程式可以看出，以连续速度移入或移出圆弧的机器将经历高度依赖于速度的加速度阶跃变化。

上图显示了过渡曲线的基本概念。 红色的螺旋部分具有逐渐变化的曲率，可平滑这种加速度变化。 该图显示了较大的过渡角，但在许多情况下，即使是几度的“过渡曲线”也可以显着降低机器振动。

请注意，过渡曲线工具会修改圆弧的位置和半径。 修改的可接受性取决于应用程序。

例

例 1

绘制字母“ 0”的命令序列可能是：

MOVE(0, 6)                 '运动从 A -> B

MOVECIRC(3, 3, 3, 0, 1)    '运动从 B -> C

MOVE(2, 0)                 '运动从 C -> D

MOVECIRC(3, -3, 0, -3, 1)  '运动从 D -> E

MOVE(0, -6)                '运动从 E -> F

MOVECIRC(-3, -3, -3, 0, 1) '运动从 F -> G

MOVE(-2, 0)                '运动从 G -> H

MOVECIRC(-3, 3, 0, 3, 1)   '运动从 H -> A

例 2

需要使用机器将化学药品滴入试管中。 喷嘴可以沿其轨道上下移动。 最有效的运动是使喷嘴在试管之间成弧形运动。

BASE(0, 1)

MOVEABS(0, 5)               '运动到第一个试管的上方

MOVEABS(0, 0)               '向下运动第一次滴入

WAIT IDLE

OP(15, ON)                  '打开点滴器

WA(20)

OP(15, OFF)

FOR x = 0 TO 5

  MOVECIRC(5, 0, 2.5, 0, 1) '试管之间的圆弧

  WAIT IDLE

  OP(15, ON)                '打开点滴器

  WA(20)

  OP(15, OFF)

NEXT x

MOVECIRC(5, 5, 5, 0, 1)     '运动到下一个位置

例 3

使用过渡曲线绘制字母“ 0”，以减小拐角处的加速力。 在每个45度弧中，过渡为10度，过渡为10度。

ta = 10 * PI / 180 '设置过渡角为10度

MOVEABS(0, 40)                    '运动从 A -> B

MOVECIRC(30, 30, 30, 0, 1, ta)    '运动从 B -> C

MOVE(20, 0)                       '运动从 C -> D

MOVECIRC(30, -30, 0, -30, 1, ta)  '运动从 D -> E

MOVE(0, -40)                      '运动从 E -> F

MOVECIRC(-30, -30, -30, 0, 1, ta) '运动从 F -> G

MOVE(-20, 0)                      '运动从 G -> H

MOVECIRC(-30, 30, 0, 30, 1, ta)   '运动从 H -> A
