* 乘

类型

数学运算符

语法

<expression1> * <expression2>

描述

用expression1乘以expression2

参数

expression1:任何有效的TrioBASIC表达式
expression2:任何有效的TrioBASIC表达式

例

例1

用10乘以2.1和9的和来计算“因子”的值。存储在' factor '中的值将是111。

Factor = 10 * (2.1 + 9)

例2

两个矩阵相乘。

'a = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}, {0, 1, 2}, {3, 4, 5}}

'b = {{1, 2, 3, 4}, {5, 6, 7, 8}, {9, 0, 1, 2}}

'c = {{38, 14, 20, 26}, {83, 38, 53, 68}, {128, 62, 86, 110}, {23, 6, 9, 12}, {68, 30, 42, 54}}



DIM a AS FLOAT(5, 3)

DIM b AS FLOAT(3, 4)

DIM c AS FLOAT(5, 4)



a = 1, 2, 3, 4, 5, 6, 7, 8, 9, 0, 1, 2, 3, 4, 5

b = 1, 2, 3, 4, 5, 6, 7, 8, 9, 0, 1, 2



c = a * b



PRINT c

这段代码的输出是:

>>38.0000, 14.0000, 20.0000, 26.0000, 

83.0000, 38.0000, 53.0000, 68.0000, 

128.0000, 62.0000, 86.0000, 110.0000, 

23.0000, 6.0000, 9.0000, 12.0000, 

68.0000, 30.0000, 42.0000, 54.0000
