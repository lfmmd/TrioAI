DISABLE_GROUP

类型

系统命令

语法

DISABLE_GROUP(parameter[,parameters…])

描述

用于创建一个轴组，如果轴组中一个或多个轴出现运动错误，该轴组将被禁用。在创建轴组之后，当发生错误时，轴组中的所有轴都将其AXIS_ENABLE设置为OFF, SERVO设置为OFF。

可以创建多个组，尽管一个轴不能属于多个组。

只有具有单独启用的轴才能在禁用组中使用。例如数字驱动器和步进驱动器。

DISABLE_GROUP(-1)

语法

DISABLE_GROUP(-1)

描述

清除所有轴组

DISABLE_GROUP(axis1…)

语法

DISABLE_GROUP(axis1 [,axis2[, axis3[, axis4.....]]])

描述

将列出的轴分配给一个轴组

参数

axis1:轴组中第一个轴的轴号
axis2:轴组中第二个轴的轴号。
axisN:轴组中第n个轴的轴号。

可以指定与系统上的轴数相同数量的的参数。

例

例1

一台机器有2个功能独立的系统，它们有自己的紧急停止和操作员保护装置。如果机器的一部分出现错误，在排除错误原因并重新启动轴组时，另一部分可以安全地保持运行。我们需要设置2个独立的轴组。

DISABLE_GROUP(-1)       '删除所有轴分组

DISABLE_GROUP(0,1,2,6)  '轴组的轴为0到2和6

DISABLE_GROUP(3,4,5,7)  '轴组的轴为3到5和7

WDOG=ON '打开使能继电器，使能远程驱动

FOR ax=0 TO 7

  AXIS_ENABLE AXIS(ax)=ON  '使能8个轴

  SERVO AXIS(ax)=ON '启动每个轴的位置闭环

NEXT ax

例2

由同一_运动控制器_操作的两台输送机需要独立运行，这样如果其中一台出现“堵塞”，就不会使另一台输送机停止运行。

DISABLE_GROUP (0)'将轴0放到轴组中

DISABLE_GROUP (1)'将轴1放到另一个轴组中

GOSUB group_enable0

GOSUB group_enable1

WDOG=ON

FORWARD AXIS(0)

FORWARD AXIS(1)



WHILE TRUE

  IF AXIS_ENABLE AXIS(0)=0 THEN

    PRINT "motion error axis 0"

    reset_0_flag=1

  ENDIF

  IF AXIS_ENABLE AXIS(1)=0 THEN

    PRINT "motion error axis 1"

    reset_1_flag=1

  ENDIF

  IF reset_0_flag=1 AND IN(0)=ON THEN

    GOSUB group_enable0

    FORWARD AXIS(0)

    reset_0_flag=0

  ENDIF

  IF reset_1_flag=1 AND IN(1)=ON THEN

    GOSUB group_enable1

    FORWARD AXIS(1)

    reset_1_flag=0

  ENDIF

WEND



group_enable0:

  BASE(0)

  DATUM(7) ' 清除轴0上的运动错误

  WA(10)

  AXIS_ENABLE=ON

  SERVO=ON

RETURN

group_enable1:

  BASE(1)

  DATUM(7) ' 清除轴1上的运动错误

  WA(10)

  AXIS_ENABLE=ON

  SERVO=ON

RETURN

例3

如果发生运动错误，机器中的一组轴需要重置，而不影响其余轴。这应该通过手动清除错误，按下按钮清除控制器的错误标志并重新启用运动来完成。

  DISABLE_GROUP(-1)      '删除任何先前的轴分组

  DISABLE_GROUP(0,1,2)   '分组轴为0到2 

  GOSUB group_enable     '使能轴并清除错误

  WDOG=ON

  SPEED=1000

  FORWARD



  WHILE IN(2)=ON   '检查轴0，但组中的所有轴将一起禁用

    IF AXIS_ENABLE =0 THEN

      PRINT "Motion error in group 0"

      PRINT "Press input 0 to reset"

      IF IN(0)=0 THEN       '检查复位按钮是否按下

        GOSUB group_enable  '清除错误并启用轴

        FORWARD             '重新启动动作

      ENDIF

    ENDIF

  WEND

  STOP            '停止程序运行子路径



group_enable:     '清除子错误并使能轴

  DATUM(0)        '清除任何运动错误

  WA(10)

  FOR axis_no=0 TO 2

    AXIS_ENABLE AXIS(axis_no)=ON  '使能轴

    SERVO AXIS(axis_no)=ON        '启动位置闭环

  NEXT axis_no

  RETURN
