PROJECT_KEY

类型

系统命令

语法

PROJECT_KEY key_string security_code_type

描述

在TRIOINIT. BAS中使用SD卡上的脚本文件，启用加载加密项目。

项目密钥由Motion Perfect在加密项目时生成

参数

key_string一个字符串，它是由Motion Perfect生成的项目密钥
security_code_type0(可选)控制器安全码
1OEM安全码
2用户安全码

例

例1

使用SD卡加载之前被Motion Perfect使用控制器安全码加密过的项目。

'============================================================

'应用程序:SDCARD启动文件

'文件名:TRIOINIT. 'BAS

'平台:MC4xx

'使用项目加密器生成特定于目标运动协调器序列号的PROJECT_KEY

'

'------------------------------------------------------------

PROJECT_KEY "MyKey"

FILE "LOAD_PROJECT" "MyEncryptedProject" '加载所需项目

例2

使用SD卡加载先前由Motion Perfect使用用户安全代码加密的项目。

'============================================================

'应用程序:SDCARD启动文件

'文件名:TRIOINIT. 'BAS

'平台:MC4xx

'使用项目加密器生成特定于目标运动协调器序列号的项目键

'

'------------------------------------------------------------

PROJECT_KEY " c8NaHIvA.tU"2

FILE "LOAD_PROJECT" "MyEncryptedProject" '加载所需的项目
