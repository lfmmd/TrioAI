DV_OUT

类型

系统命令

语法

value = DV_OUT(channel)

DV_OUT(channel) = value

描述

DV_OUT在远程I/O块(如Flexslice温度模块、热电偶和RTD类型)上设置状态或值，称为设备变量。在系统启动期间，通道是根据DV_MAP分配的，通常每个模块会有4或8个通道。

DV_OUT还返回设备变量的当前状态或值。

参数

value:从设备变量的输出中读取或写入的整数值
channel:0…最大通道号

例

读取热电偶的温度值，并与给定的设定值进行比较。使用差值来控制加热器使用低频PWM。温度值是按摄氏度x 100缩放的整数值，因此DV_IN(通道)必须除以100。

DIM temperature AS FLOAT

DIM channel_number AS INTEGER = 2

DIM p_cycle AS INTEGER = 500 ' 半秒周期时间

DIM p_mark AS INTEGER ' mark/space值作为时间，单位为msecs



DIM setpoint AS FLOAT



TICKS = 0 ' TICKS 每个伺服周期减少一个计数

WHILE TRUE

'从P367读取温度值

  temperature = DV_IN(channel_number) / 100

  setpoint = VR(0) ' degrees celcius



'与设定值比较，得出温度误差值

  temperature_error = setpoint - temperature



  loop_gain = VR(1) ' 典型值1 .. 10

  p_mark = temperature_error * loop_gain * p_cycle / 100



  IF TICKS > -p_mark THEN

    DV_OUT(channel_number) = 1

  ELSE

    DV_OUT(channel_number) = 0

  ENDIF



  IF TICKS < -p_cycle THEN

    TICKS = 0

  ENDIF



WEND
