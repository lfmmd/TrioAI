COMBINE_CONTROL

类型

轴的命令

语法

COMBINE_CONTROL(mode, source_1, source_2, Numerator_1, Denominator_1, Numerator_2, Denominator_2)

描述

COMBINE_CONTROL命令用于设置COMBINE_AXES函数的操作模式。

参数

mode:0DPOS输出= source_1位置+ source_2位置
1DPOS输出= source_1位置- source_2位置
2√ (source_1 MPOS ^2 + source_2 MPOS ^2) [1]
3DPOS输出= (source_1位置+ source_2位置)/ 2
source_10 = DPOS, 1 = MPOS
source_20 = DPOS, 1 = MPOS
numerator_1在函数中使用该值之前应用source_1
denominator_1在函数中使用该值之前应用source_1
numerator_2在函数中使用该值之前应用source_2
denominator_2在函数中使用该值之前应用source_2

[1]模式2只有3个参数:

COMBINE_CONTROL(mode, offset_1, offset_2)
mode2
offset_1应用于source_1值的偏移量
offest_2应用于source_2值的偏移量

例

例1

子例程用于设置一个轴，使其位置由其他两个轴的和定义。将第一个源轴设置为0.5倍，第二个源轴设置为x2。

DPOS out = MPOS1 * 0.5 + MPOS2 * 2

'=============================

' source1 + source2

mode_0:

source1 = 10

source2 = 11

out_axis = 12



source_mpos = 1

source_dpos = 0



'COMBINE_CONTROL(mode, source  1, source  2, Numerator  1, Denominator  1, Numerator  2, Denominator  2)

COMBINE_CONTROL(0, source_mpos, source_mpos, 1, 2, 500, 250)

COMBINE_AXES(source1, source2)



BASE(source1, source2)

MOVE(100, 120)

WAIT IDLE



PRINT DPOS AXIS(out_axis)



RETURN
