SRAMP

类型

轴参数

描述

此参数存储用于控制应用于梯形轮廓的舍入量的s斜坡因子。当移动未进行时，应将SRAMP设置为ACCEL/DECEL时间的最大一半。更改后，设置需要很短的时间才能应用。

请注意，SRAMP参数仅在VP_MODE = 0时使用，其他模式使用JERK参数平滑运动。

值

0到250毫秒之间的时间

在移动开始之前必须设置SRAMP。例如，如果您将SRAMP从0更改为200，然后在200毫秒内开始移动，则不会应用完整的SRAMP设置。

例

为了提供平稳过渡到加速度，s斜坡应用的时间为50毫秒。

VP_MODE = 0

SPEED = 160000

ACCEL = 1600000

DECEL = 1600000

SRAMP = 50



WA(50)



MOVEABS(100000)

如果没有S-ramp因子，加速度需要100 ms才能达到设定的速度。当SRAMP = 50时，加速度需要150ms，但力(扭矩)的变化率是可控的，即加加速度是有限的。
