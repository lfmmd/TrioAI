CONTROL

类型

系统参数(只读)

描述

Control参数返回系统中运动控制器的ID号:

当运动控制器被锁定时，上述数字加1000。例如，一个锁定的MC664将返回1664。

例

例1

在命令行查看被锁定控制器的控制值:

>>PRINT CONTROL

1464

>>

例2

检查程序中的控制器类型，如果失败则停止程序。:

IF CONTROL <> 664 THEN

  PRINT #terminal, "这个程序是设计用来运行MC664的"

  HALT

ENDIF
