DRIVE_MODE

类型

轴参数(MC_CONFIG)

语法

DRIVE_MODE AXIS(n) = value

描述

DRIVE_MODE设置EtherCAT远程驱动器的操作模式。如果要在上电时以所需模式初始化EtherCAT，则必须在MC_CONFIG中设置此参数。DRIVE_MODE自动设置驱动器的操作模式和轴ATYPE。

这个轴参数可以添加到MC_CONFIG中。

参数

value:1 : 循环同步位置模式（CSP）
2 : 循环同步速度模式（CSV）
3: 循环同步力矩模式（CST）
4: 开环速度（变频器驱动）
5: 参考编码器
6: 开环位置（步进驱动器）

例

例1

将设置四个EtherCAT轴，位置模式为2个轴，速度模式为1个轴，转矩模式为1个轴。 请注意，当驱动器处于CSV或CST模式时，运动控制器可以关闭位置环，或者可以开环操作轴。

'在MC_CONFIG中设置4个轴

'注意:ATYPE是自动设置的，不要在MC_CONFIG中设置

DRIVE_MODE AXIS(0)=1 '位置模式

DRIVE_MODE AXIS(1)=1 '位置模式

DRIVE_MODE AXIS(2)=2 '速度模式

DRIVE_MODE AXIS(3)=3 '扭矩模式

例2

EtherCAT轴是处于位置模式的步进驱动器，但没有用于位置验证的编码器。 模式6只是使MPOS成为DPOS的直接副本，以便REP_DIST和链接的轴正常运行。

'在MC_CONFIG中设置2个轴

'注意:ATYPE是自动设置的，不要在MC_CONFIG中设置

DRIVE_MODE AXIS(0)=6 '无反馈位置模式

DRIVE_MODE AXIS(1)=6 '无反馈位置模式
