MOVEL

类型

机器人命令

语法

MOVEL target [S:= T:= O:= ZT:= ZF:= A:= D:=]

描述

MOVEL用于沿线性路径将机器人从一个点移动到另一个点。所有轴同时到达目的地位置。

KINEMATIC_GROUP需要RPS run-time FEC

参数

参数数据类型描述
TargetTARGET机器人移动到的笛卡尔目标。
注意:关节目标不适合这个动作。如果使用关节目标则会生成错误“无效混合数据类型”。
Embedded parameters:详见RPS移动命令

例

例1

GTA(0) = 350, 0, 50, 180, 0, 0, "p1"

GTA(1) = 400, 100, 20, 180, 0, 0, "p2"

GTA(2) = 450, -300, 60, 180, 0, 0, "p3"

OBJECT_FRAME(1,"of1", -10, 0, 0, 0, 0, 15)

TOOL_OFFSET(1,"tool1", 0, 0, 50, 0, 0, 0)

ROBOT(0)

MOVEL GTA("p1")

MOVEL GTA("p2")

MOVEL GTA("p3")

STOP

这个例子命令Robot 0线性移动到目标点0。请注意，在MOVE中没有其他嵌入指令。该指令将使用以下参数将机器人MOVE到目标点1。
类型参数
Object Frame当前激活的对象框架
Tool Offset当前激活的工具偏移量
Speed []WORLD_POS_SPEED * LOOKAHEAD_FACTOR
Acceleration []WORLD_POS_MAX_ACC * JOG_ACC_FACTOR
Jerk []WORLD_POS_MAX_JERK
Blend radius []0

例2

MOVEL GTA("p1") S:=100
类型参数
Speed[]100 * LOOKAHEAD_FACTOR

例3

MOVEL GTA("p1") 0:="of1"
类型参数
Object Frame对象框架 "of1"
Speed[]WORLD_POS_SPEED * LOOKAHEAD_FACTOR

Of1未设置为激活的对象框架。这个指令只考虑了of1的这个移动。

例4

MOVEL GTA("p1") T:="tool1"
类型参数
Object Frame当前激活的对象框架
Tool Offset工具偏移“tool1”

Tool1未设置为激活的工具偏移。这个指令只考虑了tool1的这个移动。

例5

MOVEL GTA("p1") Z:=67

MOVEL GTA("p1")
类型参数
Tool Offset当前激活的工具偏移量
Blend radius[]67毫米

机器人会朝着目标点0和67前进，在到达目标点之前，它会混合下一个动作，走向目标点1。

例6

MOVEL GTA("p1") S:=150 O:="of1" T:="tool1" Z:=42

MOVEL GTA("p1") S:= 100
