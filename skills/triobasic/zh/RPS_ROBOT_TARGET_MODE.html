ROBOT_TARGET_MODE

类型

机器人参数

语法

ROBOT_TARGET_MODE = value

描述

ROBOT_TARGET_MODE用于控制使用GTAE (Global Targets Array - Extended)或TARGETE数据类型时要应用的限制级别。

ROBOT_TARGET_MODE需要RPS run-time FEC。

值

位描述值
10停用机器人数量限制2
1激活机器人数量限制
20停用机器人配置限制4
1激活机器人配置限制
30停用对象框架限制8
1激活对象框架限制
40取消工具偏移限制16
1激活工具偏移限制
50停用机器人框架限制32
1激活机器人框架限制
60停用W轴扩展范围的世界值。默认值在-180到+180之间。64
1激活W轴扩展范围的世界值，同时仍满足关节极限。值介于-无穷大和+无穷大之间(在最后的关节限制范围内)

例

例1

停用所有机器人目标限制。

ROBOT_TARGET_MODE =0 '关闭TARGETEs的所有限制

例2

监控机器人在目标中使用。

ROBOT_TARGET_MODE.1 = 1 '打开机器人数量限制。

例3

目标中使用的监控机器人和该目标的机器人配置。

ROBOT_TARGET_MODE = 6 '打开机器人数量和机器人配置限制。
