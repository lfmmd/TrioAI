THROW_ERROR

类型

开放式架构功能

语法

THROW_ERROR(error_message)

描述

该命令抛出一个带有给定错误消息的错误。它生成一个RUN_ERROR编号268。当产生错误时，会显示给定的错误信息。

此命令可以与“Open Moves”一起使用，如果无法加载某个运动，则抛出错误。它也可以在函数或程序中使用，以自定义错误消息停止程序。

参数

error_message:抛出错误时显示的消息为STRING。

例

例1

如果函数参数无效抛出错误。

FUNCTION set_frame(num as INTEGER)

   IF num > 121 THEN

      THROW_ERROR("FRAME number invalid")

   ELSE

      FRAME = num

   ENDIF

ENDFUNC

例2

如果移动时间过长，抛出错误。这将生成RUN_ERROR并停止程序。

BASE(0)

MOVEABS(100, 200, 10)

TICKS = 2000

WHILE NOT IDLE

IF TICKS < 0 THEN

THROW("Move failed to finish in time")

ENDIF

WEND
