REMOTE

类型

系统命令

语法

REMOTE(slot)

描述

启动REMOTE_PROGRAM通信协议，作为与PCMotion ActiveX通信的程序。REMOTE程序将占用一个用户进程，无论它是自动运行还是手动运行。建议REMOTE运行在高优先级进程上，可以设置REMOTE_PROC来定义REMOTE_PROGRAM运行在哪个进程上。

当您打开PCMotion连接时，REMOTE程序通常会自动启动。如果你想手动控制进程的启动，你可以手动调用它。

如果手动执行REMOTE，它所运行的程序将在REMOTE行挂起。因此，REMOTE应该是要执行的程序的最后一行。

参数

slot:0

例

一个程序，如果项目想在调试模式下运行，它将在进程20上启动REMOTE程序。

WHILE(1)

IF VR(debug)=TRUE THEN

REMOTE(0)

ELSE

WA(100)

ENDIF

WEND
