LOADED

类型

轴参数

描述

检查是否所有的移动都已加载到MTYPE缓冲区中，因此当没有缓冲的移动时将返回TRUE值，否则返回FALSE。

虽然可以将LOADED用作任何表达式的一部分，但它通常与WAIT一起使用。

值

TRUE当没有缓冲运动时
FALSE当有缓冲运动时。

例子

当NTYPE缓冲区空闲时，继续加载一系列的移动

WHILE machine_on = TRUE

  WAIT UNTIL LOADED OR machine_off = FALSE

  IF machine_on = TRUE THEN

    MOVE(TABLE(position)

    position = position + 1

  ENDIF

WEND
