RPS_KERNEL_V2

类型

机器人命令

语法

RPS_KERNEL_V2 = value

描述

RPS内核是一种控制器状态机，只有在满足特定操作配置文件的某些条件时才允许运动。有3种操作配置:Auto、Manual和External。请参考RPS操作配置文件了解更多信息。

内核作为安全模块提供了比以前的RPS解决方案更快的响应。

该命令只能在MC_CONFIG文件中设置，但可以在Motion iX (TrioBasic)程序和命令行中读取。

内核只能激活一次，不能手动去激活。这是为了保证操作配置文件内I/O操作的完整性。要停用，需要对控制器进行一次电源循环。

参数

value: booleanTrue = RPS内核被激活。
要去激活，需要重启控制器。

例

例1

激活RPS内核(从MC_CONFIG)。

RPS_KERNEL_V2 = TRUE
