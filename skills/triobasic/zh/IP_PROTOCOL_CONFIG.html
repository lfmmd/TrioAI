IP_PROTOCOL_CONFIG

类型

系统参数(MC_CONFIG)

描述

MC464:此运动控制器仅限于以太网上的7个通信端口，ip_protocol_config允许用户选择正在使用的端口，从而减少通信处理器的负载。默认情况下，除传输协议文本文件加载器端口外的所有端口都是启用的。建议使用默认开启的MC4xx协议。

MC4xx、MC5xx和MC6xx系列的所有其他运动控制器;该参数的值是一次读取的以太网数据包的数量。默认值为0，允许固件自动决定适合高效操作的数据大小。

取值(仅限MC464)

最多可选择7位，默认值为575 ($23F)。
位描述值
0Motion Perfect (Telnet)1
1PCMotion2
2Modbus4
3EthernetIP8
4IEC61131-3编程16
5Uniplay32
6传输协议文本文件加载器64
7保留位128
8保留位256
9MC4xx协议文本文件加载器512

不要禁用0位，否则命令行和Motion Perfect将无法使用。

值(所有其他运动控制器控制器)

触发固件从以太网堆栈读取数据的传入以太网数据包的数量。默认值0表示自动。

例

在MC464中，启用使用0-5位的标准端口和传输协议文本文件加载器端口。

IP_PROTOCOL_CONFIG = 1 + 2 + 4 + 8 + 16 + 32 + 64

'或

IP_PROTOCOL_CONFIG = $7F
