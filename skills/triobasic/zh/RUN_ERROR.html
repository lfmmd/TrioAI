类型

过程参数 (只读)

例

例1

使用命令行检查进程5上运行的程序为什么停止了。结果9表示除零错误。

>>?RUN_ERROR PROC(5)

9

>>

例2

在程序中，测试错误值并将结果打印到Motion Perfect终端#5。例程从ON BASICERROR GOTO error_report调用。

error_report:

SELECT_CASE RUN_ERROR

  CASE 1

    PRINT #5, "Command not recognised"

  CASE 9

    PRINT #5, "Divide by zero"

  CASE 181

    PRINT #5, "Device error: No SD card inserted"

  CASE ELSE

    PRINT #5, "Error occurred, number "; RUN_ERROR

END_CASE

STOP
