GOTO

类型

程序结构

语法

GOTO label

…

label:

描述

标识要执行的程序的下一行。

参数

label:程序中存在的有效标签。

如果标签不存在，将在运行时显示一条错误信息，并停止程序的执行。

例

在错误的输入后，使用GOTO来重复程序的某一部分。

start:

PRINT#5, "Press 'y' for YES and 'n' for NO."

GET#5, char

IF char = 121 THEN

  PRINT#5, "YES selected"

ELSEIF char = 110 THEN

  PRINT#5, "NO selected"

ELSE

  PRINT#5, "BAD selection"

  GOTO start

ENDIF
