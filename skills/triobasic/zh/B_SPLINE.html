B_SPLINE

类型

命令

语法

B_SPLINE(mode， {parameters})

描述

此功能扩展数据以生成更高分辨率的运动配置文件。它使用B样条或非均匀有理B样条(NURBS)数学方法在两种模式下运行。

参数

模式:1标准b样条
2非均匀有理b样条

Mode = 1

语法

B_SPLINE(1, data_in, points, data_out, expansion_ratio)

描述

使用B样条数学函数展开存储在TABLE区域中的现有配置文件。扩展因子是可配置的，B_SPLINE将扩展的概要文件存储到TABLE中的另一个区域。

这在理想情况下被运用在源CAM轮廓过于粗糙并且需要外推到更多点的情况下。

参数

data_in:TABLE中存储源配置文件的位置。
points:源配置文件中的点数。
data_out:TABLE中存储扩展配置文件的位置。
expansion_ratio:B_SPLINE函数的展开比。
总输出点数=(点数+1)*展开率
(即如果源配置文件为100点，扩展比例设置为10，则生成的配置文件将为1010点((100+1)* 10)。

例子

将TABLE位置0到9中的10点配置文件扩展到从TABLE地址200开始的更大的110点配置文件。

B_spline (1,0,10,200,10)

Mode = 2

语法

B_SPLINE(2, dimensions, curve_type, weight_op, points, knots, expansion, in_data, out_data)

描述

非均匀有理b样条，通常称为NURBS，已经成为表示由CAD系统设计的几何曲面信息的行业标准方式

NURBS为表示解析形状(如圆锥截面和二次曲面)以及自由形式实体(如车身和船体)提供了统一的数学基础。

NURBS很小，便于数据携带，可以缩放以增加沿曲线的目标点的数量，从而提高精度。一系列NURBS用于描述复杂的形状或表面。

NURBS被表示为一系列带有结点的XYZ点+结点的权重。

参数

dimensions:定义轴的数量。
预留将来使用的必须是3。
curve_type:NURBS曲线类型的分类。
预留将来使用的必须是3。
weight_op:设置结的权重
0 =所有权重设置为1。
knots:定义的结数。
points:数据点数。
expansion:定义扩展后的曲线在表格中所拥有的点数。
总输出点=点数*展开。最小值= 3。
in_data:输入数据的位置。
out_data:表起始位置存储输出点X0, Y0, Z0等。

例子

从9组X Y Z数据点开始，扩展5组，得到45组X Y Z数据点(135个Table点)。然后将配置文件从XYZ组拆分为单独的轴，以便可以使用CAMBOX执行配置文件。

weight_op =0 '0将所有权重设置为1.0

points =9 '数据点的个数

knots =13 '结点个数的

expansion=5 '膨胀系数

in_data =100 '数据点

out_data =1000 '构造输出的表位置
