MOVEADD

类型

轴命令

语法

MOVEADD(distance)

描述

MOVEADD在已经在轴上运行的移动上插入一个额外的位置变化。移动的距离与主移动的用户单位相同，即如果轴的UNITS设置为mm，则MOVEADD添加的距离以mm为单位，速度以mm/s为单位，加速度以mm/s2为单位。

要通过编程距离移除额外的移动部分，请使用CANCEL(5)。

该动作类似于在通过ADDAX连接到主轴的虚拟轴上进行MOVE。在MOVEADD的情况下，不需要虚拟轴。

参数

distance:添加动作的相对距离。正值将使轴前进，负值将使轴后退。

例

例1

随着连续运动在轴上运行，在为添加的运动设置速度，加速度和加加速度值后，进行额外的移动以使轴向前移动40毫米。

FORWARD

WA(100)



FORCE_SPEED = 50

FORCE_ACCEL = 2000

FORCE_DECEL = 2000

FORCE_JERK = 50000

MOVEADD(40)

例2

从产品上的传感器捕获对位后，需要调整位置。 可以在MOVEADD中使用计算出的配准偏移来相对于链接轴前进或后退轴。

BASE(10)

MOVELINK(1000, 1000, 0, 0, 11)

WAIT UNTIL MARK OR IDLE

IF MARK THEN

  adjustment_dist = calc_reg_adjust(REG_POS)

  FORCE_SPEED = 150

  FORCE_ACCEL = 2000

  FORCE_DECEL = 2000

  FORCE_JERK = 50000

  MOVEADD(adjustment_dist)

  missed_mark_count = 0

ELSE

  missed_mark_count = missed_mark_count + 1

ENDIF
