PROGRAM_NAME

类型

过程参数 (只读)

描述

以字符串值的形式返回正在执行的进程名。

该命令可以与PROC和PROCNUMBER配合使用。请看下面的例子。

例

例1

使用Basic库函数，打印正在运行的程序名。

Basic库文件(LIB.BAL):

FUNCTION printprogname(process_number AS INTEGER) AS STRING

  DIM progname AS STRING(20)

    

  progname = PROGRAM_NAME PROC(process_number)

    

  RETURN progname

ENDFUNC

演示程序(DEMO_PROG.BAS):

PRINT "Running: " + printprogname(PROCNUMBER)
