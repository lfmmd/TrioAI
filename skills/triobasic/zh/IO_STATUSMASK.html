IO_STATUSMASK

类型

I / O命令

语法

value = IO_STATUSMASK(slot, address, read_write, vr_index or mask value [, status index])

描述

该命令用于读取或写入远端Ethercat I/O设备的状态掩码。使用状态掩码系统，由设备的IO_STATUS触发的错误可以被掩盖，从而防止SYSTEM_ERROR。如果在同一设备上设置IO_STATUS和IO_STATUSMASK相同位，则会触发系统错误。

状态位的表示取决于设备的实现。

参数

value:-1成功
0失败
slot:EtherCAT I/O模块连接的槽位号
address:读取状态的I/O设备的网络地址。
Function:0读状态掩码
1写状态掩码

欧姆龙“block-type”设备对所有I/O都有一个通用状态值，因此只有status_index 0有效。倍福E-bus设备的每个通道/点有一个状态值。因此，可以使用状态索引读取每个通道的状态值。这里status_index的有效范围是0..(通道数- 1)。
