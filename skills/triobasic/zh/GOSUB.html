GOSUB ..RETURN

类型

程序结构

语法

GOSUB label

…

label:

commands

RETURN

描述

存储GOSUB命令后的行位置，然后分支到指定的标签。到达RETURN语句后，控制返回到存储行。

GOSUB..RETURN循环可以在每个程序中嵌套多达8个深度。

参数

commands:想要执行的TrioBASIC语句
label:在程序中出现的有效标签。

如果该标签不存在，将在运行时显示一条错误消息，并停止程序的执行。

你不能在没有GOSUB的情况下执行RETURN，因为会显示一个runtime错误，你的程序会停止。

例：

例1

使用子程序调用显示位置。

WHILE machine_active

  GOSUB routine1

  GOSUB routine2

WEND

STOP '防止在机器停止时运行到子程序



routine1:

  PRINT "Measured Position="; MPOS; CHR(13);

  RETURN



routine2:

  PRINT "Demand Position="; DPOS; CHR(13);

  RETURN

例2

使用子程序计算一个值。

y = 1

z = 4

GOSUB calc

PRINT "New value = "; x

STOP



calc:

  x = y + z / 2

RETURN
