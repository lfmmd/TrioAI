DV_IN

类型

系统命令(只读)

语法

value = DV_IN(channel)

描述

DV_IN从远程I/O块(如Flexslice温度模块、热电偶和RTD类型)返回值，称为设备变量。在系统启动期间，通道是根据DV_MAP分配的，通常每个模块会有4或8个通道。

参数

value:从设备的输入中读取的值
channel:0…最大通道号

例

将4个温度值复制到局部变量中，通过除以100将其缩放为摄氏度。热电偶片的原始值为度x 100作为整数。

DIM temperature_ch0, temperature_ch1, temperature_ch2,_ temperature_ch3 AS FLOAT

DIM data_index AS INTEGER = 0



'从P367读取4个温度值

temperature_ch0 = DV_IN(data_index) / 100

temperature_ch1 = DV_IN(data_index + 1) / 100

temperature_ch2 = DV_IN(data_index + 2) / 100

temperature_ch3 = DV_IN(data_index + 3) / 100





'与设定值比较，设置输出通道开启加热器

  IF temperature_ch0 < sp1 THEN

      DV_OUT(data_index) = 1

  ELSE

      DV_OUT(data_index) = 0

  ENDIF



  IF temperature_ch1 < sp2 THEN

      DV_OUT(data_index + 1) = 1

  ELSE

      DV_OUT(data_index + 1) = 0

  ENDIF



  IF temperature_ch2 < sp3 THEN

      DV_OUT(data_index + 2) = 1

  ELSE

      DV_OUT(data_index + 2) = 0

  ENDIF



  IF temperature_ch3 < sp4 THEN

      DV_OUT(data_index + 3) = 1

  ELSE

      DV_OUT(data_index + 3) = 0

  ENDIF
