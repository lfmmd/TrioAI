<小于

类型

比较运算符

语法

<expression-1> < <expression-2>

描述

如果<expression-1>小于<expression-2>返回TRUE，否则返回FALSE。

比较运算符通常用于IF、REPEAT和WHILE等条件语句中，但也可以在表达式有效的任何地方使用。

参数

Expression1:任何有效的TrioBASIC表达式
Expression2:任何有效的TrioBASIC表达式

例

检查模拟量输入1的值是否小于10，如果小于10，则执行子程序“rollup”。

IF AIN(1) < 10 THEN GOSUB rollup

查看

= 等于, <> 不等于, <= 小于等于, >= 大于等于, > 大于
