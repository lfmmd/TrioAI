. 位操作

类型

数学运算

语法

<expression>.bit_number

描述

获取表达式的指定位，如果读取则返回读取的值（0或1），如果写入则允许设置或取消设置。

通常，这用于访问变量的特定位，但可以与任何数字表达式，甚至是文字值一起使用。

由于位运算符也被用作小数点，所以要注意不要将其解释为值的一部分。

位操作符前后不应该有空格。

参数：

expression:任何有效的TrioBASIC 表达式
bit_number:要返回的表达式的位数。

例

例 1

检查 AXISSTATUS 是否有远程驱动器错误（位3）

IF AXISSTATUS.3 = 1 THEN

  PRINT "Remote drive error"

ENDIF

例 2

设置 VR(10) 为 54.2，然后读取54的 2位。

VR(10) = 54.2

PRINT (54).2

例 3

展示如何在程序中设置和清除局部变量的位。

DIM a AS INTEGER



a = 0



a.8 = 1

PRINT a '打印 256

a.2 = 1

PRINT a '打印 260

a.8 = 0

PRINT a '打印 4

例 4

命令 CLEAR_BIT, READ_BIT 和 SET_BIT专门用于在VR变量中操作位，这也可以使用位操作符来实现。

VR(23).6 = 0 '等同于 CLEAR_BIT(6, 23)

PRINT VR(13).4 '等同于 PRINT READ_BIT(4, 13)

VR(7).3 = 1 '等同于 SET_BIT(7, 3)

例 5

展示轴1的位3状态（远程驱动错误）。

PRINT AXISSTATUS AXIS(1).3
