DIM.. AS..

类型

程序变量

语法

DIM name AS BOOLEAN [= initial_value]

DIM name AS FLOAT [= initial_value]

DIM name AS INTEGER [= initial_value]

DIM name AS STRING(length) [= "initial_value"]

DIM name AS BOOLEAN[(size1 [,size2 [,size3]])]

DIM name AS FLOAT[(size1 [,size2 [,size3]])]

DIM name AS INTEGER[(size1 [,size2 [,size3]])]

DIM name AS STRING([size1 [,size2 ,]]length)

描述

默认情况下，局部变量是FLOAT类型，不需要声明。可以使用DIM声明声明其他类型的值。可以声明BOOLEAN、FLOAT、INTEGER和STRING。也可以制作数值类型的数组。

如果COMPILE_MODE = 1，则必须声明所有局部变量。

局部变量可以在INCLUDE文件中声明。

注意，对于字符串数组，括号中的最后一个值始终是字符串可能包含的字符数。

如果提供了一个初始值，这个值只会在创建变量时被赋值。

变量类型

BOOLEAN1位二进制值(TRUE或FALSE)
FLOAT64位浮点数(默认)
INTEGER64位带符号整数值
STRINGASCII文本

BOOLEAN

DIM name AS BOOLEAN[(size [,size [,size]])]

描述

将变量声明为BOOLEAN值。这可以与TRUE和FALSE一起使用，任何写入BOOLEAN变量的非零值都会将其状态设置为TRUE。

参数

name:任何用户定义的包含小写字母、数字或下划线(_)字符的名称。
size:数组中每个维度的元素个数，最多3个维度。

大小必须为数字。你不能使用局部变量，VR等来设置这个值。

例

使用局部变量作为标志来跟踪机器的ok状态。

DIM machine_ok AS BOOLEAN



machine_ok = TRUE



WHILE machine_ok = TRUE

  IF MOTION_ERROR <> 0 AND IN(0) = TRUE THEN

    machine_ok =FALSE

  ENDIF

WEND

FLOAT

DIM name AS FLOAT[(size [,size [,size]])]

描述

声明一个变量为浮点数。

参数

name:任何用户定义的包含小写字母、数字或下划线(_)字符的名称。
size:数组中每个维度的元素个数，最多3个维度。

大小必须为数字。你不能使用局部变量，VR等来设置这个值。

例

使用一个位置数组来运行一系列的移动。

DIM position AS FLOAT(10)



position(0) = 0

position(1) = 10.3214

position(2) = 15.123

position(3) = 20.77569

position(4) = 25.2215

position(5) = 22.37895

position(6) = 21.7897

position(7) = 20.1457

position(8) = 15.4457

position(9) = 0



FOR x = 0 TO 9 

  MOVEABS(position(x))

NEXT x

INTEGER

DIM name AS INTEGER[(size [,size [,size]])]

描述

将变量声明为整数值。如果一个浮点数被赋值给一个整型变量，那么它的小数点部分将被截断。

参数

name:任何用户定义的包含小写字母、数字或下划线(_)字符的名称。
size:数组中每个维度的元素数量，最多3个维度。

大小必须为数字。你不能使用局部变量，VR等来设置这个值。

例

例1

声明一个局部变量为整数，以便在从串行端口读取字符时使用。

DIM character AS INTEGER

DIM message AS STRING(200)



WHILE KEY#1

  GET#1, character

  message = message + CHR(character)

WEND

例2

声明一个三维的整数数组，并加载值。

DIM i, j, k AS INTEGER

DIM me AS INTEGER(20, 20, 20)



FOR i = 0 TO 19

  FOR j = 0 TO 19

    FOR k = 0 TO 19

      me(i, j, k) = i * 10000 + j * 100 + k

    NEXT k

  NEXT j

NEXT i



PRINT me(2, 5, 18)

STRING

DIM name AS STRING(length)

DIM name AS STRING(size, length)

DIM name AS STRING(size1, size2, length)

描述

将变量声明为字符串，以便您可以在PRINT语句，逻辑条件的一部分或TrioBASIC中使用文本的任何地方使用它。变量可以由任何生成字符串的函数或参数来赋值，也可以手工赋值。

您可以使用STR函数将数值更改为字符串。

参数

name:任何用户定义的包含小写字母、数字或下划线(_)字符的名称。
size数组中每个维度的元素个数，最多2个维度。
length:变量可以容纳的最大字符数

长度必须是一个常量。不能用局部变量、VR等来设置这个值。

例

例1

预定义一组稍后使用的错误字符串:

DIM error1 AS STRING(20)

error1 = "Feed jammed"

DIM error2 AS STRING(20)

error2 = "Cutter jammed"

DIM error3 AS STRING(20)

error3 = "Out of material"



display_error:

IF error_number = 1 then

  PRINT error1

ELSEIF error_number = 2 then

  PRINT error2

ELSE

  PRINT error3

ENDIF

例2

从通道中读入字符，并将它们附加到字符串变量中，然后最后打印它们。

DIM captured_text AS STRING(50)



captured_text = ""

WHILE char <> 13 AND count < 50

  TICKS = 10000 'Timeout on character

  WAIT UNTIL KEY#5 OR TICKS < 0

  IF TICKS < 0 THEN

    EXIT_LOOP

  ELSE

    GET #5, char

    captured_text = captured_text + CHR(char)

    count = count + 1

  ENDIF

WEND



PRINT captured_text

例3

使用字符串变量决定执行哪个动作例程:

IF g_value = "G00" THEN 'Rapid positioning

  SPEED = fast_speed

  MOVE(x, y, z)

  WAIT IDLE

  SPEED = standard_speed

ELSEIF g_value = "G01" THEN 'Linear move

  MOVE(x, y, z)

ELSEIF g_value = "G02" THEN 'Anticlockwise circular move

  MOVECIRC(x, y, x + i_value, y + j_value, 0)

ELSEIF g_value = "G03" THEN 'Clockwise circular move

  MOVECIRC(x, y, x + i_value, y + j_value, 1)

ELSE

  PRINT "Ignoring unsupported token: "; g_value

ENDIF

例4

定义一个字符串的多维数组，并在终端0打印值。

DIM arrayofstring AS STRING(2, 3, 4)



arrayofstring(0, 0) = "abcd"

arrayofstring(0, 1) = "efgh"

arrayofstring(0, 2) = "ijkl"

arrayofstring(1, 0) = "mnop"

arrayofstring(1, 1) = "qrst"

PRINT arrayofstring(0, 0)

PRINT arrayofstring(0, 1)

PRINT arrayofstring(0, 2)
