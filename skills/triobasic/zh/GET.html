GET

类型

文件函数

语法

GET [#channel,] variable

描述

等待串行中单个字符的到来。将该字符的ASCII值赋给指定的变量。用户程序将等待，直到一个字符可用。

进程KEY用于在执行GET之前检查一个字符是否已被接收。

参数

#channel:完整通道列表见#(省略时默认为0)
variable:存储接收字符的变量，这可能是局部变量，VR或TABLE

执行GET 或 GET#0将暂停命令行，直到在该通道上发送字符。

例

例1

要求用户在通道5上输入“y”表示是或“n”表示否

start:

  PRINT #5, "Press 'y' for YES or 'n' for NO"

  GET #5, char

  IF char = 121 THEN

    PRINT #5, "YES selected"

  ELSEIF char = 110 THEN

    PRINT #5, "NO selected"

  ELSE

    PRINT #5, "BAD selection"

    GOTO start

  ENDIF

例2

清空串行缓冲区，然后请求用户输入一个名称

WHILE KEY #2

  GET #2, dump

WEND



PRINT #2, "ENTER NAME"

WAIT UNTIL KEY #2

count = 0

WHILE char <> $D 'Carriage return

  GET #2, char

  VR(count) = char

  count = count + 1

WEND
