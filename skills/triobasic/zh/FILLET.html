FILLET

类型

数学函数

语法

FILLET(data_in, data_out, options)

描述

FILLET函数有2个计算函数:

第一个函数确定将两个三维向量夹角或混合在一起的弧线的尺寸

第二种方法确定将两个点与方向混合在一起的两个2D弧线的尺寸

参数

data_in:输入数据在TABLE内存中的位置。
data_out:输出数据在TABLE内存中的位置。
options:0用于计算3D中两条直线之间的弧度。
1计算两点之间有方向的一对弧线。
2从2 × 3D向量计算“点积”和中间向量。
3用1个圆弧将2D直线和2D圆弧进行倒圆角。

Option = 0

描述

该函数计算3D弧线的起点、终点、中点和中心。使用MSPHERICAL命令可以很容易地将圆弧转换为运动。

FILLET仅在2.0220及更高版本的系统中工作，输出19个数据值，包括圆角和圆角长度。

参数

输入数据:(需要7个数据值)
Table data0x向量A
1y向量A
2z向量A
3x向量B
4y向量B
5z向量B
6半径

输出数据:(输出19个数据值)。
Table data0x A剩余
1y A剩余
2z A剩余
3结束x
4结束y
5结束z
6中间x
7中间y
8中间z
9中心x
10中心y
11中心z
12错误
13输出半径
14B剩余;
15B剩余;
16z B保持不变
17角变化
18角长度

A剩余:弧的起始点相对于入射向量的起始点的XYZ位置。

中点:圆弧上的中点相对于圆弧起点的XYZ位置。

圆心:圆弧中心相对于圆弧起点的XYZ位置。

错误:如果没有错误设置为0,1 =一个或两个向量长度为零，2 =向量共线。

输出半径:如果向量不够长，无法允许请求的半径被倒圆角(考虑到选项值)，则输出半径值将显示最大的可能值，否则将反映输入半径。

B剩余:输出向量的末端相对于弧的末端的XYZ位置。

例

计算两个3D矢量的圆角，用MOVE命令表示矢量，用MSPHERICAL命令表示倒圆角。

DEFPOS(150, 0, 0) 



TRIGGER TABLE(100, -150, 0, 0) 

TABLE(103, 50, 200, 100, 70)



FILLET(100, 200, 0) 



xin = TABLE(200): yin = TABLE(201): zin = TABLE(202)



MOVE(xin, yin, zin) 



xend = TABLE(203): yend = TABLE(204): zend = TABLE(205)

xmid = TABLE(206): ymid = TABLE(207): zmid = TABLE(208)



MSPHERICAL(xend, yend, zend, xmid, ymid, zmid, 0)



xout = TABLE(214): yout = TABLE(215): zout = TABLE(216)





MOVE(xout, yout, zout) 



fillet_ang = TABLE(217): fillet_len = TABLE(218) 



PRINT fillet_ang, fillet_len

Option = 1

描述

该函数计算2个圆弧的起点、终点和中心。弧可以使用MOVECIRC或MSPHERICAL命令很容易地转换为运动。

参数

输入数据:(需要10个数据值)。
Table data0X值点A
1Y值点A
2X方向点A
3Y方向点A
4X值点B
5Y值点B
6X方向点B
7Y方向点B
8半径控制(设置为0允许FILLET计算最大的可能半径)
9电弧方向模式控制:
0 -使用最短路线

1 -左转弯-左转弯弧线强制

2 -右转弯-右转弯弧线强制

3 -左转弯然后右转弯弧线强制

4 -右转弯然后左转弯弧线强制

在一个点的方向是使用一对±增量值指定的。用户不需要将其归一化为长度为1的值。例如，沿X轴的方向可以指定为(1,0)，负X方向的方向将是(- 1,0)，沿Y轴的方向将是(0,1)。考虑一个角度在Y轴的±PI范围。方向为(sin(角度)，cos(角度))。

输出数据:(18个数据值为输出)。
Table data0位0 -弧A方向

位1 -弧B方向
11 -左转弯弧A然后左转弯弧B

2 -右转弯弧A然后右转弯弧B

3 -左转弯弧A然后右转弯弧B

4 -右转弯弧A然后左转弯弧B
2X相对于起始弧A的结束位置
3Y相对于起始弧A的结束位置
4X中心位置相对于起始弧A
5Y中心位置相对于起始弧A
6X增量链接线性移动(如果半径无限则为0)
7Y增量链接线性移动(如果半径无限则为0)
8X相对于起始弧B的结束位置
9Y相对于起始弧B的结束位置
10X中心位置相对于起始弧B
11Y中心位置相对于起始弧B
12错误，0 =无错误
13弧A长度
14链接移动长度
15弧B长度
16总长度
17半径计算。如果半径受到“半径控制”输入的限制，这个值将被设置为限制半径。

例

计算两个两点与方向混合的圆弧的尺寸，用MCIRCLE命令表示。

max_r = 20

dir_o = 0



TABLE(3000, 100, 100, 1, 0, 160, 10, 5, 5, max_r, dir_o)



FILLET(3000, 3200, 1)



IF TABLE(3212) THEN

    PRINT "Error in data"

    STOP

ENDIF



direc1 = TABLE(3200).0

direc2 = TABLE(3200).1



end1x = TABLE(3202)

end1y = TABLE(3203)

cen1x = TABLE(3204)

cen1y = TABLE(3205)



px = TABLE(3206)

py = TABLE(3207)



end2x = TABLE(3208)

end2y = TABLE(3209)

cen2x = TABLE(3210)

cen2y = TABLE(3211)



arc1len = TABLE(3213)

midlen = TABLE(3214)

arc2len = TABLE(3215)



TRIGGER

IF arc1len > 0 THEN MOVECIRC(end1x, end1y, cen1x, cen1y, direc1)

IF midlen > 0 THEN MOVE(px, py)

IF arc2len > 0 THEN MOVECIRC(end2x, end2y, cen2x, cen2y, direc2)



WAIT IDLE

Option = 2

描述

Fillet选项2生成2 x 3D向量的“点积”。这对于确定许多事情都很有用，包括向量是否在相同或相反的方向上。注意，如果向量方向相同点积为1，如果向量方向相反点积为-1。

参数

输入数据:(需要6个数据值)。
Table data0X向量A
1Y向量A
2Z向量A
3X向量B
4Y向量B
5Z向量B

输出数据:(13个数据值为输出)。
Table data0中向量X: 0.5 Norm的和。向量A X + 0.5 Norm。向量B X
1中向量Y
2中矢量Z
3归一化A向量X
4归一化A向量Y
5归一化A向量Z
6归一化B向量X
7归一化B向量Y
8归一化B向量Z
9点积
10长度向量A
11长度向量B
12错误:如果任意一个向量长度为零，则为1，如果两个向量长度都不为零，则为0

例

Option = 3

描述

一条线和一条弧的交点可以用另一条弧来圆角。可以选择不同的圆弧半径、方向或最大半径。

参数

输入数据:(需要11个数据值)。
Table data0X点A向量的第一个点。
1Y点A
2X点B向量的第二个点和给定弧的起点。
3Y点B
4X点C给定弧的中心。
5Y点C
6X点D给定弧的终点。这个点不需要完全在给定的弧上。
7Y点D
8方向给定弧的方向。
9圆角半径请求圆角半径。
10最大半径选项如果是1，如果请求的半径无法实现，该函数将计算最大可能半径。如果为0，则无法实现的半径将产生错误。

输出数据:(输出14个数据值)。

所有值均为相对位置。

Table data0开始圆角弧X圆角弧开始的X位置
1开始圆角弧Y圆角弧开始的Y位置
2中圆角弧X圆弧中点相对于圆弧起始点的X位置
3中圆角弧度Y圆弧中点相对于圆弧起始点的Y位置
4端圆角弧X圆角弧的末端相对于圆角弧的起点的X位置
5末端圆角弧度Y圆角弧的末端相对于圆角弧的起点的Y位置
6中心圆角圆弧X圆弧中心相对于圆弧起点的X位置
7中心圆角圆弧Y圆弧中心相对于圆弧起点的Y位置
8圆角半径如果半径无法实现，且“最大半径选项”=1，则此处返回实现的半径。
9角方向角的方向
10线的长度从A点到开始圆角点的线长
11角长度
12弧长从末端圆角点到末端圆弧点的弧长
13错误值0:无错误。
1:给定直线与给定弧相切。
2:圆角半径大于给定圆弧允许的范围。
3:圆角半径大于允许的最大圆角半径。
4:给定的线太短，达不到所要求的圆角半径。
5:给定的弧线太短，达不到要求的圆角半径。

例

FOR x = 0 TO 1

  BASE(x)

  ATYPE = 0

  UNITS = 100

  ACCEL = 10000

  DECEL = ACCEL

  SERVO = ON

  SPEED = 400

NEXT x



BASE(0, 1)

ax = 0

ay = -2

bx = 0.5

by = 0

cx = 1

cy = 1

dx = -4

dy = 2

filletr = 1

direction=0



TABLE(3000, ax, ay, bx, by, cx, cy, dx, dy, filletr, direction, 1)

DEFPOS(ax, ay)

WA(50)



TICKS = 0

FILLET(3000, 3011, 3)

PRINT TICKS



err = TABLE(3011 + 7)



xin = TABLE(3011): yin = TABLE(3012)

xend = TABLE(3013): yend = TABLE(3014)

xcentre = TABLE(3015): ycentre = TABLE(3016)



TRIGGER 'Start scope



MOVEABS(xin, yin)



MOVECIRC(xend, yend, xcentre, ycentre, 1)

WAIT IDLE

eb0 = ENDMOVE_BUFFER AXIS(0)

eb1 = ENDMOVE_BUFFER AXIS(1)



PRINT eb0, eb1

PRINT dx - eb0, dy - eb1, cx - eb0, cy - eb1, direction

MOVECIRC(dx - eb0, dy - eb1, cx - eb0, cy - eb1, direction)



WA(2000)

STOP
