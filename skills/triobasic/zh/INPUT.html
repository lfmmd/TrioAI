INPUT

类型

文件函数

语法

INPUT [#channel,] variable [, variable, ...]

描述

等待在当前输入设备上接收的ASCII字符串，以回车<CR>结束。如果字符串有效，则将其数值赋给指定的变量。如果输入的字符串无效，则忽略该字符串，显示错误消息并重复输入。可以在一行中请求多个值，这些值用逗号分隔，或者用回车符<CR>分隔。

轮询KEY检查在执行INPUT之前是否接收到字符的。

参数

#channel:完整通道列表见#(省略时默认为0)
variable:存储接收字符的变量，这可能是局部变量，VR或TABLE

执行INPUT或INPUT#0将暂停命令行，直到在该通道上发送字符。

例

例1

接收单个值并将其存储在本地变量num中

INPUT num

PRINT "BATCH COUNT = "; num[0]



On terminal:

123 <CR>

BATCH COUNT = 123

例2

使用一个INPUT获取长度和宽度变量。

PRINT "ENTER LENGTH AND WIDTH?";

INPUT VR(11), VR(12)

这将显示在终端上:

ENTER LENGTH AND WIDTH ? 1200,

1500 <CR>
