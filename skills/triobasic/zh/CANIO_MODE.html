CANIO_MODE

类型

I/O参数(MC_CONFIG / FLASH*)

描述

CANIO_MODE用于设置Trio CAN I/O系统的工作模式。MC4xx_运动控制器控制器_允许单独的输入和输出模块占用重叠的地址，允许多达32个输入和输出模块连接。或者，可以设置CANIO_MODE来强制MC4/MC6_运动控制器_以与MC2xx系列相同的方式工作，仅允许16个任何类型的数字模块。

位1到位3控制CAN模拟量输入的数据大小和同步。P326 CAN模拟量模块要求固件V2.0256和CAN固件V2.2.0及以上版本。

同步和16位CAN模拟量在MC464上不可用。

该值保存在flash EPROM*中，可以在MC_CONFIG脚本中设置。

*一些运动控制器，例如MC4N，不将系统参数存储到Flash中。使用MC_CONFIG。

值

位操作
00MC4xx CAN I/O寻址(默认)
1兼容模式CAN I/O寻址
10使用12位模拟量模式(默认)
1开启16位模拟量模式(1)
2开启伺服同步模拟量模式(2)
3启用2 x伺服同步模拟量模式(2)
  • 需要CAN I/O固件版本2.2.0或更高版本。
  • 只选择第2位和第3位中的一个。如果位2和位3都设置了，则位2具有优先级。

例

例1

设置CAN模拟量系统，与伺服周期同步，每秒发送AIN信息。然后将其用作伺服轴的反馈。

'为P326设置快速CAN模式

  CANIO_MODE.1 = 1 '启用16位模拟量模式

  CANIO_MODE.2 = 0

  CANIO_MODE.3 = 1 '启用'2x伺服同步'模式



  '设置CAN波特率

  '00 = 500k

  '01 = 250k

  '10 = 125k

  '11 = 1M

  CANIO_ADDRESS.8 = 1

  CANIO_ADDRESS.9 = 1



  SERVO_PERIOD = 1000
