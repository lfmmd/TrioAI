PRMBLK

类型

系统命令

语法

PRMBLK(Function, Block_Number... )

描述

使用Motion Perfect在一个以太网TCP/IP请求访问多个轴参数。PRMBLK的格式是先建立块描述，然后根据简单的数字块索引值下载值。

如果成功，PRMBLK返回TRUE(-1);如果失败，返回FALSE(0)。

此命令不能在工具模式或同步模式下连接Motion Perfect时使用。

该命令也存在于TrioPC Motion ActiveX和DLL中。在开发需要更高效的数据传输的PC应用程序时，可以使用此参考。

参数

Function 0; 定义。

定义一个参数块。最多可以在一个块中添加32个参数，但是一行BASIC不能包含所有32个参数。然后可以使用函数1将更多参数附加到块中。

PRMBLK(0, block_number, type, list_of_parameters)
block_number:0 . .63
type:0轴参数
1系统参数
2VR变量
3TABLE变量
4程序局部变量
List_of_parameters以逗号分隔的参数名列表。每个块的参数最多为32个。

Function 1;附加。

向给定的块号追加更多的参数。在附加更多参数之前，块必须以函数0“定义”开始。

PRMBLK(1, block_number, type, list_of_parameters)
block_number:0 . .63
type:0轴参数
1系统参数
2VR变量
3表变量
4程序局部变量
List_of_parameters以逗号分隔的参数名列表。每个块的参数最多为32个。

例

轴参数表;

PRMBLK(0,3,0, dpos, mpos, mspeed)

系统参数列表;

PRMBLK(1,3,1, free, wdog, motion_error)

VR变量列表。范围用..定义;

PRMBLK(1,3,2,1..5,8,12..14,55,67)

TABLE值的列表;

PRMBLK(1,3,3,1..5,8,12..14,55,67)

程序局部变量列表;

PRMBLK(1,Block_Number,4,"Program_name",Process_number,List_of_Local_Variables_Quoted)

程序名和进程号适用于列表中的所有变量。如果指定了进程号-1，则应该使用程序的第一个运行实例(具有最低进程号)。

PRMBLK(1,3,4,"PROG1",11,"n","v1","v2")

Function2;获取计数。

这个函数返回一个块的项(参数)计数。如果块为空，则返回0。

PRMBLK(2, block_number)

如。

PRINT #5,PRMBLK(2, 3)

Function 3;获取所有。

这个函数按照定义块时指定的顺序返回所有项/参数值。返回的数据是以逗号分隔的值列表的形式。

如果类型为轴参数(类型为0)，格式为:

PRMBLK(3, block_number, axis_number)

否则函数为:

PRMBLK(3, block_number)

如。

PRMBLK(3, 3, 4) '从轴4获取轴参数

Function 4;获取变化。

此函数返回自上次使用PRMBLK命令读取后发生更改的任何项/参数值。返回的数据是以逗号分隔的参数索引/值对列表的形式，格式如下:

<Item_Index>:<Value>

参数索引是参数在块中的从零开始的位置。

如果类型为轴参数(类型为0)，格式为:

PRMBLK(4, block_number, axis_number)

否则函数为:

PRMBLK(4, block_number)

如。

PRMBLK(4, 8, 2) '从坐标轴2获取block 8定义的改变后的坐标轴参数

Function 5;获取最大数据块。

这个函数获取可用数据块的最大数字。

Function 6;获得每个块的最大项目。

这个函数获取每个块中可以指定的最大项目数。

Function 7;块列表。

这个函数列出块类型和当前在块中的任何项。

项目列表以逗号分隔的所有单独项目列表的形式返回(范围被扩展)。

PRMBLK(7，<Block_Number>)

如。

PRMBLK(7,3)

Function 8;删除块。

这个函数删除一个已定义块的定义。

PRMBLK(8，<Block_Number>)

如。

PRMBLK (8, 3)

Function 9;删除所有块。

这个函数删除所有的块定义。

PRMBLK (9)

例

将轴参数设置为一个准备下载显示的块。

'将轴参数设置为在一个块中读取

PRMBLK(0,3,0,ATYPE,UNITS,P_GAIN,I_GAIN,D_GAIN,OV_GAIN,VFF_GAIN,SPEED,ACCEL)

PRMBLK(1,3,0,DECEL,SRAMP,MERGE,CREEP,VP_SPEED,MSPEED,FWD_IN,REV_IN,FHOLD_IN)

PRMBLK(1,3,0,FS_LIMIT,RS_LIMIT,FE_LIMIT,FE_RANGE,DRIVE_FE_LIMIT,REP_DIST)

PRMBLK(1,3,0,REP_OPTION,DATUM_IN,MPOS,DPOS,ENDMOVE,REMAIN,ENCODER,DAC_OUT)



'从0、1、2、15、21、22、23轴读取参数块。

PRMBLK(3,3,0..2,15,21..23)
