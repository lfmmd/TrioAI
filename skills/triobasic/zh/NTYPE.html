NTYPE

类型

轴参数(只读)

描述

这个参数保存了第一个被缓冲的移动的类型。

NTYPE缓冲区可以使用CANCEL(1)来清除。

值

移动类型的数值

返回值列表见MTYPE。

例

例1

如果第一个移动缓冲区(NTYPE)为空，则从TABLE中应用另一个移动

IF NTYPE = 0 THEN

  MOVE( TABLE(count))

  count = count + 1

ENDIF

在MC⅘/6范围内，首选轴参数为LOADED。下面的示例将比使用NTYPE更可靠地工作。

IF LOADED THEN

  MOVE( TABLE(count))

  count = count + 1

ENDIF

例2

在命令行(终端#0)中检查进程14的MTYPE、NTYPE和PMOVE的状态。

>>?MTYPE

>>?PMOVE PROC(14)
