FLAG

类型

逻辑和位运算符

语法

value = FLAG(flag_no [,state])

描述

该FLAG命令用于设置和读取一个24位的标志位组。

提供该FLAG命令是为了帮助与早期控制器兼容，不建议用于新程序。

参数

value:用一个参数返回标志的状态
如果有2个参数，它返回-1
flag_no:标志号是0..31之间的值。
state:要设置给定标志的状态。ON或OFF。

例

根据VR值切换标志

IF FLAG(21) and VR(100)=123 THEN

  FLAG(21,OFF)

ELSE IF NOT FLAG(21) and VR(100)<>123 THEN

  FLAG(21,ON)

ENDIF
