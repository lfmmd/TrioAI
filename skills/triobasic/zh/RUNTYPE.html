RUNTYPE

类型

系统命令

语法

RUNTYPE "program", mode [,process]

描述

设置程序是否在开机时自动运行，以及在哪个进程上运行。

当执行DIR命令时，将显示每个程序的RUNTYPE的当前状态。

对于任何程序在上电时自动运行，控制器上的所有程序必须无错误编译。即使不使用它们。

通常程序的RUNTYPE是通过Motion Perfect设置的。在从SD卡加载程序时设置RUNTYPE可能很有用。

参数

program:设置上电模式的程序。
mode:1开机自动运行。
0手动运行。
prcess:运行程序的进程号。

例

当从SD卡加载一系列程序时，MAIN必须设置为上电运行，并且HMI必须在上电时在进程4上运行。以下内容来自TRIOINIT.bas文件。

FILE "LOAD_PROGRAM" "MOTION"

FILE "LOAD_PROGRAM" "HMI"

FILE "LOAD_PROGRAM" "MAIN"

RUNTYPE "HMI", 1, 4

RUNTYPE "MAIN", 1
