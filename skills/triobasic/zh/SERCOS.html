SERCOS

类型

系统函数

语法

SERCOS(function#, slot, {parameters})

描述

该功能允许从TrioBASIC编程系统控制Sercos环。Sercos环由一个主站和一个或多个从站组成，使用光纤电缆菊状连接在一起。在初始化期间，环在进入可能进行运动控制的最后循环确定性阶段之前要经过几个“通信阶段”。在最后阶段，主站发送控制信息，从站发送每个周期的状态反馈信息。

一旦伺服环在CP4中运行，就可以使用标准的TrioBASIC运动命令。

_运动控制器_伺服硬件使用Sercon 816 Sercos接口芯片，允许连接速度高达16 Mhz。如果需要，该芯片可以在寄存器级使用Sercos命令进行编程。要以这种方式编程，必须获得一份芯片数据表的副本。

Sercos命令提供了对10个独立功能的访问:

参数

function:0读Sercos ASIC
1写Sercos ASIC
2初始化命令
3连接伺服系统驱动器到轴
4读参数
5写参数
6执行伺服系统过程命令
7检查驱动器是否存在
8打印网络参数
9保留
10Sercos环状态
槽:槽位号取值范围为0 ~ 6，指定主模块位置。

Function = 0

语法

SERCOS(0, slot, ram/reg, address)

描述

这个函数从Sercos ASIC读取一个值。

请不要在未参考Sercon 816数据表的情况下使用此函数。

参数

slot:安装Sercos的模块槽。
ram/reg:0从RAM中读取值
1从寄存器中读取值。
address:RAM或寄存器中的索引地址。

例

>>?SERCOS(0, 0, 1, $0c)

Function = 1

语法

SERCOS(1, slot, ram/reg, address, value)

描述

这个函数将一个值写入Sercos ASIC

不要在没有参考Sercon 816数据表的情况下使用这个函数。

参数

slot:安装Sercos的模块槽。
ram/reg:0向RAM写入值
1写值到寄存器。
address:RAM或寄存器中的索引地址。
value:待写日期

Function = 2

语法

SERCOS(2, slot[, intensity[, baudrate[, period]]])

描述

此函数初始化用于Sercos环上通信的参数。

参数

slot:安装Sercos的模块槽。
intensity:透光强度(1 ~ 6)，默认值为3。
baudrate:通信数据速率。可设置为2、4、6、8或16。
period:以微秒为单位的伺服控制周期时间。可接受的值为2000、1000、500和250µs。

例

>> SERCOS (2,3,4,16,500)

Function = 3

语法

SERCOS(3, slot, slave_address, axis[, slave_drive_type])

描述

这个函数将伺服控制驱动器(从机)连接到一个轴。

参数

slot:安装伺服控制的模块槽。
slave_address:要链接到轴的驱动器的从机地址。
axis:用于控制该驱动器的轴号。
slave_drive_type:可选参数，用于设置从盘类型。所有标准Sercos驱动器都需要通用设置。以下其他选项仅在驱动器使用非标准Sercos功能时才需要。
0通用驱动
1Sanyo-Denki
3Yaskawa + Trio P730
4PacSci
5Kollmorgen

例

>> SERCOS (3,1,3,5,0)'将地址3的驱动器链接到轴5

Function = 4

语法

SERCOS(4, slot, slave_address, parameter_ID [, parameter_size[, element_type [, list_length_offset, [VR_start_index]]])

描述

该函数从驱动器中读取参数值

参数

slot:安装Sercos的模块槽。
slave_address:要读取的驱动器的Sercos。
parameter_ID:Sercos参数IDN
parameter_size:期望参数数据的大小:
22字节参数(默认)。
44字节参数
6参数id列表
7ASCII字符串
element_type:Sercos数据块中的元素类型:
1ID号码
2名字
3属性
4单位
5最小输入值
6最大输入值
7运行数据(默认)
list_length_offset:可选参数，用于偏移列表长度。对于返回2个额外字节的驱动器，使用-2。
VR_start_index:VR数组的开始，列表将存储在其中。

这个函数返回2字节和4字节参数的值，但是在Motion Perfect中打印列表到终端，除非定义了VR开始索引。

例

>> SERCOS(4, 0, 5, 140, 7) '请求"控制器类型"

>> SERCOS(4, 0, 5, 129) '请求制造商进行1级诊断

Function = 5

语法

SERCOS(5, slot, slave_address, parameter_ID, parameter_size, parameter_value[, parameter_value, ...])

描述

该函数将一个或多个参数值写入驱动器。

参数

slot:安装Sercos的模块槽。
slave_address:要写入的驱动器的Sercos地址。
parameter_ID:Sercos参数IDN
parameter_size:要写入的参数数据大小。2、4或6。
parameter_value:为size 2和size 4输入一个参数。尺寸为6的输入2到7个参数(列表)。

例

>> SERCOS(5, 1, 7, 2, 2, 1000)      '设置Sercos循环时间

>> SERCOS(5, 0, 2, 16, 6, 51, 130)  '设置IDN 16位置反馈

Function = 6

语法

SERCOS(6, slot, slave_address, parameter_ID[, timeout[, command_type]])

描述

该函数在驱动器上运行Sercos进程。

参数

slot:安装Sercos接口的通信槽。
slave_address:驱动器的Sercos地址。
parameter_ID:Sercos程序命令IDN。
timeout:可选超时设置(毫秒)。
command_type:定义操作的可选参数:
1运行&取消操作(默认值)
0取消命令
1运行命令

例

>> SERCOS(6, 0, 2, 99)  '清除驱动器错误

Function = 7

语法

SERCOS(7, slot, slave_address)

描述

此功能用于检测在给定Sercos从地址上是否存在驱动器。

参数

slot:安装Sercos接口的模块插槽。
slave_addr:驱动器的Sercos地址。

如果检测到驱动器返回1，如果未检测到返回-1。

例

IF SERCOS(7, 2, 3) < 0 THEN

  PRINT#5, "Drive 3 on slot 2 not detected"

END IF

Function = 8

语法

SERCOS(8, slot, required_parameter)

描述

打印Sercos网络参数。

参数

slot:安装Sercos的模块槽。
required_parameter:这个函数将打印所需的网络参数，在可能的地方。
0打印一个分号分隔的' slave Id, axis number '对列表，用于注册的网络配置(如使用函数3定义的)。在阶段1中使用:如果检测到驱动器返回1，如果未检测到驱动器返回0。
1打印波特率(2、4、6或8)，和
2打印强度(0到6之间的数字)。

例

>>?SERCOS (8,0,1)

函数= 10

语法

SERCOS(10， <slot>)

描述

该函数检查光纤环路是否在相位0闭合。如果网络关闭，返回值为1，如果网络打开，返回值为-1，如果网络上存在过度失真，返回值为-2。

参数

slot:安装Sercos的模块槽。

例

>>?SERCOS(10, 1)

IF SERCOS(10, 0) <> 1 THEN

  PRINT "Sercos ring is open or distorted"

END IF
