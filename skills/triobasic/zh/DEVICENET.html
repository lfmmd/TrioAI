DEVICENET

类型

系统命令

语法

DEVICENET(slot, function[, parameters...])

描述

命令DEVICENET用于启动和停止DeviceNet从站功能，该功能内置于_运动控制器_中。

周期性传输轮询I/O数据:

From PLC to [TABLE(poll_base) -> TABLE(poll_base + poll_in)]

To PLC from [TABLE(poll_base + poll_in + 1) -> TABLE(poll_base + poll_in + poll_out)]

参数

slot:内置CAN端口设置为-1
function:0在给定的插槽上启动DeviceNet从协议。
1停止DeviceNet协议。
2将启动波特率写入Flash EPROM

Function = 0

语法

DEVICENET(slot, 0, baud, mac_id, poll_base, poll_in, poll_out)

描述

使用指定参数启动DeviceNet协议

参数

baud:设置为125、250或500来指定波特率，单位为kHz。
mac_id:_运动控制器_在DeviceNet网络上用来标识自己的ID。0 . . 63之间。
poll_base:作为轮询数据传输的第一个TABLE位置
poll_in:轮询期间接收的字数量。范围0 . . 4
poll_out:轮询期间要发送的字数量。范围0 . . 4

Function = 1

语法

DEVICENET(slot, 1)

描述

停止DeviceNet协议的运行

Function = 2

语法

DEVICENET(slot, 2, baud)

描述

将上电时的波特率存储在闪存EPROM中。

参数

baud:设置为125,250或500来指定波特率，单位为kHz。

例

例1

在内置CAN端口上启动DeviceNet协议

DEVICENET(-1, 0, 500, 30, 0, 4, 2)

例2

关闭2号槽位CAN板上的DeviceNet协议;

DEVICENET(2, 1)

例3

将0号槽位的CAN板上电时的波特率设置为125 kbps;

DEVICENET(0, 2, 125)
