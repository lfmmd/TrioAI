MODULE_IO_MODE

类型

I/O参数(MC_CONFIG / FLASH)

描述

该参数设置任意级联模块I/O通道的起始地址。为了向后兼容，也可以关闭模块I/O。

注意，扩展的I/O映射功能可以使用MC_CONFIG参数CANIO_BASE、DRIVEIO_BASE、MODULEIO_BASE和NODE_IO。这些取代了使用MODULE_IO_MODE的需要，并提供了对控制器I/O映射中I/O点位置的精确控制。但是，如果MODULE_IO_MODE设置为2，则优先于通过CANIO_BASE和MODULEIO_BASE对CANIO和MODULEIO的定位。

该参数保存在Flash EPROM中，可以包含在MC_CONFIG脚本中。

值

0模块I/O禁用
1模块I/O在控制器I/O之后，在CAN I/O之前(默认)
2模块I/O位于I/O序列的末端
3模块I/O被禁用，CAN I/O从32开始

如果升级的是已有控制器的固件，该参数可能设置为0。在出厂安装的系统上默认设置为1。

例

具有MC464，松下模块(槽位0)，FlexAxis(槽位1)和CANIO模块的系统将具有以下I/O分配:

MODULE_IO_MODE = 1(默认)+DRIVEIO_BASE = -1 + CANIO_BASE= 0 +MODULEIO_BASE = 0
0-7内置输入
8-15内置双向I/O
16-23松下输入
24-27FlexAxis输入
28-31FlexAxis双向I/O
32-47CANIO双向I/O
48-1023虚拟I / O

MODULE_IO_MODE = 0 (off) + DRIVEIO_BASE = -1 + CANIO_BASE = 0 + MODULEIO_BASE = 0
0-7内置输入
8-15内置双向I/O
16-31CANIO双向I/O
32-1023虚拟I / O

MODULE_IO_MODE = 2 (end)
0-7内置输入
8-15内置双向I/O
16-31CANIO双向I/O
32-39松下的输入
40-43FlexAxis输入
44-47FlexAxis双向I/O
48-1023虚拟I / O
