LINPUT

类型

文件函数

语法

LINPUT [#channel,] variable

描述

等待输入字符串，并将字符串的ASCII值存储在从指定编号变量开始的变量数组中。字符串必须以回车符<CR>结束，回车符也被存储。该字符串不会被控制器回复。

可以使用VRSTRING命令从VRs中打印字符串

参数

#channel:完整通道列表见#(省略时默认为0)
variable:存储接收字符的VR变量

例

使用LINPUT接收通道5上的一串字符，并将其放入从VR(0)开始的一系列VRs中。

LINPUT #5, VR(0)

现在在通道5上输入:START<CR>将会得到:

VR(0)    83    ASCII 'S'

VR(1)    84    ASCII 'T'

VR(2)    65    ASCII 'A'

VR(3)    82    ASCII 'R'

VR(4)    84    ASCII 'T'

VR(5)    13    ASCII carriage return
