IF..THEN..ELSEIF..ELSE..ENDIF

类型

程序结构

语法

IF condition THEN

commands

ELSEIF expression THEN

commands

ELSE

commands

ENDIF

描述

IF程序结构用于在有效表达式之后执行代码块。 该结构将根据条件仅执行一个命令块。 如果多个表达式有效，则第一个将执行其命令。 如果没有有效的表达式并且存在ELSE，则将执行ELSE下的命令。

参数

condition:任何有效的TrioBASIC逻辑表达式
commands:您希望执行的TrioBASIC语句

例

例1

检查批处理是否完成，如果完成则告诉用户并处理该批处理

IF count >= batch_size THEN

  PRINT #3, CURSOR(20); "   BATCH COMPLETE   ";

  GOSUB index 'Index conveyor to clear batch

  count = 0

ENDIF

例2

当机器运行时，使用IF语句点亮警示灯

IF WDOG = ON THEN

  OP(warning, ON)

ELSE

  OP(warning, OFF)

ENDIF

例3

使用IF结构来报告机器的运行状态。

IF operating_state = 0 THEN

  PRINT#5，"机器运行"

ELSEIF operating_state = 1 THEN

  PRINT#5，"机器空闲"

ELSEIF operating_state = 2 THEN

  PRINT#5，"机器堵塞"

ELSE

  PRINT#5，"机器处于未知状态"

ENDIF
