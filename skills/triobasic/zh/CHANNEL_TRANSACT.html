CHANNEL_TRANSACT

类型

文件函数

语法

x = CHANNEL_TRANSACT (function，…)

描述

CHANNEL_TRANSACT将使用以太网上的UDP协议在2个运动控制器之间建立一个通信通道。

该功能分为两部分:服务器和客户端。服务器由CHANNEL_TRANSACT(0，…)函数启动，并处理所有接收到的数据包，直到被CHANNEL_TRANSACT(1)函数停止。CHANNEL_TRANSACT服务器将接受任何TrioBASIC文件通道，但是只有使用UDP、TCP或FIFO文件通道才有意义。

CHANNEL_TRANSACT报文的长度最多为1024字节。

CHANNEL_TRANSACT数据包具有以下结构(所有多字节值都是大端):
字节描述
1SOH (ASCII字符0x01)
10数据包报头。SOH、STX、ETX和ESC字符通过发送ESC字符后跟要转义的字符加上0x20进行转义。
字节描述
1包版本号(0)
1包号(0,1,2,3，…)
1数据格式。
格式描述
08位无符号整数
132位无符号整数
264位无符号整数
364位IEEE浮点数
1目标存储区(0 -> VR, 1 -> TABLE)
4目标存储区的开始
2剩余要发送的数据点数。不一定是这个数据包中的数据点数。
1STX (ASCII字符0x02)
length数据。
长度是这个数据包中的数据点数乘以存储数据格式所需的字节数。
数据格式长度
0数据点数
1数据点数* 4
2数据点数* 8
3数据点数* 8
1ETX (ASCII字符0x03)
1最后一个包
值描述
1这是传输所有数据点所需的最后一个数据包
0还有更多的数据包输入
1EOT (ASCII字符0x04)

参数

Function:0启动CHANNEL_TRANSACT服务器
1停止CHANNEL_TRANSACT服务器
2发送数据
3收到的TRANSACT数

Function = 0:启动CHANNEL_TRANSACT服务器

语法

x = CHANNEL_TRANSACT(0, channel)

描述

初始化TRANSACT_CHANNEL服务器，并将其绑定到先前由TrioBASIC OPEN命令打开的通道。

参数

channel:TrioBASIC文件通道。

UDP

OPEN #20 AS "DGRAM:192.168.0.250(8000)" FOR READ_WRITE

CHANNEL_TRANSACT(0, 20)

TCP客户端示例

OPEN #20 AS "TCP:192.168.0.250(8000)" FOR READ_WRITE

CHANNEL_TRANSACT(0, 20)

TCP服务器示例

OPEN #20 AS "TCPSERVER: (8000)" FOR READ_WRITE

CHANNEL_TRANSACT(0, 20)

Function = 1:停止CHANNEL_TRANSACT服务器

语法

x = CHANNEL_TRANSACT(1)

描述

停止TRANSACT_CHANNEL服务器。

例

CHANNEL_TRANSACT(1)

Function = 2:发送数据

语法

CHANNEL_TRANSACT(2, channel, source_storage_area, source_start, length, format, destination_storage_area, destination_start)

描述

使用先前由TrioBASIC OPEN命令打开的通道将数据发送到TRANSACT_CHANNEL服务器。

参数

channel:TrioBASIC文件通道。
source_storage_area:从该区域读取数据。
区域描述
0VR
1TABLE
source_start:偏移到源存储区域到要读的第一个元素。
length:要发送的数据点数。
format:数据格式。
格式描述
08位无符号整数
132位无符号整数
264位无符号整数
364位IEEE浮点数
destination_storage_area:将数据写入远程控制器上的此区域。
区域描述
0VR
1TABLE
destination_start:向存储的目标区域偏移到第一个元素要写入的位置。

例

OPEN #20 AS "DGRAM:192.168.0.250(8000)" FOR READ_WRITE

CHANNEL_TRANSACT(2, 0, 0, 160, 3, 0, 200)

Function = 3:接收到的transaction计数

语法

x = channel_transaction (3)

描述

停止TRANSACT_CHANNEL服务器。

例

'获取当前的transaction计数。

c = CHANNEL_TRANSACT(3)



'等待一个传入的事务。

WAIT UNTIL c <> CHANNEL_TRANSACT(3)

例

下面是一个TrioBASIC程序发送和接收数据的完整示例。

'声明我们将要使用的变量

DIM c, i, t1, t2 AS INTEGER



' 初始化要发送的变量.

FOR i = 0 TO 159

    VR(i) = SIN(i)

NEXT i



'打开通道

OPEN #20 AS "DGRAM:127.0.0.1(8000)" FOR READ_WRITE



' 如果channel_transaction服务器没有启动，那么…

IF NOT CHANNEL_TRANSACT(0, 20) THEN

    PRINT "Error starting CHANNEL_TRANSACT server"

    GOTO close_channel

ENDIF



'获取当前transaction计数

c = CHANNEL_TRANSACT(3)



'发送transaction. 假设我们在127.0.0.1上打开了UDP套接字，这将返回给我们

t1 = TICKS

CHANNEL_TRANSACT(2, 0, 0, 160, 3, 0, 200)



'等待传入的事务被处理.

WAIT UNTIL c <> CHANNEL_TRANSACT(3)

t2 = TICKS



'验证接收到的数据

FOR i = 0 TO 159

    IF ABS(VR(i) - VR(200 + i)) > 0.001 THEN

        PRINT "Error at "; i[0]

        GOTO stop_server

    ENDIF

NEXT i



' 打印

PRINT "Duration="; t2 - t1[0]



stop_server:

CHANNEL_TRANSACT(1)



close_channel:

CLOSE #20
