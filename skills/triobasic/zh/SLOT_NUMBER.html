SLOT_NUMBER

类型

轴参数

描述

返回轴所在的SLOT号。轴号可以灵活地分配给硬件，因此不能仅通过AXIS号找到轴的物理位置。SLOT_NUMBER返回BASE轴的值，或者如果使用AXIS(数字)修饰符，则返回与该轴关联的SLOT。

当硬件允许时，可以为任何需要时间插值注册的虚拟轴设置SLOT_NUMBER。

例

例1

使用命令行终端0查找轴的SLOT_NUMBER:

>> PRINT SLOT_NUMBER AXIS(12)

2

>>

例2

启动时，检查MC464/MC664的轴槽位配置是否正确。

IF SLOT_NUMBER AXIS(0)<>-1 THEN

  PRINT "Warning – Built-in axis configuration incorrect"

  PRINT "Axis 0 expected for this application."

ENDIF

例3

设置2个虚拟轴，使它们可以使用时间插值的注册输入。这需要在MC4N, P876和P871中找到的硬件功能。

SLOT_NUMBER AXIS(10)=1

SLOT_NUMBER AXIS(11)=1

…

REGIST(21,4) AXIS(10) ' 使用输入4进行手臂配准

REGIST(21,5) AXIS(11) ' 使用输入5进行手臂配准
