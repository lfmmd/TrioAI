VR

类型

系统命令

语法

value = VR(expression)

VR(expression) = value

描述

通过索引调用或分配一个全局浮点值，从0开始。

VR中的元素数量因控制器而异，可以使用VRSIZE关键字来确定，因此有效的索引范围是从0到VRSIZE - 1。

VR可以作为一个数组使用，也可以分割成几个数组。VR也可以用来保存字符串的ASCII，并可以分配一个字符串值。要读回字符串值，你必须使用VRSTRING。

编号的变量在程序之间是全局共享的，可以用于程序之间的通信，所以当多个程序操作同一个VR时需要小心。

所有VR都存储为64位浮点值，遵循IEEE-754标准(53位有效位，1位符号位和11位指数位)进行编码。因此，需要超过53位的整数在转换为浮点数时会产生精度损失。

参数

value:向VR写入或读取的值
expression:任何有效的数值表达式(舍入为最接近的整数)

例

例1

将值1.2555放入VR变量15中。注意用于给全局变量命名的局部变量“val”:

val = 15

VR(val) = 1.2555

例2

一个传递龙门架有10个连续放下的位置，每个位置在任何时候都可以是FULL或 EMPTY。VR(101)至VR(110)用于容纳十个1或0的数组，以表示所述位置为满(1)或空(0)，并且所述龙门架应将所述负载置于第一个空闲位置。实现这一目标的部分程序将是:

movep:

  MOVEABS(115) '移动到第一个放下的位置:

  FOR VR(0) = 101 TO 110

    IF VR(VR(0)) = 0 THEN 

      GOSUB load

    ENDIF

    MOVE(200)  '200是位置之间的间距

  NEXT VR(0)

  PRINT "All Positions Are Full"

  WAIT UNTIL IN(3) = ON

  GOTO movep



load:

  '将load放到位并标记数组

  OP(15, OFF)

  VR(VR(0)) = 1

例3

将VR(0)的值乘以轴1的测量位置分配给VR(65)

VR(65) = VR(0) * MPOS AXIS(1)

PRINT VR(65)

例4

将一个字符串写入从索引10开始的VR序列中

VR(10) = "Hello World"

PRINT VR(10)       '72 (ASCII for H)

PRINT CHR(VR(10))  'H

PRINT VRSTRING(10) 'Hello World
