MOVE

类型

轴命令

语法

MOVE(distance1 [,distance2 [,distance3 [,distance4, …]]])

替代格式

MO()

描述

增量移动。一个轴或多个轴以编程的速度和加速度移动指定的距离，从上一次指定移动的末尾开始增加。列表中的第一个参数被发送到BASE轴，第二个参数被发送到BASE数组中的下一个轴，依此类推。

在多轴形式中，用于运动的速度和加速度取自BASE组中的第一个轴。控制每个轴的速度，以便使运动的结果矢量以SPEED设置运行。

多轴插补使用BASE轴的UNITS、SPEED、加速和减速。为了正确的XYZ矢量操作，所有3个轴的UNITS必须相同。

如果需要在矢量速度计算中排除某个轴，则可以将该轴的INTERP_FACTOR设置为0。

通过简单地在每个轴上单独放置 MOVE 指令，就可以实现不插补，不同步的多轴运动。 如果需要，可以使用Axis()指令符指定单个 MOVE 的目标轴，仅给当前BASE轴一个 MOVE 指令。

使用单位换算系数轴参数缩放指定的距离值 UNITS。因此，例如，某轴的编码器脉冲每毫米400个，而该轴的UNITS为400，，则指令MOVE（12.5）将移动 12.5 mm。 当MERGE设置为ON时，同一轴组中的单个运动将合并在一起以进行连续的路径运动。

如果移动中的轴数与之前的移动不同，则它们不会进行合并。同样，FORWARD或REVERSE不会与MOVE或MOVEABS 合并。

参数

distance1:从当前位置在基准轴上移动的距离。
distance2:从当前位置在BASE数组的下一个轴上移动的距离。
distance3:从当前位置在BASE数组的下一个轴上移动的距离。
distance4:从当前位置在BASE数组的下一个轴上移动的距离。

参数的最大数量是控制器上可用的轴的数量

例

例1

一个系统的单位UNITS为 1，编码器为1000线的编码器。请注意1000线编码器每圈可以产生 4000 个脉冲。

MOVE(40000) ' Move 10启动电机。



'或者，使用UNITS来校准轴

UNITS = 4000 '每转计数。

MOVE(10)

例2

轴3,4和5是独立移动(无插补)。每个轴将移动在自己的编程SPEED，ACCEL和DECEL等。

'设置轴的速度和启用

BASE(3)

SPEED = 5000

ACCEL = 100000

DECEL = 150000

SERVO = ON

BASE(4)

SPEED = 5000 

ACCEL = 150000

DECEL = 560000

SERVO = ON

BASE(5)

SPEED = 2000

ACCEL = 320000

DECEL = 352000

SERVO = ON

WDOG=ON

MOVE(10) AXIS(5)     '开始运动

MOVE(10) AXIS(4)

MOVE(10) AXIS(3)

WAIT IDLE AXIS(5)    '等待运动完成

WAIT IDLE AXIS(4)

WAIT IDLE AXIS(3)

例3

X-Y绘图仪可以在其工作信封内的任何位置写入文本。单个字符被定义为相对于起点的一系列移动，因此无论情节起源如何，都可以使用相同的命令。字母“M”的命令子程序可能是:

write_m:

  MOVE(0, 12)  'Move A >  B

  MOVE(3, -6)  'Move B >  C

  MOVE(3, 6)   'Move C >  D

  MOVE(0, -12) 'Move D > E

  RETURN

例4

轴1、轴6、轴7要一起移动。BASE轴上的SPEED设置为轴1的速度。其他2个轴的运行速度将使所有3个轴同时停止。轴6和轴7的加速和减速将与轴1的值成正比。

'设置轴速并使能

INTERP_FACTOR AXIS(1) = 1

INTERP_FACTOR AXIS(6) = 0

INTERP_FACTOR AXIS(7) = 0

BASE(1, 6, 7)

SPEED = 70 'mm / second

ACCEL = 350 '0.2 second ramp

DECEL = 700 '0.1 second ramp

MOVE(199.5, 81.53, 120) 'start moves

WAIT IDLE '等待动作结束

所有的坐标轴将在3.0秒内完成他们的移动。等速部分为1轴189毫米，用时2.7秒。
