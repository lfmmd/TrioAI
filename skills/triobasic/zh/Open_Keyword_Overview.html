开放关键字

Trio控制器支持用户描述的关键字。本文档解释了在使用此功能开发时如何设置，使用和调试。一个新的命令REGISTER_KEYWORD允许用户注册他们的关键字。

设置开放关键字

关键字可以通过命令注册

REGISTER_KEYWORD(KeywordName, BASICFunctionName, minParams, maxParams)
参数数据类型描述
KeywordNameSTRING关键字的名称。
BASICFunctionNameSTRINGTrioBASIC函数的名称。
minParamsINTEGER函数的最小参数数量
maxParamsINTEGER函数的最大参数数量

限制

  • 新关键字不能与现有关键字同名。在这种情况下，控制器将抛出一个错误。
  • 必须在调用该关键字之前编译已注册的TrioBASIC函数。
  • TrioBASIC函数中定义的参数数量必须与传递给关键字的参数数量相匹配。
  • 任何时候只能注册5个关键字。
  • REGISTER_KEYWORD命令只能在MC_CONFIG中设置。
  • 一次只能注册5个关键字。

例

在 MC_CONFIG.MCC

REGISTER_KEYWORD("RUN_TEST0", "test_void", 0, 1)
REGISTER_KEYWORD("RUN_TEST1", "test_int", 0, 2)
REGISTER_KEYWORD("RUN_TEST2", "test_float", 0, 1)
REGISTER_KEYWORD("RUN_TEST3", "test_str", 0, 1)

在 TEST_LIB.BAL

FUNCTION test_void()
    PRINT "test_void passed"
ENDFUNC


FUNCTION test_int(param1, param2 AS INTEGER)
    IF param1 = 2 THEN
        PRINT "test_int passed"
    ELSE
        PRINT "test_int failed"
    ENDIF
ENDFUNC

FUNCTION test_float(param1 AS FLOAT)
    IF param1 = 4.12 THEN
        PRINT "test_float passed"
    ELSE
        PRINT "test_float failed"
    ENDIF
ENDFUNC

FUNCTION test_str(param1 AS STRING) AS INTEGER
    IF param1 = "TEST22" THEN
        PRINT "test_str passed"
        RETURN 1
    ELSE
        PRINT "test_str failed"
        RETURN 0
    ENDIF
ENDFUNC

在 TEST.BAS

DIM ret AS INTEGER

test_int(2, 1)
RUN_TEST1(2, 1)
RUN_TEST2(4.12)
ret = RUN_TEST3("TEST22")
PRINT ret
