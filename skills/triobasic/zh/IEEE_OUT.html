IEEE_OUT

类型

数学函数

语法

byte_n = IEEE_OUT(value, n[, size])

描述

IEEE_OUT函数返回从以IEEE 754格式编码的浮点数中提取的单个字节，通常用于在通信总线系统上传输。该函数被调用多次以依次提取每个字节，IEEE 754单精度调用4次，IEEE 754双精度调用8次。

参数

value:任何TrioBASIC浮点变量或参数。
n:要提取的字节数(0 - 7)。
size:浮点数的位数(32或64)，默认为32。

字节0是IEEE 754浮点格式的高字节。

例

例1

从MPOS中提取4个字节，并将其存储在准备通过通信总线传输的本地变量中。

a = MPOS AXIS(2)

byte0 = IEEE_OUT(a, 0)

byte1 = IEEE_OUT(a, 1)

byte2 = IEEE_OUT(a, 2)

byte3 = IEEE_OUT(a, 3)

例2

从MPOS中提取8个字节，并将其存储在准备通过通信总线传输的本地变量中。

a = MPOS AXIS(2)

byte0 = IEEE_OUT(a, 0, 64)

byte1 = IEEE_OUT(a, 1, 64)

byte2 = IEEE_OUT(a, 2, 64)

byte3 = IEEE_OUT(a, 3, 64)

byte4 = IEEE_OUT(a, 4, 64)

byte5 = IEEE_OUT(a, 5, 64)

byte6 = IEEE_OUT(a, 6, 64)

byte7 = IEEE_OUT(a, 7, 64)
