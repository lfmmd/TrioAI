ENCODER_READ

类型

轴功能

语法

value = ENCODER_READ (address)

描述

从EnDat绝对值编码器读取内部寄存器。

参数

value:从指定寄存器返回的值。如果编码器尚未初始化，则返回-1
address:要读取的EnDat编码器寄存器的地址

例

例1

初始化并检查EnDat编码器

ENCODER_BITS=25+256*12

ATYPE=47

IF ENCODER_READ($A700)=-1 then

  PRINT "初始化EnDat编码器失败

ENDIF

ENCODER_CONTROL=0

例2

从EnDat编码器读取编码器位数。这可以在设置ENCODER_BITS以找到要使用的正确值之前完成。该命令将适用于任何EnDat 2.1编码器。

>>BASE(1)

>>PRINT ENCODER_READ($A10d)AND $3F

25

>>
