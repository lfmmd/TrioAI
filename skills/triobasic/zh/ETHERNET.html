ETHERNET

类型

系统命令

语法

ETHERNET(action, slot, function[, parameters...])

描述

ETHERNET命令用于配置以太网端口的操作。

许多ETHERNET功能只能在命令行中使用;这些功能存储在闪存EPROM中，然后在上电时使用。

参数

action:指定所需的操作。
0读
1写
2命令
slot:内置以太网端口设置为-1
function:要执行的功能
0IP地址
1保留功能
2子网掩码
3MAC 地址(只读)
4默认端口号
5Token端口号
6PRP固件版本(MC464只读)
7Modbus TCP模式
8默认网关
9数据配置
10Modbus TCP端口号
11ARP缓存(只读)
12保留功能
13保留功能
14为Modbus TCP或Ethernet IP配置终点
15设置每个堆栈调用处理的最大数据包
21列出到终端的端口状态
22读取以太网端口MIB信息
23为设备特定的PHY寄存器保留
24读取以太网连接状态
25重新启动以太网服务

Function = 0: IP地址

打印或写入以太网IP地址。仅限命令行。

请重新上电或执行EX(1)应用新的IP地址。

ETHERNET(action, slot, 0[, byte1, byte2, byte3, byte4])

选择

IP_ADDRESS = xxx.xxx.xxx.xxx

参数

byte1:IP地址的第一个字节
byte2:IP地址的第二个字节
byte3:IP地址的第三个字节
byte4:IP地址的第四个字节

默认地址为192.168.0.250

例

读取当前IP地址，然后在控制器中设置一个新的IP地址，并执行EX(1)来激活该地址

在本例中执行EX(1)将关闭通信，您将只能使用新的IP地址再次通信。

>>ETHERNET(0, -1, 0)

192.168.0.250

>>ETHERNET(1, -1, 0, 192, 168, 0, 201)

>>EX(1)

>>

设置Modbus TCP数据类型。该值存储在RAM中，因此每次控制器上电时都必须初始化。这可以在TrioBASIC程序中完成，例如STARTUP

这必须在Modbus主站打开端口之前配置。

ETHERNET(action, slot, 7[, mode])

参数

模式:016位整型(默认值)
132位单精度浮点，没有地址减半
2没有地址减半的32位长字整数

如果您想使用地址减半，请参阅ETHERNET功能14

例子

初始化用于浮点数据的Modbus TCP端口。

Function = 14:Modbus或以太网IP的端点TCP

该功能允许用户对以太网IP和Modbus进行低级配置。默认值允许主机连接，无需在Controller端进行任何配置。这些设置存储在控制器中，因此每次控制器上电时都必须初始化。RAM这可以在TrioBASIC程序中完成，例如STARTUP。

如上所述，以太网IP和Modbus TCP都可以在上电时工作，而不需要任何配置(使用默认值)。如果需要不同的初始配置，必须在MC_CONFIG文件中配置系统参数IP_PROTOCOL_CTRL，以便禁用协议并在BASIC启动例程中设置所需的配置。

Function = 15:每个堆栈调用处理的最大数据包

仅限基于ARM的控制器。

以太网命令的函数' 15 '提供了对参数的访问，该参数限制了每次调用特定堆栈函数所处理的传入以太网数据包的数量。它控制速率，该值越小，处理数据包的速率越慢，但使微处理器中正在进行的其他进程过载的机会就越少。

默认值为0，表示处理所有传入的数据包。该参数是动态的，必须在程序中每次上电后设置。

ETHERNET(action, slot, 15, value)

例1

在终端0命令行中，检查函数15的设置。

>>?ETHERNET(0, -1, 15)

0

-1

例2

在一个程序中，设置一次栈调用可以处理的最大数据包数为20。这将足以限制TrioPC ActiveX上的大数据传输对运动和其他网络的影响。

ETHERNET(1, -1, 15, 20)

Function = 24:读取连接状态

返回电缆连接状态为TRUE/FALSE。如果RJ45连接器插入了实时以太网连接，则函数24返回TRUE。如果电缆未插入或连接到交换机或其他未通电的设备，则该函数返回FALSE。

这个函数是只读的

DIM state AS BOOLEAN

state = ETHERNET(0, slot, 24)

例

'监视以太网连接并在连接丢失时闪烁输出



WHILE TRUE

  IF ETHERNET(0, -1, 24) = FALSE THEN

    OP(9, 0) 

    OP(8, 1) '输出8闪烁表示连接丢失

    WA(250)

    OP(8, 0)

    WA(250)

  ELSE

    OP(9, 1) '输出9显示连接良好

    WA(250)

  ENDIF

WEND
