MOVESEQ

类型

轴命令

语法

MOVESEQ(table pointer, axes, npoints, options, radius [, output [, transition_angle]])

描述

MOVESEQ命令允许通过TABLE值加载一系列移动。这些动作可以使用圆形或球形弧线自动合并在一起。2到6个轴上的移动可以用这种方式控制。

MOVESEQ被加载到控制器移动缓冲区中，如果指定2个轴，则MOVE ->MOVECIRC-> moves，如果指定3个或更多轴，则MOVE ->MSPHERICAL-> moves。如果弧线混合在一起，则可以省略线性移动。如果“选项”位0设置为1，则加载的移动序列将是指定2轴时MOVESP->MOVECIRCSP-> moves，如果指定3+轴时MOVESP->MSPHERICALSP-> moves。

如果设置了选项位0，则必须为每个table点提供4个额外的速度控制值。

MOVE_COUNT在每次移动加载时递增。这个参数可以由用户程序设置。

参数

Table pointer:绝对点在TABLE内存中的位置。
Axes:轴数2 ~ 6。
npoints:点数，每个点位需要2或3个table值。
Options:位00序列加载MOVE, MOVECIRC等。
1序列将加载嵌入的速度移动MOVESP, MOVECIRCSP等。
位1保留
位20加载一系列标准的移动类型..
1加载一个MOVESEQ/MOVESEQSP移动类型序列。
位8…13设置为指定除“Axes”以外的数据点之间的偏移量。
Radius:要应用的合并/圆角半径。0表示不进行圆角。
Output:可选的TABLE数据输出。0表示不向TABLE输出数据。否则值为该数据的TABLE起始索引。
Transition angle:生成的任何圆角弧线的可选过渡角(以弧度为单位)规格。如果不应用过渡曲线，设置为0。

如果指定的点相距不够远，无法应用圆角，圆角半径将自动减小到可能的最大值。

MOVESEQ开始时的当前轴位置用于计算第一个圆角。

例

使用MOVESEQ绘制一个动作序列:

FOR x = 0 TO 2

    BASE(x)

    ATYPE = 0

    UNITS = 100

    ACCEL = 500

    DECEL = ACCEL

    SERVO = ON

    SPEED = 100

NEXT x



BASE(0,1,2)



DEFPOS(100,0,0)
