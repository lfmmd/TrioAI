NODE_AIO

类型

系统数组(MC_CONFIG)

语法

NODE_AIO(slot, node_address) = offset_value

描述

该二维阵列可用于设置任何EtherCAT节点模拟量I/O通道的起始地址。这可以用来定义一个用户特定的I/O映射，将来自不同来源的I/O点固定到位。

这个数组是二维的，第一维是主站槽标识符，第二维是节点在主站网络中的位置。

值

slot:EtherCAT端口槽位号。(MC4N、MC6N和Flex-6 Nano上为0)
node_address:由节点在EtherCAT网络中的位置决定的地址。第一个节点是地址0。
offset_value0自动分配EtherCAT I/O(默认)
> = 8定义节点的EtherCAT模拟量I/O从这个I/O点地址开始

根据EtherCAT配置的地址分配I/O偏移量，请使用NW_NODE_AIO

例

具有MC464、EtherCAT模块(槽位0)和节点5处16输入EtherCAT模拟量I/O模块的系统通常具有以下I/O分配:

Node_aio (0,5) = 0
0-15EtherCAT模拟量输入
32-33内置模拟量输入

如果改变了NODE_IO，那么EtherCAT模拟量I/O可以移动到更高的值。

Node_aio (0,5) = 40
0 - 1内置输入(32-33的拷贝)
32-33内置输入
40-55P876 EtherCAT模拟量输入
