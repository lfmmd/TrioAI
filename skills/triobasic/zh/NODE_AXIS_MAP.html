NODE_AXIS_MAP

类型

系统数组 (MC_CONFIG)

语法

NODE_AXIS_MAP(slot, node, axis_index) = value

描述

该3D阵列可用于覆盖任何EtherCAT节点轴的双轴驱动寻址。这可以用来定义一个用户特定的轴映射，以固定来自不同来源的轴。

这个数组是三维的，第一个维度是主站槽标识符，第二个维度是节点在主站网络中的位置，第三个维度是驱动器中从0开始的轴索引。

如果在启动EtherCAT协议时请求的轴已在使用，则会引发错误。

值

0自动分配EtherCAT轴(默认)
> = 1EtherCAT驱动器位于此轴

例

配置单轴驱动TRIO DX4从轴号1到轴号200和P374 2伺服轴分别从轴号10和11到轴号100和101。
