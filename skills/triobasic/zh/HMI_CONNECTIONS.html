HMI_CONNECTIONS

类型

系统参数

语法

HMI_CONNECTIONS

描述

返回所有当前连接的客户端的连接字符串。

值

value包含所有已连接客户端的连接字符串。每个连接字符串都在单独的行上。每行有如下结构:
<session>;<major>;<minor>;<ip>;<platform>;<osversion>;<window>
式中:
<session> 为对应的部分id(0,1，…)
<major> 是HMI 客户端的主要版本
<minor> 是HMI 客户端的次要版本
<ip> 为HMI客户端的ip地址
<platform> 是运行HMI 客户端的硬件的定义。
1 => WindowsCE
2 => Windows Desktop
<osversion> 为平台上报告的版本。主版本号存储在最高有效字节中，次版本号存储在最低有效字节中。
<window> 是HMI客户端屏幕的大小。宽度存储在最高有效字节中，高度存储在最低有效字节中。

例

报告当前连接的HMI客户端。

>>PRINT HMI_CONNECTIONS

0;1.22.4.502;127.0.0.1;2;60001;32001e0

1;1.22.3.500;192.168.2.53;1;50000;32001e0
