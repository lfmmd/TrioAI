FEED_OVERRIDE

类型

轴参数

描述

当使用嵌入式速度配置文件(如MOVECONTOUR)时，此参数可用于覆盖进给速率。

默认值为100，值为0..150%可由用户程序设置。

路径速度(进给速率)上升或减少到新的值。

值

100以嵌入配置文件参数设置的速度运行。
0-150以设定速度的百分比运行。

例

使用模拟量输入来控制手动控制的进给速率。

BASE(3) '以基准轴3控制插补轴上的进给速度

WHILE TRUE

    feed_rate = AIN(1) * 150 / 2048

    IF feed_rate < 0 THEN feed_rate = 0

    FEED_OVERRIDE = feed_rate

    WA(100)

WEND
