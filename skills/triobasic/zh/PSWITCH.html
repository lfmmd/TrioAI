PSWITCH

类型

命令

语法

循环模式

PSWITCH(switch, enable[, axis, output, state, setpos, resetpos])

关闭

PSWITCH(switch, OFF[, hold])

序列模式

PSWITCH(switch, mode, axis, output, state, table_start, table_end[, pulse_time])

描述

PSWITCH命令允许在达到预定义位置时设置输出，当达到第二个位置时重置输出。通常情况下，setpos必须比resetpos低，但是从V2.0313开始，如果设置了使能值的第1位，setpos可以更高。

有64个位置开关，每一个都可以分配到任何轴和任何输出，虚拟轴或实际轴。这两个位置定义了一个窗口，其中特定的输出将按照_状态_参数所指示的方式运行，因此无论运动的方向如何，它都将始终工作。

从固件2.0309及更高版本开始，可以使用另一种操作，在从TABLE内存加载的选定位置打开和关闭输出。

多个PSWITCH可以分配到单个输出。

实际输出是输出上所有位置开关的或 OP(设置。这意味着OP(输出，ON)可以覆盖PSWITCH。

当PSWITCHOFF开关时，输出将保持在当前状态，除非hold参数被设置为1。(Hold需要固件2.0226或更高版本)

参数

循环模式

switch:开关号在0..63之间
enable:1或ON使用Setpos < Resetpos使能软件PSWITCH
3通过Setpos > Resetpos使能PSWITCH
5在DPOS上启用PSWITCH
0或OFF关闭PSWITCH
axis:连接PSWITCH的轴，可以是任何实轴或虚轴。
output:选择要设置的输出，可以是任何实输出或虚输出。
state:1或ON在setpos时ON输出
0或OFF在setpos时OFF输出
setpos:输出被设置或复位的位置，以用户单位
resetpos:输出被设置或重置的位置，以用户单位
hold:0PSWITCH输出将保持在与PSWITCH设置为OFF时相同的状态

。(默认)
1当PSWITCH设置为OFF时，即使PSWITCH输出是ON，也会强制 OFF。

序列模式

switch:开关号在0..63之间
mode:位#描述
00:关闭PSwitch, 1:开启PSwitch
1不使用
20: mpos, 1: dpos
30:单个Pswitch, 1:多个Pswitch (位4.. 6确定多模式行为)
40:位置开/关，1:定时器模式
50:正向方向，1:负向方向
60:数字输出，1:Z-Out
axis:连接PSWITCH的轴，可以是任何实轴或虚轴。
output:选择要设置的输出，可以是任何实际输出或虚拟输出。
state:1或ON在第一个位置ON输出
0或OFF在第一个位置OFF输出
table_start:PSWITCH数据开始的Table索引
table_end:PSWITCH数据结束的Table索引
pulse_time:设置脉冲输出模式位4时的伺服周期时间
例如，SERVO_PERIOD为500,pulse_time为10，脉冲输出为5ms

例

例1

旋转轴具有一个凸轮操作的开关，该开关必须针对不同尺寸的工件进行更换。轴上还有一个接近开关，用于指示机器的TDC。使用机械凸轮，每次更换工作都很耗时，但是可以通过将PSWITCH用作软件“凸轮开关”来简化。接近开关连接到输入7，输出由输出11触发。轴由3轴系统的轴0控制。电机具有900ppr编码器。 TDC之后，输出必须从80°开始开启120°。可以假定机器是从TDC启动的。

PSWITCH命令使用单位转换因子来允许以方便的单位设置位置。因此，首先必须计算并设置单位转换系数。编码器上的每个脉冲都有四个边沿供控制器计数，因此有3600个边沿/转或10个边沿/°。如果将单位转换因子设置为10，则可以以度为单位。

接下来，我们必须为所有PSWITCH参数确定一个值。

所有这些都可以放在一起形成用于设置位置开关的Trio BASIC代码的两行：
axis我们被告知轴由轴0控制，因此轴被设为0。
output我们被告知输出11是要触发的，所以这是11。
state当输出设置时，它应该是ON。
setpos输出是在上止点后以80°发射，因此设置位置是80，因为我们以度为单位工作。
resetpos输出将在80°~ 200°的范围内开启，大于200°时关闭，复位位置为200。

switch:

  UNITS AXIS(0) = 10 '设置单位转换系数(°)

  REPDIST = 360

  REP_OPTION = ON

  PSWITCH(0, ON, 0, 11, ON, 80, 200)

这个程序使用设置为360度的重复距离，且重复选项设置为ON，因此轴位置就会保持在0..360度的范围内。

例2

7号PSWITCH已在轴5上运行，控制输出14。它必须被禁用，同时输出设置为OFF。

PSWITCH(7，OFF，1)

或者相同的PSWITCH必须被禁用，但输出直到某个事件之后才改变。后面的事件由输入23上的复位按钮控制。

PSWITCH(7, OFF, 0)

WA(1) '等待一个伺服周期禁用PSWITCH

IF READ_OP(14) = ON THEN

  WAIT UNTIL IN(23) = ON

  OP(14, OFF)

ENDIF

例3

想象与例1相同的情况，其中输出11必须向“凸轮开关”系统提供下降沿信号，因此输出必须从TDC后的80°~ 200°的范围外断开。

所有PSWITCH参数的值:

axis_number = 0 op_number = 11 op_state = OFF setpos = 80 ' TDC后80度 reset_pos = setpos + 120 '度

PSWITCH(24, ON, axis_number, op_number, op_state, setpos, reset_pos)
