AXIS_ERROR_COUNT

类型

轴参数。

描述

每当数字轴上出现通信错误时，AXIS_ERROR_COUNT参数就递增。在支持的情况下，此值可用于表示数字轴上的错误率。并不是所有的数字轴类型都有计算错误的能力。在每种类型的数字通信总线的描述中可以找到进一步的信息。

值

自上次复位以来的通信错误次数。

例

初始化错误计数器

AXIS_ERROR_COUNT = 0

在终端查看最新的错误计数值。

>>?AXIS_ERROR_COUNT AXIS(3)

10.0000

>>

记录一个轴的总体错误率。

TICKS = 600000

AXIS_ERROR_COUNT = 0

REPEAT

  IF TICKS<0 THEN

    VR(10) = AXIS_ERROR_COUNT '十分钟内统计的错误数

    TICKS = 600000

    AXIS_ERROR_COUNT = 0

  ENDIF

  …

  …

UNTIL FALSE
轴参数。
