IS_NAN

类型

数学函数

语法

value = IS_NAN(expression)

描述

检查表达式以测试它是否是NaN(不是数字，而是不表示有效浮点值的位模式)。

如果电池支持的存储器失去电池电量，位模式将是随机的，NaN将导致BASIC错误。

NaN值也可以从外部源设置，如PC或PLC通过现场总线设置值。

参数

expression任何有效的数值表达式

例

例1

检查MC464中的所有VRs，将NaN设置为0。

FOR i = 0 TO 65535

    IF IS_NAN(VR(i)) THEN

        VR(i) = 0

    ENDIF

NEXT i
