VALIDATE_ENCRYPTION_KEY

类型

系统命令

语法

VALIDATE_ENCRYPTION_KEY (security_code_type, validation_key)

描述

VALIDATE_ENCRYPTION_KEY用于检查控制器是否编写了正确的用户或OEM安全代码。如果没有编写正确的安全代码，那么VALIDATE_ENCRYPTION_KEY将产生一个runtime错误(参数超出范围)，从而停止程序的功能。

Motion Perfect有一个工具来生成验证密钥

不要将用户或OEM安全代码放入程序中，因为这些代码必须保密。

参数

security_code_type1OEM安全代码
2用户安全码
validation_key一个字符串，它是由Motion Perfect生成的验证密钥

例

在运行主程序之前测试用户安全码是否有效

'验证用户安全码

VALIDATE_ENCRYPTION_KEY(2,"1Wgltam0wzrbCVJwUgEnGU")

RUN "MAIN_PROGRAM"

查看

SET_ENCRYPTION_KEY, PROJECT_KEY
