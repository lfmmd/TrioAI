CHANNEL_READ

类型

文件函数

语法

x = CHANNEL_READ(channel, storage_buffer[, delimiter_buffer[, escape_character[, crc]]])

描述

CHANNEL_READ将从通道读取字节并将其存储到存储缓冲区中。

VR和STRING实现在解析通道中的字符方面具有相同的行为，但是它们返回的数据不同。

如果存储缓冲区在VR中，那么第一个值指定CHANNEL_READ停止的原因:0表示文件结束，1表示第一个分隔符字符，2表示第二个分隔符字符，等等，命令返回读取的字符数。字符串以空结束，因此VRSTRING命令可以用来将缓冲区作为字符串查看。

如果存储缓冲区是一个命名的字符串变量，则命令返回CHANNEL_READ停止的原因:如果返回值小于零，则CHANNEL_READ找到了一个分隔符，-1表示第一个分隔符，-2表示第二个分隔符，以此类推，否则通道中没有更多的字符可读，或者字符串变量中没有更多的空间可写。读取的字符数可以使用LEN命令对命名字符串变量进行读取。

当CHANNEL_READ读取大小字节、通道为空或从通道读取的字符在分隔符缓冲区中指定时，它将停止。

如果接收到转义字符，则不解释下一个字符。这允许在不停止CHANNEL_READ的情况下接收分隔符。

计算的CRC结果将存储在VR(crc)中。

参数

channel通信或文件通道。
storage_buffer一个命名的字符串变量，或者两个指定VR基准和长度的数值表达式。
delimiter_buffer一个字符串表达式，或两个指定VR基准和长度的数字表达式。
escape_character当接收到这个字符时，不解释后面的字符。
crc存储CRC数据的VR地址 。

例1

从文件中读取数字，每行一个数字，使用VR存储和分隔符缓冲区。

'在RAM中创建一个包含数字1到10的临时文件，每个数字一行

OPEN #40 AS "ram:test" FOR OUTPUT(1)

FOR i = 1 TO 10

  PRINT #40, i

NEXT i

CLOSE #40



'设置分隔符

VR(10) = 13 'carriage return

VR(11) = 10 'line feed



'测试VR功能

OPEN #40 AS "ram:test" FOR INPUT

PRINT "-------------------- START VR ---------------------"

REPEAT

  '读取通道40

  'VR(100)具有结束状态

  'VR(101)-VR(199)保持数据

  'VR(10)-VR(11) 保持分隔符

  c = CHANNEL_READ(40, 100, 100, 10, 2)



  '如果我们有字符，那么打印它们

  IF c > 0 THEN

    PRINT c[0], VR(100)[0], VRSTRING(101)

  ENDIF

  IF VR(100) = 1 THEN

    PRINT "--- CARRIAGE RETURN ----"

  ELSEIF VR(100) = 2 THEN

    PRINT "--- LINE FEED ----"

  ENDIF

UNTIL NOT KEY #40

PRINT "-------------------- STOP VR ---------------------"

CLOSE #40

例2

从文件中读取数字，每行一个数字，使用字符串存储和分隔符缓冲区。

'在RAM中创建一个包含数字1到10的临时文件

'每个数字一行

OPEN #40 AS "ram:test" FOR OUTPUT(1)

FOR i = 1 TO 10

  PRINT #40, i

NEXT i

CLOSE #40



'声明缓冲区

DIM b AS STRING(100)

DIM d AS STRING(2)



'设置分隔符

d = CHR(13) + CHR(10)



'测试字符串功能

OPEN #40 AS "ram:test" FOR INPUT

PRINT "-------------------- START STRING ---------------------"

REPEAT

  '读取通道 40

  s = CHANNEL_READ(40, b, d)

  c = LEN(b)



  '如果有字符，打印它们

  IF c > 0 THEN

    PRINT c[0], s[0], b

  ENDIF

  IF s = -1 THEN

    PRINT "--- CARRIAGE RETURN ----"

  ELSEIF s = -2 THEN

    PRINT "--- LINE FEED ----"

  ENDIF

UNTIL NOT KEY #40

PRINT "-------------------- STOP STRING ---------------------"

CLOSE #40

'Create a temp file in RAM that contains a large buffer

OPEN #40 AS "ram:test" FOR OUTPUT(1)

FOR j = 1 TO 10

  FOR i = 32 TO 127

    PRINT #40, CHR(i)

  NEXT i

NEXT j

CLOSE #40



'Declare the buffers

DIM b AS STRING(100)

DIM d AS STRING(2)



'Set the delimiters

d = CHR(13) + CHR(10)



'Test string functionality

OPEN #40 AS "ram:test" FOR INPUT

PRINT "-------------------- START STRING ---------------------"

REPEAT

  'Read channel 40

  s = CHANNEL_READ(40, b, d)

  c = LEN(b)



  'If we have characters then print them

  IF s < 0 THEN

    IF s = -1 THEN

      PRINT "--- CARRIAGE RETURN ----"

    ELSEIF s = -2 THEN

      PRINT "--- LINE FEED ----"

    ENDIF

  ELSEIF c = 100 THEN

    PRINT c[0], s[0], b

  ELSE 

    PRINT c[0], s[0], b

    PRINT "--- END OF BUFFER ----"

  ENDIF
