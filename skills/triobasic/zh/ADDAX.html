ADDAX

类型

轴命令

语法

ADDAX(axis)

描述

ADDAX命令用于叠加2个或更多运动，以建立更复杂的运动曲线：

ADDAX命令从指定轴获取给定位置变化，并将其添加到在基准轴上运行的任何运动中。

发出ADDAX命令后，两个轴之间的链接将保持不变，直到断开为止，并且在指定轴上进行的任何进一步移动都将添加到基准轴上。

指定的轴可以是任何轴，并且不必实际存在于系统中。

因此，ADDAX命令允许轴执行指定的两个轴叠加后的运动。

在使用编码器时，如果设置 SERVO = OFF，那么 MPOS 会直接复制到DPOS . 这允许 ADDAX 对编码器输入求和。

参数

axis:要叠加的轴。
-1断开与另一个轴的链接。

ADDAX命令以编码器脉冲对运动进行求和。

例

例 1

在具有不同 UNITS 的轴上使用ADDAX, 轴0将会移动 1*1000+2*20=1040 个脉冲。

UNITS AXIS(0) = 1000

UNITS AXIS(1) = 20



'将轴1叠加在轴0上

ADDAX(1) AXIS(0)

MOVE(1) AXIS(0)

MOVE(2) AXIS(1)

例 2

物料被随机放置在一条连续移动的皮带上，并在生产线的后续阶段转移到第二条带有挡板的传送带上。检测系统指示物料是否位于其标定的前方或后方，以及偏离的距离。

  expected = 2000 '设定预期位置

  BASE(0)

  ADDAX(1)

  CONNECT(1, 2) '与齿轮皮带的连续齿轮连接

  REPEAT

    GOSUB getoffset '获取偏差值

    MOVE(offset) AXIS(1) '使校正沿虚拟轴移动

  UNTIL IN(2) = OFF '直到IN(2)=OFF

  RAPIDSTOP

  ADDAX(-1) '解除ADDAX连接

  STOP



Getoffset: '子程序来记录工件的位置并计算偏移量

  BASE(0)

  REGIST(3)

  WAIT UNTIL MARK

  seenat = REG_POS

  offset = expected - seenat

  RETURN

在此示例中，轴0连接到第二个皮带的编码器，并且轴1上的叠加 MOVE用于施加偏移。

例 3

XY打标机必须在箱子沿着传送带移动时对箱子进行标记。 使用 CONNECT可使X标记轴跟随传送带。虚拟轴用于编程标记的绝对位置，然后使用ADDAX将其叠加到X轴上。

ATYPE AXIS(3) = 0 '将轴3设置为虚拟轴

SERVO AXIS(3) = ON

DEFPOS(0) AXIS(3)

ADDAX (3) AXIS(0) '轴3要求连接到轴0

WHILE IN(2) = ON

  REGIST(3) '色标输入检测传送带上的盒子

  WAIT UNTIL MARK OR IN(2) = OFF

  IF MARK THEN

    CONNECT(1, 2) AXIS(0) '将轴0连接到正在移动的皮带

    BASE(3, 1) '轴3和轴1执行绘图运动

    '画一个M

    MOVEABS(1200, 0) 'Move A > B

    MOVEABS(600, 1500) 'Move B > C

    MOVEABS(1200, 3000) 'Move C > D

    MOVEABS(0, 0) 'Move D > E

    WAIT IDLE

    BASE(0)

    CANCEL '让轴0停止跟随皮带的运动

    WAIT IDLE

    MOVEABS(0) '将轴0移动到原点

  ENDIF

WEND
