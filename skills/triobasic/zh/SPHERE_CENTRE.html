SPHERE_CENTER

类型

轴命令

语法

SPHERE_CENTRE(table_mid, table_end, table_out)

描述

从任何中点(x, y, z)和终点(x, y, z)返回圆弧中心点(x, y, z)的坐标。x, y和z在TABLE存储区域中返回，并可以根据需要打印到终端。注意，中点和终点位置是相对于起始位置的。

参数

TABLE mid:中点x,y,z在table中的位置
TABLE end:结束点x,y,z在table中的位置
TABLE out:table中存放输出数据的位置:
偏移量0 - X

偏移量1 - Y

偏移量2 - Z

偏移量3 -角度

偏移量4 -半径

偏移量5 -如果错误设置为1，否则设置为0

例

TABLE(10,-200,400,0)

TABLE(20,-500,500,0)

SPHERE_CENTRE(10,20,30)

x = TABLE(30)

y = TABLE(31)

z = TABLE(32)

ang = TABLE(33)

rad = TABLE(34)

err = TABLE(35)

PRINT x,y,z,ang,rad,err
