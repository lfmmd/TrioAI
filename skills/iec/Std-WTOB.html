WTOB

Function

Copies and splits the WORD type value to the 2 destination array items.

Inputs

Name

Type

Description

pbyDataSrc[]

UINT

Array containing the source of the copy.

uiSize

UINT

The number of items to be copied from pbyDataSrc.

pwDataDes[]

DINT

The array where will be copied to.

Outputs

Name

Type

Description

OK

BOOL

TRUE if successful.

Remarks

Every source array element will be split into 2 destination array elements.

ST Language

OK := WTOB (pbyDataSrc, 3, pwDataDes);
