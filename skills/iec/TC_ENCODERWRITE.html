TC_ENCODERWRITE

Type:

Motion Function.

Function:

Write an internal register to an EnDat absolute encoder.

Inputs:

Axis : USINT;

Axis number

Address : DINT;

The address of the EnDat encoder register to be write

Value : DINT;

Value writed to the specified register

Outputs: 

Q : SINT;

 

Description:

Write an internal register to an EnDat absolute encoder.
