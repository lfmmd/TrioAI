TC_EC_STOP

Type:

System Command

Function:

Stops the EtherCAT network.

Inputs:

  

Execute : BOOL;

Rising edge requests execution

Slot : INT;

Slot number of EtherCAT port

Outputs: 

Done : BOOL;

TRUE when function has completed normally

Error : BOOL;

TRUE if a program error is detected

ErrorID : UINT;

Returned error number

Description:

This function block stops the EtherCAT network.
