TC_MbTCP_OpenClient

Type

System Function.

Function

Open a Modbus client connection.

Inputs

Execute: BOOL;Rising edge requests execution
Slot: INT;Slot number. Currently, the default value is -1. If it is not -1, an error will be reported
IpAddress: STRING;The server's IP address is 4 octets separated by dot.
PortNumber: UINT;Optional port number. Default is port 502 if none given.
CmdTimeOut: LINT;Timeout (ms) for executing functions
CharTimeOut: LINT;Maximum time interval permitted between individual received characters or ability to transmit characters within a message before an error is triggered

Outputs

Done: BOOL;TRUE when function has completed normally
HandleNum: DINT;Connection handle
Error: BOOL;TRUE if a program error is detected
ErrorID: UINT;Returned error number

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to open Modbus TCP client connection. ErrorID includes 316, 781, 790, 791 and 801. and the corresponding error descriptions are as follows.
ErrorIDDescription
316IP address error
781Slot error
790Timeout error
791Connection opening error
801PortNumber error
