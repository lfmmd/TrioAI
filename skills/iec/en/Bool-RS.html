RS

Function Block

Reset dominant bistable.

Inputs

NameTypeDescription
SETBOOLCondition for forcing to TRUE.
RESET1BOOLCondition for forcing to FALSE (highest priority command).

Outputs

NameTypeDescription
Q1BOOLOutput to be forced.

Remarks

The output is unchanged when both inputs are FALSE. When both inputs are TRUE, the output is forced to FALSE (reset dominant).

ST Language

MyRS is declared as an instance of RS function block:

MyRS (SET, RESET1);

Q1 := MyRS.Q1;
