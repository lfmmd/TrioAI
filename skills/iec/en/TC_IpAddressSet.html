TC_IpAddressSet

Type

System Parameter.

Function

Set the IP address of the slot.

Inputs

ParameterDescription
Execute: BOOL;Rising edge requests execution.
Slot: SINT;Slot number, the default value is -1.
IPV4_1: USINT;The first byte of the IP address.
IPV4_2: USINT;The second byte of the IP address.
IPV4_3: USINT;The third byte of the IP address.
IPV4_4: USINT;The fourth byte of the IP address.

Outputs

ParameterDescription
Done: BOOL;TRUE when function has completed normally.
Busy: BOOL;TRUE if function is running.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to set the IP address of the given slot.

ST Language

MyTC_IpAddressSet(Execute:= Execute, Slot:= Slot, IPV4_1:= IPV4_1, IPV4_2:= IPV4_2, IPV4_3:= IPV4_3, IPV4_4:= IPV4_4);

Done:=MyTC_IpAddressSet.Done;

Busy:=MyTC_IpAddressSet.Busy;

Error:=MyTC_IpAddressSet.Error;

ErrorID:=MyTC_IpAddressSet.ErrorID;
