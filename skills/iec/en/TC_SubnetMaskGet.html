TC_SubnetMaskGet

Type

System Parameter.

Function

Get the subnet mask for the slot.

Inputs

ParameterDescription
Enable: BOOL;Set TRUE to enable the function.
Slot: SINT;Slot number, the default value is -1.

Outputs

ParameterDescription
Valid: BOOL;TRUE when function has completed normally.
Busy: BOOL;TRUE if function is running.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.
Mask_1: USINT;The first byte of the Subnet Mask.
Mask_2: USINT;The second byte of the Subnet Mask.
Mask_3: USINT;The third byte of the Subnet Mask.
Mask_4: USINT;The fourth byte of the Subnet Mask.

Description

When the Enable input is TRUE, the Function Block issues a command to get the subnet mask for the given slot.

ST Language

MyTC_SubnetMaskGet(Enable:= Enable, Slot:= Slot);

Valid:=MyTC_SubnetMaskGet.Valid;

Busy:=MyTC_SubnetMaskGet.Busy;

Error:=MyTC_SubnetMaskGet.Error;

ErrorID:=MyTC_SubnetMaskGet.ErrorID;

Mask_1:=MyTC_SubnetMaskGet.Mask_1;

Mask_2:=MyTC_SubnetMaskGet.Mask_2;

Mask_3:=MyTC_SubnetMaskGet.Mask_3;

Mask_4:=MyTC_SubnetMaskGet.Mask_4;
