TC_MbNodeAddrGet

Type

System Function.

Function

Get the Modbus node address.

Inputs

Enable: BOOL;Rising edge requests execution.

Outputs

Done: BOOL;TRUE when function has completed normally.
NodeAddr: USINT;Returned Modbus node address.

Description

When the Enable input changes from FALSE to TRUE, the Function Block issues a command to get the Modbus node address.

ST Language

MyTC_MbNodeAddrGet(Enable:= Enable);

Done:=MyTC_MbNodeAddrGet.Done;

NodeAddr:=MyTC_MbNodeAddrGet.NodeAddr;

FBD Language
