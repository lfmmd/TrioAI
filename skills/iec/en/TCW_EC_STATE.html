TCW_EC_STATE

Type

System Function.

Function

Set the state of EtherCAT network.

Inputs

Slot : INT;Slot number of EtherCAT port
State : INT;State of EtherCAT network
1Initial
2Pre-Operation
4Safe-Operation
8Operation
Display : BOOLWrites state change information to the standard output stream. True - Enable(Default), False -Disable
Mpemode : BOOLMPE mode，True – Enable, False - Disable (Default)
Express : BOOLExpress mode，True – Enable, False - Disable (Default)

Outputs

Q : DINT;

Description

This function controls the EtherCAT State Machine. (ESM) It requests the master change to given EtherCAT ‘state’, and hence changes all slaves to the same state. When a change to a higher state is made, the EtherCAT network will progress to the new state through the in-between states to allow correct starting of the network.
