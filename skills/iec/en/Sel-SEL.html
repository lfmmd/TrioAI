SEL

Function

Select one of the two inputs.

Inputs

NameTypeDescription
SELECTDINTSelection command.
IN1ANYFirst input.
IN2ANYSecond input.

Outputs

NameTypeDescription
QANYIN1 if SELECT is FALSE; IN2 if SELECT is TRUE

Truth table

SELECTQ
0IN1
1IN2

Remarks

In LD language, the selector command is the input rung. The output rung keeps the same state as the input rung. In IL language, the first parameter (selector) must be loaded in the current result before calling the function. Other inputs are operands of the function, separated by commas.

ST Language

Q := SEL (SELECT, IN1, IN2);

FBD Language

LD Language

The input rung is the selector.
ENO has the same value as SELECT.
