TCW_AxisParameter

Type

Axis Parameter

Function

Writes to the named axis parameter.

Inputs

AxisNo : USINT;Axis number
ParamValue : Various;Parameter value

Outputs

ErrorID : UINT;Error ID number

Description

Writes the specified value to the AxisParameter. See the function block tooltips in the Motion Perfect editor for parameter names and data sizes.

ST Language

MyTCW_AxisParameter(AxisNo, ParamValue);

ErrorID := MyTCW_AxisParameter.ErrorID;
