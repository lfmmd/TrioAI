TCW_MOVELINKMODIFY

Type

Axis Parameter

Function

Reads the current value of MOVELINK_MODIFY.

Inputs

AxisNo: USINT;Axis number
POS: LREAL;New value of MVELINK_MODIFY

Outputs

ErrorID: UINT;Returned error number

Description

This function writes new value of MOVELINK_MODIFY, which allows MOVELINK motion commands to be modified whilst they are running by applying the specified positional offset.
