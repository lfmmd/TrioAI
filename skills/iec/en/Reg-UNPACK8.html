UNPACK8

Function block

Extract bits of a byte.

Inputs

NameTypeDescription
INUSINT8-bit register.

Outputs

NameTypeDescription
Q0BOOLLeast significant bit.
...
Q7BOOLMost significant bit.

Remarks

In LD language, the output rung is the Q0 output. The operation is executed only in the input rung (EN) is TRUE.
