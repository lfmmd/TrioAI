TC_READOP

Type

I/O Function

Function

Applies a new READ_OP request for the digital output specified.

Inputs

Index : INT;Output number

Outputs

Q : SINT;Output state

Description

Sets the digital outputs to the binary pattern given in Value.

Q := MyTC_ReadOP.Q;
