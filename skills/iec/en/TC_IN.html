TC_IN

Type

System Function

Function

Reads the state of an input.

Inputs

Index : INT;Input number to return the value of

Outputs

Q : BOOL;The state of the selected input

Description

TC_IN is used to read the state of the specified input. The value can then be stored in a variable specified at the output parameter.

ST Language

TC_IN( Index(*INT*) );

Examples

Reads IN2 and saves it to ‘IN2_val’ variable.

IN2_val:=TC_IN(2);

FBD Language

Examples

Reads IN0 and saves it to ‘Start_PB’ variable.

LD Language

Examples

Reads IN1 and saves it to ‘Stop_PB’ variable.
