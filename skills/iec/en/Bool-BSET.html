BSET

Function

Sets the specified bit state of the data.

Inputs

NameTypeDescription
wDataUINTThe variable which will be operated.
uiBitUINTThe specified bit which will be set.

Outputs

NameTypeDescription
Q1BOOLTRUE, the specified bit is available, or FALSE.

Remarks

Sets the specified bit value. ‘wData’ is a variable with 16-bit data type, so the ‘uiBit’ must be less than 16.

ST Language

wMyVar is a variable which will be operated by function ‘BSET’.

bRes := BSET(wMyVar, 10); // Set bit 10 of wMyVar
