TCR_WDOG

Type

System Parameter

Function

Reads the state of the WDOG system variable.

Inputs

None

Outputs

WDOG : DINT;WDOG state

Description

Reads the current watchdog state, returning its value in the WDOG output.

ST Language

MyTCR_Wdog(WDOG);

WDOG:= MyTCR_Wdog.WDOG;
