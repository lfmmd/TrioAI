TCR_VR

Type

Motion Parameter

Function

Reads from a VR variable.

Inputs

Index : INT;VR Index number

Outputs

Q : LREAL;VR value

Description

Reads from the VR number indicated in the Index input, returning its value in the Q output.

ST Language

MyTCR_VR(Index);

Q:= MyTCR_VR.Q;

LD Language
