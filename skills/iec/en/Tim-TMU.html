Function Block

Up-counting stop watch.

Inputs

NameTypeDescription
INBOOLThe time counts when this input is TRUE.
RSTBOOLTimer is reset to 0 when this input is TRUE.
PTTIMEProgrammed time.

Outputs

NameTypeDescription
QBOOLTimer elapsed output signal.
ETTIMEElapsed time.

Time diagram

Remarks

The timer counts up when the IN input is TRUE. It stops when the programmed time is elapsed. The timer is reset when the RST input is TRUE. It is not reset when IN is false.

PT can be entered as a TIME literal. e.g. T#100ms, T#3s250ms, T#3h5m, T#16d8h56s.

ST Language
