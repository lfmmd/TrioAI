BTOW

Function

Copies the lower 8bit data of resource data to WORD data type destination.

Inputs

NameTypeDescription
pbyDataSrc[]UINTArray containing the source of the copy.
uiSizeUINTThe number of items to be copied from pbyDataSrc.
pwDataDes[]DINTThe array destination where will be copied to.

Outputs

NameTypeDescription
OKBOOLTRUE if successful.

Remarks

The upper byte of every item in source array will be ignored.

ST Language

OK := BTOW (pbyDataSrc, 6, pwDataDes);
