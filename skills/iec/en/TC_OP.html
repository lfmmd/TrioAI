TC_OP

Type

I/O Function

Function

Applies a new OP request for the digital output specified.

Inputs

Index : INT;Output number
Value : BOOL;Value to be copied to the specified Output number

Outputs

Q : BOOL;The state of the selected output

Description

Sets the digital outputs to the binary pattern given in ‘Value’ input parameter.

ST Language

TC_OP( Index(*INT*), Value(*BOOL*) );

Examples

Switch ON output 8.

TC_OP(8,TRUE);

Examples
