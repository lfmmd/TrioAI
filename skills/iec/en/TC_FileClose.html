TC_FileClose

Type

System Function.

Function

Close the specified file.

Inputs

Execute: BOOL;Rising edge requests execution.
Name: STRING;The name of the file

Outputs

Done: BOOL;TRUE when function has completed normally.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to close the specified file.

ST Language

MyTC_FileClose(Execute:= Execute, Name:= Name);

Done:=MyTC_FileClose.Done;

Error:=MyTC_FileClose.Error;

ErrorID:=MyTC_FileClose.ErrorID;
