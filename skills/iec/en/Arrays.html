Arrays

You can declare an array variable by specifying dimensions. All dimensions are 0 based, so in case of single dimension array the first element is identified by index 0 (e.g. MyArrayName[0]).

To declare an array, enter its dimension in the corresponding column of the variable editor. For a multi-dimension array, enter dimensions separated by comas (e.g. 2, 10, 4).

Use in Structured Text

To specify an item of an array in ST language, enter the name of the array followed by the indices entered between "[" and "]" characters. For multi-dimension arrays, enter indices separated by comas. Indices may be either constant or complex expressions.

Example

VAR

  MyArray: ARRAY[2, 8] OF REAL

END_VAR

MyArray[1, 7] := value;

result := MyArray[j, i + 2];

Use in FBD and LD languages

For get blocks, the first input is the array to read, and the remaining inputs are indices. The output is the value of the array element indexed.

For put blocks, the first input is the value to assign, the second input is the array, and the remaining inputs are indices.

Arrays have at most 3 dimensions.

All dimensions are 0 based.

The total number of items in an array (merging all dimensions) cannot exceed 65535.
