TC_IsFileOpen

Type

System Function.

Function

Check whether the file is open.

Inputs

ParameterDescription
Enable: BOOL;Rising edge requests execution.
Name: STRING;The name of the file.

Outputs

ParameterDescription
Done: BOOL;TRUE when function has completed normally.
Q: TC_FILE_OPEN_MODE_T;The open state of the file. See Q.

Q

ValueDescription
tcNotOpenedFile not open.
tcOpenedRdFile read-only.
tcOpenedRWFile read/write.
tcOpenedWrFile write-only.

Description

When the Enable input changes from FALSE to TRUE, the Function Block issues a command to check whether the file is open.

ST Language

MyTC_IsFileOpen(Enable:= Enable, Name:= Name);

Done:=MyTC_IsFileOpen.Done;

Q:=MyTC_IsFileOpen.Q;
