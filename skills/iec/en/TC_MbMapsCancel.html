TC_MbMapsCancel

Type

System Function.

Function

Cancel all previous maps.

Inputs

Execute: BOOL;Rising edge requests execution.
Slot: SINT;Slot number, the default value is -1.

Outputs

Done: BOOL;TRUE when function has completed normally.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to cancel all previous maps.

ST Language

MyTC_MbMapsCancel(Execute:= Execute, Slot:= Slot);

Done:=MyTC_MbMapsCancel.Done;

Error:=MyTC_MbMapsCancel.Error;

ErrorID:=MyTC_MbMapsCancel.ErrorID;
