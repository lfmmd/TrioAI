TC_MbRTU_CloseConnection

Type

System Function.

Function

Close Modbus RTU client connection if open.

Inputs

Execute: BOOL;Rising edge requests execution
Slot: INT;Slot number. Currently, the default value is -1. If it is not -1, an error will be reported
HandleNum: DINT;The connection handle output by TC_MbRTU_OpenConnection

Outputs

Done: BOOL;TRUE when function has completed normally
Error: BOOL;TRUE if a program error is detected
ErrorID: UINT;Returned error number

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to close Modbus RTU client connection according to HandleNum if open. ErrorID includes 781, 782 and 792, and the corresponding error descriptions are as follows.
ErrorIDDescription
781Slot error
782Connection closing error
792HandleNum error
