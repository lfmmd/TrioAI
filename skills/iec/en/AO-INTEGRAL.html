INTEGRAL

Function Block

Calculates the integral of a signal.

Inputs

NameTypeDescription
RUNBOOLRun command: TRUE=integrate / FALSE=hold.
R1BOOLOverriding reset.
XINREALInput signal.
X0REALInitial value.
CYCLETIMESampling period (should not be less than the target cycle timing).

Outputs

NameTypeDescription
QDINTRunning mode report: NOT (R1).
XOUTREALOutput signal.

Remarks

In LD language, the input rung is the RUN command. The output rung is the Q report status.

ST Language

MyIntg is a declared instance of INTEGRAL function block.

MyIntg (RUN, R1, XIN, X0, CYCLE); Q := MyIntg.Q; XOUT := MyIntg.XOUT;
