TCR_SYSTEMERROR

Type

System Parameter.

Function

Returns a value which indicates which active errors have been reported by the firmware.

Inputs

Outputs

Q : SINT;System error code

Description

TCR_SYSTEMERROR returns a value which indicates which active errors have been reported by the firmware. Any bits set in SYSTEM_ERROR will cause the WDOG to be automatically set to OFF, thus halting all motion.
