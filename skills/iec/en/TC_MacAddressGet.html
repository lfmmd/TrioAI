TC_MacAddressGet

Type

System Parameter.

Function

Get the Mac address of the slot.

Inputs

ParameterDescription
Enable: BOOL;Set TRUE to enable the function.
Slot: SINT;Slot number, the default value is -1.

Outputs

ParameterDescription
Valid: BOOL;TRUE when function has completed normally.
Busy: BOOL;TRUE if function is running.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.
MacVendor: UDINT;Returned vendor number.
MacSN: UDINT;Returned network card serial number.

Description

When the Enable input is TRUE, the Function Block issues a command to get the Mac address of the given slot.

ST Language

MyTC_MacAddressGet(Enable:= Enable, Slot:= Slot);

Valid:=MyTC_MacAddressGet.Valid;

Busy:=MyTC_MacAddressGet.Busy;

Error:=MyTC_MacAddressGet.Error;

ErrorID:=MyTC_MacAddressGet.ErrorID;

MacVendor:=MyTC_MacAddressGet.MacVendor;

MacSN:=MyTC_MacAddressGet.MacSN;
