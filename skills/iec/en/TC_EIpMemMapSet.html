TC_EIpMemMapSet

Type

System Function.

Function

Set the memory mapping for Ethernet IP communication.

Inputs

ParameterDescription
Execute: BOOL;Rising edge requests execution.
Slot: SINT;Slot number, the default value is -1.
EndPoint: TC_EIP_ENDPOINT_T;Specify which end point to read from or write to. See EndPoint.
StartPos: UINT;The start position of the data location.
DataLocation: TC_DATA_LOCATION_T;The location of the data on the controller. See DataLocation.
DataFormat: TC_DATA_FORMAT_T;The precision of the data. See DataFormat.
NumEntries: UINT;The number of the data locations.

EndPoint

ValueDescription
tcEIP_Input100Ethernet IP Assembly Object, Instance 100 (input).
tcEIP_Output101Ethernet IP Assembly Object, Instance 101 (output).
tcEIP_Input102Ethernet IP Assembly Object, Instance 102 (input).
tcEIP_Output103Ethernet IP Assembly Object, Instance 103 (output).

DataLocation

ValueDescription
tcAtRegisterRSVRegister (reserved use).
tcAtIoInputI/O input.
tcAtIoOutputI/O output.
tcAtVRVR (default value).
tcAtDigitalInputDigital I/O Input.
tcAtDigitalOutputDigital I/O Output.
tcAtAnalogueInputAnalogue I/O Input.
tcAtAnalogueOutputAnalogue I/O Output.

DataFormat

ValueDescription
tcInt16bitsInteger 16 bit (default value).
tcInt32bitsInteger 32 bit.
tcFloat32bitsFloating point 32 bit.
tcFloat64bitsFloating point 64 bit.
tcUint16bitUnsigned integer 16 bit.

Outputs

ParameterDescription
Done: BOOL;TRUE when function has completed normally.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.

Description

This function allows the user to configure Ethernet IP at a low level. The default values allow a master to connect without any configuration on the Controller side. In case a different initial configuration is needed, the system parameter IP_PROTOCOL_CTRL must be configured in the MC_CONFIG file in order to disable the protocol and set the configuration required within a IEC program.

ST Language

MyTC_EIpMemMapSet(Execute:= Execute, Slot:= Slot, EndPoint:= EndPoint, StartPos:= StartPos, DataLocation:= DataLocation, DataFormat:= DataFormat, NumEntries:= NumEntries);

Done:=MyTC_EIpMemMapSet.Done;

Error:=MyTC_EIpMemMapSet.Error;

ErrorID:=MyTC_EIpMemMapSet.ErrorID;
