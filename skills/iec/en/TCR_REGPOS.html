TCR_REGPOS

Type

Axis Parameter

Function

Read the current value of axis parameter ‘REG_POS’.

Inputs

AxisNo : UINT;Specify axis number.

Outputs

REG_POS : LREAL;The absolute position in user UNITS at which the registration event occurred.

Description

Stores the latest position at which a registration mark was seen on each axis in user UNITS. This parameter is used with the first (A) hardware registration channel, or Z mark only.

If the value of the input parameter ‘AxisNo’ is out of range, the value of ‘REG_POS’ is 0 and the return value of ‘TCR_ErrorID’ is 19.

ST Language

(LREAL):=TCR_REGPOS(AxisNo:= (UINT));
