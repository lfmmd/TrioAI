TC_SerialSend

Type

System Function.

Function

Send message to the specified port.

Inputs

ParameterDescription
Execute: BOOL;Rising edge requests execution.
Channel: TC_SERIALPORT_TO_T;Port number. See Channel.
Data[]: USINT;The array used to store the sent message.
Len: USINT;The length of the sent message.

Channel

ValueDescription
tcRS232PortRS232 port 1.
tcRS485PortRS485 port 2.

Outputs

ParameterDescription
Done: BOOL;TRUE when function has completed normally.
Busy: BOOL;TRUE if function is running.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to send message to the specified port.

ST Language

MyTC_SerialSend(Execute:= Execute, Channel:= Channel, Data:= Data, Len:= Len);

Done:=MyTC_SerialSend.Done;

Busy:=MyTC_SerialSend.Busy;

Error:=MyTC_SerialSend.Error;

ErrorID:=MyTC_SerialSend.ErrorID;
