BOUT

Function

Sets the specified bit state.

Inputs

NameTypeDescription
xEnableBOOLThe specified bit value to be forced.
wDataUINTThe variable which will be operated.
uiBitUINTThe specified bit

Outputs

NameTypeDescription
Q1BOOLTRUE, the specified bit is available, or FALSE.

Remarks

Forces the specified bit value with the input ‘xEnable’. ‘wData’ is a variable with 16-bit data type, so the ‘uiBit’ must be less than 16.

ST Language

wMyVar is a variable which will be operated by function ‘BOUT’.

bRes:=BOUT(TRUE, wMyVar, 10); // Force the bit 10 of wMyVar
