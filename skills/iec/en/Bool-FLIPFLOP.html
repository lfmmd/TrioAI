FLIPFLOP

Function Block

Flipflop bistable.

Inputs

NameTypeDescription
INBOOLSwap command (on rising edge).
RSTBOOLReset to FALSE.

Outputs

NameTypeDescription
QBOOLOutput.

Remarks

The output is systematically reset to FALSE if RST is TRUE.
The output changes on each rising edge of the IN input, if RST is FALSE.

ST Language

MyFlipFlop is declared as an instance of FLIPFLOP function block:

MyFlipFlop (IN, RST);

Q := MyFlipFlop.Q;
