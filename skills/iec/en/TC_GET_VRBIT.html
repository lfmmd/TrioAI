TC_GET_VRBIT

Type

System Function.

Function

Gets the value of a bit of the VR.

Inputs

Index: DINT;VR number
Bit: USINT;Bit number

Outputs

Q: SINT;-1: Incorrect VR number or Incorrect Bit number
0 or 1: the value of a bit

Description

This function gets the value of a bit of the VR.

ST Language

(SINT):=TC_GET_VRBIT(Index:= (DINT), Bit:= (USINT));
