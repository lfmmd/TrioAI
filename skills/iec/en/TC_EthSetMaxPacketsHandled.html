TC_EthSetMaxPacketsHandled

Type

System Function.

Function

Set the maximum packets handled in one stack call.

Inputs

ParameterDescription
Execute: BOOL;Rising edge requests execution.
Slot: SINT;Slot number, the default value is -1.
PacketsNum: USINT;The maximum packets.

Outputs

ParameterDescription
Done: BOOL;TRUE when function has completed normally.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to set the maximum packets handled in one stack call. The default value is 0, which means handle all incoming packets.

ST Language

MyTC_EthSetMaxPacketsHandled(Execute:= Execute, Slot:= Slot, PacketsNum:= PacketsNum);

Done:=MyTC_EthSetMaxPacketsHandled.Done;

Error:=MyTC_EthSetMaxPacketsHandled.Error;

ErrorID:=MyTC_EthSetMaxPacketsHandled.ErrorID;
