TC_FifoRdOpen

Type

System Function.

Function

The file will be opened for reading and will be managed as a circular buffer. This is only valid for files stored in internal RAM.

Inputs

ParameterDescription
Execute: BOOL;Rising edge requests execution.
Channel: TC_FILE_CH_T;The channel associated with the file. tcFileCh0 to tcFileCh8 for files and string variables.
Name: STRING;The name of the file to be opened.
Mode: TC_FIFO_READ_MODE_T;The read mode. See Mode.
Terminator: STRING;The terminator character. See Terminator.

Mode

The mode defaults to tcFifoReadCharacters.
ValueDescription
tcFifoReadCharactersReturn the number of characters in the FIFO.
tcFifoReadLinesReturn the number of lines in the FIFO.

Terminator

The terminator defaults to carriage return.

If mode is tcFifoReadLines and this character is added to the FIFO then the line count will be incremented by 1. If mode is tcFifoReadLines and this character is removed from the FIFO then the line count is decremented by 1.

Outputs

ParameterDescription
Done: BOOL;TRUE when function has completed normally.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to open a file for reading and the file will be managed as a circular buffer.

ST Language

MyTC_FifoRdOpen(Execute:= Execute, Channel:= Channel, Name:= Name, Mode:= Mode, Terminator:= Terminator);

Done:=MyTC_FifoRdOpen.Done;

Error:=MyTC_FifoRdOpen.Error;

ErrorID:=MyTC_FifoRdOpen.ErrorID;
