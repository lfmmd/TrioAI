TC_MbTcpPortSet

Type

System Function.

Function

Set the port number for the channel which is used by Modbus TCP.

Inputs

Execute: BOOL;Rising edge requests execution.
PortNum: UINT;The port used for the channel in the controller. (default 502)

Outputs

Done: BOOL;TRUE when function has completed normally.

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to set the port number for the channel which is used by Modbus TCP.

The default value is used by Modbus and should not be changed unless absolutely necessary.

ST Language

MyTC_MbTcpPortSet(Execute:= Execute, PortNum:= PortNum);

Done:=MyTC_MbTcpPortSet.Done;
