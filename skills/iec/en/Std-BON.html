BON

Function

Checks the specified bit state of a 16bit data.

Inputs

NameTypeDescription
wDataUINTThe data/variable which will be checked.
uiBitUINTThe specified bit numbers.

(*) SRC/DST cannot be a STRING.

Outputs

NameTypeDescription
QBOOLTRUE if successful.

ST Language

wData := 0x55;

OK := BON (wData, 2); //it will return TRUE 

OK := BON (wData, 1); //it will return FALSE
Function
