TCR_LOADED

Type

Axis Parameter

Function

Checks if all the movements have been loaded into the MTYPE buffer.

Inputs

AxisNo: USINT;Axis number

Outputs

LOADED: BOOL;Returns TRUE when move buffer is empty

Description

This function checks if all the movements have been loaded into the MTYPE buffer. It will return a TRUE value when there are no buffered movements, otherwise FALSE.
