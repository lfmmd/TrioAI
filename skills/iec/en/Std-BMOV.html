BMOV

Function

Copies the specified length of data from source array to destination array.

Inputs

NameTypeDescription
pbyDataSrc[]ANY (*)Array containing the source of the copy.
uiSizeUINTNumber of items to be copied.
pbyDataDes[]ANY (*)Array containing the destination of the copy.

(*) SRC/DST cannot be a STRING.

Outputs

NameTypeDescription
QBOOLTRUE if successful.

Remarks

Arrays of string are not supported by this function.

In LD language, the operation is executed only if the input rung (EN) is TRUE. The function is not available in IL language.

The function copies uiSize consecutive items starting at the pbyDataSrc array to pbyDataDes array. These two arrays should be of the same data type.

ST Language

OK := BMOV (SRC, NB, DST);
