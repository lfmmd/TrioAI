TC_FileOpen

Type

System Function.

Function

Open a file.

Inputs

ParameterDescription
Execute: BOOL;Rising edge requests execution.
Channel: TC_FILE_CH_T;The channel associated with the file. tcFileCh0 to tcFileCh8 for files and string variables.
Location: TC_FILE_LOCATION_T;Specifies what the following "name" represents. See Location.
Name: STRING;The name of the file to be opened.
Access: TC_FILE_ACCESS_T;The operations permitted on the file. See Access.

Location

ValueDescription
tcProjectA file stored in the controller's file system.
tcRAMA file stored in the controller's RAM (volatile).
tcSDA file stored on the controller's SD-card (non-volatile).
tcStringA string variable.

Access

ValueDescription
tcAppendThe file will be opened for writing. The file must exist. All data is written to the end of the file.
tcReadOnlyThe file will be opened for reading.
tcWriteOnlyThe file will be opened for writing. If the file does not exist, then it will be created. If the file already exists, then it will be cleared.

Outputs

ParameterDescription
Done: BOOL;TRUE when function has completed normally.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to Open a file.

ST Language

MyTC_FileOpen(Execute:= Execute, Channel:= Channel, Location:= Location, Name:= Name, Access:= Access);

Done:=MyTC_FileOpen.Done;

Error:=MyTC_FileOpen.Error;

ErrorID:=MyTC_FileOpen.ErrorID;
