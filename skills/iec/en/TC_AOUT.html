TC_AOUT

Type

I/O Function.

Function

Writes a value to an analogue output.

Inputs

Index: INT;The analogue input channel number
Value: INT;Analogue output value

Outputs

ErroeID: UINT;Returned error number

Description

This function writes a value to an analogue output. The value sent is the decimal equivalent of the binary number to be written to the D to A converter. When the input parameters are incorrect, the ErrorID will change to 19.
