TC_MbNodeAddrSet

Type

System Function.

Function

Set the Modbus node address.

Inputs

Execute: BOOL;Rising edge requests execution.
NodeAddr: USINT;Modbus node address.

Outputs

Done: BOOL;TRUE when function has completed normally.

Description

When the Execute input changes from FALSE to TRUE, the Function Block issues a command to set the Modbus node address.

ST Language

MyTC_MbNodeAddrSet(Execute:= Execute, NodeAddr:= NodeAddr);

Done:=MyTC_MbNodeAddrSet.Done;

FBD Language
