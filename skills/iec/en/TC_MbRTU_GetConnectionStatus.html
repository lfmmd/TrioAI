TC_MbRTU_GetConnectionStatus

Type

System Function.

Function

Return connection status.

Inputs

Enable: BOOL;Function Block is being enabled as long as ‘Enable’ is TRUE
Slot: INT;Slot number. Currently, the default value is -1. If it is not -1, an error will be reported
HandleNum: DINT;The connection handle output by TC_MbRTU_OpenConnection

Outputs

Status: INT;0: closed
1: open
Error: BOOL;TRUE if a program error is detected
ErrorID: UINT;Returned error number

Description

When the Enable input is TRUE, the Function Block gets Modbus RTU client connection status according to HandleNum. ErrorID includes 781, 783 and 792, and the corresponding error descriptions are as follows.
ErrorIDDescription
781Slot error
783Connection status getting error
792HandleNum error

ST Language

Inst_TC_MbRTU_GetConnectionStatus(Enable:= (BOOL), Slot:= (INT), HandleNum:= (DINT));

(INT):=Inst_TC_MbRTU_GetConnectionStatus.Status;

(BOOL):=Inst_TC_MbRTU_GetConnectionStatus.Error;

(UINT):=Inst_TC_MbRTU_GetConnectionStatus.ErrorID;
