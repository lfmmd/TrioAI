WTOB

Function

Copies and splits the WORD type value to the 2 destination array items.

Inputs

NameTypeDescription
pbyDataSrc[]UINTArray containing the source of the copy.
uiSizeUINTThe number of items to be copied from pbyDataSrc.
pwDataDes[]DINTThe array where will be copied to.

Outputs

NameTypeDescription
OKBOOLTRUE if successful.

Remarks

Every source array element will be split into 2 destination array elements.

ST Language

OK := WTOB (pbyDataSrc, 3, pwDataDes);
Function
