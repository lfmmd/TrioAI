ALARM_M

Function Block

Alarm with manual reset.

Inputs

NameTypeDescription
INBOOLProcess signal.
ACKBOOLAcknowledge command.
RSTBOOLReset command.

Outputs

NameTypeDescription
QBOOLTRUE if alarm is active.
QACKBOOLTRUE if alarm is acknowledged.

Sequence

Remarks

Combine this block with the LIM_ALRM block for managing analog alarms.

ST Language

MyALARM is declared as an instance of ALARM_M function block.

MyALARM (IN, ACK, RST);

Q := MyALARM.Q;

QACK := MyALARM.QACK;
