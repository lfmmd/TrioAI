BLINKA

Function Block

Blink asymmetrically.

Inputs

NameTypeDescription
RUNBOOLEnabling command.
TM0TIMEDuration of FALSE state on output.
TM1TIMEDuration of TRUE state on output.

Outputs

NameTypeDescription
QBOOLOutput blinking signal.

Time diagram

Remarks

The output signal is FALSE when the RUN input is FALSE. In LD language, the input rung is the IN command. The output rung is the Q output signal.

ST Language

MyBlinker is a declared instance of BLINKA function block.

MyBlinker (RUN, TM0, TM1);

Q := MyBlinker.Q;
