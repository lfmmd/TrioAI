TC_EthConnectSta

Type

System Function.

Function

Read Ethernet connection state.

Inputs

ParameterDescription
Enable: BOOL;Rising edge requests execution.
Slot: SINT;Slot number, the default value is -1.

Outputs

ParameterDescription
State: BOOL;Return the cable connection state as TRUE/FALSE.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.

Description

When the Enable input changes from FALSE to TRUE, the Function Block issues a command to read Ethernet connection state of the given slot. If the RJ45 connector has a live Ethernet connection plugged in, then the State output returns TRUE. If the cable is unplugged or is connected to a switch or other device that is powered off, then the State output returns FALSE.

ST Language

MyTC_EthConnectSta(Enable:= Enable, Slot:= Slot);

State:=MyTC_EthConnectSta.State;

Error:=MyTC_EthConnectSta.Error;

ErrorID:=MyTC_EthConnectSta.ErrorID;
