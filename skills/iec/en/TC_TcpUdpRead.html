TC_TcpUdpRead

Type

System Function.

Function

Read the received characters from a TCP/UDP connection.

Inputs

Enable: BOOL;Rising edge requests execution.
Channel: TC_TCP_UDP_CH_T;The channel associated with TCP and UDP connections (tcTcpUdpCh0 to tcTcpUdpCh9 on ARM base controllers).
RcvLength: INT;The length of the received characters.
CalcCRC: BOOL;Whether to calculate CRC or not.
CrcInVR: INT;Position in the VR data where the CRC will be stored.

Outputs

Done: BOOL;TRUE when function has completed normally.
Error: BOOL;TRUE if a program error is detected.
ErrorID: UINT;Returned error number.
RcvStr: STRING;Returned received characters.

Description

When the Enable input changes from FALSE to TRUE, the Function Block issues a command to read the received characters from a TCP/UDP connection.

ST Language

MyTC_TcpUdpRead(Enable:= Enable, Channel:= Channel, RcvLength:= RcvLength, CalcCRC:= CalcCRC, CrcInVR:= CrcInVR);

Done:=MyTC_TcpUdpRead.Done;

Error:=MyTC_TcpUdpRead.Error;

ErrorID:=MyTC_TcpUdpRead.ErrorID;

RcvStr:=MyTC_TcpUdpRead.RcvStr;
