TC_TcpUdpCharKey

Type

System Function.

Function

Check whether the channel has received a character.

Inputs

Channel: TC_TCP_UDP_CH_T;The channel associated with TCP and UDP connections (tcTcpUdpCh0 to tcTcpUdpCh9 on ARM base controllers).

Outputs

Q: INT;TRUE when receive a character.

Description

The Function issues a command to check the channel whether receive a character.

ST Language

Q:=TC_TcpUdpCharKey(Channel:= Channel);
