PLF

Function Block

Outputs a pulse at the falling edge of the input ‘xExecute’.

Inputs

NameTypeDescription
xExecuteBOOLIts falling edge will output a cycle pulse on ‘xOut’.

Outputs

NameTypeDescription
xOutBOOLOutput a pulse if ‘xExecute’ falling edge detected.

Remarks

The output pulse will stay until next call.

ST Language

Output a pulse with the instance of PLF ‘Inst_PLF’ to output a pulse:

bExe := TRUE;

Inst_PLF(bExe);

bRes := Inst_PLF.xOut;

bExe := FALSE;

Inst_PLF(bExe); 

bRes := Inst_PLF.xOut; // Output TRUE

Inst_PLF(bExe); 

bRes := Inst_PLF.xOut; // Output changes to FALSE
