Non-printable characters

You can use special '$' sequences in strings to specify non-printable characters, according to the IEC standard:
CodeMeaning
$$"$" character.
$'Single quote.
$TTab stop (ASCII code 9).
$RCarriage-return character (ASCII code 13).
$LLine feed character (ASCII code 10).
$NCarriage-return plus line-feed characters (ASCII codes 13 and 10).
$PPage break character (ASCII code 12).
$xxAny character (xx is the ASCII code expressed on two hexadecimal digits.
