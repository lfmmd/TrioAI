TC_SELECTUSERFRAME

Type

Motion Function

Function

Selects a previously defined USER_FRAME to become active.

Inputs

EN : BOOL;Set TRUE to enable the function
AxisNo : USINT;Axis number
ID : USINT;Tool offset identity number

Outputs

ENO : BOOL;TRUE if function is enabled
Error : BOOL;TRUE if a program error is detected
ErrorID : UINT;Returned error number

Description

When the EN input is TRUE, the function block applies the command to the axis indicated by the AxisNo input.

A programming error, such as parameter out of range, will set the Error output and return an error code in the ErrorID output. For the error ID, see the Trio Programming error list.

ST Language

MyTC_SelectUserFrame(EN, AxisNo, ID);

ENO := MyTC_SelectUserFrame.ENO;

Error := MyTC_SelectUserFrame.Error;

ErrorID := MyTC_SelectUserFrame.ErrorID;
