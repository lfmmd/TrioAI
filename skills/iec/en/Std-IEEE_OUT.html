IEEE_OUT

Function

Returns a 32bits DWORD value extracted from the floating point value.

Inputs

NameTypeDescription
Data_inREALA REAL data type value/variable.

(*) SRC/DST cannot be a STRING.

Outputs

NameTypeDescription
QDINTThe 32 bits value which extracted from input REAL type value.

Remarks

It is usually used for data transition on Modbus.

ST Language

dValue := IEEE_OUT (fData);
