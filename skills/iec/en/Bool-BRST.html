BRST

Function

Resets the specified bit state.

Inputs

NameTypeDescription
wDataUINTThe variable which will be operated.
uiBitUINTThe specified bit to be reset.

Outputs

NameTypeDescription
Q1BOOLTRUE, the specified bit is available, or FALSE.

Remarks

Resets the specified bit value. ‘wData’ is a variable with 16-bit data type, so ‘uiBit’ must be less than 16.

ST Language

wMyVar is a variable which will be operated by function ‘BRST’.

bRes := BRST(wMyVar, 10); // Reset bit 10 of wMyVar
