ODD

Function

Test if an integer is odd.

Inputs

NameTypeDescription
INDINTInput value.

Outputs

NameTypeDescription
QBOOLTRUE if IN is odd. FALSE if IN is even.

Remarks

In LD language, the input rung (EN) enables the operation, and the output rung is the result of the function. In IL language, the input must be loaded before the function call.

ST Language

Q := ODD (IN);

FBD Language

LD Language

The function is executed only if EN is TRUE.
