Boolean Operations

Standard operators for managing Booleans:

Operator

Description

AND

performs a Boolean AND

OR

performs a Boolean OR

XOR

performs an exclusive OR

NOT

performs a Boolean negation of its input

S

forces a Boolean output to TRUE

R

forces a Boolean output to FALSE

Available blocks for managing Boolean signals:

Block

Description

RS

resets dominant bistable

SR

sets dominant bistable

ALT

alters the value of ‘xExecute’ on rising edge.

BOUT

sets the specified bit state.

BRST

resets the specified bit state.

BSET

sets the specified bit state.

R_TRIG

rising edge detection

F_TRIG

falling edge detection

SEMA

semaphore

FLIPFLOP

flipflop/bistable

PLF

outputs a pulse on falling edge.

PLR

outputs a pulse on rising edge.
