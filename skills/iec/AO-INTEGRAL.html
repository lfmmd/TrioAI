INTEGRAL

Function Block

Calculates the integral of a signal.

Inputs

Name

Type

Description

RUN

BOOL

Run command: TRUE=integrate / FALSE=hold.

R1

BOOL

Overriding reset.

XIN

REAL

Input signal.

X0

REAL

Initial value.

CYCLE

TIME

Sampling period (should not be less than the target cycle timing).

Outputs

Name

Type

Description

Q

DINT 

Running mode report: NOT (R1).

XOUT

REAL

Output signal.

Remarks

In LD language, the input rung is the RUN command. The output rung is the Q report status.

ST Language

MyIntg is a declared instance of INTEGRAL function block.

MyIntg (RUN, R1, XIN, X0, CYCLE);
Q := MyIntg.Q;
XOUT := MyIntg.XOUT;

LD Language

IL Language

MyIntg is a declared instance of INTEGRAL function block.

Op1: CAL MyIntg (RUN, R1, XIN, X0, CYCLE)
     LD  MyIntg.Q
     ST  Q
     LD  MyIntg.XOUT
     ST  XOUT
