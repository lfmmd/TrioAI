TCR_JOGSPEED

Type:

Axis Parameter

Function:

Reads the current value of JOGSPEED of an axis.

Inputs:

  

AxisNo : USINT;

Axis number

Outputs: 

JOGSPEED : REAL;

Current value of JOGSPEED of the axis

Description:

Reads the current value of JOGSPEED of an axis.

ST Language:

TCR_JOGSPEED ( AxisNo (*USINT*) );

Example 1:

Input 0, 1, 2 are used for setting up jogging parameters on axes 0. Input 0 for forward jog, input 1 for reverse jog and input 2 for fast jog. Jog speed is set for 15 units. All inputs are inverted, meaning that the jog commands are active when the corresponding input is high. To execute fast jog, both the fast jog input and either forward jog or reverse jog input need to be active.

Some local variables are defined to read back the jogging parameters.

Local Variables

Text Mode

VAR
 IN_fastJog : INT ;
 IN_fwdJog : INT ;
 IN_revJog : INT ;
 Inst_TC_EXECUTE : TC_EXECUTE ;
 Inst_TC_EXECUTE1 : TC_EXECUTE ;
 Inst_TC_EXECUTE2 : TC_EXECUTE ;
 jogSpeed : REAL ;
END_VAR

Grid Mode

 

// Invert inputs for FWDJOG, REVJOG, and FASTJOG
Inst_TC_EXECUTE( TRUE , 'INVERT_IN(0,ON)' );
Inst_TC_EXECUTE1( TRUE , 'INVERT_IN(1,ON)' );
Inst_TC_EXECUTE2( TRUE , 'INVERT_IN(2,ON)' );

// Write jogging parameters
TCW_JOGSPEED ( 0 , 15 );
TCW_FWDJOG ( 0 , 0 );
TCW_REVJOG ( 0 , 1 );
TCW_FASTJOG ( 0 , 2 );

// Read jogging parameters
IN_fwdJog:= TCR_FWDJOG ( 0 );
IN_revJog:= TCR_REVJOG ( 0 );
IN_fastJog:= TCR_FASTJOG ( 0 );
jogSpeed:= TCR_JOGSPEED ( 0 );

FBD Language:

Example 1:

Axis 1 is set to be jogged by input 6 in forward direction and input 7 in reverse direction. Fast jog is set on input 5. Using the TC_EXECUTE function block, all inputs are inverted, meaning that the jog command is active when the corresponding input is high. The jog speed is set as 25 units/sec. Fast jog takes the normal speed as jogging speed when active.

These inputs are then read and stored in local variables using TCR_FASTJOG, TCR_FWDJOG, TCR_REVJOG function blocks. The jog speed is also stored in a local variable using TCR_JOGSPEED.
