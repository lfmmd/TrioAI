TC_CLEAR_TABLEBIT

Type:

System Function.

Function:

Resets a bit of the TABLE.

Inputs:

Index: DINT;

TABLE number

Bit: USINT;

Bit number

Outputs: 

Q: SINT;

0: Incorrect TABLE number or Incorrect Bit number

1: function has completed

Description:

This function resets a bit of the TABLE.
