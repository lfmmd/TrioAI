:= MOVE

Operator

M oves the value of a constant or variable to another variable.

Inputs

Name

Type

Description

IN

ANY

A constant value or a variable.

Outputs

Name

Type

Description

Q

ANY

A variable which will be assigned by ‘IN’.

Remarks

The data types of ‘IN’ and ‘Q’ must be the same.

ST Language

Q := -IN;
Q := - (IN1 + IN2);
