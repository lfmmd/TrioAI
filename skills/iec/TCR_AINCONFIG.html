TCR_AINCONFIG

Type:

I/O Function.

Function:

Read the mode of operation for an analogue input.

Inputs:

Channel : INT;

Analogue input channel number.

Outputs:

Q : DINT;

0

Bipolar, -10 V .. +10 V

1

Unipolar, 0 .. +10 V

2

Bipolar, -5 V .. +5 V

3

Unipolar, 0 .. +5 V

Description:

Read the mode of operation for an analogue input. This operation can be performed on any external analogue input from a Trio CAN Analogue input module P326/P328.

This function requires CANIO firmware 2.3.2 or later.

The CAN analogue module P326 has 8 x 12 bit inputs.

The CAN analogue module P328 has 8 x 16 bit inputs.

ST Language:

TCR_AINCONFIG (Channel, Value);

Examples:

Read the range of the analog input 5.

TC R _AINCONFIG ( 5 );
