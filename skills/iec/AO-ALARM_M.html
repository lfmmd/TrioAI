ALARM_M

Function Block

Alarm with manual reset.

Inputs

Name

Type

Description

IN

BOOL

Process signal.

ACK

BOOL

Acknowledge command.

RST

BOOL

Reset command.

Outputs

Name

Type

Description

Q

BOOL

TRUE if alarm is active.

QACK

BOOL

TRUE if alarm is acknowledged.

Sequence

Remarks

Combine this block with the LIM_ALRM block for managing analog alarms.

ST Language

MyALARM is declared as an instance of ALARM_M function block.

MyALARM (IN, ACK, RST);
Q := MyALARM.Q;
QACK := MyALARM.QACK;
