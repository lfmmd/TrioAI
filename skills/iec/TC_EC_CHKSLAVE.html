TC_EC_CHKSLAVE

Type:

System Command

Function:

Checks slave online on the EtherCAT network.

Inputs:

  

Execute : BOOL;

Rising edge requests execution

Slot : INT;

Slot number of EtherCAT port

SlavePosition: INT;

Position of the slave

Outputs: 

ENO : BOOL;

TRUE when function has completed normally

Error : BOOL;

TRUE if a program error is detected

ErrorID : UINT;

Returned error number

Online: BOOL;

TRUE when the slave is online

Description:

This function block checks whether given slave is online.
