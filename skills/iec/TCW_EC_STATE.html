TCW_EC_STATE

Type:

System Function.

Function:

Set the state of EtherCAT network.

Inputs:

Slot : INT;

Slot number of EtherCAT port

State : INT;

State of EtherCAT network

 

1

Initial

 

2

Pre-Operation

 

4

Safe-Operation

 

8

Operation

Display : BOOL

Writes state change information to the standard output stream. True - Enable (Default), False -Disable

Mpemode : BOOL

MPE mode ，True – Enable, False - Disable (Default)

Express : BOOL

Express mode ，True – Enable, False - Disable (Default)

Outputs: 

Q : DINT;

 

Description:

This function controls the EtherCAT State Machine. (ESM)  It requests the master change to given EtherCAT ‘state’, and hence changes all slaves to the same state. When a change to a higher state is made, the EtherCAT network will progress to the new state through the in-between states to allow correct starting of the network.
