XCH

Function

Exchanges the 2 input variable values.

Inputs

Name

Type

Description

fDataSrc1

ANY (*)

The first input variable for exchanging.

fDataSrc2

ANY (*)

The second input variable for exchanging.

(*) SRC/DST cannot be a STRING.

Outputs

Name

Type

Description

OK

BOOL

TRUE if successful.

Remarks

The 2 inputs must be of the same data type variable.

ST Language

  fData1 := 1234;

  fData2 := 5678;

OK := XCH (fData1, fData2); // fData1 = 5678, fData2=5678
