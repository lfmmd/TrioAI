BON

Function

Checks the specified bit state of a 16bit data.

Inputs

Name

Type

Description

wData

UINT

The data/variable which will be checked.

uiBit

UINT

The specified bit numbers.

(*) SRC/DST cannot be a STRING.

Outputs

Name

Type

Description

Q

BOOL

TRUE if successful.

ST Language

  wData := 0x55;

OK := BON (wData, 2); //it will return TRUE

OK := BON (wData, 1); //it will return FALSE
