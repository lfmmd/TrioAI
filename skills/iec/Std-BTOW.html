BTOW

Function

Copies the lower 8bit data of resource data to WORD data type destination.

Inputs

Name

Type

Description

pbyDataSrc[]

UINT

Array containing the source of the copy.

uiSize

UINT

The number of items to be copied from pbyDataSrc.

pwDataDes[]

DINT

The array destination where will be copied to.

Outputs

Name

Type

Description

OK

BOOL

TRUE if successful.

Remarks

The upper byte of every item in source array will be ignored.

ST Language

OK := BTOW (pbyDataSrc, 6, pwDataDes);
