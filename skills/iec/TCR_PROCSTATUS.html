TCR_PROCSTATUS

Type:

System Parameter.

Function:

Return the status of the specified process.

Inputs:

ProcNum : INT;

Specify process number.

Outputs: 

ProcStatus : DINT;

Return the status of process.

-1

ERROR

The value of parameter ‘ProcNum’ is out of range.

0

STOPPED

No program or task is running on the process.

1

RUNNING

A task or program is running.

2

STEPPING

Process is stepping line by line.

3

PAUSED

Execution paused (pause button or breakpoint).

4

PAUSING

Preparing to go into paused state.

5

STOPPING

Preparing to stop.

6

SUSPEND

Operating system requested process to suspend.

7

SUSPENDED

Process has temporarily been suspended.

8

SLEEP

Process sleeping, for example during WA, WAIT IDLE, etc.

Description:

Return the status of the specified process.

If the value of the input parameter ‘ProcNum’ is out of range, the value of ‘ProcStatus’ is -1 and the return value of ‘TCR_ErrorID’ is 19.

ST Language:

(DINT):=TCR_PROCSTATUS(ProcNum:= (INT));
