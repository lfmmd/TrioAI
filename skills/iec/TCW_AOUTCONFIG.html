TCW_AOUTCONFIG

Type:

I/O Function.

Function:

Set the mode of operation for an analogue output.

Inputs:

Channel : INT;

Analogue output channel number.

Value : UINT;

0

Bipolar, -10 V .. +10 V

1

Unipolar, 0 .. +10 V

2

Bipolar, -5 V .. +5 V

3

Unipolar, 0 .. +5 V

Outputs:

Q : DINT;

 

Description:

Set the mode of operation for an analogue output. This operation can be performed on any external analogue input from a Trio CAN Analogue input/output module P326 and P328.

This function requires CANIO firmware 2.3.2 or later.

The CAN analogue module P326/P328 has 4 x 12 bit outputs.

ST Language:

TCW_AOUTCONFIG (Channel, Value);

Examples:

Set the range of the analog output 5.

TC W _A OUT CONFIG ( 2, 1 );
