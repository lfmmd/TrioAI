TC_SELECTUSERFRAMEB

Type:

Motion Function

Function:

Selects a secondary USER_FRAME to be used when SYNC mode 20 is activated.

Inputs:

  

EN : BOOL;

Set TRUE to enable the function

AxisNo : USINT;

Axis number

ID : USINT;

Tool offset identity number

Outputs: 

ENO : BOOL;

TRUE if function is enabled

Error : BOOL;

TRUE if a program error is detected

ErrorID : UINT;

Returned error number

Description:

When the EN input is TRUE, the function block applies the command to the axis indicated by the AxisNo input.

A programming error, such as parameter out of range, will set the Error output and return an error code in the ErrorID output.  For the Error ID, see the Trio Programming error list.

ST Language:

MyTC_SelectUserFrameB(EN, AxisNo, ID);

ENO := MyTC_SelectUserFrameB.ENO;

Error := MyTC_SelectUserFrameB.Error;

ErrorID := MyTC_SelectUserFrameB.ErrorID;
