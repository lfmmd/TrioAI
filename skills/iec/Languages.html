Languages

Motion Perfect supports the following IEC 61131-3 languages:

  • Ladder Diagram (LD)
  • Structured Text (ST)
  • Function Block Diagram (SFD)
  • Sequential Function Chart (SFC)

Each of the languages has its own editor and can interact with the IEC 61131-3 environment shared between all programs running on the same IEC 61131 task.

Details of the languages and in-built functions can be found the IEC Language Reference.
