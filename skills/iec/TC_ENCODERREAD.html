TC_ENCODERREAD

Type:

Motion Function.

Function:

Read an internal register from an EnDat absolute encoder.

Inputs:

Axis : USINT;

Axis number

Address : DINT;

The address of the EnDat encoder register to be read

Outputs:

Q : SINT;

Value returned from the specified register

Description:

Read an internal register from an EnDat absolute encoder.
