TCR_EC_STATE

Type:

System Function.

Function:

Get the state of EtherCAT network.

Inputs:

Slot : INT;

Slot number of EtherCAT port

Outputs:

Q : DINT;

State of EtherCAT network

 

1

Initial

 

2

Pre-Operation

 

4

Safe-Operation

 

8

Operation

Description:

This function block gets the state of EtherCAT network. The output value shows the ethercat state.
