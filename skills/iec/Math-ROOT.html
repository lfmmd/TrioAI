ROOT

Function

Calculates the N th root of the input.

Inputs

Name

Type

Description

IN

REAL

Real value

N

DINT

Root level

Outputs

Name

Type

Description

Q

REAL

Result: Nth root of IN

Remarks

In LD, the operation is executed only if the input rung (EN) is TRUE. The output rung (ENO) keeps the same value as the input rung. In IL, the input must be loaded in the current result before calling the function.

ST Language

Q := ROOT (IN, N);

FBD Language

LD Language

The function is executed only if EN is TRUE.
ENO keeps the same value as EN.
