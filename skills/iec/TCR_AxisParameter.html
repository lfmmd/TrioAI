TCR_AxisParameter

Type:

Axis Parameter

Function:

Reads the value of the given axis parameter.

Inputs:

  

AxisNo : USINT;

Axis number

Outputs: 

ParamValue : Various;

Parameter value

Description:

Reads the value of the parameter specified in the AxisParameter input, returning its value in the ParamValue output. See the function block tooltips in the Motion Perfect editor for parameter names and data sizes.

ST Language:

MyTCR_AxisParameter(AxisNo, ParamValue);

ParamValue:= MyTCR_AxisParameter.ParamValue;
