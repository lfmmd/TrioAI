TC_MbTCP_GetClientStatus

Type:

System Function.

Function:

Return connection status.

Inputs:

Enable: BOOL;

Function Block is being enabled as long as ‘Enable’ is TRUE

Slot: INT;

Slot number. Currently, the default value is -1. If it is not -1, an error will be reported

HandleNum: DINT;

The connection handle output by TC_MbTCP_OpenClient

Outputs: 

Status: INT;

0: closed

1: open

Error: BOOL;

TRUE if a program error is detected

ErrorID: UINT;

Returned error number

Description:

When the Enable input is TRUE, the Function Block gets Modbus TCP client connection status according to HandleNum. ErrorID includes 781, 783 and 792, and the corresponding error descriptions are as follows.

ErrorID

Description

781

Slot error

783

Connection status getting error

792

HandleNum error

ST Language:

Inst_TC_MbTCP_GetClientStatus(Enable:= (BOOL), Slot:= (INT), HandleNum:= (DINT));

(INT):=Inst_TC_MbTCP_GetClientStatus.Status;

(BOOL):=Inst_TC_MbTCP_GetClientStatus.Error;

(UINT):=Inst_TC_MbTCP_GetClientStatus.ErrorID;
