TC_SET_TABLEBIT

Type:

System Function.

Function:

Sets a bit of the TABLE.

Inputs:

Index: DINT;

TABLE number

Bit: USINT;

Bit number

Outputs: 

Q: SINT;

0: Incorrect TABLE number or Incorrect Bit number

1: function has completed

Description:

This function sets a bit of the TABLE.
