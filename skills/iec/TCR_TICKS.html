TCR_TICKS

Type:

Motion Parameter

Function:

Reads from the process TICKS value.

Inputs:

  

None

 

Outputs: 

TICKS : LINT;

TICKS value

Description:

Reads from the TICKS value associated with the current process.  Value is returned in TICKS.
