TCW_WDOG

类型

系统函数。

函数

写入WDOG参数。

请注意输入输出数据类型为DINT。0等于OFF。1或非0等于ON。

输入

WDOG : DINT;WDOG状态

输出

Q : DINT;

描述

设置WDOG状态。

ST 语言

TCW_WDOG( WDOG(*DINT*) );

例 1

轴0和轴1在初始配置中被设置。一旦完成后，使用TCW_WDOG指令切换WDOG ON。轴相对于当前位置移动1000个单位。

// Axis 0 & 1 setup

FOR i:=0 TO 1 DO    

    TCW_UNITS(i,1); // Set UNITS    

    TCW_SPEED(i,4000); // Set SPEED    

    TCW_ACCEL(i,20*TCR_SPEED(0)); // Set ACCEL    

    TCW_DECEL(i,TCR_ACCEL(i)); // Set DECEL    

    TCW_DGAIN(i,30); // Set D_GAIN    

    TCW_SERVO(i,TRUE); // Switch on SERVO

END_FOR;



// Switch on WDOG

TCW_WDOG(1); 



// Move axis0 and axis1 (relative) 1000 units

Inst_TC_MOVE1_0(TRUE,0,1000); 

Inst_TC_MOVE1_1(TRUE,1,1000); 

例 2

当轴0和轴1完成移动后关闭WDOG

// Switch off WDOG once moves are done

IF Inst_TC_MOVE1_0.Done=TRUE AND Inst_TC_MOVE1_1.Done=TRUE THEN

    TCW_WDOG(0);

END_IF;

FBD 语言

例 1

FBD_Init是一个配置轴参数的子程序。WDOG在执行处理轴移动的FBD_Move1子程序前被打开。

例 2

程序从轴参数初始化开始。一旦设置完成，WDOG在轴继续运动前切换为ON。

LD 语言

例 1

在设置所有轴参数的第一个梯级上调用初始化程序。一旦初始化完成，WDOG被切换为ON。只有当Stop按钮被按下时，WDOG切换为OFF。

例 2

程序从坐标轴参数的初始化开始。一旦设置完成，WDOG在继续轴移动前被切换到ON。

例 3

一旦轴完成移动，切换WDOG为OFF。
