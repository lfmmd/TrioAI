TC_VALIDENCKEY

类型

系统命令

函数

检查控制器是否存在正确用户或OEM安全码。

输入

EN : BOOL;上升沿触发
Key : INT;安全码类型：
ValidationKey : STRING;一个字符串，由Motion Perfect生成的验证密钥

输出

ENO : BOOL;如果函数在运行，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

TC_VALIDENCKEY用于检查控制器是存在正确用户或OEM安全码。

如果正确的安全码没有加载，TC_VALIDENCKEY 将会产生一个允许错误（参数超限）并在运行时停止程序。

Motion Perfect 有产生有效码的工具。

不要将用户或OEM安全码放在程序中，它们需要保密。

ST 语言

Inst_TC_VALIDENCKEY( EN(*BOOL*), Key(*INT*), ValidationKey(*STRING*) );
