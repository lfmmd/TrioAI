TC_EC_CHKSLAVE

类型

系统命令。

函数

检查EtherCAT网络中的从机是否在线。

输入

Execute : BOOL;上升沿 触发
Slot : INT;EtherCAT端口从机号
SlavePosition: INT;从机位置

输出

ENO : BOOL;当功能正常完成为TRUE
Error : BOOL;当程序检测到错误为TRUE
ErrorID : UINT;返回错误号
Online: BOOL;当从机在线为TRUE

描述

该函数用于检查EtherCAT网络中的从机是否在线。

ST 语言

Inst_TC_EC_CHKSLAVE( EN(*BOOL*), Slot(*INT*), SlavePosition(*INT*) );
