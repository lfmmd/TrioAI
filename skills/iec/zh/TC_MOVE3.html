TC_MOVE3

类型

运动函数

函数

为AxisNo指定的3个轴发出一个新的移动（距离1，距离2，距离3）运动请求。

输入

Execute : BOOL;上升沿触发
AxisNo : USINT;基轴轴号
Dist1 : LREAL;第一个轴要移动的距离
Dist2 : LREAL;第二个轴要移动的距离
Dist3 : LREAL;第三个轴要移动的距离

输出

Busy : BOOL;如果函数正在运行，则为TRUE
Done : BOOL;函数正常完成时为TRUE
Buffered : BOOL;当运动命令在NTYPE缓冲区中时为TRUE
Active : BOOL;当运动命令在MTYPE缓冲区中时为TRUE
Aborted : BOOL;如果函数因取消而终止，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当执行输入从FALSE变为TRUE（上升沿）时，功能块尝试将运动命令加载到所需的轴缓冲器中。如果缓冲区不可用，该功能将在每次PLC扫描时重新尝试。加载运动命令后，输出将指示相应的运动状态；在NTYPE、MTYPE、aborted (Cancelled)或done中。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

函数必须在处理扫描和执行输入为TRUE时，输出才能正确显示移动的状态。

ST 语言

Inst_TC_MOVE3( Execute(*BOOL*), AxisNo(*USINT*), Dist1(*LREAL*), Dist2(*LREAL*), Dist3(*LREAL*) );

例 1

程序通过调用ST_Init子程序初始化轴0和轴1的参数来启动。在切换WDOG为ON后，一个轴0和轴1的插值移动被执行。一旦移动完成，WDOG切换为OFF。

ST_Init();



//打开 WDOG

TCW_WDOG(1); 



//轴0，轴1，轴2的插值移动

Inst_TC_MOVE2(TRUE,0,2000,4000,6000);



//一旦移动被执行关闭 WDOG

IF Inst_TC_MOVE2.Done=TRUE THEN

    TCW_WDOG(0);

END_IF;
