TC_DEFPOS

类型

运动函数。

函数

对“AxisNo”指定的单轴或多轴应用新的DEFPOS请求。

输入

Execute : BOOL;上升沿触发
AxisNo : USINT;基准轴的轴号
Count : USINT;在位置数组中指定值的数量
Positions[] : LREAL[];包含要应用的位置值的数组

输出

Done : BOOL;如果函数已启用，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当执行输入从FALSE变为TRUE（上升沿）时，功能块发出命令以在速度轮廓软件中执行。数组位置中的值用于Count轴，从“AxisNo”指定的轴开始。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅Trio编程错误列表。

ST 语言

Inst_TC_DEFPOS( Execute(*BOOL*), AxisNo(*USINT*), Count(*USINT*), Positions[](*LREAL*) );

例 1

该程序首先初始化所有轴参数。一旦WDOG启用，一个新的基本数组将被设置在 dough_cnv 数组中。轴0和轴2的位置都在数组pos的下一个函数块中给出，作为TC_DEFPOS2命令的输入参数。

网格模式

//初始化轴参数

ST_Init();



//切换WDOG

TCW_WDOG(1); 



//设置基数组

Inst_TC_BASE(TRUE,2,dough_cnv);



//为dough conveyors定义位置

Inst_TC_DEFPOS(TRUE,0,2,pos);
