TC_DEFINETOOLOFFSET

类型

运动函数。

函数

为“ID”指定的标识发出新的TOOL_OFFSET定义请求。

输入

EN : BOOL;TRUE启用该功能
ID : USINT;定义刀具补偿的标识号（0–31）
XOFF : LREAL;X轴上的偏移：从世界坐标到用户坐标
YOFF : LREAL;Y轴上的偏移：从世界坐标到用户坐标
ZOFF : LREAL;Z轴上的偏移：从世界坐标到用户坐标

输出

ENO : BOOL;如果函数已启用，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当EN输入为TRUE时，功能块将TOOL_OFFSET命令应用于ID指示的标识。这些偏移应用于标识，但在执行TC_SELECTTOOLOFFSET之前不会被选择。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅Trio编程错误列表。

ST 语言

MyTC_DefineToolOffset(EN, ID, XOFF, YOFF, ZOFF);

ENO := MyTC_DefineToolOffset.ENO;

Error := MyTC_DefineToolOffset.Error;

ErrorID := MyTC_DefineToolOffset.ErrorID;
