TC_USERFRAMETRANS

类型

运动函数

函数

执行指定的表数据上的单USER_FRAME_TRANS。

输入

EN : BOOL;设置为TRUE以启用该功能
UF1 : USINT;用户Frame输入；USER_FRAME识别提供的点
UF2 : USINT;用户Frame输出；USER_FRAME识别转换的点
TO1 : USINT;工具补偿输入；TOOL_OFFSET识别提供的点
TO2 : USINT;工具补偿输出；TOOL_OFFSET识别转换的点
DataIn : DINT;输入位置的表索引
DataOut : LINT;用于生成位置开始的表索引
Scale : LREAL;表格值的比例因子（默认值1000）

输出

ENO : BOOL;如果函数已启用，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当EN输入为TRUE时，功能块应用该命令。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

MyTC_UserFrameTrans(EN, UF1, UF2, TO1, TO2, DataIn, DataOut, Scale);

ENO := MyTC_UserFrameTrans.ENO;

Error := MyTC_UserFrameTrans.Error;

ErrorID := MyTC_UserFrameTrans.ErrorID;
