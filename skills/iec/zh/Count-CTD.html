CTD/CTDr

Function Block (功能可)

减计数器。

输入

名称类型描述
CDBOOL启动计数，每当CD为TRUE时技术减小。
LOADBOOL重载命令。当LOAD设置为TRUE时计数器置为PV。
PVDINT预设值。

输出

名称类型描述
QBOOL当计数器为空时该值为TRUE，如：当CV = 0.
CVDINT计数器的当前值。

备注

当程序启动时，计数器为空（CV=0）。计数器不包括对CD输入的脉冲检测。使用R_TRIG 或 F_TRIG来检测CD输入信号脉冲。在梯形图中，CD为输入梯级。输出梯级为Q。

CTUr, CTDr, CTUDr 功能块的操作与其他计数器完全相同，除了所有的布尔输入（CU，CD，RESET，LOAD）都包含隐式上升沿检测。某些目标系统可能不支持这些计数器。

从3.03版本固件开始，CV在程序启动时自动初始化（预加载）PV的值。

ST 语言

MyCounter 被声明为CTD功能块的一个实例：

MyCounter (CD, LOAD, PV);

Q := MyCounter.Q;

CV := MyCounter.CV;
