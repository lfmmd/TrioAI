TC_IDLE

类型

运动函数

函数

评估轴是否空闲。

输入

EN : BOOL;TRUE以启用该函数
AxisNo : USINT;轴号

输出

ENO : BOOL;如果函数已启用，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号
Idle: BOOL当轴空闲时为TRUE, 如果运动正在进行则为FALSE

描述

当EN输入为TRUE时，函数块将命令应用到AxisNo指示的轴上。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

MyTC_Idle(EN, AxisNo);

ENO := MyTC_Idle.ENO;

Error := MyTC_Idle.Error;

ErrorID := MyTC_Idle.ErrorID;

Idle := MyTC_Idle.Idle;
