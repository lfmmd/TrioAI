ASCII

Function (函数)

获取字符串中字符的ASCII代码。

输入

名称类型描述
INSTRING输入字符串
POSDINT字符在字符串中的位置。（第一个有效位置是1）。

输出

名称类型描述
CODEDINT所选字符的ASCII码，如果位置无效，则为0。

备注

在LD语言中，输入梯级（EN）启用操作，输出梯级保持与输入梯级相同的值。在IL语言中，在调用函数之前，必须在当前结果中加载第一个参数（IN）。另一个输入是函数的操作数。

ST 语言

CODE := ASCII (IN, POS);

FBD 语言

LD 语言

仅当EN为TRUE时才执行该函数。

ENO保持与EN相同的值。
