PLR

Function Block (功能块)

在‘xExecute’ 输入的上升沿输出一个脉冲。

输入

名称类型描述
xExecuteBOOL上升沿将在 ‘xOut’输出一个周期脉冲。

输出

名称类型描述
xOutBOOL如果 ‘xExecute’上升沿被检测到，输出一个脉冲。

备注

输出脉冲将会保持到下一个调用。

ST 语言

使用PLR ‘Inst_PLR’的实例输出一个脉冲来输出脉冲:

bExe := FALSE;

Inst_PLR(bExe);

bRes:=Inst_PLR.xOut;

bExe := TRUE;

Inst_PLR(bExe); 

bRes:=Inst_PLR.xOut; //Output TRUE

Inst_PLR(bExe); 

bRes:=Inst_PLR.xOut; // Output changes to FALSE
