+ ADD

Operator (操作)

执行所有输入的加法运算。

输入

名称类型描述
IN1ANY第一个输入
IN2ANY第二个输入

输出

名称类型描述
QANY结果: IN1 + IN2.

备注

所有输入和输出必须具有相同的类型。在FBD语言中，块可以有多达16个输入。在LD语言中，输入梯级（EN）启用操作，输出梯级保持与输入梯级相同的值。在IL语言中，ADD指令在当前结果和操作数之间执行加法运算。当前结果和操作数必须具有相同的类型。

加法可以与字符串一起使用。结果是输入字符串的串联。

ST 语言

Q := IN1 + IN2;

MyString := 'He' + 'll ' + 'o';   (* MyString is equal to 'Hello' *)
