TC_BASE

类型

运动函数。

函数

对“AxisNo[]”指定的轴或轴应用新的BASE请求。

输入

Execute : BOOL;上升沿触发
Count : USINT;AxisNo数组中指定的轴数
AxisNo[] : USINT[];在移动命令中使用的轴的轴号

输出

Done : BOOL;如果函数正常完成，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当执行输入从FALSE变为TRUE（上升沿）时，功能块发出命令以在速度轮廓软件中执行。AxisNo数组中的轴数将变成在TC_BASE执行后要以任何曲线运动的轴。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅Trio编程错误列表。

ST 语言

Inst_TC_BASE( Execute(*BOOL*), Count(*USINT*), Axes[](*USINT*) );

例 1

设置基数，轴0和轴2定义为dough_cnv。局部变量的定义如下表所示。
局部变量
文本模式VAR<br> Inst_TC_BASE : TC_BASE ;<br> dough_cnv : ARRAY [0 .. 1] OF USINT := USINT#0,USINT#2 ; <br>END_VAR
网格模式

Inst_TC_BASE(TRUE,2,dough_cnv);
