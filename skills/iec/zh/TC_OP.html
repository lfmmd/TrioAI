TC_OP

类型

I/O 函数。

函数

为指定的数字量输出应用新的输出请求。

输入

Index : INT;输出数
Value : BOOL;要复制到指定输出号的值

输出

Q : BOOL;被选中输出的状态

描述

将数字量输出设置为“Value”输入参数中给出的二进制模式。

ST 语言

TC_OP( Index(*INT*), Value(*BOOL*) );

例

打开输出8。

TC_OP(8,TRUE);
