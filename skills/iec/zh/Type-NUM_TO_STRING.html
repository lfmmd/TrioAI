NUM_TO_STRING

Function (函数)

将数字转换为字符串值。

输入

名称类型描述
INANY输入数。
WIDTHDINT输出字符串的期望长度（见备注）
DIGITSDINT小数点后位数

输出

名称类型描述
QSTRING转换的字符串值

备注

此函数将任何数值转换为字符串。与ANY_TO_STRING函数不同，它允许您指定所需的长度和小数点后的位数。

如果WIDTH为0，则使用必要的长度格式化字符串。

如果WIDTH大于0，则字符串将以标题空白字符完成，以便与宽度值匹配。

如果WIDTH小于0，则字符串将以尾随空白字符完成，以便与WIDTH的绝对值匹配。

如果DIGITS为0，则不添加小数部分或小数点。

如果DIGITS大于0，则添加相应的小数位数必要时添加 '0' 位

如果值对于指定的宽度太长，则字符串将填充“*”字符。

例

Q := NUM_TO_STRING (123.4, 8, 2);        (* Q is '  123.40' *)

Q := NUM_TO_STRING (123.4, -8, 2);       (* Q is '123.40  ' *)

Q := NUM_TO_STRING (1.333333, 0, 2);     (* Q is '1.33' *)

Q := NUM_TO_STRING (1234, 3, 0);         (* Q is '***' *)
