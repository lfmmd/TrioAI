TC_ENCODERRATIO

类型

运动函数

函数

对AxisNo 指定的轴发布一个新的ENCODER_RATIO运动请求。

输入

EN : BOOL;TRUE 以启用该函数
AxisNo : USINT;轴号
Numerator: LINT;MPOS数 （函数的输出）
Denominator: LINT;输入数

输出

ENO : BOOL;当函数启用则为TRUE
Error : BOOL;如果程序错误被检测到则为TRUE
ErrorID : UINT;返回错误号

描述

当EN输入为TRUE时，功能块向指定的轴应用命令。

编程错误将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

MyTC_EncoderRatio(EN, AxisNo, Numerator, Denominator);

ENO := MyTC_EncoderRatio.ENO;

Error := MyTC_EncoderRatio.Error;

ErrorID := MyTC_EncoderRatio.ErrorID;
