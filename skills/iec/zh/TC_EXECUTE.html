TC_EXECUTE

类型

命令

函数

允许执行任何TrioBASIC命令，尤其是字符串输入参数。

输入

EN : BOOL;设置TRUE去启动函数
Command : STRING;要执行的TrioBASIC命令

输出

ENO : BOOL;如果函数启动为TRUE
Error : BOOL;如果程序检测到错误为TRUE
ErrorID : UINT;返回错误号

描述

当EN输入为TRUE时，功能块执行命令输入的字符串所指定的TrioBASIC命令。

该命令在EN输入参数的上升沿初始化。

编程错误将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

Inst_TC_EXECUTE( EN(*BOOL*), Command(*STRING*) );

例 1

在‘print_status_Cmd’变量的上升沿，命令字符串被执行。在这个案例中，在终端7中打印‘OK’。
局部变量
文本模式VAR<br> print_status_Cmd : STRING(255) ;<br>END_VAR
网格模式

Inst_TC_EXECUTE(print_status_Cmd,'PRINT #7,"OK"');

例 2

使用循环，命令字符串被合并用于TC_EXECUTE 命令。在这个案例中，TC_EXECUTE 只有1个实例，在不同的命令中重复使用。当TC_EXECUTE 在上升沿‘EN’输入执行时，‘EN’ 输入需要重置为False。
局部变量
文本模式VAR<br> cmd : STRING(255) ; <br> i : INT ;<br> Inst_TC_EXECUTE : TC_EXECUTE ; <br>END_VAR
网格模式

FOR i := 0 TO 5 DO

        //构建字符串作为命令

        cmd:=any_to_string(i);

        cmd:=concat('INVERT_IN(',cmd);

        cmd:=concat(cmd,',ON)');

        

        //执行命令

        Inst_TC_EXECUTE(TRUE,cmd);

        

        //重置执行状态以让实例重用于其他输入

        Inst_TC_EXECUTE(FALSE,cmd);        

    END_FOR;

例 3

TC_EXECUTE 用于反转每个输入指令的输入参数中指定的6个输入。在这个案例中，每个TC_EXECUTE 指令被分配为一个实例。
局部变量
文本模式VAR<br> Inst_TC_EXECUTE : TC_EXECUTE ;<br> Inst_TC_EXECUTE1 : TC_EXECUTE ;<br> Inst_TC_EXECUTE2 : TC_EXECUTE ;<br> Inst_TC_EXECUTE3 : TC_EXECUTE ;<br> Inst_TC_EXECUTE4 : TC_EXECUTE ;<br> Inst_TC_EXECUTE5 : TC_EXECUTE ;<br>END_VAR
网格模式

Inst_TC_EXECUTE(TRUE,'INVERT_IN(0,ON)');

Inst_TC_EXECUTE1(TRUE,'INVERT_IN(1,ON)');

Inst_TC_EXECUTE2(TRUE,'INVERT_IN(2,ON)');

Inst_TC_EXECUTE3(TRUE,'INVERT_IN(8,ON)');

Inst_TC_EXECUTE4(TRUE,'INVERT_IN(9,ON)');

Inst_TC_EXECUTE5(TRUE,'INVERT_IN(10,ON)');
