RETURN语句 (RET RETC RETNC RETCN)

Statement (表达式)

跳到程序的末尾。

备注

RETURN语句跳到程序的末尾。在FBD语言中，return语句由“<return>”符号表示。符号的输入必须连接到有效的布尔信号。仅当输入为真时才执行跳转。在LD语言中，“<RETURN>”符号用作横档末端的线圈。仅当横档状态为TRUE时才执行跳转。在IL语言中，使用RET、RETC、RETCN和RETNC指令。

当在SFC步骤的动作块中使用时，返回语句跳到动作块的末尾。

ST 语言

IF NOT bEnable THEN

   RETURN;

END_IF;

如果b使能为FALSE，则不会执行程序的其余部分。
