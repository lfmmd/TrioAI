TC_SYNC

类型

运动函数

函数

为AxisNo指定的轴发出一个新的SYNC运动请求。

输入

EN : BOOL;设置为TRUE以启用该功能
AxisNo : USINT;轴号
Control : USINT;控制值
SyncPos : LINT;同步位置
SyncAxis : USINT;主轴跟随
SyncTime : DINT;轴同步的持续时间
SyncPosX : LINT;同步位置X
SyncPosY : LINT;同步位置Y
SyncPosZ : LINT;同步位置Z

输出

ENO : BOOL;如果函数已启用，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当EN输入为TRUE时，函数块将命令应用到AxisNo指示的轴上。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

MyTC_Sync(EN, AxisNo, Control, SyncPos, SyncAxis, SyncTime, SyncPosX, SyncPosY, SyncPosZ);

ENO := MyTC_Sync.ENO;

Error := MyTC_Sync.Error;

ErrorID := MyTC_Sync.ErrorID;
