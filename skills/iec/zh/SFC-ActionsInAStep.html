一个SFC步骤中的操作

每个步骤都有一个操作块列表，这些操作块是根据步骤的活动执行的指令。操作可以是简单的布尔或SFC操作，包括使用step活动分配布尔变量或控制子SFC程序，或使用另一种语言（FBD、LD、ST或IL）输入的操作块。

运行时检查

以下是可以在SFC步骤中用于执行运行时安全检查的语法：
句法描述
StepTimeout (...);检查步骤活动持续时间是否超时。

简单布尔操作

下面是可以在SFC步骤中用于执行简单布尔操作的语法：
句法描述
BoolVar (N);在步骤激活时强制变量BoolVar为TRUE，在步骤取消激活时强制变量BoolVar为FALSE。
BoolVar (S);当步骤被激活时，将变量BoolVar设置为TRUE
BoolVar (R);当步骤被激活时，将变量BoolVar设置为FALSE
/ BoolVar;在步骤激活时强制变量BoolVar为FALSE，在步骤取消激活时强制变量BoolVar为TRUE。

警报

使用以下语法可以管理超时报警变量：
句法描述
BoolVar (A, duration);指定要与步骤关联的超时变量。
BoolVar必须是一个简单的布尔变量
持续时间是超时时间，表示为常数或单个时间变量
（复杂表达式不能用于此参数）
超时后，报警变量将变为真，并且在重置报警变量之前，无法跨越步骤后的转换。

简单的SFC操作

以下是您可以在SFC步骤中用于控制子SFC程序的可能语法：
句法描述
Child (N);在步骤被激活时启动子程序，在步骤被取消激活时停止（终止）子程序。
Child (S);启动步骤时启动子程序
Child (R);当步骤被激活时停止（终止）子程序

程序操作块

可以输入其他语言（FBD、LD、ST或IL）的程序来描述SFC步骤动作。有三种主要的编程动作块，对应于以下标识符：
标识符描述
P1仅在步骤激活时执行一次。
N在步骤处于活动状态时在每个周期执行。
P0仅在步骤变为非活动时执行一次。

工作台提供了用ST、LD或FBD语言输入P1、N和P0动作块的模板。或者，可以使用以下语法，将用ST语言编程的动作块直接插入到简单动作列表中：

ACTION ( qualifier ) :         statements...     END_ACTION;

式中 标识符qualifier 是 P1, 0 或 P0.
