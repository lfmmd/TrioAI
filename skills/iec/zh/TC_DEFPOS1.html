TC_DEFPOS1

类型

运动函数

函数

对“AxisNo”指定的一个轴应用新的DEFPOS请求。

输入

Execute : BOOL;上升沿触发
AxisNo : USINT;轴号
Pos : LREAL;要应用的位置值

输出

Done : BOOL;函数正常完成时为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当执行输入从FALSE变为TRUE（上升沿）时，功能块发出命令以在速度轮廓软件中执行。位置的值应用于“AxisNo”指定的轴。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅Trio编程错误列表。

ST 语言

Inst_TC_DEFPOS1( Execute(*BOOL*), AxisNo(*USINT*), Pos(*LREAL*) );

例 1

在初始化并激活WDOG后，轴2的位置通过TC_DEFPOS1命令来设置。

//初始化轴参数

ST_Init();



// 切换 WDOG

TCW_WDOG(1); 



// 定义轴2起始位置

Inst_TC_DEFPOS1(TRUE,2,25);
