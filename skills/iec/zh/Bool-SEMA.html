SEMA

Function Block (功能块)

信号量

输入

名称类型描述
CLAIMBOOL获取信号量。
RELEASEBOOL释放信号量。

输出

名称类型描述
BUSYBOOL如果信号量正忙，则为TRUE

备注

功能块实现以下算法：

BUSY := mem; if CLAIM then    mem := TRUE; else if RELEASE then    BUSY := FALSE;    mem := FALSE; end_if;

在LD语言中，输入横档是CLAIM命令。输出横档是正忙的输出信号。

ST 语言

MySema是SEMA函数块的声明实例：

MySema (CLAIM, RELEASE);

BUSY := MyBlinker.BUSY;
