TCW_FWDJOG

类型

轴参数。

函数

按住输入的数字以沿轴向前点动。

输入

AxisNo : USINT;轴数
FWD_JOG : INT;输入参数用于点动

输出

ErrorID : UINT;Error ID number错误ID号

描述

保持输入数字作为轴向前点动的输入。当TCW_FWDJOG 启动时，轴以JOGSPEED向前移动。查看 TCR_JOGSPEED 和TCW_JOGSPEED以设置JOGSPEED。

使用的输入是低电平的。建议在输入侧使用INVERT_IN以便输入的0V禁用点动。

如果TCW_FWDJOG与TCW_REVJOG同时被相同的输入分配并被激活时，TCW_FWDJOG会覆盖TCW_REVJOG。如果它们被分配到不同的输入并逐一激活，那么第一个命令有限，第二个命令被忽略。

ST 语言

TCW_FWDJOG( AxisNo(*USINT*), FWD_JOG(*INT*) );

例 1

输入0，1，2被用于设置0轴上的点动参数。输入0为正向向前点动，输入1为反向点动，输入2为快速点动。点动速度被设置为15个单位。所有的输入都是反向的，这意味着相应的输入处于高位时，点动命令才被激活。要执行快速点动，快速点动输入和正向点动或反向点动都要处于激活状态。

一些局部变量被定义以读取点动参数。

Inst_TC_EXECUTE(TRUE,'INVERT_IN(0,ON)');

// 写入点动参数

TCW_JOGSPEED(0,15);

TCW_FWDJOG(0,0);

TCW_REVJOG(0,1);

TCW_FASTJOG(0,2);



// 读取点动参数

IN_fwdJog:=TCR_FWDJOG(0);

IN_revJog:=TCR_REVJOG(0);

IN_fastJog:=TCR_FASTJOG(0);

jogSpeed:=TCR_JOGSPEED(0);

FBD 语言

例 1

轴1的点动被设置为输入6正向，输入7反相。输入5设置为快速点动。使用TC_EXECUTE 函数块，所有的输入都是反向的，这意味着当相应的输入处于高位时，点动命令处于激活状态。点动速度设置为25个单位/秒。快速点动在激活时以正常速度作为点动速度。

这些输入而后通过TCR_FASTJOG, TCR_FWDJOG, TCR_REVJOG 模块读取并储存在局部变量中。点动速度也使用TCR_JOGSPEED储存在一个局部变量中。
