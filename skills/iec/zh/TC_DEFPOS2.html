TC_DEFPOS2

类型

运动函数。

函数

对“AxisNo”指定的两个轴应用一个新的DEFPOS请求。

输入

Execute : BOOL;上升沿触发
AxisNo : USINT;轴号
Pos1 : LREAL;应用于第一轴的位置值
Pos2 : LREAL;应用于第二轴的位置值

输出

Done : BOOL;函数正常完成时为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当执行输入从FALSE变为TRUE（上升沿）时，功能块发出命令以在速度轮廓软件中执行。POS1和POS2的值应用于“AxisNo”指定开始的轴。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅Trio编程错误列表。

ST 语言

Inst_TC_DEFPOS2( Execute(*BOOL*), AxisNo(*USINT*), Pos1(*LREAL*), Pos2(*LREAL*) );

例 1

该程序首先调用初始化子程序ST_Init 来设置所有轴参数。一旦WDOG启用，一个新的基本数组将被设置为在 dough_cnv 中定义轴0和轴2。轴0和轴2的位置都在TC_DEFPOS2 命令的pos1和pos2的输入参数中给出。

Grid Mode

//初始化轴参数

ST_Init();



//切换 WDOG

TCW_WDOG(1); 



//设置基数组

Inst_TC_BASE(TRUE,2,dough_cnv);



//为dough conveyors定义位置

Inst_TC_DEFPOS2(TRUE,0,25,50);
