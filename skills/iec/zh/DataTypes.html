数据类型

基本数据类型

类型描述
BOOL布尔值（位）-可以为FALSE或TRUE-存储在1位上。
SINT  (*)8位上的小有符号整数（从-128到+127）。
USINT (*)8位上的小无符号整数（从0到+255）。
BYTE和USINT一样。
INT  (*)16位有符号整数（从-32768到+32767）。
UINT (*)16位无符号整数（从0到+65535）。
WORD和UINT一样。
DINT32位上的有符号整数（从-2147483648到+2147483647）。
UDINT (*)32位上的有符号整数（从0到+4294967295）。
DWORD等同于UDINT
LINT  (*)64位上的长有符号整数。
REAL (*)单精度实数-存储在32位。
LREAL  (*)双精度实数-存储在64位。
TIME一天中的时间-少于24小时-精度为1毫秒。
STRING  (*)具有声明的最大长度的可变长度字符串。
声明的最大长度不能超过255个字符。

_(*)_有些数据类型可能不是所有目标都支持。

结构

结构是定义为一组成员的复杂数据类型。结构的成员可以有各种数据类型。结构的一个构件可以具有尺寸，也可以是另一个结构的实例。

定义结构时，它可以用作其他数据类型来声明变量。

结构的构件可以具有初始值。 在这种情况下，具有此结构类型的所有已声明变量的对应成员将使用该成员的初始值初始化。

要用语言指定结构化变量的成员，请使用以下符号：

VariableName.MemberName

枚举数据类型

您可以定义一些新的数据类型，它们是命名值的枚举。例如：

type: LIGHT

values:  GREEN, ORANGE, RED

然后，在程序中，可以使用枚举值之一，前缀为类型名：

Light1 := LIGHT#RED;

具有枚举数据类型的变量只能用于赋值、比较和SEL/MUX函数。

“位字段”数据类型

可以定义从整数数据类型派生的新数据类型，这些类型的某些位具有一些可读的名称。因此，您可以在程序中使用Var名称.Bit名称符号。此类数据类型不能从LINT类型派生。
