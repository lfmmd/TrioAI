TC_STOP

类型

程序函数。

函数

在多任务系统中，停止一个给定名称和可选PROCESS数的程序或者 IEC61131-3 任务。

输入

Execute : BOOL;上升沿触发
ProgramName : STRING;要停止BASIC程序或IEC任务的名称。
ProcNumber : USINT;程序正在运行的进程号，或者-1去停止包含给定名称的所有进程、任务。当进程号未知时，-1应该被使用。

输出

Done : BOOL;如果函数正常结束，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当执行输入从FALSE变为TRUE（上升沿）时，函数块为多任务OS发布STOP指令来执行。ProcNumber中的值将使得进程中的单个实体停止。如果程序中只有一个实例的过程号或者程序未知，进程号-1将停止进程下的所有实体。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

Inst_TC_Stop(Execute, ProgramName, ProcNumber);

Done := Inst_TC_Stop.Done;

Error := Inst_TC_Stop.Error;

ErrorID := Inst_TC_Stop.ErrorID;
