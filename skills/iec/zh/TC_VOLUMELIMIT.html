TC_VOLUMELIMIT

类型

运动函数

函数

配置新的3D体积限制。

输入

EN : BOOL;设置为TRUE以启用该功能
AxisNo : USINT;轴号
Mode : USINT;体积限制模式
TableIndex : DINT体积限制表数据的位置

输出

ENO : BOOL;如果函数已启用，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当EN输入为TRUE时，函数块将命令应用到AxisNo指示的轴上。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

MyTC_VolumeLimit(EN, AxisNo, Mode, TableIndex);

ENO := MyTC_VolumeLimit.ENO;

Error := MyTC_VolumeLimit.Error;

ErrorID := MyTC_VolumeLimit.ErrorID;
