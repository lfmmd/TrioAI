TC_EC_START

类型

系统命令。

函数

启动EtherCAT网络。

输入

Execute : BOOL;上升沿触发
Slot : INT;EtherCAT 端口从机号

输出

Busy : BOOL;当函数运行时为TRUE
Done : BOOL;当函数正常完成时为TRUE
Error : BOOL;当程序检测到错误时为TURE
ErrorID : UINT;返回错误号
ECstate: UINT;EtherCAT网络状态

描述

该函数用于启动EtherCAT网络。 它初始化EtherCAT网络，并将其置于可操作模式。

在通常使用中EtherCAT网络将自动启动而不需要开始程序中的任何命令。一些EtherCAT函数在调试与设置EtherCAT系统中也许是有用的。

ST 语言

Inst_TC_EC_START( Execute(*BOOL*), Slot(*INT*) );
