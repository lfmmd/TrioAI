TC_DISABLEGROUP

类型

运动函数。

函数

.对'AxisNo[]'指定的单轴或多轴应用一个新的DISABLE_GROUP 请求。

输入

EN : BOOL;为TRUE以启用该函数
AxisCount : USINT;轴数组中指定的轴数
Axes[] : USINT[];放置在禁用组中的轴的轴号

输出

ENO : BOOL;启用函数时为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当EN输入为TRUE时，功能块应用指示轴的命令。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅Trio编程错误列表。

ST 语言

MyTC_DisableGroup(EN, AxisCount, Axes[]);

ENO := MyTC_DisableGroup.ENO;

Error := MyTC_DisableGroup.Error;

ErrorID := MyTC_DisableGroup.ErrorID;
