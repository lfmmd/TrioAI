TC_DEFPOS3

类型

运动函数。

函数

对“AxisNo”指定的三个轴应用新的DEFPOS请求。

输入

Execute : BOOL;上升沿触发
AxisNo : USINT;轴号
Pos1 : LREAL;应用于第一轴的位置值
Pos2 : LREAL;应用于第二轴的位置值
Pos3 : LREAL;应用于第三轴的位置值

输出

Done : BOOL;函数正常完成时为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当执行输入从FALSE变为TRUE（上升沿）时，功能块发出命令以在速度轮廓软件中执行。POSS1、POS2和POS3的值应用于“AxisNo”指定的轴。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅Trio编程错误列表。

ST 语言

MyTC_Defpos3(Execute, AxisNo, Pos1, Pos2, Pos3);

Done := MyTC_Defpos3.Done;

Error := MyTC_Defpos3.Error;

ErrorID := MyTC_Defpos3.ErrorID;
