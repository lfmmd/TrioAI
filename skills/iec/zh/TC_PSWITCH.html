TC_PSWITCH

类型

运动函数。

函数

为AxisNo指定的轴发出一个新的PSWITCH请求。

输入

Execute : BOOL;上升沿触发
AxisNo : USINT;轴号
Mode : USINT;PSwitch模式
Switch : USINT;PSwitch 号：0..63。
Output : USINT;数字输出数
OpState : BOOL;PSwitch在范围内时需要输出状态
SetPosition : LREAL;输出将采用定义状态的起始位置
ResetPosition : LREAL;输出将进入相反状态的结束位置

输出

Done : BOOL;函数正常完成时为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当执行输入从FALSE变为TRUE（上升沿）时，功能块运行该命令。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

Inst_TC_PSWITCH( Execute(*BOOL*), AxisNo(*USINT*), Mode(*USINT*), Switch(*USINT*), Output(*UINT*), OpState(*BOOL*), SetPosition(*LREAL*), ResetPosition(*LREAL*) );
