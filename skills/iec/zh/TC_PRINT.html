TC_PRINT

类型

命令。

函数

TC_PRINT命令允许程序输出一系列字符至一个通道。一个通道可以是一串端口或与运动协调器的其他类型连接。

该命令是IEC61131-3的TrioBasic内的‘Print’命令对应。

请参考TrioBASIC内的‘Print’命令了解更多细节。

输入

Channel : INT;使用的通道。参考TrioBASIC内的‘Print’命令了解更多细节。
Text : STRING;要打印的文本。与字符串变量不同，包含单个引号(‘’)符号的字符串文本值也是一个有效输入，例如： ‘这是一个文本’

输出

Q : DINT;执行结果。非零值指定错误。

描述

设置数据输出为Value中给出的二进制形式。

ST 语言

TC_PRINT( Channel(*INT*), Text(*STRING*) );

例

将初始化进程的状态打印至通道0。

IF NOT bInitialised THEN

    // Print status

    TC_PRINT(0,'Initialisation completed');

    

    // Set flag

    bInitialised:=TRUE;

END_IF;

FBD 语言

例 1

“初始化完成”被打印到通道7。

例 2

‘text’ 变量的内容打印到通道5。‘text’ 是一个包含“初始化完成”字符串作为初始值的局部变量。
