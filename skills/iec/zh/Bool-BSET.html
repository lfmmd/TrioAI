BSET

Function (函数)

设置数据的指定位状态。

输入

名称类型描述
wDataUINT将被操作的变量。
uiBitUINT将被设置的位。

输出

名称类型描述
Q1BOOL指定位可读为TRUE，否则为FALSE。

备注

设置指定位值。‘wData’ 是一个16为数据类型的变量，所以 ‘uiBit’ 必须小于16。

ST 语言

wMyVar 是一个被函数 ‘BSET’ 操作的变量。

bRes:=BSET(wMyVar, 10); // Set the bit 10 of wMyVar
