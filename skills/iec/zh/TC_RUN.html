TC_RUN

类型

程序函数

函数

在多任务系统的给定进程中，运行一个TrioBASIC 程序或者一个 IEC61131-3 任务。

输入

Execute : BOOL;上升沿触发
ProgramName : STRING;要运行TrioBASIC程序或IEC任务的名称。 与STRING变量不同，包含单个引号(‘’)符号的STRING文本值也是一个有效输入，例如： ‘This is a text’ 。
注意：这时不区分大小写的，如“PROG1”与“prog1”是相同的。
ProcNumber : USINT;进程号介于0和最大进程或-1之间，以便让OS分配进程。

输出

Done : BOOL;当函数完成时为TRUE
Error : BOOL;当检测到程序错误时为TRUE
ErrorID : UINT;返回错误号

描述

当执行输入从FALSE变为TRUE（上升沿）时，函数块为多任务OS发布RUN指令来执行。 如果可用，ProcNumber中的值将分配给程序、任务。如果进程被占用，则会产生错误。进程号-1允许OS自动调用下一个进程。最高可用号将会使用。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅Trio编程错误列表。

ST 语言

Inst_TC_RUN( EN(*BOOL*), ProgName(*STRING*), ProcNum(*INT*) );

例 1

一旦‘JogMachine_CMD’ 为TRUE时，IEC任务 ‘JOGS’ 将被执行。进程号将自动分配。
