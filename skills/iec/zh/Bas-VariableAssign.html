:= 赋值

Operator (操作)

变量赋值

输入

名称类型描述
INANY任何变量或复杂表达式

输出

名称类型描述
QANY强制变量

备注

输出变量和输入表达式必须具有相同的类型。强制变量不能具有只读属性。在LD和FBD语言中，1块可用于执行“1增益”数据复制。在LD语言中，输入横档（EN）启用赋值，输出横档保持输入横档的状态。在IL语言中，LD指令加载第一个操作数，ST指令将当前结果存储到变量中。当前结果和ST的操作数必须具有相同的类型。对于执行布尔求反的布尔操作数，LD和ST指令都可以被N修改。

ST 语言

Q := IN; (* copy IN into variable Q *)

Q := (IN1 + (IN2 / IN 3)) * IN4; (* assign the result of a complex expression *)

result := SIN (angle); (* assign a variable with the result of a function *)

time := MyTon.ET; (* assign a variable with an output parameter of a function block *)
