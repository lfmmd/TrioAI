TC_FRAMETRANS

类型

运动函数

函数

为AxisNo指定的轴提出一个新的FRAME_TRANS运动请求。

输入

EN : BOOL;TRUE 以启用该函数
Frame : DINT;要运行的FRAME号
DataIn : DINT;输入位置表中的起始位置
DataOut : DINT;生成位置表中的起始位置
Option : DINT;1= AXIS_DPOS到DPOS（正向运动学）
0= DPOS到AXIS_DPOS（反向运动学）
TableData : DINT;表中FRAME配置所在的第一个位置。

输出

ENO : BOOL;启用功能时为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当EN输入为TRUE时，功能块使用由FRAME指示的帧号应用命令。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

MyTC_FrameTrans(EN, Frame, DataIn, DataOut, Option, TableData);

ENO := MyTC_FrameTrans.ENO;

Error := MyTC_FrameTrans.Error;

ErrorID := MyTC_FrameTrans.ErrorID;
