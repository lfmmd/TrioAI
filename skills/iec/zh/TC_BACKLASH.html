TC_BACKLASH

类型

运动函数。

函数

为轴指定一个新的BACKLASH运动请求。

输入

Execute : BOOL;上升沿触发
AxisNo : USINT;轴号
Enable : BOOL;设置为TRUE以启用间隙功能
Distance : LINT;方向改变时应用的间隙距离
Speed : LREAL;间隙间隙修正速度（单位：Units/s）
Accel : LREAL;间隙修正加速度（单位：Units/s/s）

输出

Done : BOOL;如果函数已启用，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当执行输入从FALSE变为TRUE（上升沿）时，功能块发出命令以在速度轮廓软件中执行。如果Execute为TRUE，则该函数使用给定的参数设置间隙间隙操作。如果启用为FALSE，则在AxisNo定义的轴上取消间隙操作。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅

TrioBasic 错误清单。
