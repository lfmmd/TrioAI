EXIT

Statement (表达式)

从循环语句中退出。

备注

EXIT语句指示必须完成当前循环（WHILE、REPEAT或FOR）。在END_WHILE、END_REPEAT或END_FOR关键字或EXIT所在的循环之后继续执行。退出只退出一个循环，不能同时退出多个嵌套循环级别。

循环指令可能导致无限循环，从而阻塞目标循环。

ST 语言

本程序寻找数组中的第一个非空单元：

iFound = -1; (* means: not found *)

FOR iPos := 0 TO (iArrayDim - 1) DO

   IF iPos <> 0 THEN

      iFound := iPos;

      EXIT;

   END_IF;

END_FOR;

FBD 语言

不可用。

LD 语言

不可用。
