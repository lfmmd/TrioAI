CTUD/CTUDr

Function Block (功能块)

加/减计数器。

输入

名称类型描述
CUBOOL启动计数，每当CU为TRUE时计数增加。
CDBOOL启动计数，每当CD为TRUE时计数减小。
RESETBOOL重置命令。当调用RESET设置为TRUE时计数器置为0。
LOADBOOL重载命令。当调用LOAD设置为TRUE时计数器置为PV。
PVDINT预设值。

输出

名称类型描述
QUBOOL当计数器的当前值大于或等于时该值为TRUE，如：当CV= PV
QDBOOL计数器为空时该值为TRUE，如：当CV=0
CVDINT计数器的当前值。

备注

当程序启动时，计数器为空（CV=0）。计数器不包括对CU和CD输入的脉冲检测。使用R_TRIG 或 F_TRIG来检测CU或CD输入信号脉冲。在梯形图中，CU为输入梯级。输出梯级为QU。

CTUr, CTDr, CTUDr 功能块的操作与其他计数器完全相同，除了所有的布尔输入（CU，CD，RESET，LOAD）都包含隐式上升沿检测。某些目标系统可能不支持这些计数器。

ST 语言

MyCounter被声明为 CTUD 功能块的一个实例。

MyCounter (CU, CD, RESET, LOAD, PV);

QU := MyCounter.QU;

QD := MyCounter.QD;

CV := MyCounter.CV;
