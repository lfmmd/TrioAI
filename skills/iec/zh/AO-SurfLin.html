Function Block (功能块)

曲面上的线性插值。

输入

名称类型描述
XREAL要插值的点的X坐标。
YREAL要插值的点的Y坐标。
XAxisREAL[]X轴已知点的X坐标。
YAxisREAL[]Y轴已知点的Y坐标。
ZValREAL[,]由坐标轴定义的点的Z坐标。

输出

名称类型描述
ZREAL对应于X，Y输入点的插值Z值
OKBOOL如果成功，则为TRUE。
ERRDINT如果失败，则返回错误代码-如果正常，则返回0。

备注

该函数在XAxis定义的点列表和YAxis一维数组之间进行线性曲面插值。输出z值是在轴中定义的四个舍入点的z值的内插。定义点的Z值在ZVal矩阵（二维数组）中传递。

ZVal维数必须表示为：ZVal[iX，iY]

x和y轴中的值必须从最小到最大排序。必须在每个轴上定义至少两个点。ZVal必须符合XAxis数组和YAxis数组的维数。例如：

XAxis : ARRAY [0..2] of REAL;

YAxis : ARRAY [0.3] of REAL;

ZVal : ARRAY [0..2,0..3] of REAL;

如果输入点在XAxis和YAxis限制定义的矩形之外，则将Z输出绑定到相应的值，并报告错误。

如果函数失败，错误输出将给出错误原因：
错误码含义
0OK
1输入数组维度无效
2X轴的无效点
3Y轴的无效点
4X，Y点超出定义的轴
