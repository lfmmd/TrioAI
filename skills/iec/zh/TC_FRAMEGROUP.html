TC_FRAMEGROUP

类型

运动函数

函数

为AxisNo指定的轴提出一个新的FRAME_ GROUP运动请求。

输入

EN : BOOL;TRUE 以启用该功能
ID : USINT;FRAME组标识号
TableIndex : DINT;表索引点到FRAME参数
AxisCount : USINT;FRAME组中的轴数
Axes[] : USINT[];包含轴号的数组

输出

ENO : BOOL;启用功能时为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述

当EN输入为TRUE时，功能块将命令应用到AxisNo指示的轴组。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

MyTC_FrameGroup(EN, ID, TableIndex, AxisCount, Axes);

ENO := MyTC_FrameGroup.ENO;

Error := MyTC_FrameGroup.Error;

ErrorID := MyTC_FrameGroup.ErrorID;
