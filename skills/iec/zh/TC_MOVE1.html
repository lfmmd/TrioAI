TC_MOVE1

类型

运动函数

函数

为AxisNo指定的轴发出一个新的移动（距离）运动请求。

输入

Execute : BOOL;上升沿触发
AxisNo : USINT;基轴轴号
Dist : LREAL;要移动的距离

输出

Busy : BOOL;如果函数正在运行，则为TRUE
Done : BOOL;函数正常完成时为TRUE
Buffered : BOOL;当运动命令在NTYPE缓冲区中时为TRUE
Active : BOOL;当运动命令在MTYPE缓冲区中时为TRUE
Aborted : BOOL;如果函数因取消而终止，则为TRUE
Error : BOOL;如果检测到程序错误，则为TRUE
ErrorID : UINT;返回的错误号

描述：

当执行输入从FALSE变为TRUE（上升沿）时，功能块尝试将运动命令加载到所需的轴缓冲器中。如果缓冲区不可用，该功能将在每次PLC扫描时重新尝试。加载运动命令后，输出将指示相应的运动状态；在NTYPE、MTYPE、aborted (Cancelled)或done中。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

函数必须在处理扫描和执行输入为TRUE时，输出才能正确显示移动的状态。

ST 语言

Inst_TC_Move1( Execute(*BOOL*), AxisNo(*USINT*), Dist(*LREAL*) );

例 1

轴0和轴1在初始配置中被设置。一旦完成后，使用TCW_WDOG指令切换WDOG ON。轴相对于当前位置移动1000个单位。

// 轴0 & 1 设置

FOR i:=0 TO 1 DO    

    TCW_UNITS(i,1); // Set UNITS    

    TCW_SPEED(i,4000); // Set SPEED    

    TCW_ACCEL(i,20*TCR_SPEED(i)); // Set ACCEL    

    TCW_DECEL(i,TCR_ACCEL(i)); // Set DECEL    

    TCW_DGAIN(i,30); // Set D_GAIN    

    TCW_SERVO(i,TRUE); // Switch on SERVO

END_FOR;



// 切换 WDOG

TCW_WDOG(1); 



// 移动轴0和轴1（相对）1000个单位

Inst_TC_MOVE1_0(TRUE,0,1000); 

Inst_TC_MOVE1_1(TRUE,1,1000);
