LIFO

Function Block (功能块)

管理后进先出堆栈。

输入

名称类型描述
PUSHBOOL推入一个新值（在上升沿）。
POPBOOL弹出一个新值（在上升沿）。
RSTBOOL重置列表。
NEXTINANY要推送的值。
NEXTOUTANY堆栈顶部的值-调用后更新！
BUFFERANY存储值的数组。

输出

名称类型描述
EMPTYBOOL如果堆栈为空，则为TRUE。
OFLOBOOL如果PUSH命令溢出，则为TRUE。
COUNTDINT堆栈中的值数。
PREADDINT堆栈顶部缓冲区中的索引。
PWRITEDINT下一个推送位置的缓冲区中的索引。

备注

NEXTIN、NEXTOUT和BUFFER必须具有相同的数据类型，并且不能是字符串。

NEXTOUT参数指定一个变量，该变量在调用块后由堆栈顶部的值填充。

值存储在缓冲区数组中。数据永远不会移动或重置。只更新读写指针和推送值。堆栈的最大大小是数组的维数。

第一次调用块时，它会记住应该在哪个数组上工作。如果稍后使用另一个缓冲区输入调用同一实例，则该调用将被视为无效且不产生任何结果。在这种情况下，输出报告一个空堆栈。

在LD语言中，输入梯级是PUSH输入。输出梯级是空输出。

ST 语言

MyLIFO是后进先出功能块的声明实例。

MyLIFO (PUSH, POP, RST, NEXTIN, NEXTOUT, BUFFER); EMPTY := MyLIFO.EMPTY; OFLO := MyLIFO.OFLO; COUNT := MyLIFO.COUNT; PREAD := MyLIFO.PREAD; PWRITE := MyLIFO.PWRITE;
