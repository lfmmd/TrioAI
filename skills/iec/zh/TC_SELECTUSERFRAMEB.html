TC_SELECTUSERFRAMEB

类型

运动函数

函数

选择激活同步模式20时要使用的辅助USER_FRAME。

输入

EN : BOOL;如果函数启动，则为TRUE
AxisNo : USINT;如果监测到程序错误，则为TRUE
ID : USINT;返回错误码

输出

ENO : BOOL;如果函数启动，则为TRUE
Error : BOOL;如果监测到程序错误，则为TRUE
ErrorID : UINT;返回错误码

描述

当EN输入为TRUE时，函数块将ADDAX命令应用到AxISNO指示的轴上。

编程错误（如参数超出范围）将设置错误输出并返回错误ID号。有关错误ID参考，请参阅TrioBasic 错误清单。

ST 语言

MyTC_SelectUserFrameB(EN, AxisNo, ID);

ENO := MyTC_SelectUserFrameB.ENO;

Error := MyTC_SelectUserFrameB.Error;

ErrorID := MyTC_SelectUserFrameB.ErrorID;
