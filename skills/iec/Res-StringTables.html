Non-printable characters

You can use special '$' sequences in strings to specify non-printable characters, according to the IEC standard:

Code

Meaning

$$

"$" character.

$'

Single quote.

$T

Tab stop (ASCII code 9).

$R

Carriage-return character (ASCII code 13).

$L

Line feed character (ASCII code 10).

$N

Carriage-return plus line-feed characters (ASCII codes 13 and 10).

$P

Page break character (ASCII code 12).

$xx

Any character (xx is the ASCII code expressed on two hexadecimal digits.
